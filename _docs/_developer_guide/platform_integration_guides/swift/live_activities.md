# Live Activities for iOS

Live Activities are persistent, interactive notifications displayed on your lock screen, allowing you to keep an eye on things in real time.

As a developer, you can use the Braze Swift SDK to easily manage Live Activity push tokens. Then, you can use the Braze REST API to send Live Activity updates, and have all subscribed devices receive the update as quickly as possible.
