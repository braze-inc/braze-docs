# Shopgate

> Shopgate is a leading mobile commerce and omnichannel provider, enabling merchants to create beautiful apps as well as improve the efficiency of brick and mortar stores via intelligent fulfillment methods and clienteling options.

*This integration is maintained by Shopgate*

  

## About this integration

The integration allows Braze and Shopgate merchants to create powerful and highly customized app campaigns through multiple channels such as notifications, in-app content and location based marketing.

  

## Use cases

 - Custom app onboarding including permission management 
 - Push campaigns
 - In-app messages
 - Dynamic content using Braze banners and content cards
 - App inbox

  

## Prerequisites

Before you start, you need the following:

| Prerequisite | Description |
|--|--|
| Shopgate Enterprise Account | This integration is limited to Shopgate Enterprise contracts only |
| Invite Shopgate | Invite Shopgate to your Braze account in order to set up your apps |


  

## Integrating Shopgate

To learn how to integrate Shopgate with Braze, checkout [Shopgate's integration guide](https://support.shopgate.com/en/knowledge/braze-integration) directly on their website. After integrating, you’ll be able to leverage the following Shopgate features in Braze::
 - Custom app onboarding including permission management 
 - Push campaigns
 - In-app messages
 - Dynamic content using Braze banners and content cards
 - App inbox
