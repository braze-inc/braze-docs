# Figma

> [Figma](https://www.figma.com/) is a collaborative design platform that allows you to build, design, and prototype products. 

## About the integration

The Braze and Figma integration allows you to send images and visual assets from Figma directly into the Braze media library.

## Prerequisites

| Requirement | Description |
|---|---|
| Figma account | A Figma account is required to take advantage of this partnership. |
| Braze media library access | You must have "Manage Media Library Assets" permission to add, edit, and delete media library assets in Braze. |
| Braze workspace access | You must have access to the workspaces you're trying to upload these images to in Braze. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integration

### Step 1: Install the Figma to Braze Export plugin

In Figma, you can find the Figma to Braze Export plugin in the **Plugins** section. 

### Step 2: Connect to Braze

After installing, select **Connect to Braze** to connect your Braze account, then select **Continue**.

Next, select your Braze workspace from the **Braze workspace** dropdown or enter the workspace name.

### Step 3: Select your Figma assets

Select the images and visual assets to export to Braze. To select multiple assets, you can press <kbd>Shift</kbd> or drag and drop your cursor over the assets.

### Step 4: Export to Braze

Select **Export to Braze**. Your images and visual assets are uploaded into the Braze media library. Note that the asset name in the media library matches the name in the Figma file.
