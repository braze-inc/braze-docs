## Integration

[Update these steps to match the OpenAI integration flow.]

1. Log in to the OpenAI portal and select **Connect** within the integration page for Braze.
2. Enter your Braze instance and Braze API key and select **Connect Braze**.

If there are additional Braze accounts to integrate:
1. Select **Edit Connection** within the integration page for Braze.
2. From the dropdown, select **+Add New Connection**.
3. Under **Connection Name**, enter a new name to identify this separate account (for example, Braze Training Account).
4. Enter a separate Braze instance and API key for this new account.
5. Select **Connect**.

Contact [OpenAI support](https://help.openai.com/) with any issues or questions regarding your integration.
