# February 2017

## Add Canvas steps after launch

You can now [add Canvas steps]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/#creating-a-canvas) to existing steps even after the Canvas is launched.

## Bulk CSV for Engagement Reports

You can now export campaign data CSV in bulk with [Engagement Reports]({{site.baseurl}}/user_guide/data_and_analytics/reporting/engagement_reports/#engagement-reports). You can export Canvas data via Engagement Reports as well.

## Filter by last received in-app message

For more personalized user targeting, you can now use "Last Received In-App Message" as a time-based filter. To learn more about user targeting with Braze, visit [Segmentation]({{site.baseurl}}/user_guide/engagement_tools/segments/creating_a_segment/).

## Event user log upgrade

It's now easier for you to see what is happening in your SDK integration with our upgraded [event user logs]({{site.baseurl}}/user_guide/administrative/app_settings/developer_console/event_user_log_tab/#event-user-log-tab).
