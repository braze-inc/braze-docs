# Custom events reports

> Custom event reports provide insight on the occurrences of one or more [custom events]({{site.baseurl}}/user_guide/data/activation/custom_data/custom_events/) over time. You can also view this data by specific segments.

## Export custom event reports

You can export the following CSVs:

- Custom Events by Date
    - (Optional) Custom Events for Different Segments
- Custom Events by Hour
    - (Optional) Custom Events for Different Segments
- Custom Events per MAU
- Custom Events for Different Segments
- Custom Events per KPI formula
    - (Optional) Custom Events for Different Segments
