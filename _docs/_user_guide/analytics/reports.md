## Choosing a report

| Report | Best for | Description |
| --- | --- | --- |
| [Campaign analytics]({{site.baseurl}}/user_guide/analytics/reports/campaign_analytics/) | Per-channel campaign results | View real-time results for each campaign, broken down by messaging channel. |
| [Canvas analytics]({{site.baseurl}}/user_guide/analytics/reports/canvas_analytics/) | Canvas performance | View top-line stats, variant performance, and step-level metrics for a Canvas. |
| [Report Builder]({{site.baseurl}}/user_guide/analytics/reports/report_builder/) | Cross-campaign comparison | Compare results across multiple campaigns or Canvases in a single view, with customizable metrics and drilldowns. |
| [Query Builder]({{site.baseurl}}/user_guide/analytics/reports/query_builder/) | Custom SQL analysis | Write custom SQL queries against your Braze data in Snowflake, or use pre-built templates for common analyses. |
| [Engagement reports]({{site.baseurl}}/user_guide/analytics/reports/engagement_reports/) | Scheduled email exports | Set up recurring CSV exports of engagement statistics for selected campaigns and Canvases. |
| [Custom events report]({{site.baseurl}}/user_guide/analytics/reports/custom_events_report/) | Custom event trends | Monitor custom event frequency over time, broken down by segment or normalized by KPI formulas. |
| [Revenue report]({{site.baseurl}}/user_guide/analytics/reports/revenue_report/) | Revenue and purchases | Track revenue, purchases, and product breakdowns over time, with optional segment filters. |
| [Funnel reports]({{site.baseurl}}/user_guide/analytics/reports/funnel_reports/) | Conversion funnel analysis | Analyze the journeys your customers take after receiving a campaign or Canvas, including drop-off points. |
| [Retention reports]({{site.baseurl}}/user_guide/analytics/reports/retention_reports/) | Long-term retention impact | Measure how effectively a campaign or Canvas re-engages users over time (up to 30 days). |
| [Configure reporting]({{site.baseurl}}/user_guide/analytics/reports/configure_reporting/) | Weekly email digest | Subscribe to a weekly analytics email and choose which custom events to include. |
