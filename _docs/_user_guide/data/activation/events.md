To track unique user actions like starting a tutorial or adding a wishlist item, you can define custom events. For common industries like eCommerce, recommended events provide pre-built schemas and best-practice data structures to help you get started faster. You can also record transactions with revenue data using purchase events. <br><br>Together, these events allow you to analyze trends over time and trigger timely, relevant engagement."
