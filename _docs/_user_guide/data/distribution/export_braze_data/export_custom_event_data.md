# Export custom event aggregates

> This page covers the **Custom Events** report page of the dashboard, which allows you to view the occurrences of one or more custom events over time. If you view detailed statistics for custom events or custom events by the hour, you'll have the option of also viewing the data by specific segments.

You can find the **Custom Events Report** under **Analytics**.

![Custom events]({% image_buster /assets/img_archive/Export_events.png %})

You can export the following CSVs:

- Custom Events by Date
    - (Optional) Custom Events for Different Segments
- Custom Events by Hour
    - (Optional) Custom Events for Different Segments
- Custom Events per MAU
- Custom Events for Different Segments
- Custom Events per KPI formula
    - (Optional) Custom Events for Different Segments

{% alert tip %}
For help with CSV and API exports, refer to [Export troubleshooting]({{site.baseurl}}/user_guide/data/export_braze_data/export_troubleshooting/).
{% endalert %}
