# キャンペーンとキャンバスの複製

> 既存のキャンペーンとキャンバスを複製して、成功した戦略を継続的に反復してさらに向上させることができます。キャンペーンとキャンバスを複製すると、配信の詳細とオーディエンスセグメントの再設定にかかる時間を最小限に抑えることができます。

## 複製

キャンペーンまたはキャンバスを複製すると、新しいメッセージのタイトルが自動的に`"Copy of [Original Name]"` になり、編集ページに移動します。これらの複製は、編集して起動するまで下書きとして残ります。

{% alert tip %}
また、[ ワークスペース間でキャンペーン]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/copying_across_workspaces/) をコピーすることもできます。コピーは、編集して起動するまで下書きとして残ります。
{% endalert %}

{% tabs %}
{% tab campaign %}
キャンペーンを複製するには:

1. [**メッセージング**] > [**キャンペーン**] に進みます。
2. キャンペーンにカーソルを合わせて、<i class="fas fa-gear"></i>**その他のアクション**を選択します。
3. [**複製**] を選択します。
{% endtab %}

{% tab canvas %}
キャンバスを複製するには、次のようにします。

1. [**メッセージング**] > [**キャンバス**] に進みます。
2. キャンバスの行にカーソルを合わせ、<i class="fas fa-ellipsis-vertical"></i>**その他のアクション**を選択します。
3. [**複製**] を選択します。

{% endtab %}
{% endtabs %}
