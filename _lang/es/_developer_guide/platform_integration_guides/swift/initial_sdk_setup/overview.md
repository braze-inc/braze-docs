> ¿Quieres actualizarte al SDK Swift para iOS desde nuestro antiguo SDK Objective-C? Es muy fácil. Consulta nuestra [guía de](https://braze-inc.github.io/braze-swift-sdk/documentation/braze/appboy-migration-guide/) migración para más detalles.

Una vez que te hayas integrado utilizando una de las opciones de la lista, hayas seguido los pasos para [completar la integración]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/initial_sdk_setup/completing_integration/) y hayas habilitado otras personalizaciones del SDK (opcional), podrás integrar, habilitar y personalizar canales y características adicionales para adaptarlos a las necesidades de tus futuras campañas.  

{% alert note %} Carthage no es compatible como administrador de paquetes para el SDK Swift de iOS. {% endalert %}
