# Divulgación de software de código abierto para los servicios Braze

_(en vigor desde el 29 de agosto de 2018; sujeto a cambios)_

Los Servicios Braze incluyen código de terceros licenciado a Braze para su uso y redistribución bajo licencias de código abierto. A continuación se incluye una lista de divulgaciones y exenciones de responsabilidad en relación con dicho software con licencia de código abierto que se ha incorporado a los servicios de Braze. Sin perjuicio de cualquiera de los términos y condiciones de tu contrato de suscripción o licencia con Braze, los términos de ciertas licencias de código abierto pueden ser aplicables a tu uso de los Servicios Braze, tal y como se establece a continuación.

Esta lista de código abierto (la "Lista") ha sido generada por Braze utilizando software de terceros con licencia de Braze que identifica el código abierto de los Servicios a partir de una fecha concreta, y sólo pretende ser una revelación de un momento concreto. En consecuencia, se te advierte de que la Lista puede actualizarse de vez en cuando y puede no estar completa. Suponiendo que no modifiques el código fuente abierto utilizado en los Servicios Braze, el uso de los Servicios Braze no te obligará a conceder a terceros ninguno de tus derechos de propiedad intelectual en virtud de una licencia de código abierto, ni te obligará a poner a disposición de terceros ninguno de tus códigos fuente en virtud de una licencia de software como servicio (SaaS).
