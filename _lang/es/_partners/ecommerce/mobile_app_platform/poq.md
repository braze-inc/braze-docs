# Poq

[Poq](https://poqcommerce.com/) permite a las empresas lanzar, gestionar y escalar rápidamente aplicaciones nativas de iOS y Android, ofreciendo experiencias móviles de alto rendimiento que impulsan el comercio y dan vida a la promesa de tu marca.

*Esta integración es mantenida por Poq.*

## Acerca de la integración

La integración de Poq y Braze te permite conectar la experiencia de tu aplicación móvil nativa con la plataforma de interacción con los clientes de Braze. Mediante los SDK de Braze para iOS y Android, Braze se integra directamente en una aplicación impulsada por Poq, lo que permite la segmentación, personalización y entrega de campañas dirigidas en dispositivos móviles.

## Qué se admite

Se admiten los siguientes canales y características.

| Canal/Característica | Descripción |
| :---- | :---- |
| Notificaciones push | Envía notificaciones push dirigidas y personalizadas a los usuarios de la aplicación a través de Braze, incluyendo notificaciones push enriquecidas. |
| Mensajes dentro de la aplicación | Muestra mensajes dentro de la aplicación de Braze en la experiencia nativa de la aplicación para interactuar con los usuarios en el momento adecuado de su recorrido. |
| Análisis y seguimiento de eventos | Poq envía un conjunto completo de eventos de la aplicación a Braze, cubriendo todo el recorrido del cliente. Estos datos potencian la segmentación, los desencadenantes de campañas y la personalización en Braze. |
| Tarjetas de contenido y buzón de entrada de mensajes | Entrega Tarjetas de contenido persistentes de Braze dentro de la aplicación, ya sea en una pantalla dedicada de buzón de entrada de mensajes, en páginas creadas con el constructor de contenido dinámico de Poq o en espacios predefinidos a lo largo de la aplicación. Las Tarjetas de contenido se filtran mediante pares clave-valor configurados en Braze. |
| Banners | Muestra banners de Braze dentro de la aplicación, ya sea en páginas creadas con el constructor de contenido dinámico de Poq o en espacios predefinidos a lo largo de la aplicación. El contenido de los banners se obtiene mediante un ID de ubicación. |

## Requisitos previos

Necesitas lo siguiente para usar esta integración.

| Requisitos | Descripción |
| :---- | :---- |
| Aplicación Poq | Se requiere una aplicación Poq para aprovechar esta integración. |
| Claves de API de la aplicación Braze | Se requiere una clave de API de la aplicación Braze para cada plataforma (iOS y Android). Obtén estas claves desde **Configuración** > **APIs e identificadores** > **Identificadores de aplicación** en el dashboard de Braze. |
