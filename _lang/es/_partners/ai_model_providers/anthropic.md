# Antrópico

> [Anthropic](https://www.anthropic.com/) es una empresa de seguridad e investigación en IA que está desarrollando Claude, un asistente de IA de nueva generación construido para ser útil, honesto y seguro en una amplia gama de tareas lingüísticas.

_Esta integración es mantenida por Anthropic._

## Sobre la integración

La integración de Braze y Anthropic te permite conectar tu clave de API de Anthropic a Braze para que puedas utilizar los modelos de Claude al crear agentes de IA personalizados. Con esta integración, tus agentes pueden generar textos personalizados, tomar decisiones en tiempo real o actualizar los campos del catálogo utilizando los modelos Claude de Anthropic.

## Requisitos previos

| Requisitos | Descripción |
|---|---|
| Cuenta antrópica con clave de API | Una cuenta Antrópica con una clave de API. Para obtener ayuda, ponte en contacto con tu administrador o con [el servicio de asistencia de Anthropic](https://support.anthropic.com/). |
| instancia de Braze | Puedes encontrar tu instancia de Braze en la [página de resumen de la API]({{site.baseurl}}/api/basics/#endpoints) o a través de tu administrador de incorporación de Braze. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integración

Para conectar tu clave de API de Anthropic a Braze:

1. Ve a **Integraciones de socios** > **Socios tecnológicos** en el panel de Braze y busca Anthropic.
2. Introduce tu clave de API de Anthropic.
3. Seleccione **Guardar**.

Después de guardar, puedes seleccionar los modelos de Claude cuando [crees un agente personalizado]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) en la Consola de Agentes.

Ponte en contacto con [el servicio de asistencia de Anthropic](https://support.anthropic.com/) si tienes algún problema o pregunta sobre tu integración.
