Tutorías:
Un recorrido general por un concepto instructivo. Debe contener CONOCIMIENTOS PRÁCTICOS. Se centra en un único tema (por ejemplo, cómo crear una campaña, cómo crear un Canvas, etc.) Artículo orientado a objetivos o tareas que va PASO A PASO resolviendo un problema ESPECÍFICO (cómo dirigirse a usuarios concretos, cómo segmentar en función de la ubicación, etc.).

### Ejemplo de código

Si estás explicando un concepto técnico, anótalo aquí y muestra un ejemplo de código.

```html
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
```

Asegúrate de definir los parámetros o elementos que los usuarios podrían tener que ajustar a partir de la muestra de código. Muchos usuarios se limitan a copiar y pegar.

| Variable | Descripción |
| -------- | ----------- |
| Título de la página | Puedes titular tu página como quieras. Tienes que tener esto. |
| Mi primer encabezado | Recomendamos ponerlo en mayúsculas. Esto también es opcional. |
