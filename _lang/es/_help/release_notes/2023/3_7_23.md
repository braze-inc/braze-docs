# Lanzamiento el 7 de marzo de 2023

## Eliminar la posibilidad de duplicar los lienzos de experiencia originales

A partir del 28 de febrero de 2023, ya no podrás crear ni duplicar Lienzos utilizando la experiencia original de Canvas. Braze recomienda a los clientes que utilicen la experiencia Canvas original que se pasen a Canvas Flow. Es una experiencia de edición mejorada para construir y gestionar mejor los lienzos. Más información sobre la [clonación de tus lienzos en el flujo de Canvas]({{site.baseurl}}/user_guide/engagement_tools/canvas/managing_canvases/cloning_canvases/).

## Actividades en vivo para iOS (acceso anticipado)

[Las Actividades en vivo]({{site.baseurl}}/developer_guide/push_notifications/live_notifications/?sdktab=swift) son notificaciones persistentes e interactivas que se muestran en tu pantalla de bloqueo y te permiten estar al tanto de todo en tiempo real. Como aparecen en la pantalla de bloqueo, las Actividades en vivo garantizan que no se pierdan tus notificaciones. Como son persistentes, puedes mostrar contenido actualizado a tus usuarios sin ni siquiera tener que desbloquear su teléfono.

## Creación de tarjetas para tarjetas de contenido

Ahora puedes elegir cuándo Braze evalúa la elegibilidad de la audiencia y la personalización para las nuevas campañas de tarjeta de contenido, especificando cuándo se crea la tarjeta.

Están disponibles las siguientes opciones:

- **En el lanzamiento de la campaña:** El comportamiento anterior por defecto para las Tarjetas de Contenido. Braze calcula la audiencia elegible y la personalización cuando se lanza la campaña, luego crea la tarjeta y la almacena hasta que el usuario abre tu aplicación.
- **En la primera impresión:** La próxima vez que el usuario abra tu aplicación (es decir, cuando inicie una nueva sesión), Braze determinará para qué tarjetas de contenido es elegible, plantillas de personalización como Liquid o Contenido conectado, y luego creará la tarjeta.

Para más información, consulta la [creación de tarjetas]({{site.baseurl}}/user_guide/message_building_by_channel/content_cards/create/card_creation/).

## Restablecer estilos para mensajes dentro de la aplicación Editor de arrastrar y soltar

En el editor de arrastrar y soltar para mensajes dentro de la aplicación, ahora puedes restablecer rápidamente los estilos a su estilo predeterminado después de hacer cambios. Para más información, consulta [Restablecer los estilos predeterminados]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/#resetting-styles-to-default).

## Dominios personalizados para acortar enlaces

El acortamiento de enlaces también te permite utilizar tu propio dominio para personalizar el aspecto de tus URL acortadas, lo que ayuda a mostrar una imagen de marca coherente. Una vez configurados, [los dominios personalizados]({{site.baseurl}}/user_guide/message_building_by_channel/sms/campaign/link_shortening/#custom-domains) pueden asignarse a uno o varios grupos de suscripción SMS.

## Notificación push web para Safari móvil

Safari v16.4 es compatible con la notificación push web móvil, lo que significa que ahora puedes reactivar la interacción de los usuarios móviles con notificaciones push en iOS y iPadOS. Sigue nuestra guía dedicada para saber cómo admitir [la notificación push web en Safari para iOS y iPadOS]({{site.baseurl}}/developer_guide/push_notifications/?sdktab=safari).

## Casos de uso del componente de actualización de usuarios

El componente de actualización de usuarios en Canvas te permite actualizar los atributos, eventos y compras de un usuario en un compositor JSON, pero ¿no estás muy seguro de cómo aprovechar al máximo esta característica? Hemos añadido [tres casos de uso de ejemplo]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/user_update/#use-cases) para darte algunas ideas.

## Búsqueda de usuarios

Este nuevo artículo describe cómo utilizar [la búsqueda de]({{site.baseurl}}/user_guide/engagement_tools/segments/creating_a_segment/) usuarios para buscar a un usuario concreto directamente desde el compositor para comprobar si tus filtros y segmentos están configurados correctamente. Esto también puede ser útil a la hora de solucionar problemas de una campaña o Canvas que no se está enviando como se esperaba; por ejemplo, si los usuarios no están recibiendo un mensaje cuando deberían.

La búsqueda de usuarios está disponible cuando:

- Crear un segmento
- Configurar una campaña o un público de Canvas
- Configuración del paso de rutas de audiencia

## Añadir a lista de bloqueo o eliminar datos personalizados

En este nuevo artículo se describe cómo retirar un objeto de datos personalizados [añadiendo a la lista de bloqueo o eliminando datos personalizados]({{site.baseurl}}/user_guide/data_and_analytics/custom_data/managing_custom_data/#blocklisting-custom-data).

En ocasiones puedes identificar atributos personalizados, eventos personalizados o eventos de compra que consumen demasiados puntos de datos, que ya no son útiles para tu estrategia de marketing o que se registraron por error. Para impedir que estos datos se envíen a Braze, puedes bloquear un objeto personalizado de datos mientras tu equipo de ingeniería trabaja para eliminarlo del backend de tu aplicación o sitio web.

## Nuevas asociaciones Braze

### Sisu Data - Inteligencia empresarial

La integración de [Sisu Data]({{site.baseurl}}/partners/data_and_analytics/business_intelligence/sisu_data/) y Braze te permite comprender en todas las campañas o a nivel de campaña por qué están cambiando las métricas (por ejemplo, la tarifa abierta, la tasa de clics, la tasa de conversión, etc.) y qué impulsa los resultados más óptimos. Una vez identificados estos segmentos, los usuarios de Braze pueden materializar los resultados en su almacén de datos o enviarlos directamente de Sisu a Braze para reorientar y reactivar la interacción de los usuarios.

### Loplat - Ubicación contextual

La integración de Braze y [loplat]({{site.baseurl}}/partners/message_personalization/location/loplat/) te permite utilizar los servicios de ubicación de loplat (POI de tienda y geovalla personalizada) para desencadenar campañas de marketing geo-contextual y crear eventos personalizados mediante segmentación offline. Cuando los usuarios visitan la ubicación seleccionada que has configurado en loplat X, la información sobre la campaña y la ubicación se envía inmediatamente a Braze.

### ActionIQ - Plataforma de datos de los clientes

La integración de Braze y [ActionIQ]({{site.baseurl}}/partners/data_and_analytics/customer_data_platform/actioniq/) permite a las marcas sincronizar y mapear sus datos de ActionIQ directamente con Braze, potenciando la entrega de experiencias del cliente extraordinarias basadas en toda la amplitud de sus datos de clientes. La integración habilita a los usuarios para:

- Asigne segmentos de audiencia o atributos personalizados a Braze directamente desde ActionIQ
- Transmite los eventos seguidos por ActionIQ a Braze en tiempo real para desencadenar campañas personalizadas y específicas

### Komo - Contenido dinámico

La integración de Braze y [Komo]({{site.baseurl}}/partners/additional_channels_and_extensions/extensions/landing_pages/komo/) te permite recopilar datos propios y zero-party data a través de los Komo Engagement Hubs. Estos hubs son micrositios dinámicos que ofrecen contenidos interactivos y características de gamificación. Los datos de usuario recogidos de estos centros se transmiten a la API de Braze.

- Ingesta de datos de usuario propios y de terceros desde Komo a Braze en tiempo real.
- Ingesta de datos de estudios de mercado y preferencias de los usuarios cuando responden a encuestas, cuestionarios y preguntas tipo test.
- Construir progresivamente perfiles de usuario en Braze a medida que el usuario sigue interactuando y compartiendo más datos sobre sí mismo.
- Estandarizar el aspecto de los correos electrónicos transaccionales enviados a través de Braze

### Phrase - Localización

La integración de [Phrase]({{site.baseurl}}/partners/message_personalization/localization/phrase/) y Braze te permite traducir plantillas de correo electrónico y bloques de contenido sin salir de la interfaz de Braze. Con la integración de Phrase TMS para Braze, puedes aumentar el compromiso de los clientes e impulsar el crecimiento en nuevos mercados con una localización perfecta.

### Nift - Fidelización

La integración de Braze y [Nift]({{site.baseurl}}/partners/additional_channels_and_extensions/extensions/rewards/nift/) te permite desencadenar automáticamente "agradecimientos" con regalos de Nift en momentos clave del ciclo de vida del cliente e identificar qué clientes utilizaron su regalo. Las tarjetas de regalo de Nift pueden utilizarse para acceder a productos y servicios suministrados por marcas que confían en la tecnología de emparejamiento de Nift para captar nuevos clientes de forma rentable a escala.

### Sageflo - Plantillas de mensajes

La integración de Braze y [Sageflo]({{site.baseurl}}/partners/message_orchestration/templates/sageflo/) permite a los equipos locales enviar fácilmente sus propios correos electrónicos utilizando plantillas aprobadas por marketing, imágenes y segmentos de audiencia mediante integraciones API con Braze.

### Airbyte - Automatización del flujo de trabajo

La integración de Braze y [Airbyte]({{site.baseurl}}/partners/data_and_analytics/workflow_automation/airbyte/) permite a los usuarios crear una canalización de datos para recopilar y analizar datos de Braze conectando todas tus aplicaciones y bases de datos a un almacén central. Una vez recopilados los datos en el almacén central, los equipos de datos pueden explorar los datos de Braze con eficacia utilizando sus herramientas de inteligencia empresarial preferidas.

### GrowthLoop - Automatización del flujo de trabajo

La integración de Braze y [GrowthLoop]({{site.baseurl}}/partners/data_and_analytics/customer_data_platform/growthloop/) te permite segmentar los datos de clientes directamente desde el almacén de datos y enviarlos a Braze, garantizando que los usuarios puedan optimizar el profundo conjunto de características de Braze junto con su única fuente de verdad. Agiliza los esfuerzos de marketing para la segmentación y activación de clientes, reduciendo el tiempo que se tarda en segmentar, lanzar, probar y medir los resultados de las campañas dirigidas enviadas a Braze.

### Mozart Data - Automatización del flujo de trabajo

La integración de [datos de Mozart]({{site.baseurl}}/partners/data_and_analytics/workflow_automation/mozart_data/) y Braze te permite:

- Utiliza Fivetran para importar datos de Braze a Snowflake
- Crear transformaciones combinando datos de Braze con datos de otras aplicaciones y analizar eficazmente los comportamientos de los usuarios.
- Importar datos de Snowflake a Braze para crear nuevas oportunidades de interacción con los clientes.
- Combinar los datos de Braze con los de otras aplicaciones para obtener una comprensión más holística de los comportamientos de los usuarios.
- Intégralo con una herramienta de inteligencia empresarial para explorar más a fondo los datos almacenados en Snowflake.
