guide_top_header: "Tratamiento de palabras clave"
guide_top_text: "Las palabras clave son un aspecto fundamental de la mensajería SMS automatizada. Con las palabras clave, sus usuarios pueden enviar mensajes a una lista preestablecida de comandos de una sola palabra que realizan alguna acción, como activar o desactivar la recepción de mensajes SMS. Con Braze, también puede establecer palabras clave personalizadas y activar la exclusión difusa para personalizar aún más el recorrido del usuario. <br><br> En estos artículos se explica cómo aborda Braze el procesamiento y la gestión de palabras clave, así como algunas prácticas recomendadas."
description: "Esta página de destino incluye artículos relacionados con el procesamiento de palabras clave de SMS, como las palabras clave de inclusión y exclusión, la gestión de palabras clave personalizadas, la gestión de números de teléfono desconocidos y la exclusión difusa."
