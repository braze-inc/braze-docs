# Promoción estacional con imagen de fondo

> Utiliza el editor de arrastrar y soltar mensajes dentro de la aplicación para promocionar una oferta de temporada o un trato para impulsar la interacción de los usuarios.

{% multi_lang_include drag_and_drop/templates.md section='Requisitos SDK' %}

## Crear una promoción de temporada con una imagen de fondo

### Paso 1: Elige tu plantilla

Cuando crees un mensaje dentro de la aplicación arrastrando y soltando, selecciona **Promoción estacional con imagen de fondo** para tu plantilla y, a continuación, selecciona **Crear mensaje**. Esta plantilla es compatible tanto con aplicaciones móviles como con navegadores web.

![El editor de mensajes dentro de la aplicación con la plantilla para una promoción de temporada.][img1]

### Paso 2: Configura tus estilos de mensaje

{% multi_lang_include drag_and_drop/templates.md section='estilo del mensaje' %}

### Paso 3: Personaliza tu componente botón

Para empezar a crear tu promoción de temporada, selecciona el componente botón en el editor. Después, utiliza el menú lateral para seleccionar a dónde se lleva a los usuarios cuando seleccionan el botón. La plantilla predeterminada es cerrar el mensaje, pero puedes elegir navegar a una página específica dentro de tu aplicación (como el producto que estás promocionando).

![El editor de mensajes dentro de la aplicación con un menú lateral para personalizar el elemento botón.][img2]

También puedes añadir mensajes adicionales al anuncio de tu producto en la sección **Páginas**, y luego enlazarlos para conseguir un flujo secuencial. Por ejemplo, puedes elaborar una secuencia de mensajes que describan brevemente las capacidades del producto y terminen con un botón que lleve a los usuarios a la página del producto. Aprende a hacerlo en [Conectar páginas juntos]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Paso 4: Estiliza tu mensaje

Personaliza el aspecto de tu promoción de temporada utilizando los [componentes de arrastrar y soltar de los mensajes dentro de la aplicación][3]. Añade tu propia imagen de fondo sustituyendo la URL predeterminada de la imagen de fondo en el menú **contenedor de mensajes** o elimina la URL y selecciona tu imagen de la [biblioteca multimedia]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

![El editor de mensajes dentro de la aplicación con un menú lateral para seleccionar la imagen de fondo.][img3]
