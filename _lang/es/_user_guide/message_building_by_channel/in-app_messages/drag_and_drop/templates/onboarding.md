# Flujo de incorporación

> Utiliza la plantilla de mensajes dentro de la aplicación **del flujo de incorporación** para recopilar atributos, información y preferencias de los usuarios que impulsen tu estrategia de campaña. 

{% multi_lang_include drag_and_drop/templates.md section='Requisitos SDK' %}

## Crear un flujo de incorporación

### Paso 1: Elige tu plantilla

Cuando crees un mensaje dentro de la aplicación arrastrando y soltando, selecciona **Flujo de incorporación** para tu plantilla y, a continuación, selecciona **Crear mensaje**. Esta plantilla es compatible tanto con aplicaciones móviles como con navegadores web.

![El editor de mensajes dentro de la aplicación con la plantilla de flujo de incorporación.]({% image_buster /assets/img/drag_and_drop/templates/onboarding_editor.png %})

### Paso 2: Configura tus estilos de mensaje

{% multi_lang_include drag_and_drop/templates.md section='estilo del mensaje' %}

### Paso 3: Personaliza el orden de tu flujo de incorporación

Para empezar a crear tu flujo de incorporación, utiliza la sección **Páginas** para añadir o eliminar mensajes a tu flujo. Cuando tengas el número de mensajes que desees, selecciona los botones de cada página. Después, utiliza el menú lateral para seleccionar a dónde van los usuarios cuando seleccionan el botón: si salen del flujo o van al siguiente mensaje.

Comprueba que tus mensajes están conectados en orden secuencial para que los usuarios se muevan por el flujo correctamente. Para saber cómo hacerlo, consulta [Conectar páginas juntos]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Paso 4: Estiliza tu mensaje

Personaliza el aspecto de tu flujo de incorporación utilizando los [componentes de arrastrar y soltar de los mensajes dentro de la aplicación]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components).
