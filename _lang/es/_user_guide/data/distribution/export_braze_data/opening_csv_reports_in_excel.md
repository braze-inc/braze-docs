# Abrir exportaciones CSV en Excel

> Esta página explica cómo abrir exportaciones CSV en Excel.

## Establecer Excel como programa predeterminado

Aunque los archivos CSV suelen abrirse automáticamente en Excel de forma predeterminada, no siempre es así. Consulta los artículos de solución de problemas [de](https://support.apple.com/guide/mac-help/choose-an-app-to-open-a-file-on-mac-mh35597/mac) [Windows](https://support.microsoft.com/en-us/windows/change-which-programs-windows-7-uses-by-default-62fd162f-8c82-0436-806f-c60d69dcf495) y Apple para conocer los pasos a seguir para establecer Excel como programa predeterminado.

## Convertir CSV a XLSX o XLS

Para convertir un CSV a XLSX o XLS, o eliminar la coma entre los valores de los datos, consulta [esta guía](https://www.ablebits.com/office-addins-blog/convert-csv-excel/#import-csv-wizard) para importar CSV a Excel.

## Eliminación de los ceros a la izquierda de los ID de usuario

A veces puede ocurrir que se eliminen los ceros a la izquierda de los ID de usuario en la exportación CSV. Esto ocurre porque Excel trata los números de un CSV como datos en lugar de como texto. Para resolverlo, ejecuta el [Asistente para importar texto de Excel](https://www.ablebits.com/office-addins-blog/converting-csv-excel-issues/#leading-zeros).
