guide_top_text: "<h2>¿Qué es?</h2>Braze Cloud Data Ingestion (CDI) te permite configurar una conexión directa desde tu solución de almacenamiento en la nube a Braze para sincronizar datos relevantes de usuarios o catálogos y eliminar usuarios. Cuando se sincronizan con Braze, estos datos pueden aprovecharse para casos de uso como la personalización o la segmentación. La integración flexible de Cloud Data Ingestion admite estructuras de datos complejas, como JSON anidado y matrices de objetos. <br><br>**Capacidades de ingesta de datos en la nube de Braze:**<br> - Crea una integración sencilla directamente desde tu almacén de datos o solución de almacenamiento de archivos a Braze en sólo unos minutos.<br>- Sincroniza de forma segura los datos de usuario, incluyendo atributos, eventos y compras desde tu almacén de datos a Braze.<br>- Cierra el bucle de datos con Braze combinando la ingesta de datos en la nube con la compartición de datos Currents o Snowflake.<br><br>**La ingesta de datos en la nube puede sincronizar datos de**:<br> - Amazon Redshift<br> - Tejidos de datos<br> - Google BigQuery<br> - Tejido Microsoft<br> - S3<br> - Snowflake"
