# Solución de problemas de BrazeAI Operator

> Problemas comunes y soluciones para BrazeAI Operator<sup>TM</sup>.

## Problemas comunes

| Problema | Solución de problemas |
| --- | --- |
| Sin respuesta | Intenta actualizar la página y volver a abrir el panel del operador. |
| Respuestas fuera de tema | Reformula la pregunta de forma más específica. Menciona la característica o el flujo de trabajo sobre el que se te pregunta. |
| Mensajes de error | Si el operador no puede transmitir contenido, es posible que aparezca un mensaje que dice «Intenta de nuevo». Es posible que el operador no esté disponible temporalmente o que la conexión se haya interrumpido. Vuelve a intentarlo dentro de unos minutos. |
| Límite de uso diario superado | Se ha alcanzado el límite de uso de la empresa. Espera a que se restablezca el límite (se restablece cada 24 horas). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Limitaciones

- **Acceso a los datos:** El operador puede ver el contenido de la página activa. También puede buscar algunos datos específicos de la empresa, como campañas anteriores, catálogos y atributos personalizados. Sin embargo, actualmente no puede generar nuevos informes ni ejecutar consultas de informes.
- **Límites de uso:** El operador tiene un límite de uso diario para toda la empresa que se restablece cada 24 horas. Si se alcanza el límite, aparecerá un banner con el mensaje «Límite de uso diario excedido» en el chat y no se podrán realizar más solicitudes hasta que se restablezca el límite.
