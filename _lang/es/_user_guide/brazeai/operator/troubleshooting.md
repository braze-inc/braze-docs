# Solución de problemas de BrazeAI Operator

> Problemas habituales y soluciones para BrazeAI Operator<sup>TM</sup>.

## Problemas habituales

| Problema | Solución |
| --- | --- |
| Sin respuesta | Actualiza la página y vuelve a abrir el panel de Operator. |
| Respuestas fuera de tema | Reformula la pregunta de forma más específica. Menciona la función o el flujo de trabajo en cuestión. |
| Mensajes de error | Si Operator no puede transmitir contenido, puede aparecer la solicitud "Intentar de nuevo". Operator puede estar temporalmente no disponible o la conexión se ha interrumpido. Vuelve a intentarlo tras unos minutos. |
| Límite de uso diario superado | Se ha alcanzado el límite de uso de la empresa. Espera a que el límite se restablezca (cada 24 horas). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Limitaciones

- **Acceso a datos:** Operator puede ver el contenido de la página activa. También puede consultar algunos datos específicos de la empresa, como campañas anteriores, catálogos y atributos personalizados. Sin embargo, actualmente no puede generar nuevos informes ni ejecutar consultas de informes.
- **Límites de uso:** Operator tiene un límite de uso diario a nivel de empresa que se restablece cada 24 horas. Cuando se alcanza el límite, aparecerá el mensaje "Límite de uso diario superado" en el chat y no se podrán hacer más solicitudes hasta que se restablezca.
