# Solución de problemas de BrazeAI Operator

> Problemas comunes y soluciones para BrazeAI Operator<sup>TM</sup>.

## Problemas comunes

| Problema | Solución de problemas |
| --- | --- |
| Sin respuesta | Intenta actualizar la página y volver a abrir el panel de Operator. |
| Respuestas fuera de tema | Reformula la pregunta de forma más específica. Menciona la característica o el flujo de trabajo sobre el que preguntas. |
| Mensajes de error | Si Operator no puede transmitir contenido, es posible que aparezca un mensaje que dice «Inténtalo de nuevo». Es posible que Operator no esté disponible temporalmente o que la conexión se haya interrumpido. Vuelve a intentarlo dentro de unos minutos. |
| Límite de uso diario superado | Se ha alcanzado el límite de uso de toda la empresa. Espera a que se restablezca el límite (se restablece cada 24 horas). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Limitaciones

- **Acceso a los datos:** Operator puede ver el contenido de la página activa, excepto el contenido dentro de editores de arrastrar y soltar. También puede buscar algunos datos específicos de la empresa, como campañas anteriores, catálogos y atributos personalizados. Sin embargo, actualmente no puede generar nuevos informes ni ejecutar consultas de informes.
- **Límites de uso:** Operator tiene un límite de uso diario para toda la empresa que se restablece cada 24 horas. Si se alcanza el límite, aparecerá un banner con el mensaje «Límite de uso diario superado» en el chat y no se podrán realizar más solicitudes hasta que se restablezca el límite.
