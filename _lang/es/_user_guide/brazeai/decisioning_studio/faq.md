# Preguntas más frecuentes

> Este artículo da respuesta a algunas preguntas frecuentes sobre Decisioning Studio.

### ¿Qué es un agente decisor?

Un agente de decisión es una configuración personalizada de BrazeAI Decisioning Studio™ hecha a medida para cumplir un objetivo de negocio específico. Esto se define por la métrica de éxito, las dimensiones y las opciones que elijas. El agente de toma de decisiones descubre automáticamente la acción óptima para cada cliente con el fin de maximizar la métrica empresarial que hayas elegido.

### ¿Qué métricas puedo optimizar?

Puedes optimizar para cualquier métrica empresarial que se alinee con tus objetivos, como ingresos, conversiones, ingresos medios por usuario (ARPU), valor de duración del cliente (CLV), ganancias, renovaciones de contratos o cualquier otro KPI empresarial.

### ¿Qué son las dimensiones en Decisioning Studio?

Las dimensiones pueden considerarse como los *tipos de palancas* de las que puede tirar el agente decisor para maximizar la métrica del éxito. Las dimensiones típicas incluyen la oferta, la línea del asunto, la creatividad, el canal o la hora de envío.

### ¿Qué es un banco de acción?

El banco de acciones define las *opciones concretas* a las que tiene acceso el agente decisor para cada dimensión "palanca". Por ejemplo, para una dimensión de canal, definirías los canales específicos a los que tiene acceso el agente decisor. Para una dimensión de oferta, definirías las ofertas concretas que el agente decisor puede probar.

### ¿Puede el agente decisor realizar acciones que yo no haya configurado?

No. El agente decisor sólo puede realizar acciones que tú configures y añadas al banco de acciones. Esto significa que todas las acciones posibles están definidas por las combinaciones de lo que pongas en el banco de acciones.

### ¿Qué son las limitaciones?

Las restricciones limitan las acciones del agente decisor para respetar las reglas críticas de la empresa. Por ejemplo, podría ser impedir que se seleccione una oferta concreta para los clientes de una zona geográfica no elegible, o establecer un presupuesto máximo para el agente decisor.

### ¿Cuál es la diferencia entre Decisioning Studio Go y Decisioning Studio Pro?

Decisioning Studio Pro incluye el soporte de los Servicios de Decisión de IA del equipo de ciencia de datos de Braze, que te ayudará a diseñar y configurar tu agente para maximizar los resultados de tu negocio. Para más información, consulta [Decisioning Studio Go frente a Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
