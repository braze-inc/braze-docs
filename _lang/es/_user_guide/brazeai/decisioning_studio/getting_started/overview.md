# Resumen

> Esta referencia ofrece un resumen de los pasos necesarios para configurar Decisioning Studio, incluida la conexión de orígenes de datos, la configuración de la orquestación y el diseño de tu agente de decisión.

BrazeAI Decisioning Studio™ te permite diseñar y desplegar agentes de decisión que optimicen cualquier métrica empresarial. Un agente decisor es una configuración personalizada y adaptada a un objetivo de negocio concreto.

Para ello, debes conectar los orígenes de datos, configurar la orquestación y diseñar tus agentes de decisión.

{% alert tip %}
Para los clientes de Decisioning Studio Pro, tu equipo de servicios expertos en IA te ayudará a configurar Decisioning Studio para obtener un rendimiento óptimo.
{% endalert %}

## Configuración de Decisioning Studio

Para configurar Decisioning Studio, deberás completar los siguientes pasos:

### Paso 1: Conectar orígenes de datos

Conecta los perfiles de clientes y los datos de interacción para que los agentes decisores que crees comprendan quién es cada cliente y cómo se comporta.

Normalmente, sólo tendrás que conectar tus orígenes de datos una vez, durante la configuración inicial de Decisioning Studio. Si amplías tus casos de uso más adelante, puede que necesites añadir nuevos orígenes de datos.

{% alert tip %}
Cualquier dato que ya esté en la [Plataforma de Datos Braze]({{site.baseurl}}/user_guide/data/braze_data_platform) estará disponible automáticamente para Decisioning Studio.
{% endalert %}

Para obtener información detallada, consulta la documentación de tu nivel de Decisioning Studio:
- [Estudio de decisión Go: Conectar orígenes de datos]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Decisioning Studio Pro: Conectar orígenes de datos]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Paso 2: Configurar la orquestación

Integra Decisioning Studio con tu plataforma de interacción con los clientes (CEP) para que tus agentes puedan orquestar acciones. El CEP es la plataforma utilizada para entregar experiencias personalizadas a tus clientes basadas en las decisiones del agente.

Por lo general, sólo tendrás que configurar esta orquestación una vez.

Para obtener información detallada, consulta la documentación de tu nivel de Decisioning Studio:
- [Estudio de decisión Go: Configurar la orquestación]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Decisioning Studio Pro: Configurar la orquestación]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Paso 3: Diseña a tus agentes

Configura tus agentes de decisión para definir qué resultados quieres maximizar y qué acciones puede realizar el agente para conseguirlos. Consulta [Diseñar Agentes Decisores]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/) para una orientación detallada sobre el diseño de agentes.

Para una orientación específica por niveles:
- [Estudio de decisión Go: Diseña tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Decisioning Studio Pro: Diseña tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Para los clientes de Decisioning Studio Pro, tu equipo de Servicios de Decisión de IA te ayudará a diseñar y poner en marcha tus agentes de decisión.
{% endalert %}

### Paso 4: Pon en marcha tu agente decisor

Pon en marcha tu agente decisor y deja que aprenda continuamente y optimice los resultados de tu negocio.

Para obtener información detallada, consulta la documentación de tu nivel de Decisioning Studio:
- [Estudio de decisión Go: Lanza tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Decisioning Studio Pro: Lanza tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
