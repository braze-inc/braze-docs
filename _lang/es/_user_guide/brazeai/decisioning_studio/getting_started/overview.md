# Resumen

> Esta referencia ofrece un resumen de los pasos necesarios para configurar Decisioning Studio, incluyendo la conexión de orígenes de datos, la configuración de la orquestación y el diseño de tu agente de decisión.

BrazeAI Decisioning Studio™ te permite diseñar e implementar agentes de toma de decisiones que optimizan cualquier métrica empresarial. Un agente de decisión es una configuración personalizada diseñada para cumplir un objetivo de negocio específico.

Para ello, debes conectar los orígenes de datos, configurar la orquestación y diseñar tus agentes de decisión.

{% alert tip %}
Para los clientes de Decisioning Studio Pro, el equipo de servicios expertos en IA te ayudará a configurar Decisioning Studio para obtener un rendimiento óptimo.
{% endalert %}

## Configuración de Decisioning Studio

Para configurar Decisioning Studio, debes completar los siguientes pasos:

### Paso 1: Conectar orígenes de datos

Conecta los perfiles de los clientes y los datos de interacción para que los agentes de toma de decisiones que crees comprendan quién es cada cliente y cómo se comporta.

Por lo general, solo es necesario conectar los orígenes de datos una vez, durante la configuración inicial de Decisioning Studio. Si amplías tus casos de uso más adelante, es posible que tengas que añadir nuevos orígenes de datos.

{% alert tip %}
Todos los datos que ya se encuentran en la [plataforma de datos Braze]({{site.baseurl}}/user_guide/data/braze_data_platform) están automáticamente disponibles para Decisioning Studio.
{% endalert %}

Para obtener instrucciones detalladas, consulta la documentación correspondiente a tu nivel de Decisioning Studio:
- [Decisioning Studio Go: Conectar orígenes de datos]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Decisioning Studio Pro: Conectar orígenes de datos]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Paso 2: Configurar la orquestación

Realiza la integración de Decisioning Studio con tu plataforma de interacción con los clientes (CEP) para que tus agentes puedan realizar la orquestación de acciones. El CEP es la plataforma que se utiliza para entregar experiencias personalizadas a tus clientes en función de las decisiones del agente.

Por lo general, solo es necesario configurar esta orquestación una vez.

Para obtener instrucciones detalladas, consulta la documentación correspondiente a tu nivel de Decisioning Studio:
- [Decisioning Studio Go: Configurar la orquestación]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Decisioning Studio Pro: Configurar la orquestación]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Paso 3: Diseña tus agentes

Configura tus agentes de decisión para definir qué resultados deseas maximizar y qué acciones puede realizar el agente para conseguirlos. Consulta [Diseño de agentes de decisión]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/) para obtener orientación detallada sobre el diseño de agentes.

Para obtener orientación específica para cada nivel:
- [Decisioning Studio Go: Diseña tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Decisioning Studio Pro: Diseña tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Para los clientes de Decisioning Studio Pro, el equipo de servicios de toma de decisiones basados en IA te ayudará a diseñar y poner en marcha tus agentes de toma de decisiones.
{% endalert %}

### Paso 4: Lanza tu agente de toma de decisiones

Pon en marcha tu agente de toma de decisiones y deja que aprenda y se optimice continuamente para obtener los mejores resultados para tu negocio.

Para obtener instrucciones detalladas, consulta la documentación correspondiente a tu nivel de Decisioning Studio:
- [Decisioning Studio Go: Inicia tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Decisioning Studio Pro: Inicia tu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
