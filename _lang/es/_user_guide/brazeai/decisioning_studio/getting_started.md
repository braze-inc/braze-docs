# Primeros pasos con Decisioning Studio

> Esta referencia ofrece un resumen de los pasos necesarios para configurar Decisioning Studio, incluida la conexión de orígenes de datos, la configuración de la orquestación y el diseño de tu agente de decisión.

BrazeAI Decisioning Studio™ te permite diseñar y desplegar agentes de decisión que optimicen cualquier métrica empresarial. Un agente decisor es una configuración personalizada y adaptada a un objetivo de negocio concreto.

Para ello, debes conectar los orígenes de datos, configurar la orquestación y diseñar tus agentes de decisión.

{% alert tip %}
Para los clientes de Decisioning Studio Pro, tu equipo de servicios expertos en IA te ayudará a configurar Decisioning Studio para obtener un rendimiento óptimo.
{% endalert %}
