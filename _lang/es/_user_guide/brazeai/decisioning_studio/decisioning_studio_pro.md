# BrazeAI Decisioning Studio™ Pro

## Acerca de Decisioning Studio Pro

Decisioning Studio Pro es más flexible y potente que Decisioning Studio Go. Pro incluye la personalización completa del banco de acciones, la posibilidad de optimizar en función de cualquier métrica (como ingresos, conversiones o interacción) y es compatible con cualquier plataforma de interacción con los clientes (CEP).

La configuración de un agente Pro requiere la colaboración con nuestro equipo de servicios de toma de decisiones de IA, el equipo de ciencia de datos de Braze, que guía y asiste en el diseño del agente para aportar el máximo valor a tu empresa.

Para saber más sobre las diferencias entre Decisioning Studio Go y Decisioning Studio Pro, consulta [Decisioning Studio Go frente a Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
