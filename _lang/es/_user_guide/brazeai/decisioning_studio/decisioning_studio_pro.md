# BrazeAI Decisioning Studio™ Pro

## Acerca de Decisioning Studio Pro

Decisioning Studio Pro es más flexible y potente que Decisioning Studio Go. Pro incluye la personalización completa del banco de acciones, la capacidad de optimizar cualquier métrica (como ingresos, conversiones o interacción) y es compatible con cualquier plataforma de interacción con los clientes (CEP).

La configuración de un agente Pro requiere la colaboración con nuestro equipo de servicios de toma de decisiones basados en IA, el equipo de ciencia de datos de Braze, que te guía y ayuda en el diseño del agente para obtener el máximo valor para tu negocio.

Para obtener más información sobre las diferencias entre Decisioning Studio Go y Decisioning Studio Pro, consulta [Decisioning Studio Go frente a Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
