# BrazeAI Decisioning Studio™ Go

## Acerca de Decisioning Studio Go

Decisioning Studio Go incluye configuración creativa autoservicio con un banco de acciones limitado, características basadas únicamente en la interacción con los clientes, optimización de clics y solo es compatible con las tres plataformas de interacción con los clientes (CEP) siguientes:

* Braze
* Salesforce Marketing Cloud
* Klaviyo

Para obtener más información sobre las diferencias entre Decisioning Studio Go y Decisioning Studio Pro, consulta [Decisioning Studio Go frente a Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
