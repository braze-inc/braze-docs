# Utilizar API públicas para contenidos conectados

> Hay varias API disponibles públicamente que pueden utilizarse para el Contenido conectado. Las API públicas te permiten insertar datos personalizados en los mensajes. Esta página recopila una lista de API públicas que podrían utilizarse para el Contenido conectado. Sin embargo, existen muchas más API, que proporcionan una amplia variedad de usos potenciales de los Contenidos Conectados.  

Ponte en contacto con nosotros en [success@braze.com](mailto:success@braze.com) para decirnos si tienes una API que compartir.

{% alert note %}
Las API públicas pueden estar sujetas a restricciones de uso y limitación de tasas. Asegúrate de leer la documentación de la API y de ponerte en contacto con los proveedores de la API sobre el uso que pretendes darle.
{% endalert %}

## Noticias e información

|	 API 	| Descripción |
| --------- | --- |
| [OpenWeatherMap](http://openweathermap.org/api) | Proporciona datos meteorológicos actuales, previsiones para 5 y 16 días, y datos históricos. |
| [Búsqueda de artículos en el NYT](https://developer.nytimes.com/docs/articlesearch-product/1/overview) | Proporciona datos de artículos del NYT que incluyen titular, tema, URL, fecha, resumen, etc. |
| [La API de The Guardian](http://open-platform.theguardian.com/documentation/) | Proporciona los datos del artículo Guardian, que incluyen titular, tema, URL, fecha, resumen, etc.|
