# Archivar campañas y lienzos

> Archivar tus campañas y lienzos puede ser una forma sencilla y eficaz de organizar tu espacio de trabajo. A medida que continúes creando más mensajes, el archivado puede ayudarte a centrarte en las iniciativas de mensajería activas y relevantes.

## Archivo

Cuando archivás una campaña o Canvas, no se entregarán más mensajes y tus mensajes dentro de la aplicación no se mostrarán. Aún puedes ver los resultados después de archivarlos, pero no podrás editar la campaña ni Canvas. Para ello, primero tendrás que [descomprimirlo](#unarchiving-campaigns-and-canvases).

{% tabs local %}
{% tab campaign %}
Para archivar una campaña:

1. Ve a **Mensajes** > **Campañas**.
2. Pasa el ratón por encima de la fila de la campaña y selecciona el menú <i class="fas fa-ellipsis-vertical"></i>.
3. Selecciona **Archivo**.
{% endtab %}

{% tab canvas %}
Para archivar un Canvas:

1. Ve a **Mensajería** > **Canvas**.
2. Coloca el cursor sobre la fila del Canvas y selecciona el<i class="fas fa-ellipsis-vertical"></i>menú.
3. Selecciona **Archivo**.
{% endtab %}
{% endtabs %}

## Desarchivar

Cuando desarchivas una campaña o un Canvas, esta no se publicará, solo cambiará su estado a **Activo**. Después de desarchivar, puedes editar y revisar cómo está configurado el mensaje sin que se envíe ningún mensaje.

Para reanudar la campaña o Canvas y comenzar a enviar mensajes, selecciona el<i class="fas fa-ellipsis-vertical"></i>menú de la fila correspondiente y selecciona **Reanudar**.
