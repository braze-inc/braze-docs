# Duplicar campañas y lienzos

> Puedes duplicar campañas y lienzos preexistentes para iterar continuamente y desarrollar tus estrategias exitosas. Duplicar campañas y lienzos ayuda a minimizar el tiempo que dedicarías a reconfigurar los detalles de entrega y los segmentos de audiencia.

## Duplicar

Después de duplicar una campaña o un Canvas, el nuevo mensaje se titulará `"Copy of [Original Name]"`automáticamente  y se te redirigirá a su página de edición. Estos duplicados permanecerán como borradores hasta que los edites y los publiques.

{% alert tip %}
También puedes [copiar tu campaña]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/copying_across_workspaces/) en diferentes espacios de trabajo. La copia permanecerá como borrador hasta que la edites y la publiques.
{% endalert %}

{% tabs %}
{% tab campaign %}
Para duplicar una campaña:

1. Ve a **Mensajes** > **Campañas**.
2. Coloca el cursor sobre la campaña y selecciona<i class="fas fa-gear"></i>  **Más acciones**.
3. Selecciona **Duplicar**.
{% endtab %}

{% tab canvas %}
Para duplicar un Canvas:

1. Ve a **Mensajería** > **Canvas**.
2. Pasa el ratón por encima de la fila Canvas y selecciona <i class="fas fa-ellipsis-vertical"></i> **Más acciones**.
3. Selecciona **Duplicar**.

{% endtab %}
{% endtabs %}
