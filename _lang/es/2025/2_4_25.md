# Lanzamiento el 4 de febrero de 2025.

## Mejoras en Braze Docs

### Guía para contribuir
Nuestras recientes actualizaciones de la [Guía para colaboradores]({{site.baseurl}}/contributing/your_first_contribution) facilitan a los usuarios sin conocimientos técnicos contribuir a Braze Docs.

### Renovación de datos y análisis
Para que te resulte más fácil encontrar lo que buscas, hemos separado los artículos que antes se agrupaban bajo «Análisis de& datos» en [Datos]({{site.baseurl}}/user_guide/data) y [Análisis]({{site.baseurl}}/user_guide/analytics). 

### Guía del desarrollador
Hemos realizado una limpieza exhaustiva de todos los documentos de la [Guía para desarrolladores de Braze]({{site.baseurl}}/developer_guide/home), lo que ha incluido la fusión de las instrucciones divididas en varias páginas en una sola página.

También hay una nueva [página de referencia del SDK]({{site.baseurl}}/developer_guide/references) que enumera toda la documentación de referencia y los repositorios de cada SDK de Braze.

#### SDK Braze para Unreal Engine
Hemos realizado la migración y reescritura de todo el contenido del repositorio GitHub README del SDK de Unreal Engine Braze a su [sección dedicada en Braze Docs]({{site.baseurl}}/developer_guide/sdk_integration/?sdktab=unreal%20engine).

## Flexibilidad de los datos

### Panel de uso de la API

{% multi_lang_include release_type.md release="General availability" %}

[El panel de control de uso de]({{site.baseurl}}/user_guide/analytics/dashboard/api_usage_dashboard) [la API]({{site.baseurl}}/user_guide/analytics/dashboard/api_usage_dashboard) te permite supervisar el tráfico entrante de la API REST en Braze para comprender tus tendencias en el uso de nuestras API REST y realizar la solución de problemas.

### Añadir etiquetas a atributos personalizados

{% multi_lang_include release_type.md release="General availability" %}

Puedes [añadir etiquetas a un atributo personalizado]({{site.baseurl}}/user_guide/data/custom_data/custom_attributes#adding-tags) después de crearlo si tienes el permiso «Gestionar eventos, atributos y compras». Las etiquetas pueden utilizarse para filtrar la lista de atributos.

### Selecciones de catálogo y puntos finales de campos de catálogo asíncronos 

{% multi_lang_include release_type.md release="General availability" %}

Los siguientes puntos finales ya están disponibles de forma generalizada:
* [POST: Crear campos de catálogo]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/post_create_catalog_fields)
* [DELETE: Eliminar campo del catálogo]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/delete_catalog_field)
* [DELETE: Eliminar selección del catálogo]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/delete_catalog_selection)
* [POST: Crear selección de catálogo]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/post_create_catalog_selections)

### Uso de una dirección de correo electrónico para desencadenar campañas o lienzos

{% multi_lang_include release_type.md release="General availability" %}

Ahora puedes especificar un destinatario por dirección de correo electrónico para desencadenar tus [campañas]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/targeting_users/) y [Canvases]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/?tab=target%20audience#step-2c-set-your-target-entry-audience).

### Uso de un número de teléfono como identificador de usuario a través de la API

{% multi_lang_include release_type.md release="General availability" %}

Ahora puedes utilizar un número de teléfono, además de un alias de usuario y una dirección de correo electrónico, como identificador de usuario a través del [punto `/users/identify`final de la API.]({{site.baseurl}}/api/endpoints/user_data/post_user_identify)

### Obtener un rastro SAML
Hemos añadido [pasos sobre cómo obtener un rastro SAML]({{site.baseurl}}/user_guide/administrative/access_braze/single_sign_on/set_up#obtaining-a-saml-trace), lo que te ayudará a resolver problemas relacionados con el SAML SSO con el servicio de asistencia de forma más eficiente.
 
### Centros de datos específicos para cada región
A medida que Braze crece para prestar servicio en nuevas áreas, hemos añadido un [artículo sobre los centros de datos de Braze]({{site.baseurl}}/user_guide/data/data_centers) para aclarar nuestro enfoque operativo.

 
## Desbloquear la creatividad
 
### Notificaciones de bajadas de precio y notificaciones de reposición de existencias

{% multi_lang_include release_type.md release="General availability" %}

Ahora puedes notificar a los clientes cuando un artículo vuelve a estar disponible configurando [notificaciones de reposición de existencias]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/back_in_stock_notifications) a través de Canvas y el catálogo.

También puedes crear [notificaciones de bajada de precios]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/price_drop_notifications) para avisar a los clientes cuando el precio de un artículo haya bajado, configurando las notificaciones de bajada de precios en un catálogo y en Canvas.

### Vista previa para selección 

{% multi_lang_include release_type.md release="General availability" %}

Después de crear una selección, puedes [ver qué resultados arrojaría dicha selección]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/selections/#test-and-preview) para un usuario aleatorio o para un usuario específico.

### Elementos del catálogo de plantillas, incluido Liquid 

{% multi_lang_include release_type.md release="General availability" %}

Puedes crear [plantillas para elementos del catálogo que incluyan Liquid]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/using_catalogs/#using-liquid).

### Plantillas de Canvas
Hemos añadido nuevas plantillas de Canvas para [la incorporación de usuarios con un cuestionario de preferencias]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/preference_survey) y [crear la posibilidad de registrarse por correo electrónico con doble adhesión voluntaria]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/email_signup).

### Administración de clientes potenciales con Salesforce Sales Cloud para B2B
Una forma en que los especialistas en marketing B2B pueden utilizar Braze es mediante la integración con Salesforce Sales Cloud. Lee más sobre cómo implementar este [caso de uso]({{site.baseurl}}/user_guide/getting_started/b2b_use_cases/b2b_salesforce_sales_cloud).
 
## Canales robustos

### Listas de supresión

{% multi_lang_include release_type.md release="Beta" %}
 
[Las listas de supresión]({{site.baseurl}}/user_guide/engagement_tools/segments/suppression_lists) especifican grupos de usuarios que nunca recibirán mensajes. Los administradores pueden crear listas de supresión con filtros de segmentación para reducir un grupo de usuarios de la misma manera que lo harías para la segmentación.

## Nuevas asociaciones Braze

### Constructor - Contenido dinámico
[Constructor]({{site.baseurl}}/partners/ecommerce/product_search_recommendations/constructor/) es una plataforma de búsqueda y descubrimiento de productos que utiliza inteligencia artificial y aprendizaje automático para entregar búsquedas personalizadas, recomendaciones de productos y experiencias de navegación para sitios web de comercio electrónico y de comercio minorista.
 
### Trustpilot - Contenido dinámico
[Trustpilot]({{site.baseurl}}/partners/trustpilot/) es una plataforma de reseñas en línea que habilita a tus clientes para que compartan sus opiniones y te permite administrar y responder a las reseñas.
