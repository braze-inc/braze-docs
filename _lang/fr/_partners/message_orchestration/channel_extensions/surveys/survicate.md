# Survicate

![Exemple illustrant à quoi pourrait ressembler une enquête HTML intégrée (première question) dans un e-mail Braze.][2]{: style="float:right;max-width:40%;border:0; margin-left:8px;"}

> [Survicate][1] est un logiciel de sondage NPS et d’avis clients qui vous aide à obtenir un flux constant d'informations sur les utilisateurs à toutes les étapes de l'entonnoir. 

L'intégration de Braze et Survicate vous permet d'inclure des liens d'enquête dans vos e-mails ou d'intégrer directement des extraits de code d'enquête afin d'augmenter le taux de réponse. Une fois les enquêtes terminées, retournez sur Survicate pour identifier et analyser les attributs et les réponses des personnes ayant répondu à l'enquête.

## Conditions préalables

| Condition | Description |
| ----------- | ----------- |
| Compte Survicate | Un compte Survicate est nécessaire pour bénéficier de ce partenariat. |

## Cas d'utilisation

Survicate peut couvrir un large éventail de cas d'utilisation pour aider à découvrir des informations exploitables afin de prendre des décisions plus intelligentes et d'offrir une meilleure expérience client :
- Mesurer la satisfaction des clients
- NPS (Net Promoter Score)
- Recueillir les commentaires sur les produits
- Réaliser des études d'utilisateurs ou de marketeurs

## Intégration

Avec Survicate, vous pouvez collecter des retours d'expérience dans Braze en quelques minutes sans qu’aucune compétence technique ne soit requise.

### Étape 1 : Créez votre enquête dans Survicate

Connectez-vous à votre compte Survicate et cliquez sur le bouton **Créer une nouvelle enquête**. Sélectionnez le type d'**enquête par e-mail ou par lien partageable**, et commencez à concevoir votre enquête à partir de zéro ou choisissez parmi plus de 100 modèles d'enquête prêts à l'emploi.

### Étape 2 : Connectez votre enquête à Braze

Une fois que l'enquête est prête, passez à l'onglet **Configurer** dans le composeur d'enquête et sélectionnez **Braze** dans la liste déroulante des outils sous le champ **Identifier les répondants avec.**  Désormais, vous n'aurez plus besoin de demander aux répondants leurs coordonnées.

![][3]{: style="max-width:70%;" }

### Étape 3 : Partagez votre enquête

Accédez à l'onglet **Partage** et choisissez l'option préférée pour le placement d'une enquête. Vous pouvez soit obtenir un lien direct (que vous pourrez ensuite cacher sous un bouton ou un lien hypertexte dans Braze), soit intégrer la première question dans le corps de l'e-mail à l'aide du code HTML.

![][4]{: style="max-width:70%;" }

### Étape 4 : Ajoutez l'enquête à votre campagne d'e-mailing dans Braze

Dans Braze, ajoutez le lien ou le code HTML de votre enquête dans le contenu de votre campagne d'e-mail. Commencez à sonder vos clients et analysez les résultats dans Survicate !
