## Wie Sie diese Kampagne/Canvas erstellen

Wenn Sie diesen Teil des Artikels schreiben, gliedern Sie das, was sie tun sollten, nach Strategien und Schritten. Sie können dies deutlich machen, indem Sie die Schritte wie folgt benennen. Schreiben Sie die Schritte nicht explizit aus (i.e., Klicken Sie auf diesen Button; klicken Sie auf diesen Button). Weisen Sie sie stattdessen an und leiten Sie sie an, indem Sie die Schritte beschreiben, mit denen die Ziele des Anwendungsfalls erreicht werden.

### Schritt 1: Richten Sie Ihr Canvas für passive Nutzer:innen ein (Ratschläge zur Zielgruppe)

Wenn Sie über die Zielgruppe sprechen, müssen Sie sie nicht über die Verwendung von Segmenten oder Filtern belehren. Zeigen Sie ihnen stattdessen, wie sie ihre Zielgruppe auf Nutzer:innen eingrenzen und sicherstellen können, dass sie nicht an andere Personen weitergegeben werden, und warum das wichtig ist. So etwas wie "Sie können Ihre passiven Nutzer:innen auf verschiedene Weise eingrenzen - z.B. indem Sie Nutzer:innen auswählen, die die App seit ein paar Tagen nicht mehr benutzt oder eine E-Mail geöffnet haben, wenn Sie das möchten. Stellen Sie sicher, dass Sie einen Filter anwenden, wenn Sie Personen ausschließen möchten, die schon so lange nicht mehr in der App sind, dass sie die Datenpunkte nicht mehr wert sind."

### Schritt 2: Senden Sie Ihre Nachricht zum richtigen Zeitpunkt

Sprechen Sie darüber, wann Sie Ihre Nachricht senden sollten, wenn das wichtig ist. Vielleicht aber auch nicht! Lassen Sie sie wenigstens wissen, dass es möglich ist, in Ortszeit zu senden. Oder lassen Sie sie wissen, wie Sie diese spezielle Situation per API triggern können, oder sogar, wenn Sie ein Canvas verwenden und es besser ist, den Schritt auf der Grundlage einer Aktion zu triggern (z. B. eine E-Mail zu öffnen und mindestens einen Tag lang nicht zu klicken).

### Schritt 3: Aufbau Ihrer Nachricht

Hier können Sie uns zu den besten Praktiken beraten. Ich empfehle Ihnen, dies zu tun, auch wenn Sie nur einige Dinge auflisten. Ich würde auch einige Bilder einfügen.

### Schritt 4: Alles andere

Wenn Sie einen zusätzlichen Schritt hinzufügen möchten, z.B. über Warteschritte oder etwas Ähnliches in einem Canvas sprechen, würde ich das hier und in den nachfolgenden Schritten fortsetzen. Machen Sie den Artikel aber nicht zu lang - Sie wollen die Zielgruppe ja nicht verlieren.
