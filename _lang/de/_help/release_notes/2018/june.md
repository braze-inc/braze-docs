# Juni 2018

## Passwortwiederverwendung

Administratoren können jetzt die Mindestanzahl neuer, eindeutiger Passwörter festlegen, die andere Braze-Benutzer in ihrem Unternehmen verwenden müssen, bevor sie ein früheres Passwort wieder verwenden können. Dies kann über das Panel [Authentifizierungsregeln][98] ] in den **Unternehmenseinstellungen** auf der Registerkarte **Sicherheitseinstellungen** gesteuert werden.

## Tracking für Canvas Schritte deinstallieren
