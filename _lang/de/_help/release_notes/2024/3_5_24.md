# Veröffentlichung am 5\. März 2024

## Google EU-Zustimmungsrichtlinie für Nutzer

Google aktualisiert seine [EU-Zustimmungsrichtlinie](https://www.google.com/about/company/user-consent-policy/) als Reaktion auf die Änderungen des [Digital Markets Act (DMA](https://ads-developers.googleblog.com/2023/10/updates-to-customer-match-conversion.html)), der ab dem 6\. März 2024 in Kraft tritt. Diese neue Änderung verlangt von den Werbetreibenden, dass sie ihren Endnutzern aus dem EWR und dem Vereinigten Königreich bestimmte Informationen offenlegen und die erforderlichen Einwilligungen von ihnen einholen. Im Rahmen dieser bevorstehenden Änderung können Sie [beide Einwilligungssignale in Braze als benutzerdefinierte Attribute erfassen]({{site.baseurl}}/partners/canvas_steps/google_audience_sync/#collecting-consent-for-eea-and-uk-end-users). Braze synchronisiert die Daten aus diesen benutzerdefinierten Attributen mit den entsprechenden Einwilligungsfeldern in Google.

## Flexibilität der Daten

### Duplizierte Benutzer zusammenführen

{% multi_lang_include release_type.md release="Früher Zugang" %}

Im Braze Dashboard können Sie jetzt [nach doppelten Benutzern suchen und diese zusammenführen]({{site.baseurl}}/user_guide/engagement_tools/segments/user_profiles/duplicate_users), um die Effektivität Ihrer Kampagnen und Canvases zu maximieren. Sie können Benutzerprofile einzeln zusammenführen oder eine Massenzusammenführung durchführen, bei der alle Profile mit übereinstimmenden Identifikatoren in das zuletzt aktualisierte Benutzerprofil zusammengeführt werden.

### Suche nach archivierten Inhalten

Im Braze-Dashboard können Sie jetzt [archivierte Inhalte in Ihre Suchergebnisse]({{site.baseurl}}/user_guide/administrative/access_braze/global_search/#filter-for-archived-content) einbeziehen, indem Sie **Archivierte Inhalte anzeigen** wählen.

### Unterstützung der Nachrichtenarchivierung für AWS S3 und Google Cloud Storage

Sie können die [Nachrichtenarchivierung]({{site.baseurl}}/user_guide/data_and_analytics/export_braze_data/message_archiving/) verwenden, um eine Kopie Ihrer an Benutzer gesendeten Nachrichten zu Archivierungs- oder Compliance-Zwecken in Ihrem AWS S3-Bucket, Azure Blob Storage-Container oder Google Cloud Storage-Bucket zu speichern.

### SQL-Tabellenreferenz

Besuchen Sie die [SQL-Tabellenreferenz]({{site.baseurl}}/user_guide/engagement_tools/segments/sql_segments/sql_segments_tables/), um die Tabellen und Spalten zu sehen, die im Query Builder oder beim Erstellen von SQL-Segmenterweiterungen abgefragt werden können.

## Kreativität entfesseln

### Tonsteuerung für KI-Texte

Sie können jetzt einen [Nachrichtenton]({{site.baseurl}}/user_guide/brazeai/generative_ai/ai_copywriting/#steps) wählen, um den Stil der mit dem KI-Texterstellungsassistenten erstellten Texte zu bestimmen.

## Robuste Kanäle

### Kartenerstellung

Sie können festlegen, wann Braze die Eignung der Zielgruppe und die Personalisierung für neue Content Card-Kampagnen und Canvas-Schritte bewertet, indem Sie angeben, wann die Karte [erstellt]({{site.baseurl}}/user_guide/message_building_by_channel/content_cards/create/card_creation/) wird. 

### Vorschau der Benutzerpfade

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Erleben Sie die Canvas-Reise, die Sie für Ihre Benutzer erstellt haben, einschließlich einer Vorschau des Timings und der Nachrichten, die sie erhalten werden. Diese [Testläufe]({{site.baseurl}}/preview_user_paths/) dienen der Qualitätssicherung, um sicherzustellen, dass Ihre Nachrichten an die richtige Zielgruppe gesendet werden, und zwar noch vor dem Versand der Canvas.

### Schnelle Push-Kampagnen

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Wenn Sie eine Push-Kampagne in Braze erstellen, können Sie mehrere Plattformen und Geräte auswählen, um eine Nachricht für alle Plattformen in einem einzigen Bearbeitungsvorgang zu erstellen, der [Quick Push]({{site.baseurl}}/user_guide/message_building_by_channel/push/creating_a_push_message/quick_push/) genannt wird. Diese Funktion ist nur für Kampagnen verfügbar.

### Angepasster List-Unsubscribe-Header

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Wenn Sie eine [benutzerdefinierte Kopfzeile für die Abmeldeliste]({{site.baseurl}}/user_guide/administrative/app_settings/email_settings/#custom-list-unsubscribe-header) in Ihre E-Mail-Nachrichten einfügen, können sich Ihre Empfänger abmelden. Auf diese Weise können Sie Ihren eigenen konfigurierten Endpunkt für die Abmeldung mit einem Klick und ein optionales "mailto:" hinzufügen. Braze benötigt eine Eingabe für die URL, um einen benutzerdefinierten List-Unsubscribe-Header zu unterstützen, da die HTTP-Abmeldung mit einem Klick eine Anforderung von Yahoo und Gmail für Massenversender ist.

### Mehrere Seiten für In-App-Nachrichten

{% multi_lang_include release_type.md release="Früher Zugang" %}

[Durch das Hinzufügen von Seiten zu Ihrer In-App-Nachricht]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/#multi-page) können Sie die Benutzer durch einen sequentiellen Ablauf führen, z. B. einen Onboarding-Flow oder eine Willkommensreise. Sie können Seiten im Bereich **Seiten** der Registerkarte **Erstellen** verwalten.

### Pfade für einen Experimentierpfad randomisieren

Um [die Pfadzuweisung]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/experiment_step) für einen Experimentierpfad-Schritt immer [zu randomisieren]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/experiment_step), wählen Sie **Randomisierte Pfade in Experimentierpfade** in dem Schritt. Diese Option ist nicht verfügbar, wenn Sie einen der beiden Pfade "Gewinnen" oder "Personalisierte Pfade" verwenden.

### Formular zur E-Mail-Erfassung

Mit [E-Mail-Capture-Nachrichten]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/traditional/customize/email_capture_form/) können Sie die Benutzer Ihrer Website ganz einfach auffordern, ihre E-Mail-Adresse zu übermitteln, die dann in ihrem Benutzerprofil für alle Ihre Nachrichtenkampagnen zur Verfügung steht.
