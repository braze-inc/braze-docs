# Veröffentlichung am 2\. April 2024

## WhatsApp

### Mehrere Geschäftskonten

Jetzt können Sie zu jedem Arbeitsbereich mehrere WhatsApp Business-Konten und Abonnementgruppen hinzufügen. Eine vollständige Anleitung finden Sie unter [Mehrere WhatsApp Business-Konten und Telefonnummern]({{site.baseurl}}/user_guide/message_building_by_channel/whatsapp/overview/multiple_subscription_groups/).

### Preise lesen

WhatsApp testet neue Ansätze, beginnend mit Verbrauchern in Indien, um wertvollere Erfahrungen zu schaffen und die Einbindung in Marketingkonversationen von Unternehmen zu maximieren. Dazu kann es gehören, die Anzahl der Marketinggespräche, die eine Person in einem bestimmten Zeitraum von einem Unternehmen erhält, zu begrenzen und mit einer kleinen Anzahl von Gesprächen zu beginnen, die weniger wahrscheinlich gelesen werden. Weitere Informationen finden Sie unter [Meta-Ressourcen]({{site.baseurl}}/user_guide/message_building_by_channel/whatsapp/meta_resources/).

## Flexibilität der Daten

### Amazon S3 Buckets mit Braze synchronisieren

{% multi_lang_include release_type.md release="Früher Zugang" %}

Sie können jetzt Cloud Data Ingestion für S3 verwenden, um einen oder mehrere S3-Buckets in Ihrem AWS-Konto direkt mit Braze zu integrieren. Wenn neue Dateien in S3 veröffentlicht werden, wird eine Nachricht an SQS gesendet, und Braze Cloud Data Ingestion nimmt diese neuen Dateien auf. Weitere Informationen finden Sie unter [Integration von Dateispeichern]({{site.baseurl}}/user_guide/data_and_analytics/cloud_ingestion/file_integrations/).

### Shopify OAuth

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Shopify ist ein weltweit führendes Handelsunternehmen, das zuverlässige Tools für die Gründung, das Wachstum, die Vermarktung und die Verwaltung von Einzelhandelsgeschäften jeder Größe anbietet. Wenn Sie nun Shopify für Braze einrichten, können Sie [OAuth für Ihren Arbeitsbereich aktivieren]({{site.baseurl}}/partners/message_orchestration/channel_extensions/ecommerce/shopify/setting_up_shopify/).

### Verwenden Sie Expo für iOS-Push-Benachrichtigungen

Wir [haben eine Anleitung hinzugefügt]({{site.baseurl}}/developer_guide/platform_integration_guides/react_native/push_notifications/?tab=expo), wie Sie umfangreiche Push-Benachrichtigungen und Push Stories in Ihre iOS-App integrieren können, indem Sie Expo mit React Native verwenden.

### Fernstart iOS Live-Aktivitäten

Jetzt können Sie Ihre Live-Aktivitäten auf iOS über den [Endpunkt`/messages/live_activity/start` ]({{site.baseurl}}/api/endpoints/messaging/live_activity/start/) aus der Ferne starten. Eine vollständige Übersicht finden Sie unter [Live-Aktivitäten: Starten Sie eine Aktivität]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/live_activities/live_activities/#step-2-start-the-activity).

## Neue Lötpartnerschaften

### Volkszählung - Cohort Import

Sie können jetzt [Kohortenbenutzer aus Braze in Census importieren]({{site.baseurl}}/partners/data_and_infrastructure_agility/cohort_import/census/), eine Datenaktivierungsplattform, die Cloud Data Warehouses wie Snowflake und BigQuery mit Braze verbindet. Ihre Marketingteams können die Leistungsfähigkeit ihrer First-Party-Daten nutzen, um dynamische Zielgruppensegmente zu erstellen, Kundenattribute zur Personalisierung von Kampagnen zu synchronisieren und alle Daten in Braze auf dem neuesten Stand zu halten.

## SDK Aktualisierungen

Die folgenden SDK-Updates wurden veröffentlicht. Die wichtigsten Updates sind unten aufgeführt. Alle anderen Updates finden Sie in den entsprechenden SDK Changelogs.

- [React Native 9.1.0](https://github.com/braze-inc/braze-react-native-sdk/blob/master/CHANGELOG.md)
  - Die Mindestversion von React Native wurde auf 0.71.0 aktualisiert.
  - Die minimale iOS-Version wurde auf 12.0 aktualisiert.
  - Die iOS Bindungen wurden aktualisiert, um Braze Swift SDK 8.1.0 zu verwenden.
  - Die Android-Bindungen wurden aktualisiert, um Braze Android SDK 30.1.1 zu verwenden.
