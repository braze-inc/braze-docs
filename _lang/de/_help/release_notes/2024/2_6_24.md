# Veröffentlichung am 6\. Februar 2024

## Manifest zum Datenschutz löten

Braze hat unser eigenes Datenschutzmanifest veröffentlicht, zusammen mit neuen flexiblen APIs, die die deklarierten Tracking-Daten automatisch an spezielle `-tracking` Endpunkte umleiten. Weitere Informationen finden Sie im [Datenschutzmanifest von Braze]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/privacy_manifest).

## Google EU-Zustimmungsrichtlinie für Nutzer

Google aktualisiert seine [EU-Zustimmungsrichtlinie](https://www.google.com/about/company/user-consent-policy/) als Reaktion auf die Änderungen des [Digital Markets Act (DMA](https://ads-developers.googleblog.com/2023/10/updates-to-customer-match-conversion.html)), der am 6\. März 2024 in Kraft tritt. Diese neue Änderung verlangt von den Werbetreibenden, dass sie ihren Endnutzern aus dem EWR und dem Vereinigten Königreich bestimmte Informationen offenlegen und die erforderlichen Einwilligungen von ihnen einholen. Im Rahmen dieser bevorstehenden Änderung können Sie [beide Einwilligungssignale in Braze als benutzerdefinierte Attribute erfassen]({{site.baseurl}}/partners/canvas_steps/google_audience_sync/#collecting-consent-for-eea-and-uk-end-users). Braze synchronisiert die Daten aus diesen benutzerdefinierten Attributen mit den entsprechenden Einwilligungsfeldern in Google.

## Robuste Kanäle

### Benutzerdefinierte Codeblöcke

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Mit [benutzerdefinierten Codeblöcken]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/editor_blocks/#custom-code) können Sie HTML, CSS und JavaScript für eine In-App-Nachricht hinzufügen, bearbeiten oder löschen.

### Verringern Sie die Größe der Nutzlast von Push-Benachrichtigungen

Der neue Hilfe-Artikel [Größe der Nutzlast von Benachrichtigungen]({{site.baseurl}}/help/help_articles/push/reducing_payload_size#reducing-push-notification-payload-size) enthält einige Tipps, wie Sie die Größe der Nutzlast Ihrer Push-Benachrichtigungen reduzieren können, wenn Sie eine Kampagne oder einen Canvas-Schritt aufgrund von Größenbeschränkungen für Push-Nutzlasten nicht starten können.

### BCC-Adressen zu Ihrer Kampagne oder Ihrem Canvas hinzufügen

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Sie können eine [BCC-Adresse]({{site.baseurl}}/user_guide/administrative/app_settings/email_settings/?tab=bcc%20address#outbound-email-settings) an eine E-Mail-Nachricht anhängen. Dadurch wird eine identische Kopie der Nachricht, die Ihr Benutzer erhält, an Ihren BCC-Posteingang gesendet. Auf diese Weise können Sie Kopien der Nachrichten aufbewahren, die Sie an Ihre Benutzer geschickt haben, um Compliance-Anforderungen zu erfüllen oder den Kundensupport zu unterstützen.

### Links zum Abbestellen von E-Mails mit einem Klick

Wenn Sie eine [Kopfzeile für die Abmeldeliste]({{site.baseurl}}/user_guide/administrative/app_settings/email_settings/#list-unsubscribe-header) verwenden, können sich Ihre Empfänger mit einem Klick von Marketing-E-Mails abmelden, indem Sie eine Schaltfläche für **die Abmeldung** innerhalb der Mailbox-Benutzeroberfläche und nicht im Nachrichtentext anzeigen.

## Neue Lötpartnerschaften

### Criteo - Canvas Publikumssynchronisation

Mit der [Braze Audience Sync to Criteo]({{site.baseurl}}/partners/canvas_steps/criteo_audience_sync/) können Marken Benutzerdaten aus ihrer eigenen Braze-Integration zu Criteo-Kundenlisten hinzufügen, um Werbung auf der Grundlage von Verhaltensauslösern, Segmentierung und mehr zu liefern. Alle Kriterien, die Sie normalerweise verwenden, um eine Nachricht (Push, E-Mail, SMS, Webhook usw.) in einem Braze Canvas auf der Grundlage Ihrer Benutzerdaten auszulösen, können jetzt verwendet werden, um eine Anzeige für diesen Benutzer in Ihren Criteo-Kundenlisten auszulösen.

### Movable Ink - Dynamischer Inhalt

Die [Movable Ink]({{site.baseurl}}/partners/message_personalization/dynamic_content/movable_ink#movable-ink) Customer Data API-Integration ermöglicht es Marketingfachleuten, in Braze gespeicherte Kundenereignisdaten zu aktivieren, um personalisierte Inhalte in Movable Ink zu generieren.

### Scuba Analytics - Analytik

[Scuba Analytics]({{site.baseurl}}/partners/data_and_infrastructure_agility/analytics/scuba#scuba-analytics) ist eine umfassende, auf maschinellem Lernen basierende Plattform für die Zusammenarbeit mit Daten, die für schnelle Zeitreihendaten entwickelt wurde. Mit Scuba können Sie selektiv Benutzer (auch Akteure genannt) exportieren und in Ihre Braze-Plattform laden. In Scuba werden benutzerdefinierte Akteurseigenschaften verwendet, um Verhaltenstrends zu analysieren, Ihre Daten über verschiedene Plattformen hinweg zu aktivieren und mithilfe von maschinellem Lernen Vorhersagemodelle zu erstellen.
