# April 5, 2022 Veröffentlichung

## Nach Kampagne oder Canvas-Attribution filtern
Sie können jetzt nach Nutzern filtern, die auf eine bestimmte SMS-Kampagne oder einen Canvas-Schritt, eine Schlüsselwortkategorie oder einen Tag geantwortet haben. Weitere Informationen finden Sie unter [SMS-Retargeting][8].

## Dashboard zur E-Mail-Leistung
Mit dem [E-Mail-Performance-Dashboard][1] können Sie die aggregierten Performance-Metriken für Ihren gesamten E-Mail-Kanal sowohl für Kampagnen als auch für Canvases für den von Ihnen ausgewählten Datumsbereich anzeigen.

Weitere Informationen über die in Braze verfügbaren Analyse-Dashboards finden Sie im Abschnitt [Ihre Analyse-Dashboards][2].

## Globale Stil-Einstellungen

Einführung [globaler Stil-Einstellungen][3] für den Drag & Drop-Editor! Jetzt können Sie das Aussehen Ihrer E-Mail-Kampagnen und Canvases ganz einfach personalisieren, indem Sie ein Standardthema hinzufügen, die grundlegende Textgestaltung festlegen und vieles mehr.

## Webhaken von Braze zu Braze
Mit einem Braze to Braze Webhook können Sie Webhooks verwenden, um mit der Braze REST API zu kommunizieren und alles zu tun, was unsere API Ihnen erlaubt. Im Wesentlichen handelt es sich um einen Webhook, der von Braze zu Braze kommuniziert. Weitere Informationen finden Sie in unserem Artikel [Braze to Braze Webhooks][4].

## Abwertungen

### Windows SDK
Ab dem 24\. März 2022 ist das [Braze Windows SDK][6] veraltet und es können keine neuen Windows-Apps mehr im Braze Dashboard erstellt werden. 

### Baidu Push-Integration
Ab dem 24\. März 2022 ist die [Braze Baidu Push-Integration][7] veraltet und es können keine neuen Baidu-Apps mehr im Braze Dashboard erstellt werden. 

## Neue Lötpartnerschaften

### Tealium für Currents

Die Integration von Braze und [Tealium][5] ermöglicht Ihnen die nahtlose Steuerung des Informationsflusses zwischen den beiden Systemen. Mit Currents können Sie jetzt auch Daten mit Tealium verbinden, um sie über den gesamten Wachstumsbereich hinweg nutzbar zu machen.
