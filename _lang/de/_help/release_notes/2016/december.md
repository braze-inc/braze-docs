# Dezember 2016

## E-Mail-Kopfzeilen

E-Mail-Kopfzeilen fügen die explizite Einstellung von E-Mail-Kopfzeilen über den Sendeendpunkt hinzu. Weitere Informationen über das Versenden von E-Mails mit Braze finden Sie unter [E-Mail-Best Practices][14].

## Canvas Aktualisierungen

Mit diesem Update können Sie einen Schritt mit mehreren Schritten verbinden oder mehrere Schritte zu einem einzigen, zusammengefassten Schritt verbinden und Verbindungspunkte zwischen ihnen zeichnen. So können Benutzer, die einen Kauf getätigt haben, einem anderen Ablauf folgen als Benutzer, die dies nicht getan haben. Weitere Informationen zu Canvas finden Sie unter [Erstellen einer Verzweigung][15].
