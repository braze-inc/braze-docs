# Januar 2017

## Neue Konvertierungsereignisse für E-Mail-Nachrichten

Sie können jetzt ein Konvertierungsereignis für "geöffnete E-Mail aus der Kampagne" oder "angeklickte E-Mail aus der Kampagne" angeben. Erfahren Sie mehr über die [Zustellungsarten von Kampagnen][16].

## CSV-Export mit Kontrollgruppe
Wenn Sie eine CSV-Datei mit Benutzerkampagnen exportieren, können Sie jetzt Kontrollgruppen in die Daten aufnehmen. Erfahren Sie mehr über den [Export von Dashboard-Daten][17].

## Willkommens- und Double-Opt-In-E-Mails
Sie können jetzt einen neu erstellten aktionsbasierten Auslöser "E-Mail-Adresse hinzugefügt" verwenden, mit dem Sie ganz einfach Double-Opt-in-E-Mails oder Willkommens-E-Mails senden können, wenn ein Benutzer zum ersten Mal seine E-Mail-Adresse hinzufügt. Erfahren Sie mehr über [bewährte E-Mail-Verfahren][18].

## Liste-abbestellen E-Mail-Kopfzeile
Sie können jetzt List-Unsubscribe-Kopfzeilen konfigurieren, die einen Abonnementstatus von "abonnierte/eingeloggte Benutzer" oder "nur eingeloggte Benutzer" haben. Dadurch wird in Google Mail neben dem Namen des Absenders ein Link zum Abbestellen eingefügt, was die Zustellbarkeit verbessern kann.

## Aktualisierungen der Benutzerfreundlichkeit
Sie können jetzt auf eine Schaltfläche klicken, um die Canvas-Kopfzeile für kleinere Bildschirme zu minimieren, und den Mauszeiger über die vollständigen Namen Ihrer Canvases und Kampagnen in Ihrem Benutzerprofil in Braze bewegen.
