# Juni 2017

## Update auf externe API

Wir haben die externe API aktualisiert, um das Feld collapse_id für Apple Push zu unterstützen, so dass Sie Ihren Kunden präzise Nachrichten zukommen lassen können. Erfahren Sie mehr über die [Braze Messaging API][49].

## Update für Web-Push-Benachrichtigungen

Wir haben die Vorschau für Web-Push-Benachrichtigungen aktualisiert, einschließlich eines Dropdowns und einer korrekten Darstellung der Bildhöhe, so dass Sie Web-Push-Benachrichtigungen leichter testen und versenden können. Erfahren Sie mehr über [Web Push][47].

## Verbesserte Sicherheitseinstellungen

Wir haben unseren Kunden die Möglichkeit gegeben, Regeln für die Passwortkomplexität, das Dashboard-IP-Whitelisting und die Zwei-Faktor-Authentifizierung festzulegen, um Ihnen mehr Kontrolle zu geben. Erfahren Sie mehr über [die Sicherheitseinstellungen][46].

## Gleichzeitige CSV-Importe

Sie können jetzt mehrere CSV-Dateien hochladen und gleichzeitig verarbeiten, so dass Sie schneller arbeiten können. Erfahren Sie mehr über [Benutzerimporte][50].

## Attribute der Leinwandvorlage

Wir haben die Möglichkeit hinzugefügt, [Schablonenattribute][45] in Nachrichten einzubinden, die aus Canvas-Schritten gesendet werden, damit Sie effizienter arbeiten können.

## Aktualisierungen der Zielführung

Sie können jetzt Stunden als Zeiteinheit auswählen, wenn Sie nach Datum und Uhrzeit filtern, so dass Sie Ihre Nachrichten noch präziser ausrichten können. Erfahren Sie mehr über [Filter][44].
