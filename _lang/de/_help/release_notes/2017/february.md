# Februar 2017

## Canvas-Schritte nach dem Start hinzufügen

Sie können jetzt [Canvas-Schritte][23] zu bestehenden Schritten [hinzufügen][23], auch nachdem der Canvas gestartet wurde.

## Bulk CSV für Engagementberichte

Sie können jetzt mit [Engagement Reports][22] Kampagnendaten im CSV-Format in großen Mengen exportieren. Sie können Canvas-Daten auch über Engagement Reports exportieren.

## Filtern nach der zuletzt empfangenen In-App-Nachricht

Für eine persönlichere Ausrichtung der Nutzer können Sie jetzt "Zuletzt empfangene In-App-Nachricht" als zeitbasierten Filter verwenden. Wenn Sie mehr über das Benutzer-Targeting mit Braze erfahren möchten, besuchen Sie [Segmentierung][21].

## Ereignis-Benutzerprotokoll-Upgrade

Mit unseren aktualisierten [Ereignisbenutzerprotokollen][20] können Sie jetzt noch einfacher sehen, was in Ihrer SDK-Integration passiert.

## Vorlagen für News Feeds

Es gibt jetzt zwei News Feed Kartenfelder,{% raw %} {{card.${name}}} {% endraw %} und {% raw %}  {{card.${api_id}}}{% endraw %}. Diese Karten können zu Liquid Templating hinzugefügt werden. Erfahren Sie mehr über [personalisierte Nachrichten und Vorlagen][19].
