# März 2017

## Neuer Retargeting-Filter

Wir haben einen neuen Filter "Von Leinwand konvertiert" hinzugefügt, um nahtlose Retargeting-Optionen zu bieten. Erfahren Sie mehr über die Verwendung von [Retargeting-Filtern][27].

## Webhook-Vorlagen

Sie können jetzt Webhook-Nachrichten als Vorlagen zur einfachen Wiederverwendung speichern oder mit einer vorkonfigurierten Vorlage für Plattformen wie SMS mit Twilio oder Chatbots wie Facebook Messenger beginnen. Erfahren Sie mehr über die [Erstellung eines Webhooks][26].

## Unterstützung mehrerer E-Mails für Engagement-Berichte

[Engagement-Berichte][25] können jetzt an mehrere E-Mail-Adressen gesendet werden.

## Geben Sie die Zeit für die aktionsbasierte Lieferung an

Sie haben jetzt die Möglichkeit, bei der Verwendung von Auslösern, die auf einer verzögerten Aktion basieren, zu einem bestimmten Zeitpunkt nach einer Verzögerung zu senden. Diese Funktion ist für Canvas und Kampagnen verfügbar. Erfahren Sie mehr über den Versand von [aktionsbasierten Versandkampagnen][24].
