# April 4, 2023 Veröffentlichung

## Dokumentation Brotkrümel 
Vielleicht ist Ihnen aufgefallen, dass die Braze Docs-Website jetzt oben auf jedem Artikel Brotkrümel hat, die Ihnen zeigen, wo Sie sich auf der Website befinden. Diese sind nur eine weitere Option, die Ihnen die Navigation erleichtert!

## Kataloge im Browser erstellen
Sie können Kataloge verwenden, um in Ihren Braze-Kampagnen über Liquid auf Nicht-Benutzerdaten zu verweisen. Mit Braze können Sie jetzt einen Katalog direkt in Ihrem Browser erstellen, anstatt eine CSV-Datei zu importieren. Weitere Informationen finden Sie unter [Erstellen eines Katalogs]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog) für mehr.

## Benutzerdefiniertes SQL im Query Builder
Mit dem Query Builder können Sie Berichte mit Braze-Daten in Snowflake erstellen. Jetzt können Sie [benutzerdefiniertes SQL verwenden]({{site.baseurl}}/user_guide/data_and_analytics/query_builder/custom_sql), um neue Erkenntnisse zu gewinnen.

{% alert important %}
Der SQL-Editor befindet sich im Early Access. Wenn Sie daran interessiert sind, am Early Access teilzunehmen, wenden Sie sich bitte an Ihren Customer Success Manager.
{% endalert %}

## Merkmal Flagge FAQ

## Nachricht Extras Flüssigkeitstag für Currents
Mit dem [`message_extras` Liquid-Tag]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/liquid/advanced_filters/message_extras) können Sie Ihre Sendeereignisse mit dynamischen Daten aus Connected Content, benutzerdefinierten Attributen (z. B. Sprache, Land) und Canvas-Eigenschaften versehen. Dieses Liquid-Tag fügt Schlüssel-Wert-Paare an das entsprechende Sendeereignis in Currents an.

{% alert important %}
Dieses Liquid-Tag ist derzeit in der Beta-Phase für E-Mail-, SMS- und Push-Send-Ereignisse. Wenden Sie sich an Ihren Braze Customer Success Manager, wenn Sie an einer Teilnahme an der Beta interessiert sind.
{% endalert %}

## Shopify Historisches Backfill
Mit [Shopify Historical Backfill](https://www.braze.com/docs/partners/message_orchestration/channel_extensions/ecommerce/shopify/shopify_backfill/) kann Braze alle Kunden, Bestellungen und Kaufereignisse der letzten 90 Tage vor Ihrer Shopify-Integrationsverbindung importieren.

## WhatsApp
WhatsApp ist eine beliebte Peer-to-Peer-Nachrichtenplattform, die weltweit genutzt wird und konversationsbasierte Nachrichten für Unternehmen anbietet. Der [WhatsApp-Messaging-Kanal]({{site.baseurl}}/user_guide/message_building_by_channel/whatsapp) bietet eine direkte Möglichkeit, Nutzer auf der WhatsApp-Plattform durch Kampagnen, Opt-Ins und Opt-Outs, Schnellantworten und vieles mehr zu erreichen.

### WhatsApp API-Objekt
Als Teil der WhatsApp-Unterstützung von Braze ermöglicht Ihnen das Objekt `whats_app`, WhatsApp-Nachrichten über unsere [Messaging-Endpunkte]({{site.baseurl}}/api/endpoints/messaging) zu ändern oder zu erstellen. Die vollständige Spezifikation finden Sie in der [Objektdokumentation`whats_app` ]({{site.baseurl}}/api/objects_filters/messaging/whats_app_object).
