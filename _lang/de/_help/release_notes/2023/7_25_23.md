# Veröffentlichung am 25\. Juli 2023

## Canvas-Zulassung 
Die neue Einstellung für den Canvas-Genehmigungs-Workflow fügt einen Überprüfungsprozess hinzu, bevor ein neuer Canvas gestartet wird. Beachten Sie, dass diese Funktion standardmäßig ausgeschaltet ist, so dass Sie die Kontrolle über ihre Implementierung haben. Weitere Details zur Aktivierung dieses Workflows finden Sie unter [Canvas Genehmigungen und Berechtigungen]({{site.baseurl}}/user_guide/engagement_tools/canvas/managing_canvases/canvas_approval/).
 
## Funktionskennzeichen in Canvas
Mit Feature-Flags können Sie experimentieren und Ihre Hypothesen über neue Funktionen bestätigen, indem Sie sie für verschiedene Gruppen Ihrer Nutzer aus- und einschalten. Die neue Feature-Flag-Komponente in Canvas ermöglicht es Ihnen, Ihr Publikum in einem Canvas zu segmentieren, je nachdem, ob ein Feature-Flag für einen Benutzer aktiviert oder deaktiviert ist. Darüber hinaus können Sie mit den Experimentierpfaden diese Konversionen optimieren, indem Sie verschiedene Botschaften oder Pfade gegeneinander testen und ermitteln, welche am effektivsten ist. In der [Übersicht der Feature-Flags]({{site.baseurl}}/developer_guide/platform_wide/feature_flags/about/) finden Sie Informationen über Feature-Flags im Allgemeinen, oder Sie erfahren mehr über die [Verwendung von Feature-Flags in Canvas]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/feature_flags/).

## Artikel Segmente verwalten
Der neue Artikel [Segmente verwalten]({{site.baseurl}}/user_guide/engagement_tools/segments/managing_segments/) behandelt die Aktionen, mit denen Sie Ihre Segmente konfigurieren können, z. B. das Filtern einer Liste von Segmenten, das Erstellen von Segmenten und das Bearbeiten von Segmenten.
 
## Zeilenkomponente Inhaltsblock
Sie können jetzt eine Zeilenkomponente als Inhaltsblock speichern, der dann in allen E-Mail-Kampagnen und E-Mail-Nachrichten in einem Canvas verwendet werden kann. Weitere Informationen zum Ziehen und Ablegen von Inhaltsblöcken finden Sie unter [Inhaltsblöcke]({{site.baseurl}}/user_guide/message_building_by_channel/email/drag_and_drop/dnd_content_blocks/).

## <sup>BrazeAITM</sup>
<sup>BrazeAITM</sup> bietet eine Sammlung von leicht zugänglichen, benutzerfreundlichen Tools, die die Einstiegshürden für Kreativität, Personalisierung und Optimierung Ihrer Engagement-Strategie senken. Erfahren Sie mehr über die Funktionen und Möglichkeiten von <sup>BrazeAITM</sup> in unserer [Dokumentation]({{site.baseurl}}/user_guide/sage_ai).

## Aktualisieren eines Benutzerprofils nach Rufnummer
Mit dem [`/users/track`]({{site.baseurl}}/api/endpoints/user_data/post_user_track) Endpunkt können Sie ein Benutzerprofil anhand seiner Telefonnummer aktualisieren. 

{% alert important %}
Diese Funktion befindet sich derzeit im Early Access. Wenden Sie sich an Ihren Braze-Kundenbetreuer, wenn Sie sich für die Teilnahme am Early Access interessieren.
{% endalert %}

## Whitespace nach E-Mail-Vorspann 
Das neue Kontrollkästchen **Whitespace nach Preheader hinzufügen** blendet den Text oder HTML des E-Mail-Textes im E-Mail-Preheader aus. Erfahren Sie mehr über das Hinzufügen von E-Mail-Kopfzeilen unter [Erstellen einer E-Mail]({{site.baseurl}}/user_guide/message_building_by_channel/email/html_editor/creating_an_email_campaign/#step-3-compose-your-email).
