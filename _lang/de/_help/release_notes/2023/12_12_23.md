# Veröffentlichung am 12\. Dezember 2023

## Aktualisierungen der Android-Push-Integration

Am 20\. Juni 2023 hat Google seine Cloud Messaging API zum Senden von Push-Benachrichtigungen an Android-Apps abgeschafft. Die [Standard-Android-Push-Integration](https://www.braze.com/docs/developer_guide/platform_integration_guides/android/push_notifications/android/integration/standard_integration/) zeigt Ihnen jetzt, wie Sie stattdessen die Firebase Cloud Messaging API von Google einrichten können.

Weitere Informationen über die Google Cloud Messaging API-Abschreibung finden Sie in den [Firebase FAQ](https://firebase.google.com/support/faq#fcm-23-deprecation).

## Robuste Kanäle

### WhatsApp Antwortnachrichten

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Wenn Sie [eine WhatsApp-Nachricht]({{site.baseurl}}/whatsapp_response_messaging/) in einer Kampagne oder einem Canvas [erstellen]({{site.baseurl}}/whatsapp_response_messaging/), können Sie Antwortnachrichten erstellen, um innerhalb eines Zeitfensters von 24 Stunden auf die WhatsApp-Nachrichten der Benutzer zu antworten. Antwortnachrichten können besonders hilfreich bei Canvases sein, die Interaktionen zwischen Ihrer Marke und ihren Nutzern fördern, wie z. B. Opt-in-Kampagnen.

### Begrenzung der WhatsApp-Frequenz

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Sie können jetzt [Regeln für die Frequenzbegrenzung]({{site.baseurl}}/user_guide/engagement_tools/campaigns/building_campaigns/rate-limiting/#frequency-capping) für WhatsApp einrichten. Die Frequenzbegrenzung wird auf der Ebene der Kampagne oder der Canvas-Komponente angewendet und kann für jeden Arbeitsbereich unter **Einstellungen** > **Frequenzbegrenzungsregeln** eingerichtet werden.  

## Flexibilität der Daten

### Dashboard für Umsätze

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Mit dem [Conversions-Dashboard]({{site.baseurl}}/user_guide/data_and_analytics/analytics/conversions_dashboard/) können Sie die Conversions über Kampagnen, Canvases und Kanäle hinweg analysieren und dabei verschiedene Attributionsmethoden verwenden. Wenn Sie Ihre Conversions messen, können Sie den Zeitrahmen, das Conversion-Ereignis und das Conversion-Fenster festlegen.

### E-Mail-Insights-Berichte

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Es gibt eine neue Registerkarte, Email Insights, die sich im [Dashboard Email Performance]({{site.baseurl}}/email_engagement_dashboard/) befindet und zwei neue Berichte enthält:

- **Engagement von Briefkastenanbietern:** Zeigt die Anzahl der Klicks und Öffnungen nach Postfachanbieter an. Sie können einen Mailbox-Anbieter auswählen und nach bestimmten Empfangsdomänen aufschlüsseln.
- **Engagement nach Wochentag:** Zeigt an, wann die Nutzer ihre E-Mails nutzen.

### Aktualisierung der Zeitreihengrafik der Abonnementgruppe

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

Das **Zeitseriendiagramm für Abonnementgruppen**, das auf der Seite **Abonnementgruppen** erscheint, zeigt jetzt die Anzahl der Abonnements pro Benutzer und nicht mehr pro E-Mail oder Telefonnummer. Dies stimmt besser mit der Art und Weise überein, wie Braze die Statistiken in anderen Bereichen des Dashboards berechnet.

## KI & ML Automatisierung

### Empfehlungen für AI-Artikel

{% multi_lang_include release_type.md release="Allgemeine Verfügbarkeit" %}

[KI-Artikel-Empfehlungen]({{site.baseurl}}/ai_item_recommendations) ist eine auf Deep Learning basierende Produktempfehlungsmaschine, die das kollektive Kaufverhalten der Nutzer nutzt, um Artikel zu empfehlen. Sie können KI-Artikel-Empfehlungen verwenden, um die beliebtesten Produkte zu berechnen oder personalisierte KI-Empfehlungen für einen bestimmten Katalog zu erstellen. Nachdem Sie Ihre Empfehlung erstellt haben, können Sie die Personalisierung nutzen, um diese Produkte in Ihre Nachrichten einzufügen.

## Neue Lötpartnerschaften

### Facebook Lead Ads über Zapier - Leads erfassen

Mit der [Facebook Lead Ads-Integration über Zapier]({{site.baseurl}}/partners/data_and_infrastructure_agility/leads_capture/facebook_via_zapier/) können Sie Ihre Leads aus Facebook in Braze importieren und ein benutzerdefiniertes Ereignis verfolgen, wenn Leads erfasst werden.

### SmarterSends - Nachrichtenvorlagen

Die Partnerschaft zwischen Braze und [SmarterSends]({{site.baseurl}}/partners/message_orchestration/channel_extensions/email_templates/smartersends/) ermöglicht es Ihnen, die Leistungsfähigkeit von Braze mit den hyperlokalisierten Inhalten Ihrer verteilten Nutzer zu kombinieren, um Ihre Marketingkampagnen zu optimieren.

### Recurly - Zahlungen

Die Integration von [Recurly]({{site.baseurl}}/partners/data_and_infrastructure_agility/payments/recurly/) und Braze vereinfacht den Austausch von Abonnementdaten mit Braze und ermöglicht eine gezielte Kommunikation mit Kunden.

## SDK Aktualisierungen

Die folgenden SDK-Updates wurden veröffentlicht. Die wichtigsten Updates sind unten aufgeführt. Alle anderen Updates finden Sie in den entsprechenden SDK Changelogs.

- [Flutter SDK 8.0.0-8.1.0](https://pub.dev/packages/braze_plugin/changelog)
  - Aktualisiert die native Android-Bridge von Braze Android SDK 27.0.1 auf 29.0.1.
  - Aktualisiert die native iOS-Bridge von Braze Swift SDK 6.6.1 auf 7.2.0.
  - Ändert das Verhalten für Feature Flags Methoden.
    - `BrazePlugin.getFeatureFlagByID(String id)` gibt nun `null` zurück, wenn das Feature-Flag nicht existiert.
    - `BrazePlugin.subscribeToFeatureFlags(void Function(List<BrazeFeatureFlag>) onEvent))` wird nur in den folgenden Situationen ausgelöst:
      - Wenn eine Aktualisierungsanfrage mit Erfolg oder Misserfolg abgeschlossen wird.
      - Bei der ersten Anmeldung, wenn zuvor Daten aus der aktuellen Sitzung zwischengespeichert wurden.
  - Die minimal unterstützte Android SDK-Version ist 21.
