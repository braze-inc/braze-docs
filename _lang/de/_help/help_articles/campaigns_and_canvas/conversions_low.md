# Niedrige Kampagnen- oder Canvas-Konversionen

Conversions treten auf, wenn Ihr Nutzer eine Aktion innerhalb Ihrer Nachricht ausführt, die Sie bei der Erstellung Ihrer Kampagne festgelegt haben.

Ihre Konversionen sind vielleicht nicht so hoch, wie Sie es erwarten, wenn Sie sie mit früheren Kampagnen oder Ihren Erwartungen vergleichen. Konversionen sind eine knifflige Angelegenheit, aber sie hängen von ein paar einfachen Funktionen in unserer Plattform ab: Ereignisverfolgung und Konversionsfristen.

Um schnell herauszufinden, woran das liegt, empfehlen wir Ihnen, Ihre Ereignisverfolgung und Konvertierungsfristen zu überprüfen.

## Ereignisverfolgung

Wenn eine Kampagne einen Sitzungsstart oder ein benutzerdefiniertes Ereignis auslöst, müssen Sie sicherstellen, dass dieses Ereignis oder diese Sitzung häufig genug stattfindet, um die Nachricht auszulösen. Überprüfen Sie diese Daten auf den Seiten [Übersicht][1] (für Sitzungsdaten) oder [Benutzerdefinierte Ereignisse][2]:

![Die Seite Benutzerdefinierte Ereignisse mit den Statistiken für benutzerdefinierte Ereigniszählungen.][43]

## Fristen für die Umstellung

Für jedes Konvertierungsereignis, das Sie pro Kampagne auswählen, legen Sie die [Frist][44]] fest. Das bedeutet, dass Sie ein Zeitlimit festlegen, innerhalb dessen eine Konversion stattfinden muss, damit sie für die jeweilige Kampagne gezählt wird.

Vergewissern Sie sich, dass Sie die Informationen zur [Berechnung der Conversions][45] überprüft haben, um Ihre Kampagnenmetriken zu verstehen. Informationen zu Benutzerumwandlungen in Canvas finden Sie in den [Canvas-FAQ]({{site.baseurl}}/user_guide/engagement_tools/canvas/faqs/#how-are-user-conversions-tracked-in-a-canvas).
