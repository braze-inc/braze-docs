## November 2018

### Assistent für den Canvas-Eingang

Das Canvas UI wird vereinfacht, um verpasste Aufgaben und daraus resultierende Fehler zu vermeiden. Die Canvas-Konfigurationen werden jetzt in einem Assistenten angezeigt, der dem Design des Assistenten für Kampagnen ähnelt. Weitere Informationen finden Sie in den Schritten zur [Erstellung eines Canvas]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/)!

Möchten Sie mehr über dieses neue Feature erfahren? Schauen Sie sich unseren Braze-Lernkurs über den [Eingangs-Assistenten](https://learning.braze.com/the-new-canvas-entry-step/264889/scorm/20z5ij5ublxbk) an!

### In-App-Nachricht Web Modal CSS

Der [CSS-Editor des Web Modals]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/create/#web-modal-css) wurde aktualisiert und erlaubt Ihnen mit unseren neuen [CSS-Templates]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/in_app_message_color_templates/#css-template) mehr Kontrolle über das Aussehen Ihrer In-App-Nachrichten.
