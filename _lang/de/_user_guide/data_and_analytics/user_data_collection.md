{% alert important %}
Braze sperrt Nutzer:innen (Dummy-Nutzer:innen) mit mehr als 5 Millionen Sitzungen und nimmt ihre SDK-Events nicht mehr auf, da sie in der Regel das Ergebnis von Fehlintegration sind. Wenn Sie feststellen, dass dies bei einem rechtmäßigen Nutzer:in passiert ist, wenden Sie sich an Ihren Braze-Konto Manager:in.
{% endalert %}
