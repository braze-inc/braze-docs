  - name: Anwendungsprogramm-Schnittstelle (API)

Braze-Kunden verwenden es, um Benachrichtigungen zu verwalten, gezielte Messaging-Kampagnen einzurichten und Analysen einzusehen. Entwickler:in verwalten damit Einstellungen für die Integration von Apps, wie API-Schlüssel und Zugangsdaten für Push-Benachrichtigungen."
  - name: Braze Engagement Tools
    description: "Braze bietet zahlreiche <a href=\"https://www.braze.com/docs/user_guide/engagement_tools/\">Tools zur Einbindung</a>, um Ihre Kunden und Nutzer mit unseren Kampagnen und Canvas-Tools zu erreichen. Außerdem können Sie mit unseren Templates &amp; Media-Tools die Konsistenz optimieren (und Bilder und andere Inhalte hochladen). Von dort aus können Sie Segmente und Geofences erstellen, um Ihre Zielgruppe nach Standorten oder anderen Attributen zu adressieren."
  - name: Lötende Teams

Dies ermöglicht es Braze-Administratoren, den Zugriff auf bestimmte Funktionen nach Gruppenmitgliedschaft zu beschränken."
  - name: Kampagnen
    description: "Eine anpassbare Nachrichtenmethode, um Ihren Kunden eine persönliche Antwort zukommen zu lassen. Sie können über verschiedene Messaging-Kanäle <a href=\"https://www.braze.com/docs/user_guide/engagement_tools/campaigns/\">Kampagnen erstellen</a>, um Ihre eindeutigen Nachrichten zu versenden." 
  - name: Canvas
    description: "Eine einzige einheitliche Schnittstelle, über die Marketer Kampagnen mit mehreren Nachrichten und Schritten einrichten können, um eine zusammenhängende Reise zu gestalten. Mit <a href=\"https://www.braze.com/docs/user_guide/engagement_tools/canvas/\">Canvas</a> können Sie diese Erfahrungen auch vergleichen und optimieren, indem Sie umfassende Analysen für das gesamte Nutzererlebnis nutzen."
  - name: Connected-Content
    description: "<a href=\"https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/connected_content/\">Connected Content</a> erweitert die Personalisierung des Marketings, um die Kundenbindung und die Konversionsrate zu erhöhen. Mit dieser Funktion können Sie alle über die API zugänglichen Informationen direkt in die Nachrichten einfügen, die Sie an Benutzer senden. Mit Connected-Content können Sie Inhalte entweder direkt von Ihrem Webserver oder über öffentlich zugängliche APIs beziehen."
  - name: Content-Cards
    description: "Mit <a href=\"https://www.braze.com/docs/user_guide/message_building_by_channel/content_cards/about/\">Content Cards</a> können Sie Ihren Kunden direkt in den Apps, die sie lieben, einen gezielten, dynamischen Strom reichhaltiger Inhalte senden, ohne dass diese ihr Erlebnis unterbrechen. Content Cards können an iOS-, Android- und Web-Benutzer gesendet werden."
  - name: Currents
    description: "<a href = \"https://www.braze.com/docs/user_guide/data_and_analytics/braze_currents/\">Currents</a>, unser Daten-Streaming-Export, ist in bestimmten Braze-Paketen enthalten. Braze Currents ermöglicht Ihnen die Integration durch Datenspeicherung unter Verwendung von Flat Files oder durch unsere Partner für Verhaltensanalyse und Kundendaten unter Verwendung von gebündelten JSON-Payloads an einen bestimmten Endpunkt."
  - name: Angepasste Attribute
    description: "<a href=\"https://www.braze.com/docs/user_guide/data_and_analytics/custom_data/custom_attributes/\">Benutzerdefinierte Attribute</a> sind eine Sammlung der einzigartigen Eigenschaften Ihrer Benutzer. Sie eignen sich am besten zum Speichern von Attributen über Ihre Benutzer oder von Informationen über geringwertige Aktionen innerhalb Ihrer Anwendung. Sie können Benutzern innerhalb des Dashboards benutzerdefinierte Attribute zuweisen. Sie können Ihre Nutzer nach diesen Attributen filtern und segmentieren, sowohl für <a href=\"https://www.braze.com/docs/developer_guide/platform_integration_guides/swift/analytics/setting_custom_attributes/\">iOS-</a> als auch für <a href=\"https://www.braze.com/docs/developer_guide/platform_integration_guides/android/analytics/setting_custom_attributes/\">Android-Kampagnen</a>."
  - name: Angepasste Events
    description: "<a href=\"https://www.braze.com/docs/user_guide/data_and_analytics/custom_data/custom_events/#custom-events\">Benutzerdefinierte Ereignisse</a> in Braze sind Aktionen, die von Ihren Benutzern ausgeführt werden. Sie eignen sich am besten, um hochwertige Benutzerinteraktionen mit Ihrer Anwendung zu verfolgen."
  - name: Datenpunkt

B. E-Mail, Vorname, Nachname, Land, Wohnort usw.) protokolliert werden, wenn eine Sitzung beginnt und wenn eine Sitzung endet."
  - name: Deeplinking

  - name: "Inaktive Nutzer:innen"

  - name: Endpunkt

Dadurch wird sichergestellt, dass die getriggerte Nachricht für die:den Nutzer:in immer noch relevant ist."
  - name: Frequency-Capping

  - name: HIPAA

Die HIPAA-Anforderungen umfassen administrative, physische und technische Sicherheit."
  - name: In-App-Nachrichten
    description: "<a href=\"https://www.braze.com/docs/user_guide/message_building_by_channel/in-app_messages/\">In-App-Nachrichten</a> sind mobile Nachrichten, die innerhalb Ihrer Anwendung erscheinen. Sie helfen Ihnen, Inhalte an Ihre Nutzer zu übermitteln, ohne sie mit einer Push-Benachrichtigung zu unterbrechen. Angepasste und maßgeschneiderte In-App-Nachrichten verbessern das Nutzererlebnis und helfen Ihrer Zielgruppe, den größten Nutzen aus Ihrer App zu ziehen."
  - name: Inaktive Benutzer

  - name: Schlüssel-Werte-Paare

  - name: Liquid

Liquid erlaubt Ihnen die Verwendung von Objekten, Tags und Filtern, um <a href=\"http://www.braze.com/docs/user_guide/personalization_and_dynamic_content/liquid/supported_personalization_tags/\">eine persönliche Anpassung vorzunehmen</a>."

  - name: Push Time to Live

  - name: Race-Condition
    description: "Eine <a href=\"http://www.braze.com/docs/user_guide/engagement_tools/testing/race_conditions/\">Race Condition</a> ist ein Konzept aus der Softwaretechnik, das eine unerwünschte Situation beschreibt, die auftritt, wenn ein System versucht, mehrere Operationen gleichzeitig auszuführen. Aufgrund der Beschaffenheit des Systems müssen die Operationen jedoch in der richtigen Reihenfolge ausgeführt werden, um korrekt zu funktionieren. <br><br>In der Braze-Plattform kann die Segmentierung einer getriggerten Kampagne anhand von Nutzerdaten, die zum Zeitpunkt des Events aufgezeichnet wurden, eine Race-Condition verursachen. Dies geschieht, wenn eine Änderung des Attributs der Nutzerin oder des Nutzers, auf dessen Grundlage die Kampagne segmentiert wird, zum Zeitpunkt der Bestimmung der Segmentzugehörigkeit und des Versands der Kampagne noch nicht die:den Nutzer:in verarbeitet wurde und dazu führen kann, dass die:der Nutzer:in die Kampagne nicht erhält."

  - name: Tags

  - name: Segmentierung

Braze bietet SDK-Integration sowohl für <a href=\"https://www.braze.com/docs/developer_guide/platform_integration_guides/swift/initial_sdk_setup/overview/\">iOS</a> als auch für <a href=\"https://www.braze.com/docs/developer_guide/platform_integration_guides/android/initial_sdk_setup/android_sdk_integration/#initial-sdk-setup\">Android</a>."
  - name: Sonnenuntergang

Die Erstellung von Sunset-Richtlinien für Ihre <a href=\"https://www.braze.com/docs/user_guide/message_building_by_channel/email/best_practices/sunset_policies/\">E-Mails</a> und <a href=\"https://www.braze.com/docs/user_guide/message_building_by_channel/push/best_practices/sunset_policies/#sunset-policies-for-push/\">Push-Nachrichten</a> kann dazu beitragen, die Auswirkungen auf Ihre Öffnungsraten einzudämmen."

  - name: Webhooks
