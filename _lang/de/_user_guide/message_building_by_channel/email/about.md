# [![Braze Learning Kurs]](https://learning.braze.com/messaging-channels-email) ( [{% image_buster /assets/img/bl_icon3.png %})](https://learning.braze.com/messaging-channels-email){: style="float:right;width:120px;border:0;" class="noimgborder"} Über E-Mail

> Mit E-Mail-Nachrichten können Sie eine Menge anstellen! Nachrichten per E-Mail sind eine großartige Möglichkeit, Ihren Nutzer:innen reichhaltige Inhalte zu liefern. E-Mail-Nachrichten bieten unbegrenzte Flexibilität für die Kommunikation mit Ihren Benutzern außerhalb Ihrer App oder Website. Passen Sie Ihre E-Mail-Nachrichten an, von der Zielgruppenverwaltung bis hin zur Einbindung auffälliger multimedialer Contents. Um Beispiele für E-Mail Kampagnen zu sehen, sehen Sie sich unsere [Anwendungsbeispiele][6] an.

## E-Mails anpassen

Sie können Ihr E-Mail Messaging auf vielfältige Weise anpassen, zum Beispiel:

- [Braze-E-Mail-Templates][2]
- [Benutzerdefinierte HTML-Vorlagen][7]
- [Blöcke für den Drag-and-Drop-Editor][4]
- [Benutzer-Abonnements][5]

## Tolle Anwendungsfälle

Hier sind einige großartige Anwendungsfälle, die Sie inspirieren werden!

| Anwendungsfall | Erklärung |
| --- | --- |
| Erneute Interaktion | Erreichen Sie Nutzer außerhalb Ihrer App, auch solche, die die App nicht installiert haben. |
| Onboarding | Binden Sie neue Benutzer ein und ermutigen Sie sie, Push-Benachrichtigungen zu aktivieren oder die App in sozialen Netzwerken zu teilen. |
| Rich-Nachrichten | Ermöglichen Sie umfangreiche und dynamische HTML-Nachrichten. |
| Multimedialer Content | Einfache Platzierung von Multimedia-Inhalten, die Nutzer:innen engagieren, wie z.B. Videos und Bilder. |
| Newsletters | Versenden Sie bequem monatliche oder wöchentliche Newsletter, um das Engagement der Nutzer:innen aufrechtzuerhalten. |
| Transaktionen | Benachrichtigen Sie Benutzer über kürzlich getätigte Käufe und liefern Sie wichtige Produkt- und Versandinformationen mit [Transaktions-E-Mails][3].
