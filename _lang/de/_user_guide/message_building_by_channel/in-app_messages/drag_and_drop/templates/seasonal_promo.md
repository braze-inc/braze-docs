# Saisonale Aktion mit Hintergrundbild

> Nutzen Sie den In-App-Nachricht Drag-and-Drop-Editor, um ein saisonales Angebot oder einen Deal zu bewerben und das Engagement der Nutzer:innen zu steigern.

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Erstellen einer saisonalen Aktion mit einem Hintergrundbild

### Schritt 1: Wählen Sie Ihr Template

Wenn Sie eine In-App-Nachricht per Drag-and-Drop erstellen, wählen Sie **Saisonale Aktion mit Hintergrundbild** als Template aus und wählen dann **Nachricht erstellen**. Dieses Template unterstützt sowohl mobile Apps als auch Webbrowser.

![Der Editor für In-App-Nachrichten mit dem Template für eine saisonale Aktion.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo.png %})

### Schritt 2: Nachrichtenstile festlegen

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Schritt 3: Passen Sie Ihre Button-Komponente an

Um mit der Erstellung Ihrer saisonalen Aktion zu beginnen, wählen Sie die Button-Komponente im Editor aus. Verwenden Sie dann das Seitenmenü, um auszuwählen, wohin die Nutzer:innen geführt werden, wenn sie den Button auswählen. Standardmäßig schließt das Template die Nachricht, aber Sie können wählen, ob Sie zu einer bestimmten Seite in Ihrer App navigieren möchten (z.B. zu dem Produkt, das Sie bewerben).

![Der In-App-Nachricht-Editor mit einem Seitenmenü zum Anpassen des Button-Elements.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo_button.png %})

Sie können Ihrer Produktankündigung im Bereich **Seiten** auch weitere Nachrichten hinzufügen und diese dann miteinander verknüpfen, um einen sequentiellen Ablauf zu gewährleisten. Sie können zum Beispiel eine Folge von Nachrichten zusammenstellen, die kurz die Möglichkeiten des Produkts beschreiben und mit einem Button enden, der Nutzer:innen zur Produktseite führt. Erfahren Sie, wie das geht, in [Connect pages together]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Schritt 4: Nachricht gestalten

Passen Sie das Aussehen Ihrer saisonalen Aktion mit den [In-App-Nachrichten-Komponenten]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) per Drag-and-Drop an. Fügen Sie Ihr eigenes Hintergrundbild hinzu, indem Sie die Standard-URL des Hintergrundbildes im Menü **des Nachrichten-Containers** ersetzen, oder entfernen Sie die URL und wählen Sie Ihr Bild aus der [Bibliothek]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

![Der In-App-Nachricht-Editor mit einem Seitenmenü zum Auswählen des Hintergrundbildes.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo_image.png %})
