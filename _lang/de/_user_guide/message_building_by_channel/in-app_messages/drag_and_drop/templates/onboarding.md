# Onboarding-Ablauf

> Verwenden Sie die Vorlage für In-App-Nachricht im **Onboarding-Flow**, um Nutzer:innen-Attribute, Insights und Präferenzen zu sammeln, die Ihre Strategie für die Kampagne unterstützen. 

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Erstellen eines Onboarding-Ablaufs

### Schritt 1: Wählen Sie Ihr Template

Wenn Sie eine In-App-Nachricht per Drag-and-Drop erstellen, wählen Sie für Ihr Template den **Onboarding-Fluss** und dann **Nachricht erstellen**. Dieses Template wird sowohl für mobile Apps als auch für Webbrowser unterstützt.

]({% image_buster /assets/img/drag_and_drop/templates/onboarding_editor.png %})Der Editor für In-App-Nachrichten mit dem Onboarding Flow Template.()

### Schritt 2: Nachrichtenstile festlegen

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Schritt 3: Passen Sie die Reihenfolge Ihres Onboarding-Flusses an

Um mit dem Aufbau Ihres Onboarding-Flusses zu beginnen, verwenden Sie den Abschnitt **Seiten**, um Nachrichten zu Ihrem Fluss hinzuzufügen oder zu löschen. Wenn Sie die gewünschte Anzahl von Nachrichten haben, wählen Sie die Buttons auf jeder Seite aus. Verwenden Sie dann das Seitenmenü, um auszuwählen, wohin Nutzer:innen gehen, wenn sie den Button auswählen - ob sie den Fluss verlassen oder zur nächsten Nachricht gehen.

Vergewissern Sie sich, dass Ihre Nachrichten in der richtigen Reihenfolge verknüpft sind, damit die Nutzer:innen den Fluss korrekt durchlaufen. Wie Sie dabei vorgehen, erfahren Sie auf den [Seiten von Connect together]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Schritt 4: Nachricht gestalten

Passen Sie das Aussehen Ihres Onboarding-Flusses mit den [In-App-Nachrichten-Komponenten]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) per Drag-and-Drop an.
