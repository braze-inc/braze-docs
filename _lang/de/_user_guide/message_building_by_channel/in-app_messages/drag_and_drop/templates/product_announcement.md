# Produktankündigung

> Nutzen Sie den In-App-Nachricht per Drag-and-Drop-Editor, um die Aufmerksamkeit für neue Produkte zu erhöhen.

{% multi_lang_include drag_and_drop/templates.md section='SDK-Anforderungen' %}

## Eine Produktankündigung erstellen

### Schritt 1: Wählen Sie Ihr Template

Wenn Sie eine In-App-Nachricht per Drag-and-Drop erstellen, wählen Sie **Produktankündigung** für Ihr Template und dann **Nachricht erstellen**. Dieses Template wird sowohl für mobile Apps als auch für Webbrowser unterstützt.

![Der Editor für In-App-Nachrichten mit dem Template für eine Produktankündigung.][img1]

### Schritt 2: Nachrichtenstile festlegen

{% multi_lang_include drag_and_drop/templates.md section='Nachrichtenstil' %}

### Schritt 3: Passen Sie Ihre Button-Komponente an

Um mit der Erstellung Ihrer Produktankündigung zu beginnen, wählen Sie die Button-Komponente im Editor aus. Verwenden Sie dann das Seitenmenü, um auszuwählen, wohin die Nutzer:innen geführt werden, wenn sie den Button auswählen. Standardmäßig schließt das Template die Nachricht ab, aber Sie können wählen, ob Sie zu einer bestimmten Seite in Ihrer App navigieren möchten (z. B. zu der Seite des Produkts, das in Ihrer Ankündigung vorgestellt wird).

![Der In-App-Nachricht-Editor mit einem Seitenmenü zur Anpassung des Button-Elements.][img2]

Sie können Ihrer Produktankündigung im Bereich **Seiten** auch zusätzliche Nachrichten hinzufügen und diese dann nacheinander miteinander verknüpfen. Sie können zum Beispiel eine Folge von Nachrichten zusammenstellen, die kurz die Möglichkeiten des Produkts beschreiben und mit einem Button enden, der Nutzer:innen zur Produktseite führt. Erfahren Sie, wie das geht, in [Connect pages together]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Schritt 4: Nachricht gestalten

Passen Sie das Aussehen Ihrer Produktankündigung mit den [In-App-Nachrichten-Komponenten][3] per Drag-and-Drop an.
