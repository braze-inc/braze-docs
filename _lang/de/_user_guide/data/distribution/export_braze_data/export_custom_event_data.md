# Exportieren Sie angepasste Event-Aggregate

> Diese Seite behandelt die Berichtsseite " **Angepasste Events"** des Dashboards, auf der Sie die Vorkommen eines oder mehrerer angepasster Events im Laufe der Zeit anzeigen können. Wenn Sie sich detaillierte Statistiken für angepasste Events oder angepasste Events pro Stunde ansehen, haben Sie die Möglichkeit, sich die Daten auch nach bestimmten Segmenten anzeigen zu lassen.

Sie finden den **Bericht Angepasste Events** unter **Analytics**.

![Angepasste Events]({% image_buster /assets/img_archive/Export_events.png %})

Sie können die folgenden CSV-Dateien exportieren:

- Angepasste Events nach Datum
    - (Optional) Angepasste Events für verschiedene Segmente
- Angepasste Events nach Stunde
    - (Optional) Angepasste Events für verschiedene Segmente
- Angepasste Events pro MAU
- Angepasste Events für verschiedene Segmente
- Angepasste Events per KPI-Formel
    - (Optional) Angepasste Events für verschiedene Segmente

{% alert tip %}
Hilfe zu CSV- und API-Exporten finden Sie unter [Fehlerbehebung bei Exporten]({{site.baseurl}}/user_guide/data/export_braze_data/export_troubleshooting/).
{% endalert %}
