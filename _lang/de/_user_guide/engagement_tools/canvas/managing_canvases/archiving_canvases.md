# Leinwände archivieren

> Um das Senden eines Canvas zu stoppen oder es aus Ihrem Dashboard zu entfernen, können Sie ein Canvas archivieren. 

Rufen Sie die Seite **Canvas** auf, klicken Sie auf das Zahnradsymbol <i class="fas fa-gear"></i> neben dem ausgewählten Canvas und klicken Sie auf **Archiv**.

Sie können auch Massenaktionen durchführen, z. B. mehrere Canvase archivieren, indem Sie die Kästchen neben den Canvase markieren und den entsprechenden Button auswählen. 

![][1]

Wenn Sie einen Canvas archivieren, werden keine weiteren Nachrichten aus dem Canvas gesendet. Sie können alle zuvor archivierten Leinwände ansehen, indem Sie auf der Seite **Leinwand** den Status **Archiviert** wählen.

![][2]

## Leinwände entarchivieren

Sie können auch die Archivierung eines Canvas aufheben. Sie können entweder nach dem Canvas suchen oder die Liste Ihrer archivierten Canvases auf der **Canvas-Seite** aufrufen. Klicken Sie dann auf das Zahnradsymbol <i class="fas fa-gear"></i> neben dem ausgewählten Canvas und klicken Sie auf **Archivierung aufheben**. Beachten Sie, dass, wenn Sie die Archivierung eines Canvas aufheben, auch die zugehörigen archivierten Segmente nicht mehr archiviert werden.
