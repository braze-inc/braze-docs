# Newsfeed: Anwendungsfälle {#news-feed-use-cases}

{% multi_lang_include deprecations/braze_sdk/news_feed.md %}

> Bei Braze schätzen wir die Anpassungsmöglichkeiten, die News Feed bietet. Hier finden Sie einige unserer bevorzugten Anwendungsfälle, um Ihnen Ideen zu geben, wie Sie das Beste aus Braze herausholen können.

- App-Updates bewerben
- Begrüßungskampagnen mit schrittweisem Nachrichtenversand
- Bestimmung der Zahlungsbereitschaft eines Nutzers.
  - Wenn Sie mit Hilfe von gezielten News Feed-Karten einen Zeitplan für stetig steigende Rabatte anbieten, können Sie genau sehen, wie viel Ihr Service den verschiedenen Nutzersegmenten wert ist.
- Erfolge und Badges
  - Wenn ein Nutzer eine bemerkenswerte Aufgabe erledigt oder eine gewünschte Aktion ausführt, nachdem er von einer Marketingkampagne angesprochen wurde, können Sie dieses Segment isolieren, um seine Leistung mit einer News-Feed-Karte zu würdigen.
    - Vielleicht möchten Sie sogar eine bestimmte Belohnung wie einen In-App-Rabatt damit verknüpfen.
- Aufforderung zum Teilen in sozialen Netzwerken.
- [Deeplinks][1] mit angepassten URLs, um auf Ressourcen in Ihrer App hinzuweisen.
  - Eine Video-Messaging-App könnte zum Beispiel eine News-Feed-Karte haben wollen, die auf ein Video verweist, das an anderer Stelle in der App zu sehen ist. Sie könnten dies sogar nutzen, um Ihre gesamte App-Navigation um unseren News Feed herum zu gestalten!
   
{% alert note %}
Braze empfiehlt Kunden, die unser News-Feed-Tool verwenden, auf unseren Nachrichtenkanal Content Cards umzusteigen - er ist flexibler, anpassbarer und zuverlässiger. Außerdem ist er in Braze einfacher zu finden und zu verwenden. Kontaktieren Sie Ihren Braze-Kundenbetreuer für weitere Informationen.
{% endalert %}
