# Fehlerbehebung für BrazeAI Operator

> Häufige Probleme und Lösungen für BrazeAI Operator<sup>TM</sup>.

## Häufige Probleme

| Fehler | Fehlersuche |
| --- | --- |
| Keine Antwort | Bitte versuchen Sie, die Seite zu aktualisieren und das Operator-Panel erneut zu öffnen. |
| Antworten, die nicht zum Thema gehören | Bitte formulieren Sie die Frage genauer. Bitte nennen Sie das Feature oder den Workflow, zu dem Sie eine Frage haben. |
| Fehlermeldungen | Sollte der Operator keine Inhalte streamen können, wird möglicherweise die Aufforderung „Bitte versuchen Sie es erneut“ angezeigt. Der Operator ist möglicherweise vorübergehend nicht erreichbar oder die Verbindung wurde unterbrochen. Bitte versuchen Sie es in einigen Minuten erneut. |
| Tägliches Nutzungslimit überschritten | Das unternehmensweite Nutzungslimit wurde erreicht. Bitte warten Sie, bis das Limit zurückgesetzt wird (wird alle 24 Stunden zurückgesetzt). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Beschränkungen

- **Datenzugriff:** Der Operator kann den Inhalt der aktiven Seite einsehen. Es kann auch bestimmte unternehmensspezifische Daten abrufen, wie beispielsweise vergangene Kampagnen, Kataloge und angepasste Attribute. Derzeit ist es jedoch nicht möglich, neue Berichte zu erstellen oder Berichtsabfragen auszuführen.
- **Nutzungsbeschränkungen:** Der Operator verfügt über ein unternehmensweites Tageslimit, das alle 24 Stunden zurückgesetzt wird. Wenn das Limit erreicht ist, wird im Chat ein Banner mit der Meldung „Tägliches Nutzungslimit überschritten“ angezeigt, und es können keine weiteren Anfragen gestellt werden, bis das Limit zurückgesetzt wird.
