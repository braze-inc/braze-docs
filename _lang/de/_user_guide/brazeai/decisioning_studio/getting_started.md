# Erste Schritte mit Decisioning Studio

> Diese Referenz gibt eine Übersicht über die Schritte zur Einrichtung von Decisioning Studio, einschließlich der Verbindung von Datenquellen, der Einrichtung der Orchestrierung und des Entwurfs Ihres Decisioning Agent.

Mit BrazeAI Decisioning Studio™ können Sie Entscheidungsagenten entwerfen und einsetzen, die jede beliebige geschäftliche Metrik optimieren. Ein Decisioning Agent ist eine angepasste Konfiguration, die auf ein bestimmtes Geschäftsziel zugeschnitten ist.

Dazu müssen Sie Datenquellen verbinden, die Orchestrierung einrichten und Ihre Entscheidungsagenten entwerfen.

{% alert tip %}
Für Kunden:in von Decisioning Studio Pro unterstützt Sie Ihr Team von KI Expert Serviceleistungen; Dienste bei der Einrichtung von Decisioning Studio für optimale Performance.
{% endalert %}

## Einrichten von Decisioning Studio

Um Decisioning Studio einzurichten, führen Sie die folgenden Schritte aus:

### Schritt 1: Datenquellen verbinden

Verknüpfen Sie Kundenprofile und Engagement-Daten, damit die von Ihnen erstellten Entscheidungsagenten verstehen, wer jeder Kunde ist und wie er sich verhält.

In der Regel müssen Sie Ihre Datenquellen nur einmal verbinden, nämlich bei der Ersteinrichtung von Decisioning Studio. Wenn Sie Ihre Anwendungsfälle später erweitern, müssen Sie möglicherweise neue Datenquellen hinzufügen.

{% alert tip %}
Alle Daten, die sich bereits in der [Braze Data Platform]({{site.baseurl}}/user_guide/data/braze_data_platform) befinden, sind automatisch für Decisioning Studio verfügbar.
{% endalert %}

Ausführliche Anleitungen finden Sie in der Dokumentation zu Ihrer Version von Decisioning Studio:
- [Decisioning Studio Go: Datenquellen verbinden]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Decisioning Studio Pro: Datenquellen verbinden]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Schritt 2: Orchestrierung einrichten

Integrieren Sie Decisioning Studio in Ihre Customer-Engagement-Plattform (CEP), um Ihren Agenten die Orchestrierung von Aktionen zu ermöglichen. Die CEP ist die Plattform, mit der Sie Ihren Kund:innen personalisierte Erlebnisse auf der Grundlage der Entscheidungen des Agenten zustellen können.

Im Allgemeinen müssen Sie diese Orchestrierung nur einmal einrichten.

Ausführliche Anleitungen finden Sie in der Dokumentation zu Ihrer Version von Decisioning Studio:
- [Decisioning Studio Go: Orchestrierung einrichten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Decisioning Studio Pro: Orchestrierung einrichten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Schritt 3: Gestalten Sie Ihre Agenten

Konfigurieren Sie Ihre Entscheidungsagenten, um festzulegen, welche Ergebnisse Sie maximieren möchten und welche Aktionen der Agent durchführen kann, um diese zu erreichen. Eine ausführliche Anleitung zum [Design von Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/designing_decisioning_agents/) finden Sie unter [Design von Entscheidungsagenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/designing_decisioning_agents/).

Für stufenspezifische Anleitungen:
- [Decisioning Studio Go: Gestalten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Decisioning Studio Pro: Gestalten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Für Decisioning Studio Pro Kunden unterstützt Sie Ihr Team von KI Decisioning Serviceleistungen; Dienste; Dienste bei der Entwicklung und Einführung Ihrer Entscheidungsagenten.
{% endalert %}

### Schritt 4: Starten Sie Ihren Entscheidungsfindungsagenten

Starten Sie Ihren Decisioning Agent und lassen Sie ihn kontinuierlich lernen und für Ihre Geschäftsergebnisse optimieren.

Ausführliche Anleitungen finden Sie in der Dokumentation zu Ihrer Version von Decisioning Studio:
- [Decisioning Studio Go: Starten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Decisioning Studio Pro: Starten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
