# Übersicht

> Diese Referenz bietet eine Übersicht über die Schritte zum Einrichten von Decisioning Studio, einschließlich der Verbindung von Datenquellen, der Einrichtung der Orchestrierung und der Gestaltung Ihres Entscheidungsagenten.

Mit BrazeAI Decisioning Studio™ können Sie Entscheidungsagenten entwerfen und einsetzen, die beliebige Metriken optimieren. Ein Entscheidungsagent ist eine angepasste Konfiguration, die auf ein bestimmtes Geschäftsziel zugeschnitten ist.

Dazu müssen Sie Datenquellen verbinden, die Orchestrierung einrichten und Ihre Entscheidungsagenten entwerfen.

{% alert tip %}
Für Kunden von Decisioning Studio Pro unterstützt Sie Ihr KI-Expert:innen-Team bei der Einrichtung von Decisioning Studio, um eine optimale Performance zu erzielen.
{% endalert %}

## Einrichtung von Decisioning Studio

Um Decisioning Studio einzurichten, führen Sie bitte die folgenden Schritte aus:

### Schritt 1: Datenquellen verbinden

Verbinden Sie Kundenprofile und Engagement-Daten, damit die von Ihnen erstellten Entscheidungsagenten verstehen, wer jeder Kunde ist und wie er sich verhält.

In der Regel müssen Sie Ihre Datenquellen nur einmal verbinden, und zwar während der Ersteinrichtung von Decisioning Studio. Sollten Sie Ihre Anwendungsfälle später erweitern, müssen Sie möglicherweise neue Datenquellen hinzufügen.

{% alert tip %}
Alle Daten, die bereits in der [Braze Data Platform]({{site.baseurl}}/user_guide/data/braze_data_platform) vorhanden sind, stehen Decisioning Studio automatisch zur Verfügung.
{% endalert %}

Ausführliche Anweisungen finden Sie in der Dokumentation zu Ihrer Stufe von Decisioning Studio:
- [Entscheidungsstudio Go: Datenquellen verbinden]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Entscheidungsstudio Pro: Datenquellen verbinden]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Schritt 2: Orchestrierung einrichten

Integrieren Sie Decisioning Studio in Ihre Customer-Engagement-Plattform (CEP), damit Ihre Mitarbeiter die Orchestrierung von Maßnahmen durchführen können. Die CEP ist die Plattform, die verwendet wird, um Ihren Kunden auf Grundlage der Entscheidungen des Agenten personalisierte Erlebnisse zu bieten.

In der Regel müssen Sie diese Orchestrierung nur einmal einrichten.

Ausführliche Anweisungen finden Sie in der Dokumentation zu Ihrer Stufe von Decisioning Studio:
- [Entscheidungsstudio Go: Orchestrierung einrichten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Entscheidungsstudio Pro: Orchestrierung einrichten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Schritt 3: Entwerfen Sie Ihre Agenten

Konfigurieren Sie Ihre Entscheidungsagenten, um zu definieren, welche Ergebnisse Sie maximieren möchten und welche Maßnahmen der Agent ergreifen kann, um diese zu erreichen. Ausführliche Anleitungen zum Entwerfen von Agenten finden Sie unter [Entwerfen von Entscheidungsagenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/).

Für stufenspezifische Leitlinien:
- [Entscheidungsstudio Go: Entwerfen Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Entscheidungsstudio Pro: Entwerfen Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Für Kund:innen von Decisioning Studio Pro unterstützt Sie Ihr KI-Decisioning-Service-Team bei der Entwicklung und Einführung Ihrer Entscheidungsagenten.
{% endalert %}

### Schritt 4: Starten Sie Ihren Entscheidungsagenten

Starten Sie Ihren Entscheidungsagenten und lassen Sie ihn kontinuierlich lernen und Ihre Geschäftsergebnisse optimieren.

Ausführliche Anweisungen finden Sie in der Dokumentation zu Ihrer Stufe von Decisioning Studio:
- [Entscheidungsstudio Go: Starten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Entscheidungsstudio Pro: Starten Sie Ihren Agenten]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
