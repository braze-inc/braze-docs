> Möchten Sie von unserem älteren Objective-C SDK auf das Swift iOS SDK upgraden? Es ist ganz einfach. Weitere Informationen finden Sie in unserem [Leitfaden für die Migration](https://braze-inc.github.io/braze-swift-sdk/documentation/braze/appboy-migration-guide/).

Nachdem Sie die Integration mit einer der aufgelisteten Optionen durchgeführt, die Schritte zur [Fertigstellung der Integration]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/initial_sdk_setup/completing_integration/) befolgt und weitere SDK-Anpassungen (optional) aktiviert haben, können Sie zusätzliche Kanäle und Features integrieren, aktivieren und anpassen, um sie an die Anforderungen Ihrer zukünftigen Kampagnen anzupassen.  

{% alert note %} Carthage wird nicht als Package Manager für das Swift iOS SDK unterstützt. {% endalert %}
