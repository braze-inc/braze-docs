# Survicate

![Ein Beispiel dafür, wie eine eingebettete HTML-Umfrage (erste Frage) in einer E-Mail von Braze aussehen könnte.]({% image_buster /assets/img/survicate/survicate_asset_1.png %}){: style="float:right;max-width:40%;border:0; margin-left:8px;"}

> [Survicate](https://survicate.com/integrations/braze-survey/?utm_source=braze&utm_medium=integrations&utm_campaign=helpcenter) ist eine Plattform für Kundenfeedback, die Ihnen hilft, Insights von Kund:in über mehrere Kanäle und während der gesamten Customer Journey zu sammeln, zu analysieren und zu nutzen.  

_Diese Integration wird von Survicate gepflegt._

## Über die Integration

Mit der Integration von Braze und Survicate können Sie Umfragen direkt in Ihre E-Mails von Braze einbetten, um die Antwortraten zu erhöhen. Umfragen werden automatisch mit Braze Nutzerprofilen als angepasste Attribute oder Events synchronisiert. Insights in Realtime erleichtern Ihnen das Tracking und die Analyse von Feedback zusammen mit Kundendaten und die Erstellung gezielter Follow-ups.

## Anwendungsfälle

Braze und Survicate arbeiten zusammen, um eine Reihe von Feedback-Anwendungsfällen abzudecken und Ihnen dabei zu helfen, umsetzbare Insights der Nutzer:innen zu sammeln und das Kundenerlebnis zu verbessern:

- Messen Sie die Kundenzufriedenheit (wie CSAT, NPS oder CES)
- Sammeln Sie Produkt-Feedback
- Führen Sie Nutzer:innen- oder Marktforschung durch
- Sammeln Sie Insights in kritischen Phasen der Customer Journey
- Triggern Sie personalisierte Workflows und automatisieren Sie Folgekampagnen auf der Grundlage des Feedbacks von Kund:in

## Die wichtigsten Features der Integration

Die Integration von Survicate und Braze bietet eine Realtime-Datensynchronisation, so dass die aktuellsten Informationen aus Survicate-Umfragen sofort in Braze verfügbar sind. Auf der Grundlage der Antworten auf Umfragen können Sie diese Daten nutzen, um zeitnahe, personalisierte Maßnahmen zu ergreifen.

- **Senden Sie Umfragen an Braze als angepasste Attribute für Nutzer:innen**: Reichern Sie die Nutzerprofile von Braze mit Daten aus Umfragen an.
- **Triggern Sie angepasste Events in Braze**: Nutzen Sie Ereignisse, die auf den Antworten auf Umfragen basieren, um bestimmte Gruppen zu targeting oder Folgekampagnen zu initiieren.
- **Erstellen Sie detaillierte Segmente**: Erstellen Sie Segmente in Braze anhand von Daten aus Survicate-Umfragen, um Ihre Reichweite weiter zu personalisieren.

## Integration

### Schritt 1: Erstellen Sie Ihre Umfrage in Survicate

1. Wählen Sie in Ihrem Survicate Panel **Neue Umfrage erstellen**.
2. Wählen Sie den Kanal für Ihre Umfrage: E-Mail**, Link, Website, produktinterne Umfragen und Umfragen für mobile Apps** sind möglich. 
3. Entwerfen Sie Ihre Umfrage von Grund auf neu, verwenden Sie den KI Umfrage-Ersteller oder wählen Sie aus über 100 gebrauchsfertigen Templates.

![Vier Optionen für die Erstellung einer Umfrage: von Grund auf neu beginnen, eine Vorlage verwenden, KI-unterstützte Erstellung und Fragen importieren.]({% image_buster /assets/img/survicate/survicate_asset_3.png %})

### Schritt 2: Automatische Identifizierung der Befragten mit Braze-E-Mails

1. Nachdem Ihre Umfrage fertig ist, gehen Sie auf den Tab **Konfigurieren**.
2. Für *Bezeichner identifizieren mit* wählen Sie **Braze** aus. Dadurch werden die Antworten automatisch mit Ihren Braze Kundenprofilen verknüpft, so dass Sie in Ihrer Umfrage nicht mehr nach den Kontaktdaten fragen müssen.

![Braze wird als Befragte ausgewählt.]({% image_buster /assets/img/survicate/survicate_asset_2.png %})

### Schritt 3: Verbinden Sie die Integration

1. Suchen Sie dann auf dem **Tab Verbinden** nach Braze und wählen Sie **Verbinden** zur Integration. 
2. Geben Sie den API-Schlüssel Ihres Braze-Kontos für den Workspace und die URL der Braze-Instanz ein.

![Felder zur Eingabe des Workspace API-Schlüssels und der URL der Braze-Instanz.]({% image_buster /assets/img/survicate/image1.png %})

### Schritt 4: Teilen Sie Ihre Umfrage

1. Als nächstes wählen Sie auf dem Tab **Teilen**, wo Sie Ihre Umfrage platzieren möchten. Die Optionen umfassen:
- **Direkter Link**: Kopieren Sie den Link, um ihn in Braze als Button oder Hyperlink zu verwenden.
- **Die erste Frage einbetten**: Kopieren Sie den HTML Code, um die erste Frage der Umfrage direkt in eine E-Mail von Braze einzubetten.
- **Starten Sie eine Umfrage auf Ihrer Website oder in Ihrem Produkt**: Installieren Sie den Tracking Code einmal und stellen Sie Umfragen direkt vom Survicate Panel aus live.

### Schritt 5: Fügen Sie die Umfrage zu Ihrer Braze E-Mail Kampagne hinzu

1. Fügen Sie in Braze den Link zur Umfrage oder den HTML-Code in den Inhalt Ihrer E-Mail Kampagne ein.
2. Beginnen Sie mit dem Sammeln von Feedback und dem Tracking von Antworten direkt in Survicate.
