# Voucherify

> [Voucherify](https://www.voucherify.io/) ist eine All-in-One-Werbeplattform, die personalisierte Kampagnen und Treueprogramme ermöglicht, die das Engagement und die Bindung der Nutzer fördern. 

<iframe src="https://player.vimeo.com/video/745340934?h=17ceae8c3c" width="640" height="360" frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>

Die Integration von Braze und Voucherify ermöglicht es Ihnen, Ihre Werbekampagnen zu erweitern, indem Sie einzigartige Codes versenden:

- [Verbundener Inhalt]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_fetching_data_through_braze_connected_content): Fügen Sie Braze-Kampagnen über Braze Connected Content eindeutige Codes hinzu. Mit dieser Funktion können Sie Voucherify-Rabattcoupons, Geschenkkartenaktionen, Treuekarten und Empfehlungscodes verwenden.
- [Benutzerdefinierte Attribute]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_distribution_with_braze_custom_attributes): Mit benutzerdefinierten Attributen können Sie den Benutzerprofilen in Braze einzigartige Voucherify-Gutscheine, Geschenkkarten, Treuekarten und Empfehlungscodes zuweisen. So können Sie in E-Mail-Kampagnen angehängte Codes und Attribute versenden und sie mit Ihren Benutzern teilen.
- [Aktionscode-Listen]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_using_braze_promotion_codes_list): Verwenden Sie die von Voucherify generierten Aktionscodes und laden Sie sie in Braze hoch.

## Voraussetzungen

| Anforderung | Beschreibung |
| ----------- | ----------- |
|Voucherify Konto | Um die Vorteile dieser Partnerschaft zu nutzen, benötigen Sie ein Voucherify-Konto. |
| Braze REST API Schlüssel | Ein Braze REST API-Schlüssel mit `users.track` Berechtigungen. <br><br> Dieser kann im Braze-Dashboard unter **Einstellungen** > **API-Schlüssel** erstellt werden. |
