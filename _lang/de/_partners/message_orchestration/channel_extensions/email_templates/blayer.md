# B.Layer

> [B.Layer](https://blayer.phiture.com) ist der Phiture In-App Message Builder, mit dem CRM-Teams für mobile Apps einfach, schnell und ohne Programmieraufwand individuell gestaltete In-App-Nachrichten erstellen können. 

Die Integration von Braze und B.Layer ermöglicht es Ihnen, den B.Layer In-App Message Builder zu nutzen, um In-App-Nachrichten zu erstellen, die als Zip-Datei oder Inline-HTML in Braze exportiert werden können. Diese Integration erfordert keine zusätzlichen Entwicklerressourcen, so dass Sie Zeit und Budget sparen.

## Anwendungsfälle

Mit B.Layer gibt es unendlich viele Möglichkeiten zum Erstellen und Experimentieren, z. B. Slider für Produktempfehlungen, Onboarding über mehrere Bildschirme oder Umfragen, NPS, E-Mail-Erfassung, Sonderangebote und vieles mehr.

Sie arbeiten mit Marken wie Lifesum, Blinkist, OnX Hunt und vielen anderen zusammen, um die Nutzererfahrung ohne zusätzliche Ressourcen zu verbessern. Wir sind auch unter den Finalisten der APS Awards 2022 in der Kategorie App-Innovation.

## Integration

### Schritt 1: Erstellen Sie Ihre In-App-Nachricht

#### Markenfarben und Schriftarten festlegen

Auf B.Layer klicken Sie im Hamburger-Menü oben auf der Seite auf **Marken-Assets > Ihre Marken-Assets hinzufügen**. Hier können Sie Ihre Markenfarbe und Schriftarten zuweisen.
Sie sind bereit. Jetzt können Sie mit der Gestaltung Ihrer In-App-Nachricht beginnen.

![][4]

#### Gestalten Sie Ihre In-App-Nachricht

Um Ihre In-App-Nachricht zu gestalten, wählen Sie eine einzelne In-App-Nachricht aus. Als nächstes gestalten Sie Ihre Nachricht und fügen die benötigten Komponenten hinzu. Jede Komponente kann angepasst werden.

![][5]

### Laden Sie Ihre In-App-Nachricht herunter

Sobald Sie fertig sind, laden Sie Ihre Nachricht herunter. Ihre Nachricht kann als ZIP- oder Inline-HTML-Datei heruntergeladen werden. 

### Schritt 2: B.Layer benutzerdefinierten Code hinzufügen

Erstellen Sie in Braze eine benutzerdefinierte In-App-Nachricht. Wenn Sie über eine ZIP-Datei verfügen, ziehen Sie diese in das Feld über dem Code-Bereich und legen Sie sie dort ab. Wenn Sie eine Inline-HTML-Datei haben, fügen Sie die Inline-HTML-Datei in den HTML-Abschnitt ein.

![][6]

## Tastenverfolgung

Mit B.Layer können Sie Tasteninteraktionen oder Texteingaben als Braze-Attribut protokollieren. Das können Sie im Editor erledigen. Ein beliebtes Beispiel ist eine NPS-Umfrage.

B.Layer verwendet das Braze-Schaltflächen-Tracking, das zu den von Ihnen eingegebenen Links hinzugefügt wird (z. B. `?button=0`). Auf diese Weise können Sie die Schaltflächenklicks im Analyseteil Ihrer Kampagne sehen.
