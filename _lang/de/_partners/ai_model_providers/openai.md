# OpenAI

> [OpenAI](https://openai.com/) entwickelt fortschrittliche KI-Modelle wie GPT, die das Verstehen und Generieren natürlicher Sprache ermöglichen und Marken in die Lage versetzen, sinnvolle Kunden:in-Interaktionen aufzubauen und zu skalieren. 

_Diese Integration wird von OpenAI gepflegt._

## Über die Integration

Mit der Integration von Braze und OpenAI können Sie Ihren OpenAI API-Schlüssel mit Braze verbinden, so dass Sie OpenAI-Modelle verwenden können, wenn Sie angepasste KI-Agenten erstellen. Mit dieser Integration können Ihre Agenten personalisierte Texte erstellen, Entscheidungen in Echtzeit treffen oder Katalogfelder mit Hilfe der umfangreichen Sprachmodelle von OpenAI aktualisieren.

## Voraussetzungen

| Anforderungen | Beschreibung |
|---|---|
| OpenAI-Konto mit API-Schlüssel | Ein OpenAI-Konto mit einem API-Schlüssel. Wenden Sie sich bei Problemen oder Fragen an Ihren Administrator oder den [OpenAI-Support](https://help.openai.com/). |
| Braze-Instanz | Sie finden Ihre Braze-Instanz auf der [API-Übersichtsseite]({{site.baseurl}}/api/basics/#endpoints) oder über Ihren Braze Onboarding Manager:in. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integration

So verbinden Sie Ihren OpenAI API-Schlüssel mit Braze:

1. Gehen Sie im Braze-Dashboard zu **Partnerintegrationen** > **Technologiepartner** und finden Sie OpenAI.
2. Geben Sie Ihren API-Schlüssel von OpenAI ein.
3. Wählen Sie **Speichern**.

Nach dem Speichern können Sie OpenAI-Modelle auswählen, wenn [Sie einen angepassten Agenten]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) in der Agentenkonsole [erstellen]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/).

Wenden Sie sich an den [OpenAI-Support](https://help.openai.com/), wenn Sie Probleme oder Fragen zu Ihrer Integration haben.
