# Blings

> [Blings](https://www.blings.io/) ist eine personalisierte Videoplattform der nächsten Generation, die es Ihnen ermöglicht, interaktive und datengestützte Video-Erlebnisse in Echtzeit über alle Kanäle hinweg zuzustellen.  

_Diese Integration wird von Blings gepflegt._

## Voraussetzungen  

| Anforderung     | Beschreibung                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| Blings Konto  | Um die Vorteile dieser Partnerschaft zu nutzen, benötigen Sie ein Blings-Konto.         |  

## Integration  

### Schritt 1: Holen Sie sich Ihr Blings HTML Snippet

{% tabs %}
{% tab Blings business and free plans %}

#### Blings Business und kostenlose Pläne

Suchen und kopieren Sie Ihr HTML-Snippet direkt in der Blings App.

1. Gehen Sie auf den Tab **Verbinden** des ausgewählten MP5-Projekts.
2. Fügen Sie Braze Liquid-Tags zu den entsprechenden Variablen auf der Blings **Connect-Seite** hinzu. Die Tags füllen dynamisch Werte in das HTML Snippet ein. 

![Blings HTML Snippet.]({% image_buster /assets/img/blings/blings_connect_audience.png %}){: style="max-width:70%;"}  

{% endtab %}
{% tab Blings Enterprise plan %}

#### Blings Unternehmensplan

Fordern Sie das HTML Snippet bei Ihrer Blings Vertretung an. 

{% endtab %}
{% endtabs %}

### Schritt 2: Erstellen Sie eine Braze Kampagne  

Erstellen Sie in Braze eine neue E-Mail- oder In-App-Nachrichten-Kampagne und fügen Sie das Blings HTML-Snippet ein.

![Blings Braze-Kampagne.]({% image_buster /assets/img/blings/blings_braze_campaign.png %})  

### Schritt 3: Test und Start  

Zeigen Sie eine Vorschau der Kampagne in Braze an, um sicherzustellen, dass die personalisierten Felder korrekt ausgefüllt werden. Dann setzen Sie Ihre MP5 Kampagne in großem Umfang ein.

![Blings Braze Vorschau.]({% image_buster /assets/img/blings/blings_braze_preview.png %}){: style="max-width:70%;"}
