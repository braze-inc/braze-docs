## Flexibilidade de dados

### Painel de uso da API

{% multi_lang_include release_type.md release="General availability" %}

O dashboard de uso da API permite monitorar o tráfego de entrada da API REST na Braze para entender as tendências de uso de nossas APIs REST e solucionar possíveis problemas.

### Adição de tags a atributos personalizados

{% multi_lang_include release_type.md release="General availability" %}

Você pode [adicionar tags a um atributo personalizado]({{site.baseurl}}/user_guide/data/custom_data/custom_attributes#adding-tags) depois que ele for criado se tiver a permissão "Manage Events, Attributes, Purchases" (Gerenciar eventos, atributos e compras). As tags podem então ser usadas para filtrar a lista de atributos.

### Endpoints de seleções de catálogo e campos de catálogo assíncronos 

{% multi_lang_include release_type.md release="General availability" %}

Os seguintes endpoints já estão disponíveis de forma geral:
* [POST: Criar campos de catálogo
* [DELETE: Excluir campo de catálogo
* [DELETE: Excluir seleção de catálogo
* [POST: Criar seleção de catálogo

### Uso de um endereço de e-mail para acionar campanhas ou Canvases

{% multi_lang_include release_type.md release="General availability" %}

Agora você pode especificar um destinatário por endereço de e-mail para acionar suas [campanhas]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/targeting_users/) e [Canvases]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/?tab=target%20audience#step-2c-set-your-target-entry-audience).

### Uso de um número de telefone para identificar um usuário por meio da API

{% multi_lang_include release_type.md release="General availability" %}

Agora você pode usar um número de telefone, além de um alias e endereço de e-mail, para identificar um usuário por meio do [endpoint da API`/users/identify` ]({{site.baseurl}}/api/endpoints/user_data/post_user_identify)

### Obtendo um rastreamento SAML
Adicionamos [etapas sobre como obter um rastreamento SAML]({{site.baseurl}}/user_guide/administrative/access_braze/single_sign_on/set_up#obtaining-a-saml-trace), o que o ajuda a resolver problemas sobre o SAML SSO com o Support de forma mais eficiente.
 
### Data centers específicos da região
Como o Braze está crescendo para atender a novas áreas, adicionamos um [artigo sobre os data centers do Braze]({{site.baseurl}}/user_guide/data/data_centers) para esclarecer nossa abordagem operacional.

 
## Liberando a criatividade
 
### Notificações de queda de preço e notificações de estoque disponível

{% multi_lang_include release_type.md release="General availability" %}

Agora você pode notificar os clientes quando um item estiver em estoque, configurando [notificações de estoque em estoque]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/back_in_stock_notifications) por meio de um Canvas e de um catálogo.

Você também pode criar [notificações de queda de preço]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/price_drop_notifications) para notificar os clientes quando o preço de um item diminuir, configurando notificações de queda de preço em um catálogo e no Canvas.

### Visualização para seleção 

{% multi_lang_include release_type.md release="General availability" %}

Depois de criar uma seleção, você pode [ver o que uma seleção retornaria]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/selections/#test-and-preview) para um usuário aleatório ou um usuário específico.

### Modelo de itens de catálogo, incluindo Liquid 

{% multi_lang_include release_type.md release="General availability" %}

Você pode [modelar itens de catálogo que incluam o Liquid]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/using_catalogs/#using-liquid).

### Modelos de canva
Adicionamos novos modelos do Canvas para [integração de usuários com uma pesquisa de preferências]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/preference_survey) e [criação de uma inscrição por e-mail com dupla aceitação]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/email_signup).

### Gerenciamento de leads com o Salesforce Sales Cloud para B2B
Uma das maneiras pelas quais os profissionais de marketing B2B podem usar o Braze é por meio de uma integração com o Salesforce Sales Cloud. Leia mais sobre como implementar esse [caso de uso]({{site.baseurl}}/user_guide/getting_started/b2b_use_cases/b2b_salesforce_sales_cloud).
 
## Canais robustos

### Listas de supressão

{% multi_lang_include release_type.md release="Beta" %}
 
As listas de supressão especificam grupos de usuários que nunca receberão mensagens. Os administradores podem criar listas de supressão com filtros de segmento para restringir um grupo de usuários da mesma forma que você faria para a segmentação.

## Novas parcerias Braze

### Construtor - Conteúdo dinâmico
[O Constructor]({{site.baseurl}}/partners/ecommerce/product_search_recommendations/constructor/) é uma plataforma de pesquisa e descoberta de produtos que usa IA e aprendizado de máquina para oferecer pesquisa personalizada, recomendações e experiências de navegação para sites de comércio eletrônico e varejo.
 
### Trustpilot - Conteúdo dinâmico
[A Trustpilot]({{site.baseurl}}/partners/trustpilot/) é uma plataforma de avaliação on-line que permite que seus clientes compartilhem feedback e que você gerencie e responda às avaliações.
