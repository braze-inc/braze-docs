# Visão geral

> Esta referência fornece uma visão geral das etapas envolvidas na configuração do Decisioning Studio, incluindo a conexão de fontes de dados, a configuração da orquestração e o design do seu agente de decisão.

O BrazeAI Decisioning Studio™ permite projetar e implantar agentes de decisão que otimizam qualquer métrica de negócios. Um agente de decisão é uma configuração personalizada adaptada para atender a uma meta comercial específica.

Para isso, você deve conectar fontes de dados, configurar a orquestração e projetar seus agentes de decisão.

{% alert tip %}
Para clientes do Decisioning Studio Pro, sua equipe de Serviços Especializados em IA irá ajudá-lo a configurar o Decisioning Studio para obter a melhor performance.
{% endalert %}

## Configurando o Decisioning Studio

Para configurar o Decisioning Studio, você deverá concluir as seguintes etapas:

### Etapa 1: Conecte fontes de dados

Conecte perfis de clientes e dados de engajamento para que os agentes de decisão criados por você entendam quem é cada cliente e como ele se comporta.

Normalmente, você só precisa conectar suas fontes de dados uma vez, durante a configuração inicial do Decisioning Studio. Se você expandir seus casos de uso posteriormente, talvez seja necessário adicionar novas fontes de dados.

{% alert tip %}
Todos os dados já existentes na [Plataforma de Dados Braze]({{site.baseurl}}/user_guide/data/braze_data_platform) ficam automaticamente disponíveis para o Decisioning Studio.
{% endalert %}

Para obter orientações detalhadas, consulte a documentação do seu nível do Decisioning Studio:
- [Decisão do Studio Go: Conecte fontes de dados]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Decisioning Studio Pro: Conecte fontes de dados]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Etapa 2: Configurar orquestração

Integre o Decisioning Studio à sua plataforma de engajamento com clientes (CEP) para permitir que seus agentes coordenem ações. O CEP é a plataforma utilizada para oferecer experiências personalizadas aos seus clientes com base nas decisões do agente.

Geralmente, você precisa configurar essa orquestração apenas uma vez.

Para obter orientações detalhadas, consulte a documentação do seu nível do Decisioning Studio:
- [Decisão do Studio Go: Configurar orquestração]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Decisioning Studio Pro: Configurar orquestração]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Etapa 3: Crie seus agentes

Configure seus agentes de decisão para definir quais resultados você deseja maximizar e quais ações o agente pode realizar para alcançá-los. Consulte [Projetando agentes de decisão]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/) para obter orientações detalhadas sobre o projeto de agentes.

Para orientações específicas por nível:
- [Decisão do Studio Go: Crie seu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Decisioning Studio Pro: Crie seu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Para clientes do Decisioning Studio Pro, sua equipe de Serviços de Decisão IA irá ajudá-lo a projetar e lançar seus agentes de decisão.
{% endalert %}

### Etapa 4: Inicie seu agente de tomada de decisão

Inicie seu agente de tomada de decisões e permita que ele aprenda e otimize continuamente para os resultados do seu negócio.

Para obter orientações detalhadas, consulte a documentação do seu nível do Decisioning Studio:
- [Decisão do Studio Go: Inicie seu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Decisioning Studio Pro: Inicie seu agente]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
