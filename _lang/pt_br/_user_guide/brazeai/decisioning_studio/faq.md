# Perguntas frequentes

> Este artigo fornece respostas a algumas perguntas frequentes sobre o Decisioning Studio.

### O que é um agente de decisão?

Um agente de decisão é uma configuração personalizada do BrazeAI Decisioning Studio™ que é feita sob medida para atender a uma meta comercial específica. Isso é definido pela métrica de sucesso, pelas dimensões e pelas opções que você escolher. O agente de decisão descobre automaticamente a ação ideal para cada cliente a fim de maximizar a métrica de negócios escolhida.

### Para quais métricas posso otimizar?

É possível otimizar para qualquer métrica comercial que se alinhe às suas metas, como receita, conversões, receita média por usuário (ARPU), valor do tempo de vida do cliente (CLV), lucro, renovação de contratos ou qualquer outro KPI comercial.

### O que são dimensões no Decisioning Studio?

As dimensões podem ser consideradas como os *tipos de alavancas* que o agente tomador de decisões pode acionar para maximizar a métrica de sucesso. As dimensões típicas incluem oferta, linha de assunto, criativo, canal ou horário de envio.

### O que é um banco de ações?

O banco de ações define as *opções específicas* às quais o agente de decisão tem acesso para cada dimensão "alavanca". Por exemplo, para uma dimensão de canal, você definiria os canais específicos aos quais o agente de decisão tem acesso. Para uma dimensão de oferta, você definiria as ofertas específicas que o agente de decisão pode testar.

### O agente de decisão pode realizar ações que eu não tenha configurado?

Não. O agente de decisão só pode executar ações que você configurar e adicionar ao banco de ações. Isso significa que todas as ações possíveis são definidas pelas combinações do que você coloca no banco de ações.

### O que são restrições?

As restrições limitam as ações do agente de tomada de decisão para respeitar as regras comerciais críticas. Por exemplo, isso pode impedir que uma oferta específica seja selecionada para clientes em uma área geográfica não elegível ou definir um orçamento máximo para o agente de decisão gastar.

### Qual é a diferença entre o Decisioning Studio Acessar e o Decisioning Studio Pro?

O Decisioning Studio Pro inclui o suporte dos IA Decisioning Services da equipe de ciência de dados implantada do Braze, que o ajudará a projetar e configurar seu agente para maximizar os resultados de seus negócios. Para saber mais, consulte [Decisioning Studio Go versus Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
