# Lance seu agente

> Depois de conectar as fontes de dados, configurar a orquestração e projetar seu agente, você está pronto para lançar. Este artigo aborda a ativação do seu agente e o fechamento do ciclo de decisão de IA para que o agente possa aprender e melhorar continuamente.

## Lançando seu agente

Após concluir todas as etapas de configuração com sua equipe de Serviços de Decisão de IA:

1. Revise a configuração do seu agente para garantir que todas as configurações estejam corretas.
2. Verifique se suas conexões de dados e integrações de orquestração estão ativas.
3. Trabalhe com sua equipe de Serviços de Decisão de IA para ativar o agente.

Uma vez lançado, seu agente irá:
- Começar a receber dados de público e de clientes
- Começar a fazer recomendações personalizadas para cada cliente
- Orquestrar ações através do seu CEP configurado
- Coletar dados de feedback para aprender e melhorar ao longo do tempo

## Fechando o ciclo de decisão de IA

Uma vez lançado, seu agente precisa de dados de feedback para aprender e melhorar. Isso inclui dados de conversões, dados de engajamento e dados de ativações que informam ao agente o que aconteceu após as decisões de engajamento do cliente serem enviadas.

Para requisitos detalhados sobre como preparar esses ativos críticos de dados de feedback, veja [Preparando suas fontes de dados]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/preparing_your_data_sources/).

{% alert note %}
Se o agente estiver integrado nativamente com a plataforma de engajamento do cliente (como Braze, SFMC ou Klaviyo), pode não haver etapas adicionais de configuração necessárias para os dados de feedback, uma vez que esses podem ser enviados automaticamente com os dados do cliente.
{% endalert %}

## Monitorando seu agente

Após o lançamento, trabalhe com sua equipe de Serviços de Decisão de IA para monitorar a performance:

- **Métricas de performance**: Acompanhe sua métrica de sucesso entre os grupos de experimento
- **Progresso de aprendizado**: Observe como as recomendações do agente evoluem ao longo do tempo
- **Insights**: Entenda quais dimensões e opções estão impulsionando resultados para diferentes segmentos de clientes

## Otimização contínua

Sua equipe de Serviços de Decisão de IA continuará a trabalhar com você para:

- Analisar a performance do agente e identificar oportunidades de otimização
- Expandir dimensões ou opções conforme necessário
- Ajustar restrições com base nas mudanças nas regras de negócios
- Escalar agentes bem-sucedidos para casos de uso adicionais

{% alert tip %}
O agente aprende e melhora continuamente ao longo do tempo. Permita tempo suficiente para que o agente colete dados e otimize antes de fazer mudanças significativas na configuração.
{% endalert %}
