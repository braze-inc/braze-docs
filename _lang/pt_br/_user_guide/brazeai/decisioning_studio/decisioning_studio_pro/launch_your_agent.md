# Inicie seu agente

> Depois de conectar as fontes de dados, configurar a orquestração e projetar seu agente, você estará pronto para iniciar. Este artigo aborda a ativação do seu agente e o fechamento do ciclo de decisão da IA para que o agente possa aprender e melhorar continuamente.

## Lançamento do seu agente

Após concluir todas as etapas de configuração com sua equipe de Serviços de Decisão de IA:

1. Verifique a configuração do seu agente para garantir que todas as definições estejam corretas.
2. Verifique se suas conexões de dados e integrações de orquestração estão ativas.
3. Trabalhe com sua equipe de Serviços de Decisão de IA para ativar o agente.

Após o lançamento, seu agente irá:
- Comece a receber dados sobre o público e os dados de cliente
- Comece a fazer recomendações personalizadas para cada cliente
- Orquestre ações por meio do seu CEP configurado
- Colete dados de feedback para aprender e melhorar ao longo do tempo

## Fechando o ciclo de decisão da IA

Depois de lançado, seu agente precisa de dados de feedback para aprender e melhorar. Isso inclui dados de conversões, dados de engajamento e dados de ativações que informam ao agente o que aconteceu após o envio das decisões de engajamento do cliente.

Para obter requisitos detalhados sobre como preparar esses ativos de dados de feedback críticos, consulte [Preparando suas fontes de dados]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/preparing_your_data_sources/).

{% alert note %}
Se o agente estiver integrado nativamente à plataforma de engajamento com clientes (como Braze, SFMC ou Klaviyo), talvez não sejam necessárias etapas adicionais de configuração para os dados de feedback, uma vez que estes podem ser enviados automaticamente com os dados do cliente.
{% endalert %}

## Monitorando seu agente

Após o lançamento, trabalhe com sua equipe de Serviços de Decisão de IA para monitorar a performance:

- **Métricas de performance**: Realize o rastreamento das métricas de sucesso entre os grupos experimentais
- **Progresso na aprendizagem**: Observe como as recomendações do agente evoluem ao longo do tempo
- **Insights**: Entenda quais dimensões e opções estão gerando resultados para diferentes segmentos de clientes.

## Otimização contínua

Sua equipe de Serviços de Decisão de IA continuará trabalhando com você para:

- Analise a performance dos agentes e identifique oportunidades de otimização
- Expanda as dimensões ou opções conforme necessário
- Ajustar restrições com base em alterações nas regras de negócios
- Expanda os agentes bem-sucedidos para outros casos de uso

{% alert tip %}
O agente aprende e melhora continuamente ao longo do tempo. Dê tempo suficiente para que o agente colete dados e otimize antes de fazer alterações significativas na configuração.
{% endalert %}
