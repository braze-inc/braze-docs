# BrazeAI Decisioning Studio™ Pro

## Sobre o Decisioning Studio Pro

O Decisioning Studio Pro é mais flexível e mais poderoso do que o Decisioning Studio Go. O Pro inclui personalização completa do banco de ações, a capacidade de otimizar qualquer métrica (como receita, conversões ou engajamento) e é compatível com qualquer plataforma de engajamento com clientes (CEP).

A configuração de um agente Pro requer a colaboração com nossa equipe de Serviços de Decisão de IA — a equipe de ciência de dados avançada da Braze — que orienta e auxilia no projeto do agente para gerar o máximo valor para o seu negócio.

Para saber mais sobre as diferenças entre o Decisioning Studio Go e o Decisioning Studio Pro, consulte [Decisioning Studio Go versus Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro).
