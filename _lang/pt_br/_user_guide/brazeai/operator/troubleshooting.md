# Solução de problemas do BrazeAI Operator

> Problemas comuns e soluções para o BrazeAI Operator<sup>TM</sup>.

## Problemas comuns

| Problema | Solução |
| --- | --- |
| Sem resposta | Atualize a página e abra o painel do Operator novamente. |
| Respostas fora do tema | Formule a pergunta de forma mais específica. Mencione a funcionalidade ou o fluxo de trabalho em questão. |
| Mensagens de erro | Se o Operator não conseguir transmitir conteúdo, pode aparecer o prompt "Tentar novamente". O Operator pode estar temporariamente indisponível ou a conexão pode ter sido interrompida. Tente novamente em alguns minutos. |
| Limite de uso diário atingido | O limite de uso em toda a empresa foi atingido. Aguarde até o limite ser redefinido (a cada 24 horas). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Limitações

- **Acesso a dados:** O Operator pode ver o conteúdo da página ativa. Ele também pode buscar alguns dados específicos da empresa, como campanhas passadas, catálogos e atributos personalizados. No momento, porém, ele não pode criar novos relatórios nem executar consultas de relatórios.
- **Limites de uso:** O Operator está sujeito a um limite de uso diário em toda a empresa, redefinido a cada 24 horas. Quando o limite é atingido, a mensagem "Limite de uso diário atingido" aparece no chat e nenhuma solicitação adicional é possível até a redefinição.
