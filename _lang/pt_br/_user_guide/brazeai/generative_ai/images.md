# Geração de imagens com o <sup>BrazeAITM</sup>

> {% multi_lang_include brazeai/generative_ai/about_images.md %}

## Sobre a biblioteca de mídia

A biblioteca de mídia permite que você gerencie seus ativos em um único local centralizado. Para acessar sua biblioteca de mídia do Braze, vá para **Templates** > **Media Library**. Para obter informações mais detalhadas, consulte [Biblioteca de mídia]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

## Geração de uma imagem

1. Vá para a biblioteca de mídia e selecione <i class="fas fa-wand-magic-sparkles"></i> **AI Image Generator**.
2. Digite uma descrição da imagem que você deseja gerar, com até 300 caracteres. Quanto mais detalhada for a descrição, melhor será o resultado. Esse recurso só suporta entrada de texto - não é possível carregar uma imagem como referência.
3. Selecione **Gerar imagens**. As imagens podem levar cerca de um minuto para serem geradas.
4. Selecionar <i class="fas fa-download" title="Add image to Media Library (Adicionar imagem à biblioteca de mídia)"></i> nas imagens que deseja adicionar à sua biblioteca de mídia.
