# Gere imagens com o BrazeAI

> {% multi_lang_include brazeai/generative_ai/about_images.md %}

## Sobre a biblioteca de mídia

A biblioteca de mídia permite que você gerencie seus ativos em um único local centralizado. Para acessar sua biblioteca de mídia Braze, vá para **Modelos** > **Biblioteca de mídia**. Para saber mais, consulte [Biblioteca de mídia]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

## Gerando uma imagem

1. Acesse sua biblioteca de mídia e selecione<i class="fas fa-wand-magic-sparkles"></i>**IA Image Generator**.
2. Digite uma descrição da imagem que você deseja gerar, com até 300 caracteres. Quanto mais detalhada a descrição, melhor será o seu resultado. Este recurso suporta apenas entrada de texto — não é possível fazer upload de uma imagem como referência.
3. Selecione **Gerar Imagens**. Pode levar cerca de um minuto para as imagens serem geradas.
4. Selecionar <i class="fas fa-download" title="Add image to Media Library (Adicionar imagem à biblioteca de mídia)"></i> nas imagens que deseja adicionar à sua biblioteca de mídia.
