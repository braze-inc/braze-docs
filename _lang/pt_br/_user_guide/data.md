### Fazendo com que os dados do cliente trabalhem mais

- **Flexível e baseado em componentes:** Nosso objetivo principal é ajudar as marcas a utilizar seus dados de forma mais eficaz e completa. Criadas com uma arquitetura composta, as marcas podem aproveitar as tecnologias necessárias para fazer com que seus dados trabalhem mais, sem middleware desnecessário. 
- **Integrações de parceiros:** Priorizamos integrações com as melhores tecnologias de ecossistema (e oferecemos APIs) que facilitam o compartilhamento bidirecional de dados em tempo real.
- **Arquitetura de processamento de fluxo:** Os clientes podem acionar ações em qualquer ponto de dados ingerido no Braze para segmentação, orquestração e personalização.

### Aumentar a agilidade dos dados para impulsionar o desempenho 

- **Construção fácil para o público:** Reduzir a dependência de equipes técnicas para criar públicos e oferecer interação personalizada com o cliente em escala. 
- **Velocidade e desempenho:** Os dados e os insights de engajamento são fornecidos em tempo real, o que dá suporte ao engajamento iterativo e eficaz do cliente, bem como à tomada de decisões comerciais mais amplas.

### Manter os dados das marcas seguros, protegidos e em conformidade

- **Práticas de segurança líderes do setor:** Realizar auditorias regulares de terceiros, incluindo SOC 2 Tipo 2 e ISO 27001, para cumprir os mais altos padrões do setor. Manter um programa público de recompensa por bugs para lidar proativamente com possíveis vulnerabilidades e ter uma equipe de segurança dedicada e comprometida com a proteção dos dados dos clientes.
- **Conformidade com o setor:** Fornecer ferramentas que promovam a adesão às normas de proteção de dados, incluindo GDPR e CCPA.
- **Privacidade de dados:** Permita que os clientes gerenciem o consentimento do usuário final, processem solicitações e acionem os direitos do consumidor.
