# Exportar dados do Canvas

> Os dados de usuários podem ser exportados para um arquivo CSV. Esta página aborda como exportar dados de todo o Canvas ou de um componente específico do Canvas.

## Exportação de dados para um Canva

Para exportar dados para um canva, faça o seguinte:

1. Acesse **Envio de mensagens** > **Canvas** e selecione seu Canvas.
2. Selecione o menu suspenso **Dados de usuários** na seção **Detalhes do Canva**. 
3. Selecione uma das seguintes opções de exportação:
  - **Exportar dados de usuários via CSV** ou
  - **Exportar endereço de e-mail via CSV**.

Também é possível exportar os dados de usuários de todos os participantes de um Canva como um arquivo CSV.

## Exportação de dados para um componente (somente no fluxo de trabalho original)

Os resultados do Canvas podem ser exportados em uma base de componente individual para o fluxo de trabalho original do Canvas. Para fazer isso, selecione o componente específico e, em seguida, selecione o menu suspenso **Dados de usuários** na página **Detalhes da etapa do canva**.
