# Exportar dados do Canvas

> Os dados do usuário podem ser exportados para um arquivo CSV. Esta página aborda como exportar dados de todo o seu Canvas ou de um componente específico do Canvas.

## Exportação de dados para um Canvas

Para exportar dados de um Canvas, faça o seguinte:

1. Vá para **Messaging** > **Canvas** e selecione seu Canvas.
2. Selecione o menu suspenso **Dados do usuário** na seção **Detalhes do Canvas**. 
3. Selecione uma das seguintes opções de exportação:
  - **CSV Exportar dados do usuário** ou
  - **Exportação CSV Endereço de e-mail**.

Também é possível exportar os dados de usuário de todos os participantes de um Canvas como um arquivo CSV.

## Exportação de dados para um componente (somente fluxo de trabalho original)

Os resultados do Canvas podem ser exportados em uma base de componente individual para o fluxo de trabalho original do Canvas. Para fazer isso, selecione o componente específico e, em seguida, selecione o menu suspenso **Dados do usuário** na página **Detalhes da etapa do Canvas**.
