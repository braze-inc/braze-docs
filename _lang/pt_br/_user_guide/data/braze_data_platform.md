# Visão geral da Braze Data Platform

> A Braze Data Platform é um conjunto abrangente e criador de recursos de dados e integrações com parceiros que permite que você crie experiências personalizadas e impactantes em todo o ciclo de vida do cliente com base em três tarefas relacionadas a dados a serem realizadas: [Unificação de dados]({{site.baseurl}}/user_guide/data/unification), [ativação de dados]({{site.baseurl}}/user_guide/data/activation) e [distribuição de dados]({{site.baseurl}}/user_guide/data/distribution).

Ao usar uma combinação de recursos na plataforma de dados do Braze, você pode aproveitar seus dados para criar mensagens significativas e direcionadas que respondam ao que seus clientes fazem em tempo real.

## Unifique seus dados 

Colete e consolide dados primários de qualquer fonte usando [APIs]({{site.baseurl}}/api/home) e [SDKs]({{site.baseurl}}/developer_guide/sdk_integration). Você também pode usar ferramentas de ingestão incorporadas, como o [Cloud Data Ingestion]({{site.baseurl}}/user_guide/data/unification/cloud_ingestion), para criar uma integração direta de seu data warehouse ou solução de armazenamento de arquivos para o Braze, ou usar o [Data Transformation]({{site.baseurl}}/user_guide/data/unification/data_transformation) para criar e gerenciar integrações de webhook para transferir dados para o Braze.

## Ativar seus dados

Limpe, organize e prepare seus dados para uso. Isso envolve compreender os comportamentos e as preferências dos clientes em tempo real com perfis e segmentos de usuários. Consulte o [glossário de métricas de relatórios]({{site.baseurl}}/user_guide/data/activation/report_metrics) ao criar mensagens direcionadas para identificar como seus clientes estão respondendo a essas experiências personalizadas.

## Distribua seus dados

Envie e exporte seus dados para sistemas externos para obter insights e tomar decisões nas próximas etapas. Use o [Braze Currents]({{site.baseurl}}/user_guide/data/distribution/braze_currents) para enviar dados de eventos do Braze para um data warehouse para alimentar ferramentas de business intelligence.
