# Visão geral da Braze Data Platform

> A Braze Data Platform é um conjunto de recursos de dados e integrações de parceiros abrangentes e compostáveis que permitem que você crie experiências personalizadas e impactantes em todo o ciclo de vida do cliente com base em três tarefas relacionadas a dados a serem realizadas: [Unificação de dados]({{site.baseurl}}/user_guide/data/unification), [ativação de dados]({{site.baseurl}}/user_guide/data/activation) e [distribuição de dados]({{site.baseurl}}/user_guide/data/distribution).

Ao usar uma combinação de recursos na Braze Data Platform, você pode aproveitar seus dados para criar mensagens significativas e direcionadas que respondam ao que seus clientes fazem em tempo real.

## Unifique seus dados 

Colete e consolide dados primários de qualquer fonte, usando [APIs] e [SDKs]. Você também pode aproveitar nossas ferramentas de ingestão incorporadas, como a [Ingestão de dados na nuvem]({{site.baseurl}}/user_guide/data/unification/cloud_ingestion), para criar uma integração simples diretamente de seu data warehouse ou solução de armazenamento de arquivos para o Braze, ou usar [a Transformação de dados]({{site.baseurl}}/user_guide/data/unification/data_transformation) para criar e gerenciar integrações de webhook para coletar seus dados e transferi-los para o Braze.

## Ativar seus dados

Limpe, organize e prepare seus dados para uso. Isso envolve compreender os comportamentos e as preferências dos clientes em tempo real com perfis e segmentos de usuários. Consulte o [glossário de métricas de relatórios]({{site.baseurl}}/user_guide/data/activation/report_metrics) ao criar mensagens direcionadas para identificar como seus clientes estão respondendo a essas experiências personalizadas.

## Distribua seus dados

Transmita e exporte seus dados para sistemas externos para obter insights e tomar decisões nas próximas etapas. Use o [Braze Currents]({{site.baseurl}}/user_guide/data/distribution/braze_currents) para transmitir dados de eventos do Braze em um data warehouse para alimentar ferramentas de business intelligence.
