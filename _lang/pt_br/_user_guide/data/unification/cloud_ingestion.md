guide_top_text: "<h2>O que é?</h2>A Ingestão de Dados em Nuvem Braze (CDI) permite que você configure uma conexão direta entre sua solução de armazenamento de dados e o Braze para sincronizar dados relevantes de usuários ou catálogos e excluir usuários. Quando sincronizados com o Braze, esses dados podem ser utilizados para casos de uso como personalização ou segmentação. A integração flexível da Ingestão de Dados em Nuvem suporta estruturas de dados complexas, incluindo JSON aninhado e arrays de objetos. <br><br>**Capacidades da Ingestão de Dados em Nuvem Braze:**<br> - Crie uma integração simples diretamente do seu armazém de dados ou solução de armazenamento de arquivos para o Braze em apenas alguns minutos.<br>- Sincronize com segurança os dados dos usuários, incluindo atributos, eventos e compras do seu armazém de dados para o Braze.<br>- Feche o ciclo de dados com o Braze combinando a Ingestão de Dados em Nuvem com Currents ou Compartilhamento de Dados do Snowflake.<br><br>**A Ingestão de Dados em Nuvem pode sincronizar dados de:**<br> - Amazon Redshift<br> - Databricks<br> - Google BigQuery<br> - Microsoft Fabric<br> - S3<br> - Snowflake"
