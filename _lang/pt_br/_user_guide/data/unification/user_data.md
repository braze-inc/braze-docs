{% alert important %}
O Braze banirá ou bloqueará os usuários ("usuários fictícios") com mais de 5 milhões de sessões e não mais ingerirá seus eventos de SDK porque eles geralmente são o resultado de uma integração incorreta. Se você perceber que isso aconteceu com um usuário legítimo, entre em contato com o gerente da sua conta Braze.
{% endalert %}
