{% alert important %}
O Braze banirá ou bloqueará os usuários ("usuários fictícios") com mais de 5 milhões de sessões e não ingerirá mais seus eventos SDK porque eles geralmente são o resultado de uma integração incorreta. Se isso acontecer com um usuário legítimo, entre em contato com o gerente da sua conta Braze.
{% endalert %}
