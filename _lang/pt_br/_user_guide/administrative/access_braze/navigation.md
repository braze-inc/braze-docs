# Navegando no Braze

A navegação do Braze foi projetada para ajudá-lo a acessar com eficiência os recursos e o conteúdo em todos os dispositivos. Vamos explorar como tirar o máximo proveito disso.

{% alert note %}
Essa atualização simplifica sua experiência ao fazer a transição de todos os usuários para uma navegação unificada. A opção de alternar entre as versões de navegação não está mais disponível.
{% endalert %}

## Usando o menu

A estrutura de menu vertical mantém suas ferramentas mais usadas a apenas um toque de distância. Selecione um item do menu principal para revelar suas opções em um layout vertical empilhado. 

O usuário seleciona a página "Settings" (Configurações) para abrir o submenu como um menu vertical empilhado à direita.]({% image_buster /assets/img/navigation/vertical_menu.gif %})

## Pesquisa global

A busca global, localizada no cabeçalho, é a maneira mais rápida de encontrar qualquer coisa no Braze. Selecione a barra de pesquisa ou pressione Ctrl + K no Windows ou ⌘ + K no Mac para ir diretamente para lá. 

\![Pesquisa global aberta sem termos de pesquisa inseridos, mostrando as páginas abertas recentemente.]({% image_buster /assets/img/navigation/search_recently_opened.png %})

A pesquisa compreende sinônimos para páginas que foram renomeadas em nossa navegação atualizada. Por exemplo, ele encontrará "Data Export" (Exportação de dados) quando você pesquisar "Currents" (Correntes), pois essa página foi renomeada.

\![Resultados da pesquisa "currents" exibidos: "Exportação de dados. Anteriormente, 'Currents'".]({% image_buster /assets/img/navigation/global_search_synonym.png %})

## Minimizar a navegação lateral

Para reduzir a desordem visual e se concentrar no espaço de trabalho, você pode minimizar o painel de navegação lateral. Pressione **Minimizar menu** para ocultar a navegação, o que é especialmente útil ao trabalhar no Canvas para projetar fluxos de usuário. 

\![O usuário expande e recolhe o menu de navegação do lado esquerdo no Braze Canvas, alternando entre rótulos de texto completo e visualização somente de ícones.]({% image_buster /assets/img/navigation/expand_collapse_menu.gif %})

Mesmo quando minimizado, você pode passar o mouse sobre os ícones para ver dicas de ferramentas com nomes de itens de menu para acesso rápido.

\![Navegação do Braze minimizada, mostrando uma dica de ferramenta para "Audience" quando o foco é movido para o item de menu Audience.]({% image_buster /assets/img/navigation/minimized_menu_tooltip.png %}){:style="max-width:50%"}

## Navegação responsiva

A navegação se ajusta perfeitamente aos tamanhos dos dispositivos. Em telas menores, a navegação lateral é recolhida automaticamente. Pressione <i class="fa-solid fa-bars" aria-label="Abrir menu de navegação"></i> para abrir o menu quando necessário. 

\![Em telas menores, a navegação lateral é recolhida automaticamente. Tocar no ícone do menu abre as opções de navegação.]({% image_buster /assets/img/navigation/navigation_small_screens.png %})
