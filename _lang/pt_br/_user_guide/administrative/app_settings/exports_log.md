# Registro de exportação

> Use a página **Registro de exportações** para visualizar o status dos trabalhos de exportação e cancelar as exportações em andamento diretamente da plataforma Braze. <br><br> Atualmente, o registro de exportações suporta apenas exportações de segmentos. Para obter informações sobre a ferramenta de exportação de segmentos, consulte [Exportação de dados de segmentos para CSV]({{site.baseurl}}/user_guide/data/export_braze_data/segment_data_to_csv/).

{% alert important %} O registro de exportações está em acesso antecipado. Se estiver interessado em participar do acesso antecipado, entre em contato com seu gerente de sucesso do cliente. {% endalert %}

É possível encontrar o registro de exportações em **Configurações** > **Registro de exportações**. Aqui, você pode visualizar as exportações anteriores, o nome dos segmentos exportados, o status de cada exportação, a origem de cada exportação e quando cada exportação foi iniciada e concluída. 

\![O registro de exportação com uma lista de exportações concluídas]\[1].

## Cancelamento de uma exportação pendente

É possível cancelar exportações pendentes diretamente na página **Registro de exportações**, selecionando o menu <i class="fas fa-ellipsis-vertical"></i> e, em seguida, **Cancelar exportação**, ou selecionando o **ID de exportação** e, em seguida, **Cancelar exportação** na página da exportação.

\![O registro de exportação mostra uma exportação pendente e a opção "Cancelar exportação"]\[2].

## Compartilhamento de um registro de exportação específico

Compartilhe um registro de exportação selecionando o **ID de exportação** e, em seguida, selecionando **Compartilhar registro**.

\![Uma janela suspensa com o link para compartilhar a página]\[3]{: style="max-width:45%;"}

{% alert note %} Se tiver comentários sobre o registro de exportações, sinta-se à vontade para compartilhá-los com seu gerente de sucesso do cliente. {% endalert %}
