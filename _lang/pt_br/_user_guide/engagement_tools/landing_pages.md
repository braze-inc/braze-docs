# Sobre as páginas de destino

> As páginas de destino do Braze são páginas da Web independentes que podem impulsionar sua estratégia de aquisição e engajamento de usuários.

Use as páginas de destino para aumentar seu público, capturar dados do usuário, promover ofertas especiais e apoiar campanhas multicanais.

{% alert note %}
A disponibilidade da página de destino e do domínio personalizado depende de seu pacote Braze. Entre em contato com seu gerente de conta ou gerente de sucesso do cliente para começar.
{% endalert %}

## Pré-requisitos

Para poder acessar, criar e publicar landing pages, você precisa de [permissões]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/user_permissions/#list-of-permissions) de administrador ou de todas as permissões a seguir:

- Acessar páginas de destino
- Criar rascunhos de páginas de destino
- Publicar páginas de destino

## Perguntas frequentes

### Qual é o tamanho máximo das páginas de destino?

O tamanho do corpo da página de destino pode ser de até 500 KB.

### Há algum requisito técnico para publicar uma landing page?

Não, não há requisitos técnicos.

### Existe um editor de HTML para páginas de destino?

Sim. Use o bloco **Código personalizado** no editor de arrastar e soltar para adicionar ou editar HTML.

### Posso criar um webhook dentro de uma landing page?

Não, isso não é suportado no momento.
