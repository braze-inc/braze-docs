# Calendário de conteúdo

> O calendário de conteúdo oferece uma visão geral de alto nível para suas próximas campanhas agendadas.

Você pode analisar dados de envio de mensagens, bem como visualizar e gerenciar suas campanhas agendadas. Para abrir o calendário, selecione **envio de mensagens** > **Calendário de Conteúdo**.

{% alert note %}
Se você estiver usando a [navegação mais antiga]({{site.baseurl}}/navigation), selecione **engajamento** > **Campanhas** > **Calendário**.
{% endalert %}

## análise de dados de envio de mensagens

A seção de análise de dados de envio de mensagens contém dois relatórios para suas campanhas agendadas recentes e futuras:

- **Próximos Tipos de Mensagens:** O número de push, e-mail e campanhas de envio de mensagens no app agendadas para os próximos 14 dias.
- **Mensagens enviadas:** O número de mensagens enviadas e abertas nos últimos 14 dias.

Você pode visualizar a análise de dados de envio de mensagens para um dia específico em um relatório, colocando o cursor sobre esse dia no relatório.

![Os gráficos 'Próximos Tipos de Mensagens' e 'Mensagens Enviadas' dentro da análise de dados de envio de mensagens.]({% image_buster /assets/img/campaign_calendar/content_calendar_messaging_analytics.png %})

Para baixar uma cópia dos seus dados, selecione <i class="fa-solid fa-bars" style="color: #2e7487;"></i> **Menu de contexto do gráfico**, depois escolha o formato de arquivo de sua preferência.

## Enviar calendário

O calendário de envio exibe todas as suas campanhas agendadas para o mês atual.

{% alert note %}
Mensagens no app são sempre baseadas em ações, então você não verá nenhuma campanha de mensagem no app no calendário de envio.
{% endalert %}

Você pode mudar o mês selecionando <i class="fa-solid fa-chevron-left" style="color: #2e7487;"></i> **Ver mês anterior** ou <i class="fa-solid fa-chevron-right" style="color: #2e7487;"></i> **Ver próximo mês**. Para voltar ao mês atual, selecione **Hoje**.

![O Calendário de Envio exibindo todas as campanhas agendadas no mês atual.]({% image_buster /assets/img/campaign_calendar/content_calendar_sends.png %})

Cada cor no calendário representa um tipo diferente de campanha agendada:

- **Vermelho:** Campanhas agendadas de canal único.
- **Azul:** Campanhas agendadas multicanal.
- **Ouro:** Campanhas paradas, que estão incluídas no calendário para referência histórica.

Você pode visualizar ou modificar suas campanhas selecionando a campanha no calendário. Para saber mais, veja [Gerenciando campanhas]({{site.baseurl}}/user_guide/engagement_tools/campaigns/managing_campaigns).
