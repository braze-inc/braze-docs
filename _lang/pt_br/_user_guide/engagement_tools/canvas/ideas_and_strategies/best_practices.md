# Práticas recomendadas do Canva

> Este artigo fornece algumas práticas recomendadas para criar e personalizar as jornadas do usuário com o Canvas e o Canvas Flow.

## Identifique seu objetivo

Mergulhe no que, quem e por quê!
- O que você está tentando ajudar os usuários a realizar?
- Quem são os usuários que você está tentando alcançar?
- Por que está construindo esse Canva?

## Misturar e combinar

Desbloqueie novas combinações de jornadas de usuários com os [componentes do Canva]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/about/).
- Divida seus usuários com a [divisão de decisão]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/decision_split/) e crie fluxos de trabalho diferentes.
- Espace suas jornadas de usuário com uma etapa [de postergação]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/delay_step/).
- Adicione [mensagens autônomas]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/message_step/) em qualquer lugar que desejar em seu fluxo do Canvas. 

## Crie envios de mensagens mais ricos

Atraia seus usuários com mensagens mais ricas.

- Crie [mensagens no app]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/canvas_by_channel/in-app_messages_in_canvas/) para integração de Canvas para aproveitar ao máximo sua primeira impressão.
- Introduza [cartões de conteúdo]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/canvas_by_channel/content-cards_in_canvas/) em uma jornada do canva para ofertas promocionais e notificações por push.

## Teste suas jornadas de usuário

Determine o impacto do envio de mensagens do canva incorporando grupos de controle. Dessa forma, você poderá entender como seu Canva foi recebido!

- Nomeie cada etapa do seu Canva para identificar a jornada do usuário.
- Aproveite o componente [Experiment Paths (Caminhos experimentais]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/experiment_step/) ) na jornada do usuário para atribuir aleatoriamente aos usuários diferentes caminhos que você criar. 
- Diversifique suas jornadas de usuário com etapas de postergação e mensagem para ajudar a descobrir qual jornada é mais eficaz.
- Verifique [a análise de dados do Canva]({{site.baseurl}}/user_guide/engagement_tools/canvas/testing_canvases/measuring_and_testing_with_canvas_analytics/) para ver a performance de cada componente na jornada do usuário.
- [Edite sua tela]({{site.baseurl}}/post-launch_edits/) após o lançamento inicial.

## Agendando seus canvas

{% alert note %}
O Canvas impedirá que você use o envio agendado com um horário que já passou. No entanto, é possível lançar um Canvas exatamente no mesmo minuto em que a campanha está agendada (ou nos segundos anteriores). Isso pode fazer com que o Canvas perca o horário de entrada agendado e os usuários não entrem no Canvas. Recomendamos enviar Canvases imediatamente no caso de qualquer campanha ser editada dentro de minutos do horário de envio agendado.
{% endalert %}

Para etapas do Canvas, considere os seguintes detalhes ao agendar seu Canvas:

- Mudanças de agendamento só se aplicarão a usuários que não estão esperando para receber a etapa.
- Mudanças de público, por padrão, se aplicam a todos os usuários, a menos que você agende mudanças para se aplicarem a usuários que não estão esperando para receber a etapa.
- Editar um Canvas que está agendado para ser entregue assim que implantado e selecionar **Atualizar** fará com que ele seja enviado.
