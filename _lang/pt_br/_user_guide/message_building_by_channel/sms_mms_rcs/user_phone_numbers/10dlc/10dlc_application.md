# Informações necessárias para o aplicativo 10DLC

Se você comprou ajuda da Braze para registrar seu 10DLC, precisará nos enviar todas as informações necessárias listadas no e-mail abaixo, conforme acordado em seu contrato conosco.

Inclua todas as capturas de tela e anexos relevantes e envie tudo para [smsapplications@braze.com](mailto:smsapplications@braze.com). Se tiver alguma dúvida sobre o 10DLC, consulte o [guia do Braze SMS 10DLC]({{site.baseurl}}/assets/pdf/SMS_10DLC_Guide.pdf) ou entre em contato com o gerente de sucesso do cliente Braze.

## Informações sobre a campanha de SMS

Será necessário registrar cada grupo de assinatura de SMS que contenha um código longo dos EUA com sua própria campanha A2P 10DLC (às vezes chamada de caso de uso). Forneça as informações abaixo uma vez para cada um de seus grupos de assinatura de SMS que contenham um código longo dos EUA.	

| Campo | Opções (se aplicável) | Descrição |
| ------ | ------------------------ | --------------- |
| Caso de uso da campanha (escolha o que melhor se encaixa) | {::nomarkdown} <ul><li>Marketing</li><li>Notificações de conta</li><li>Notificações de entrega</li><li>Atendimento ao cliente</li><li>Alerta de segurança</li><li>Mensagens de alerta de fraude</li><li>Ensino superior</li><li>Misto</li><li>Pesquisa de opinião e votação</li><li>Anúncio de serviço público</li></ul>{:/} | Observação: Se você planeja usar um caso de uso "Misto" para um grupo de assinaturas, consulte seu Customer Success Manager ou Customer Onboarding Manager para obter detalhes importantes com antecedência |
| Nome da campanha | | {::nomarkdown}<p>O nome da campanha de SMS precisa ser exibido de forma consistente em suas mensagens, nos Termos de Serviço e em qualquer lugar em que os usuários se inscrevam no seu programa.</p><p>Exemplos:</p><ul><li>Alertas de marcação de brasagem</li><li>Notificações da conta Braze</li><li>Lembretes de carrinho de compras</li></ul>{:/} |
| Descrição da campanha | | Essa campanha envia mensagens promocionais e de marketing aos usuários que optam por receber SMS, incluindo ofertas/cupons especiais e notificações de lembrete de carrinho. |
| Endereço de e-mail do suporte ao cliente | | Onde os usuários finais podem entrar em contato por e-mail para obter ajuda e suporte |
| Número de telefone do suporte ao cliente | | Onde os usuários finais podem entrar em contato por telefone para obter ajuda e suporte |
| Termos de SMS & Condições URL | | {::nomarkdown} <p>Para concluir o registro US A2P 10DLC, sua página de termos de SMS deve incluir o seguinte:</p><ol><li>Nome do programa (marca)</li><li>Divulgação consistente do produto no site e no fluxo de mensagens</li><li>Frequência de mensagens (consistente com o fluxo de mensagens)</li><li>"Isenção de responsabilidade "Taxas de mensagens e dados podem ser aplicadas</li><li>Instruções de HELP/STOP OU informações de contato do atendimento ao cliente</li></ol><p>Se essa página não estiver ativa ou se alguma das divulgações exigidas estiver faltando, será necessário um rascunho dos Termos de SMS juntamente com a data prevista de implementação na sua página ativa para concluir o processo de registro.</p>{:/} |
| Como os usuários finais aderem à sua campanha de SMS? (Compartilhe como os usuários finais podem optar pelo seu programa de SMS, por exemplo, como os usuários finais podem dar consentimento para receber suas mensagens SMS. Se forem usados vários métodos de opt-in, todos eles deverão ser listados aqui). | | {::nomarkdown}<p>Exemplos:</p><ul><li>Os usuários finais optam por participar enviando uma mensagem de texto START para XXXXXXXXXXXX.</li><li>Os usuários finais optam por participar visitando o site e adicionando seu número de celular. Em seguida, eles marcam uma caixa concordando em receber SMS.</li><li>Se o método de opt-in escolhido ainda não estiver ativo. Forneça um modelo/uma captura de tela do método de opt-in.</li></ul>{:/} |
| Palavras-chave Opt-In | | {::nomarkdown} <p>Se os usuários finais puderem digitar uma palavra-chave para começar a receber SMS dessa campanha, essas palavras-chave deverão ser fornecidas. Por padrão, o Braze configura as seguintes palavras-chave:</p><ul><li>INICIAR</li><li>UNSTOP</li><li>SIM</li></ul><p>Informe-nos caso deseje adicionar alguma palavra-chave.</p> {:/} |
| Mensagem de confirmação de opt-in | | A mensagem de resposta automática enviada aos usuários finais para confirmar a adesão deve ser fornecida. A resposta de opt-in deve incluir o nome da marca, a confirmação do registro de opt-in em uma campanha de mensagens recorrentes, como obter ajuda e uma descrição clara de como optar por não participar. (por exemplo, Obrigado por participar do Braze Marketing Alerts! A frequência das mensagens varia. Podem ser aplicadas tarifas de mensagens e dados. Responda HELP para obter ajuda ou STOP para não participar). |
| Palavras-chave de exclusão | | {::nomarkdown} <p>Os usuários finais devem ser capazes de digitar uma palavra-chave para parar de receber mensagens dessa campanha. Por padrão, o Braze configura as seguintes palavras-chave:</p><ul>  <li>PARAR</li><li>PARAR</li><li>CANCELAR A INSCRIÇÃO</li><li>CANCELAR</li><li>FIM</li><li>SAIR</li></ul><p>Informe-nos caso deseje adicionar alguma palavra-chave.</p>{:/} |
| Mensagem de exclusão | | Ao receber as palavras-chave de cancelamento dos usuários finais, as operadoras esperam uma resposta gerada automaticamente, que deve fornecer a confirmação da solicitação de cancelamento e a confirmação de que nenhuma outra mensagem será enviada. Recomenda-se também que essas mensagens de recusa incluam o nome da marca (por exemplo, Braze Marketing Alerts): Você cancelou a assinatura de todas as mensagens. Responder HELP para obter ajuda). <br><br>Máximo de 320 caracteres|
| Palavras-chave de ajuda | | Os usuários finais devem ser capazes de digitar uma palavra-chave para receber ajuda. Por padrão, o Braze configura as seguintes palavras-chave: -HELP -INFO Entre em contato conosco se desejar adicionar alguma palavra-chave. |
| Mensagem de ajuda | | Ao receber as palavras-chave de exclusão dos usuários finais, as operadoras esperam uma resposta gerada automaticamente, que pode incluir o nome da marca e informações adicionais de contato de suporte (por exemplo, Braze Marketing Alerts): Para obter ajuda, ligue para 555-555-555. Msg&data tarifas podem ser aplicadas. A frequência da mensagem pode variar, envie uma mensagem de texto STOP para cancelar)<br><br>Máximo de 320 caracteres|
| Mensagem de exemplo 1 de 2 | | Atualizações de clientes/marcas: Olá e bem-vindo ao Customer Brand Updates. Visite nosso site em www.customerwebsite.com. Digite HELP para obter ajuda ou STOP para optar por não participar. |
| Mensagem de exemplo 2 de 2 | | Atualizações de clientes/marcas: Confira nossa última promoção de fim de ano com 20% de desconto em todos os itens até segunda-feira. Visite nosso site em www.customerwebsite.com. Digite HELP para obter ajuda ou STOP para optar por não participar. |
| Volume diário estimado | | Indique o número médio estimado de mensagens enviadas por dia para esse caso de uso/grupo de assinatura específico, por exemplo, 10.000. |
| Lista de verificação de chamadas para ação | | {::nomarkdown} <p>A chamada para ação deve incluir o seguinte:</p><ol><li>Descrição do produto</li><li>Divulgação da frequência de mensagens</li><li>Link para os Termos e Condições completos</li><li>Link para a Política de Privacidade completa</li><li>Instruções da palavra-chave STOP</li><li>Instruções da palavra-chave HELP</li><li>"Taxas de mensagens e dados" podem ser aplicadas à divulgação</li></ol> {:/} |
