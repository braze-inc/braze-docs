## Ótimos casos de uso

Aqui estão alguns casos de uso excelentes para você se inspirar!

| Caso de uso | Explicação |
| --- | --- |
| Reengajamento | Alcance usuários fora do seu aplicativo, inclusive aqueles que não instalaram o aplicativo. |
| Integração | Integre e incentive os novos usuários a ativar as notificações push ou a compartilhar o aplicativo nas redes sociais. |
| Mensagens ricas | Permitir mensagens HTML ricas e dinâmicas. |
| Conteúdo multimídia | Facilidade de colocação de conteúdo multimídia que envolva os usuários, como vídeos e imagens. |
| Boletins informativos | Envie convenientemente boletins informativos mensais ou semanais para manter o envolvimento do usuário. |
| Transações | Notifique os usuários sobre compras recentes e forneça informações importantes sobre produtos e remessas com [e-mails transacionais]({{site.baseurl}}/user_guide/message_building_by_channel/email/transactional_message_api_campaign/).
