# Anúncio do produto

> Use o editor de arrastar e soltar de mensagens no aplicativo para aumentar a conscientização sobre o lançamento de novos produtos.

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Criação de um anúncio de produto

### Etapa 1: Escolha seu modelo

Ao criar uma mensagem in-app do tipo arrastar e soltar, selecione **Anúncio de produto** como modelo e, em seguida, selecione **Criar mensagem**. Esse modelo é compatível com aplicativos móveis e navegadores da Web.

\![O editor de mensagens no aplicativo com o modelo para um anúncio de produto.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement.png %})

### Etapa 2: Configure seus estilos de mensagem

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Etapa 3: Personalize seu componente de botão

Para começar a criar seu anúncio de produto, selecione o componente de botão no editor. Em seguida, use o menu lateral para selecionar para onde os usuários serão levados quando selecionarem o botão. Por padrão, o modelo fecha a mensagem, mas você pode optar por navegar para uma página específica em seu aplicativo (como a página do produto que está em destaque em seu anúncio).

\![O editor de mensagens no aplicativo com um menu lateral para personalizar o elemento do botão.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement_button.png %})

Também é possível adicionar mensagens adicionais ao anúncio do produto na seção **Páginas** e vinculá-las sequencialmente. Por exemplo, você pode criar uma sequência de mensagens que descrevam brevemente os recursos do produto e terminem com um botão que leve os usuários à página do produto. Saiba como fazer isso em [Conectar páginas juntas]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Etapa 4: Estilize sua mensagem

Personalize a aparência de seu anúncio de produto usando os [componentes de mensagem no aplicativo do]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) tipo arrastar e soltar.
