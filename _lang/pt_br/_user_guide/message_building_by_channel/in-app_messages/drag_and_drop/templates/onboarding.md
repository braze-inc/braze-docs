# Fluxo de integração

> Use o modelo de mensagem in-app **do fluxo de Onboarding** para coletar atributos, insights e preferências do usuário que impulsionam sua estratégia de campanha. 

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Criação de um fluxo de integração

### Etapa 1: Escolha seu modelo

Ao criar uma mensagem in-app do tipo arrastar e soltar, selecione **Fluxo de integração** para seu modelo e, em seguida, selecione **Criar mensagem**. Esse modelo é compatível com aplicativos móveis e navegadores da Web.

\![O editor de mensagens no aplicativo com o modelo de fluxo de integração.]({% image_buster /assets/img/drag_and_drop/templates/onboarding_editor.png %})

### Etapa 2: Configure seus estilos de mensagem

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Etapa 3: Personalize a ordem do seu fluxo de integração

Para começar a criar seu fluxo de integração, use a seção **Páginas** para adicionar ou excluir mensagens ao seu fluxo. Depois que você tiver o número desejado de mensagens, selecione os botões em cada página. Em seguida, use o menu lateral para selecionar para onde os usuários vão quando selecionam o botão - se eles saem do fluxo ou vão para a próxima mensagem.

Verifique se as mensagens estão conectadas em ordem sequencial para que os usuários percorram o fluxo corretamente. Para obter etapas sobre como fazer isso, consulte [Conectar páginas juntas]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Etapa 4: Estilize sua mensagem

Personalize a aparência de seu fluxo de integração usando os [componentes de mensagem no aplicativo do]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) tipo arrastar e soltar.
