# Promoção sazonal com imagem de fundo

> Use o editor de arrastar e soltar de mensagens no aplicativo para promover uma oferta sazonal ou um negócio para aumentar o envolvimento do usuário.

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Criação de uma promoção sazonal com uma imagem de fundo

### Etapa 1: Escolha seu modelo

Ao criar uma mensagem in-app do tipo arrastar e soltar, selecione **Promoção sazonal com imagem de fundo** para o seu modelo e, em seguida, selecione **Criar mensagem**. Esse modelo é compatível com aplicativos móveis e navegadores da Web.

\![O editor de mensagens no aplicativo com o modelo para uma promoção sazonal.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo.png %})

### Etapa 2: Configure seus estilos de mensagem

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Etapa 3: Personalize seu componente de botão

Para começar a criar sua promoção sazonal, selecione o componente de botão no editor. Em seguida, use o menu lateral para selecionar para onde os usuários serão levados quando selecionarem o botão. O padrão do modelo é fechar a mensagem, mas você pode optar por navegar para uma página específica do seu aplicativo (como o produto que você está promovendo).

\![O editor de mensagens no aplicativo com um menu lateral para personalizar o elemento do botão.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo_button.png %})

Também é possível adicionar mensagens adicionais ao anúncio do produto na seção **Páginas** e vinculá-las para obter um fluxo sequencial. Por exemplo, você pode criar uma sequência de mensagens que descrevam brevemente os recursos do produto e terminem com um botão que leve os usuários à página do produto. Saiba como fazer isso em [Conectar páginas juntas]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Etapa 4: Estilize sua mensagem

Personalize a aparência de sua promoção sazonal usando os [componentes de mensagem no aplicativo do]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) tipo arrastar e soltar. Adicione sua própria imagem de fundo substituindo o URL padrão da imagem de fundo no menu **do contêiner de mensagens** ou remova o URL e selecione sua imagem na [Biblioteca de mídia]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

\![O editor de mensagens no aplicativo com um menu lateral para selecionar a imagem de fundo.]({% image_buster /assets/img/drag_and_drop/templates/seasonal_promo_image.png %})
