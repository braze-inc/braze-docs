# Promoção sazonal com imagem de fundo

> Use o editor de arrastar e soltar mensagens no app para promover uma oferta sazonal ou um negócio para impulsionar o engajamento do usuário.

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Criação de uma promoção sazonal com uma imagem de fundo

### Etapa 1: Escolha seu modelo

Ao criar uma mensagem no app do tipo arrastar e soltar, selecione **Promoção sazonal com imagem de fundo** para o modelo e, em seguida, selecione **Montar mensagem**. Esse modelo é compatível com apps móveis e navegadores da Web.

![O editor de mensagens no app com o modelo para uma promoção sazonal.][img1]

### Etapa 2: Configure seus estilos de mensagens

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Etapa 3: Personalize seu componente de botão

Para começar a criar sua promoção sazonal, selecione o componente de botão no editor. Em seguida, use o menu lateral para selecionar para onde os usuários serão levados quando selecionarem o botão. 

![O editor de mensagens no app com um menu lateral para personalizar o elemento do botão.][img2]

Também é possível adicionar mensagens adicionais ao anúncio do produto na seção **Páginas** e, em seguida, vinculá-las para obter um fluxo sequencial. Por exemplo, é possível montar uma sequência de mensagens que descrevam brevemente os recursos do produto e terminem com um botão que leve os usuários à página do produto. Saiba como fazer isso em [Conectar páginas]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together).

### Etapa 4: Estilize sua mensagem

Personalize a aparência de sua promoção sazonal usando os [componentes de mensagem no app][3] do tipo arrastar e soltar. Adicione sua própria imagem de fundo substituindo o URL padrão da imagem de fundo no menu **Estrutura da mensagem** ou remova o URL e selecione sua imagem na [Biblioteca de mídia]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/).

![O editor de mensagens no app com um menu lateral para selecionar a imagem de fundo.][img3]
