# Recurso de ícone de aplicativo personalizado (iOS 10.3) 

> Com o iOS 10.3, a Apple introduziu a capacidade de alterar o ícone da tela inicial de um aplicativo sem precisar atualizar o aplicativo na Apple App Store. O desenvolvedor agora pode permitir que o usuário altere o ícone da tela inicial dentro do aplicativo. A Apple exige que todas as imagens de ícones de aplicativos que o desenvolvedor deseja disponibilizar ao usuário sejam incluídas no binário que é enviado à Apple para análise durante a publicação do aplicativo na Apple App Store.

Para notificar seus usuários sobre esse recurso, é possível enviar uma mensagem no aplicativo ou uma notificação por push por meio do Braze para o usuário, explicando essa funcionalidade ou perguntando se ele gostaria de alterar seu ícone. O desenvolvedor só precisaria criar um link profundo no aplicativo, onde o prompt nativo do iOS pode ser exibido para fazer a alteração do ícone. Isso é semelhante à mesma orientação que fornecemos sobre a configuração de uma cartilha de notificação por push para APNs atualmente.

Além disso, essas mensagens podem tirar o máximo proveito da segmentação para tornar o texto da mensagem altamente contextual para um usuário. Você também pode aproveitar o teste A/B de mensagens para ver qual mensagem causa mais impacto no resultado desejado.
