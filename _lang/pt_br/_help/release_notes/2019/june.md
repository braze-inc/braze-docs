# Junho de 2019

## Parceria Snowflake

A Braze tem o orgulho de anunciar nossa parceria com a Snowflake!

O Snowflake é um data warehouse de nuvem SQL criado especificamente para todos os seus dados e todos os seus usuários. Com a arquitetura exclusiva e patenteada da Snowflake, é fácil reunir todos os seus dados, ativar análises rápidas e obter insights orientados por dados para todos os seus usuários.

A Braze aproveita o Data Exchange da Snowflake para construir uma presença, encontrar novos clientes e expandir o alcance por meio da crescente base de clientes da Snowflake.

Saiba mais sobre essa parceria [aqui](https://www.braze.com/perspectives/article/snowflake-partner-announcement) ou em [nossa documentação]({{site.baseurl}}/partners/snowflake/).
