# Outubro de 2019

## Análise aprimorada da variante do Canva

O Canva tem análises de dados novas e aprimoradas para visualizar a performance de cada uma de suas variantes. Há novas métricas, especialmente em relação a eventos de conversão e confiança, e novos recursos como cópia de células, IDs de API e download de um `.csv` dos resultados. Leia mais sobre a [análise aprimorada da variante do Canva]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/measuring_and_testing_with_canvas_analytics/#performance-breakdown-by-variant).

## SMS

A Braze agora fornece SMS com campanhas, canvas e [currents]({{site.baseurl}}/user_guide/data/braze_currents/event_glossary/message_engagement_events/)! Confira nosso [guia de configuração]({{site.baseurl}}/user_guide/message_building_by_channel/sms_mms_rcs/sms/sms_setup/sms_sending/) para começar e nosso [guia de envio de SMS]({{site.baseurl}}/user_guide/message_building_by_channel/sms/) para saber mais!

## Melhorias no bloco de conteúdo

A API ID de um bloco de conteúdo agora será exibida na página do bloco de conteúdo selecionado no dashboard. Além disso, mostraremos onde os blocos de conteúdo estão sendo usados.

## Criação e identificação de usuários somente com alias por meio da API

Agora é possível [usar uma solicitação de API]({{site.baseurl}}/api/endpoints/user_data/#user-attributes-object-specification) com qualquer campo no objeto Attributes para criar ou atualizar um atributo desse nome com o valor fornecido no perfil de usuário especificado.
