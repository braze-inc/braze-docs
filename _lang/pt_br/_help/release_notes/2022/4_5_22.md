# Lançamento em 5 de abril de 2022

## Filtrar por campanha ou atribuição do Canva
Agora é possível filtrar os usuários que responderam a uma campanha de SMS específica ou a uma etapa do Canva, categoria de palavra-chave ou tag. Para saber mais, consulte [Redirecionamento de SMS][8].

## Painel de desempenho de e-mail
Com o [dashboard de desempenho de e-mail][1], você pode visualizar as métricas de desempenho agregadas de todo o seu canal de e-mail, tanto de campanhas quanto de Canvas, para o intervalo de datas selecionado.

Para saber mais sobre os dashboards de análise de dados disponíveis no Braze, consulte a seção [Seus Dashboards de Análise][2].

## Configurações globais de estilo

Apresentamos [as configurações de estilo global][3] para o editor de arrastar e soltar! Agora, você pode personalizar facilmente a aparência de suas campanhas de e-mail e Canvas adicionando um tema padrão, definindo o estilo básico do texto e muito mais.

## Webhooks do Braze para o Braze
Com um webhook da Braze para a Braze, você pode usar webhooks para se comunicar com a API Braze REST, fazendo tudo o que nossa API permite que você faça. Essencialmente, esse é um webhook que está se comunicando da Braze para a Braze. Para saber mais, consulte nosso artigo sobre [webhooks da Braze para a Braze][4].

## Depreciações

### SDK do Windows
Desde 24 de março de 2022, o [SDK da Braze para Windows][6] está obsoleto, e nenhum novo aplicativo para Windows pode ser criado no dashboard da Braze. 

### Integração com o Baidu push
A partir de 24 de março de 2022, a [integração de push da Braze para Baidu][7] foi descontinuada e nenhum aplicativo Baidu novo poderá ser criado no dashboard da Braze. 

## Novas parcerias Braze

### Tealium para Currents

A integração do Braze e do [Tealium][5] permite que você controle perfeitamente o fluxo de informações entre os dois sistemas. Agora, com o Currents, você também pode conectar dados ao Tealium para transformá-los em ação em todo o stack de crescimento.
