# Lançamento em 5 de abril de 2022

## Filtrar por campanha ou atribuição do Canva
Agora é possível filtrar os usuários que responderam a uma campanha de SMS específica ou a uma etapa do Canva, categoria de palavra-chave ou tag. Para saber mais, consulte [Redirecionamento de SMS]({{site.baseurl}}/user_guide/message_building_by_channel/sms/campaign/retargeting/).

## Painel de desempenho de e-mail
Com o [dashboard de desempenho de e-mail]({{site.baseurl}}/user_guide/data_and_analytics/analytics/), você pode visualizar as métricas de desempenho agregadas de todo o seu canal de e-mail, tanto de campanhas quanto de Canvas, para o intervalo de datas selecionado.

Para saber mais sobre os dashboards de análise de dados disponíveis no Braze, consulte a seção [Seus Dashboards de Análise]({{site.baseurl}}/user_guide/data_and_analytics/analytics/understanding_your_app_usage_data/).

## Configurações globais de estilo

Apresentamos as [configurações de estilo global]({{site.baseurl}}/user_guide/message_building_by_channel/email/drag_and_drop/dnd_email_style_settings/) para o editor de arrastar e soltar! Agora, você pode personalizar facilmente a aparência de suas campanhas de e-mail e Canvas adicionando um tema padrão, definindo o estilo básico do texto e muito mais.

## Webhooks do Braze para o Braze
Com um webhook da Braze para a Braze, você pode usar webhooks para se comunicar com a API Braze REST, fazendo tudo o que nossa API permite que você faça. Essencialmente, esse é um webhook que está se comunicando da Braze para a Braze. Para saber mais, consulte nosso artigo sobre [webhooks da Braze para a Braze]({{site.baseurl}}/user_guide/message_building_by_channel/webhooks/braze_to_braze_webhooks/).

## Depreciações

### SDK do Windows
Desde 24 de março de 2022, o [SDK da Braze para Windows]({{site.baseurl}}/developer_guide/sdk_integration/initialization/?sdktab=web/) está obsoleto, e nenhum novo aplicativo para Windows pode ser criado no dashboard da Braze. 

### Integração com o Baidu push
Desde 24 de março de 2022, a [integração de push da Braze para Baidu]({{site.baseurl}}/developer_guide/platform_integration_guides/android/push_notifications/android/integration/baidu_integration/) foi descontinuada e nenhum aplicativo Baidu novo poderá ser criado no dashboard da Braze. 

## Novas parcerias Braze

### Tealium para Currents

A integração do Braze e do [Tealium]({{site.baseurl}}/partners/data_and_infrastructure_agility/customer_data_platform/tealium/tealium_for_currents/) permite que você controle perfeitamente o fluxo de informações entre os dois sistemas. Agora, com o Currents, você também pode conectar dados ao Tealium para transformá-los em ação em todo o stack de crescimento.
