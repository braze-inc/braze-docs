# lançamento de 8 de fevereiro de 2022

## Jornadas de ação dos canvas

[Jornadas de ação]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/action_paths/) permitem que você classifique seus usuários com base em suas ações. Usando jornadas de ação, você pode:
- Personalize os caminhos do usuário com base em uma ação específica. 
- Mantenha os usuários por um determinado período para priorizar sua próxima jornada com base em suas ações durante este período de avaliação. 

O acesso às jornadas de ação será implementado lentamente para todas as instâncias da Braze. Entre em contato com o gerente da sua conta Braze para saber mais.

## Descontinuação da API Transifex v2 e v2.5
A partir de 7 de abril de 2022, o [Transifex]({{site.baseurl}}/partners/message_personalization/localization/transifex/#transifex) estará descontinuando suas versões de API 2 e 2.5 para abrir caminho para a versão 3. Após esta data, v2 e v2.5 não estarão mais operacionais, e as solicitações relevantes falharão. Os usuários desta integração com parceiros são instados a atualizar sua versão da API antes desta data.
