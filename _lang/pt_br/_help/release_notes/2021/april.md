# Abril de 2021

## Guia de implementação avançada do iOS push

Este guia detalhado aborda como aproveitar as extensões de app de conteúdo de notificação por push para obter o máximo de suas mensagens por push. Estão incluídos três casos de uso personalizados criados por nossa equipe (push interativo, push de captura de dados e push baseado em progresso), trechos de código que os acompanham e orientação sobre análises de registro. Confira o [guia de implementação avançada do iOS push]({{site.baseurl}}/developer_guide/push_notifications/examples/?sdktab=swift) para saber mais.

## Suporte VFC para serviço de mensagens multimídia (MMS)

Os vCards, também conhecidos como Virtual Contact Files (VCF), são um formato de arquivo padronizado para o envio de informações de negócios/contato que podem ser facilmente importadas para catálogos de endereços/contatos. Esses arquivos VFC agora podem ser enviados por [MMS]({{site.baseurl}}/user_guide/message_building_by_channel/sms_mms_rcs/mms/create/) e adicionados à biblioteca de mídia do Braze. 

## Atualizações para exclusão de usuários

Em outubro de 2020, fizemos [melhorias]({{site.baseurl}}/help/release_notes/2020/october/) na forma como nossa exclusão de usuários lida com o número de telefone ou o endereço de e-mail do titular dos dados. 

## Novas parcerias Braze

### Airbridge - Atribuição

A [integração do Airbridge e do Braze]({{site.baseurl}}/partners/message_orchestration/attribution/airbridge/) permite que você passe todos os dados de atribuição de instalação orgânica e não orgânica para o Braze para criar campanhas de marketing mais personalizadas e entender exatamente onde os usuários foram adquiridos.
### Kubit - Análise de dados

[A Kubit]({{site.baseurl}}/partners/data_and_analytics/analytics/kubit/) é uma plataforma de análise de dados sem código e de autoatendimento que oferece insights instantâneos sobre o produto. Por meio da integração perfeita e sem código com o Braze, é possível importar informações de coortes de usuários para o Braze e lançar campanhas de engajamento para direcionamento a coortes específicas. Além disso, com o uso do Snowflake Secure Data Sharing, você pode integrar a campanha bruta e os dados de impressão do Braze com a análise de produtos no Kubit para medir o impacto dessas campanhas em tempo real. 

### Census - Plataforma de dados do cliente

O [Census]({{site.baseurl}}/partners/data_and_infrastructure_agility/workflow_automation/census#census) permite manter as equipes de sucesso do cliente, vendas e marketing na mesma página, mantendo os dados do cliente sincronizados, tudo sem a ajuda contínua do departamento de engenharia.

### Treasure Data - Plataforma de dados do cliente

O [Treasure Data]({{site.baseurl}}/partners/data_and_analytics/customer_data_platform/treasure_data/) ajuda a promover experiências relevantes para os clientes, harmonizando dados, insights e engajamento para trabalhar em perfeita harmonia. Munidas de indicadores acionáveis, as equipes de CX, incluindo marketing, vendas e atendimento ao cliente, podem otimizar efetivamente os gastos e personalizar as interações omnicanal em toda a jornada do cliente. 

## Jacquard - Testes A/B

O engajamento do cliente da Braze desenvolve relacionamentos por meio do marketing multicanal. Trabalhando em conjunto com a [Jacquard]({{site.baseurl}}/partners/message_personalization/dynamic_content/content_optimization_testing/jacquard/), a Braze pode implementar a linguagem da marca, em escala, em canais personalizados de acordo com a voz de sua marca. O mecanismo de deep learning do Jacquard lida com os testes, monitora os resultados e gera uma nova linguagem com base no que aprendeu.
