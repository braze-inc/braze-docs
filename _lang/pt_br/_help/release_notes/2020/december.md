# Dezembro de 2020

## Atualizações das propriedades do evento de envio de mensagens Currents

Nos eventos de engajamento com mensagens de e-mail do Currents, a propriedade de rastreamento `ip_pool` foi adicionada. As propriedades de rastreamento `bounce_reason` e `bounce_code` também foram adicionadas aos eventos `users.messages.email.Bounce` e `users.messages.email.SoftBounce`. <br>Para obter a lista completa, consulte o [glossário de eventos de engajamento com mensagens]({{site.baseurl}}/user_guide/data/braze_currents/event_glossary/message_engagement_events/).

## Perguntas frequentes sobre previsão de churn

Um artigo de perguntas frequentes foi adicionado à documentação existente do Predictive Churn. Para saber mais sobre esses possíveis erros, esclarecimentos sobre o momento certo e considerações sobre os dados, consulte nossas [Perguntas frequentes sobre previsão de churn]({{site.baseurl}}/user_guide/brazeai/predictive_suite/predictive_churn/prediction_faq/).

## Documentos de solução de problemas de exportações CSV e API

Um documento de solução de problemas detalhando erros comuns de CSV e API foi adicionado à documentação da Braze. Para saber mais sobre esses erros, bem como sobre algumas perguntas frequentes, consulte nossa documentação [de solução de problemas de]({{site.baseurl}}/user_guide/data/export_braze_data/export_troubleshooting/) exportação CSV e API. 

## Parceria EduMe

O EduMe é uma ferramenta de treinamento baseada em dispositivos móveis que oferece à sua força de trabalho o conhecimento de que ela precisa para o sucesso. Use recurso de conteúdos conectados da Braze para dar à sua força de trabalho ou aos seus clientes acesso a lições e cursos no EduMe. Eles poderão acessar esse conteúdo sem problemas no navegador, e você poderá acompanhar o progresso deles como um grupo ou individualmente usando a funcionalidade de relatório do EduMe. Para saber mais, consulte a documentação [do EduMe]({{site.baseurl}}/partners/additional_channels_and_extensions/extensions/learning/edume/).

## Parceria com a Pypestream

A Pypestream é uma plataforma de IA conversacional full-stack que oferece envio de mensagens patenteado e completo na nuvem para transformar marcas em entidades digitais "sempre ativas". Com a parceria Braze-Pypestream, as marcas podem orquestrar com perfeição o ciclo de vida do cliente de ponta a ponta, desde o contato inicial, encaminhado para uma experiência de conversação, até o(s) acompanhamento(s) omnicanal(is) por meio de redirecionamento inteligente. Para saber mais, consulte a documentação [do Pypestream]({{site.baseurl}}/partners/data_and_infrastructure_agility/support/pypestream#pypestream).

## Parceria Dyspatch

Com o Dyspatch, use o intuitivo construtor de e-mail do tipo arrastar e soltar para criar e-mails bonitos, responsivos e envolventes sem precisar escrever código. Colabore com sua equipe para criar e aprovar e-mails no Dyspatch e, em seguida, faça upload deles para usar na Braze, tudo em apenas algumas etapas! Para saber mais, consulte a documentação [do Dyspatch]({{site.baseurl}}/partners/message_orchestration/templates/dyspatch/).

## Parceria RudderStack

O RudderStack é uma infraestrutura de dados de clientes de código aberto para coletar e rotear dados de eventos de clientes para seu data warehouse preferido e dezenas de outros provedores de análise de dados, como o Braze. Ele está pronto para empresas e oferece uma estrutura de transformação robusta para processar seus dados de eventos em tempo real. Para saber mais, consulte nossa documentação [do RudderStack]({{site.baseurl}}/partners/data_and_infrastructure_agility/customer_data_platform/rudderstack/#rudderstack).

## Parceria Jebbit

O Jebbit é uma PaaS na qual é possível criar experiências de engajamento para que os usuários capturem dados primários. A Jebbit fez uma parceria com a Braze para que você possa passar os e-mails e atribuições dos usuários de suas campanhas Jebbit como dados de usuários para a Braze em tempo real. Esses dados podem então ser utilizados para impulsionar iniciativas de marketing, como campanhas de e-mail personalizadas e disparos. Para saber mais, consulte a documentação [do Jebbit]({{site.baseurl}}/partners/additional_channels_and_extensions/extensions/surveys/jebbit).
