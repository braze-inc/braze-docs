# Dezembro de 2016

## Cabeçalhos de e-mail

Os cabeçalhos de e-mail adicionam a configuração explícita de cabeçalhos de e-mail por meio do ponto de extremidade de envio. Consulte [Práticas recomendadas de e-mail][14] para obter mais informações sobre o envio de e-mail com o Braze.

## Atualizações do Canva

Com essa atualização, é possível fazer com que uma etapa se conecte a várias etapas, ou fazer com que várias etapas se conectem em uma única etapa mesclada, bem como desenhar pontos de conexão entre elas. Por exemplo, os usuários que fizeram uma compra podem seguir um fluxo diferente dos usuários que não fizeram. Para saber mais sobre os canvas, consulte [Criação de uma ramificação][15].
