# Fevereiro de 2017

## Adicione etapas de canva após o lançamento

Agora você pode [adicionar etapas de canva]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/#creating-a-canvas) às etapas existentes, mesmo após o lançamento do canva.

## CSV em massa para Relatórios de Engajamento

Agora você pode exportar dados de campanha CSV em massa com [Relatórios de Engajamento]({{site.baseurl}}/user_guide/data_and_analytics/reporting/engagement_reports/#engagement-reports). Você pode exportar dados da canva por meio de Relatórios de engajamento também.

## Filtrar pela última mensagem no app recebida

Para um direcionamento de usuário mais personalizado, você pode agora usar "Última Mensagem no App Recebida" como um filtro baseado no tempo. Para saber mais sobre o direcionamento de usuários com a Braze, visite [Segmentação]({{site.baseurl}}/user_guide/engagement_tools/segments/creating_a_segment/).

## fazer upgrade do registro de usuários de eventos

Agora é mais fácil para você ver o que está acontecendo na sua integração de SDK com nossos [logs de usuário de eventos]({{site.baseurl}}/user_guide/administrative/app_settings/developer_console/event_user_log_tab/#event-user-log-tab) atualizados.
