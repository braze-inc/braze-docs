# Editor de HTML atualizado para e-mail

> O editor de HTML atualizado para envio de e-mail apresenta uma série de melhorias na interface do usuário, incluindo uma melhor experiência de edição de código. 

Com o editor de HTML atualizado, você pode alternar facilmente entre os editores (HTML, clássico, texto simples e AMP) com a navegação superior do editor. Essas opções estavam anteriormente ocultas em um menu suspenso no antigo editor de HTML. Esse editor de HTML atualizado também cria uma experiência consistente em nossos editores em outros canais de envio de mensagens, permitindo que o Braze desenvolva com mais eficiência melhorias universais para esses editores no futuro.

![][1]

{% alert important %}
Para acessar a guia **Gerenciamento de links** no editor de e-mail HTML atualizado, é necessário ativar [o alias de links]({{site.baseurl}}/user_guide/message_building_by_channel/email/templates/link_aliasing/). Para ativar o aliasing de links, entre em contato com o gerente da sua conta.
{% endalert %}

Embora você ainda possa mudar para o editor de HTML antigo clicando em **Mudar para o editor de HTML antigo**, note que o editor de HTML antigo será removido no início de 2024.
