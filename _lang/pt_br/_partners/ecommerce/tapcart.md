# Tapcart

> [A Tapcart](https://www.tapcart.com/) é uma plataforma de comércio móvel líder para marcas com tecnologia Shopify, ativando os comerciantes para criar aplicativos móveis personalizados que proporcionam experiências de compras personalizadas e engajadoras que seus clientes adoram.

_Essa integração é mantida pela Tapcart._

## Pré-requisitos

| Requisito              | Descrição                                                                                                                                         |
|--------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| Conta Tapcart Enterprise | A integração está disponível apenas para **clientes do Tapcart Enterprise**.                                                                              |
| Chave de API do aplicativo Braze         | A [chave de API do identificador do Braze App](https://www.braze.com/docs/api/identifier_types) é necessária. Cada plataforma (como iOS ou Android) tem seu próprio identificador de app. No dashboard do Braze, acesse **Settings** > **APIs and Identifiers** > **App Identifiers**. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Sobre a integração

Para saber como integrar o Tapcart ao Braze, consulte [o guia de integração do Tapcart](https://help.tapcart.com/hc/en-us/articles/36413383526675-Braze-Integration) diretamente em seu site. Após a integração, você poderá aproveitar os seguintes recursos do Tapcart no Braze:

- Integração de SDK básico
- Notificações por push para iOS e Android
- Envio de mensagens sofisticadas
- Rastreamento e segmentação da jornada do usuário

Note que os seguintes recursos não são suportados no momento: Personalização de Push Stories, Web Push e mensagens no app.
