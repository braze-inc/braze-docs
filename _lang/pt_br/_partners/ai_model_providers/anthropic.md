# Antrópica

> [A Anthropic](https://www.anthropic.com/) é uma empresa de segurança e pesquisa de IA que está desenvolvendo o Claude, um assistente de IA de última geração criado para ser útil, honesto e seguro para uma ampla gama de tarefas linguísticas.

_Essa integração é mantida pela Anthropic._

## Sobre a integração

A integração entre o Braze e o Anthropic permite conectar sua chave de API do Anthropic ao Braze para que você possa usar os modelos da Claude ao criar agentes de IA personalizados. Com essa integração, seus agentes podem gerar cópias personalizadas, tomar decisões em tempo real ou atualizar os campos do catálogo usando os modelos Claude do Anthropic.

## Pré-requisitos

| Solicitações | Descrição |
|---|---|
| Conta Anthropic com chave de API | Uma conta Anthropic com uma chave de API. Para obter ajuda, entre em contato com seu administrador ou com [o suporte da Anthropic](https://support.anthropic.com/). |
| Instância do Braze | Você pode encontrar sua instância do Braze na [página de visão geral da API]({{site.baseurl}}/api/basics/#endpoints) ou com seu gerente de integração do Braze. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integração

Para conectar sua chave de API do Anthropic ao Braze:

1. Acesse **Partner Integrations** > **Technology Partners** no dashboard do Braze e encontre o Anthropic.
2. Digite sua chave de API do Anthropic.
3. Selecione **Salvar**.

Depois de salvar, você pode selecionar os modelos Claude ao [criar um agente personalizado]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) no Console do agente.

Entre em contato com [o suporte da Anthropic](https://support.anthropic.com/) em caso de problemas ou dúvidas sobre sua integração.
