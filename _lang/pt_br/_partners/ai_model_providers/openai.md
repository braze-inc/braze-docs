# OpenAI

> [A OpenAI](https://openai.com/) cria modelos avançados de IA, como o GPT, que ativam a compreensão e a geração de linguagem natural, capacitando as marcas a criar e dimensionar interações significativas com os clientes. 

_Essa integração é mantida pela OpenAI._

## Sobre a integração

A integração entre o Braze e o OpenAI permite conectar sua chave de API do OpenAI ao Braze para que você possa usar os modelos do OpenAI ao criar agentes de IA personalizados. Com essa integração, seus agentes podem gerar cópias personalizadas, tomar decisões em tempo real ou atualizar campos do catálogo usando os grandes modelos de linguagem da OpenAI.

## Pré-requisitos

| Solicitações | Descrição |
|---|---|
| Conta OpenAI com chave de API | Uma conta OpenAI com uma chave de API. Entre em contato com o administrador ou com o [suporte da OpenAI](https://help.openai.com/) em caso de problemas ou dúvidas. |
| Instância do Braze | Você pode encontrar sua instância do Braze na [página de visão geral da API]({{site.baseurl}}/api/basics/#endpoints) ou com seu gerente de integração do Braze. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integração

Para conectar sua chave de API da OpenAI ao Braze:

1. Acesse **Partner Integrations** > **Technology Partners** no dashboard do Braze e encontre o OpenAI.
2. Digite sua chave de API da OpenAI.
3. Selecione **Salvar**.

Depois de salvar, você pode selecionar modelos OpenAI ao [criar um agente personalizado]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) no Console do agente.

Entre em contato com [o suporte da OpenAI](https://help.openai.com/) em caso de problemas ou dúvidas sobre sua integração.
