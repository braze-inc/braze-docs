# Sageflo Radiate

> [O Sageflo Radiate](https://sageflo.com/radiate) é uma ferramenta de marketing distribuído que permite que as equipes locais enviem facilmente seus próprios e-mails usando modelos, imagens e segmentos de público aprovados pelo marketing por meio de integrações de API com o Braze.

_Essa integração é mantida pela Sageflo._

## Sobre a integração

Forneça às equipes de localização as ferramentas necessárias para fazer um marketing mais inteligente, aproveitando os recursos sofisticados do Braze que incluem segmentação de público, governança de frequência e conteúdo dinâmico, tudo isso incluindo proteções para a sua marca. 

## Pré-requisitos

| Requisito | Descrição |
| ----------- | ----------- |
| Conta Sageflo Radiate | É necessário ter uma conta Sageflo Radiate para usar a parceria. |
| Chave da API REST do Braze | Uma chave da API REST do Braze com permissões `templates` e `campaigns` completas. <br><br> Isso pode ser criado no dashboard do Braze em **Configurações** > **Chaves de API**. |
| Endpoint REST do Braze | [Seu URL do ponto de extremidade REST]({{site.baseurl}}/developer_guide/rest_api/basics/#endpoints). Seu endpoint de API corresponde ao URL do dashboard de sua instância do Braze. <br><br> Por exemplo, se o URL do dashboard for `https://dashboard-03.braze.com`, seu endpoint será `dashboard-03`. |

## Casos de uso

O Radiate é ideal para franquias e empresas de varejo que buscam ampliar seus esforços de marketing, capacitando equipes distribuídas a enviar e-mails para seus públicos locais pela Braze.

* Capacite as equipes distribuídas a enviar facilmente e-mails de marketing e SMS
* Crie conexões com os clientes voltadas para a comunidade
* Mantenha a consistência da marca com proteções integradas
* Diminua a carga de trabalho da sua equipe de marketing nacional

## Integração

Sua equipe de contas da Sageflo orientará a configurar a integração. Você precisará fornecer suas credenciais da API da Braze, e a Sageflo trabalhará com sua equipe de marketing para configurar segmentos de público para locais e filiais específicos. 

Após a conexão, a Sageflo se encarregará de:

* Configurar o ambiente Radiate e a conexão com o Braze
* Configurar segmentos de público baseados em local no Braze
* Defina as configurações de campanha, local e grupo de usuários
* Modelos de mapas do Braze para uso em campanhas do Radiate
* Programar e conduzir o treinamento de usuários
