# Tapcart

> [Tapcart](https://www.tapcart.com/) é uma plataforma de comércio móvel líder para marcas impulsionadas pelo Shopify, permitindo que os comerciantes criem aplicativos móveis personalizados que oferecem experiências de compra personalizadas e envolventes que seus clientes adoram.

_Esta integração é mantida pelo Tapcart._

## Pré-requisitos

| Requisito              | Descrição                                                                                                                                         |
|--------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| Conta Empresarial do Tapcart | A integração está disponível apenas para **clientes da Tapcart Enterprise**.                                                                              |
| Chave de API do App Braze         | A [chave de API do identificador do App Braze](https://www.braze.com/docs/api/identifier_types) é necessária. Cada plataforma (como iOS ou Android) tem seu próprio identificador de aplicativo. No painel do Braze, acesse **Configurações** > **APIs e Identificadores** > **Identificadores de Aplicativo**. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Sobre a integração

Para aprender como integrar o Tapcart com o Braze, confira [o guia de integração do Tapcart](https://help.tapcart.com/hc/en-us/articles/36413383526675-Braze-Integration) diretamente em seu site. Após a integração, você poderá aproveitar os seguintes recursos do Tapcart no Braze:

- Integração de SDK Base
- Notificações por push para iOS e Android
- Mensagens push ricas
- Rastreamento e segmentação da jornada do usuário

Observe que os seguintes recursos não são suportados atualmente: Histórias Push, Web Push, personalização de Mensagens In-App.
