# ThoughtSpot

> A ThoughtSpot é a nuvem de análise moderna, uma plataforma de análise de última geração que fornece análise em tempo real para sua pilha de dados moderna, capacitando seus colegas, parceiros e clientes a transformar dados em insights práticos.

A integração Braze e ThoughtSpot aproveita os blocos TML da ThoughtSpot que permitem que os usuários do Braze acelerem a análise de dados de comportamento do usuário com modelos pré-construídos de planilhas e modelos. Essa integração ativa a capacidade de os usuários pesquisarem ilimitadamente os dados de interação do Braze e descobrirem insights práticos. 

## Pré-requisitos

Para começar a usar a ThoughtSpot na Braze, seus dados precisam ser enviados para um data warehouse na nuvem antes que a ThoughtSpot possa consultá-los em tempo real.

| Requisito | Descrição |
| ----------- | ----------- |
| Conta da ThoughtSpot | É necessário ter uma conta da ThoughtSpot para usar a parceria. |
| Data warehouse na nuvem| Os dados da Braze são armazenados no data warehouse na nuvem usando o Braze Currents. |

## Blocos TML

Os usuários do Braze podem acessar e pesquisar facilmente todos os seus dados de interação digital. Nossos modelos permitem que os usuários configurem suas análises rapidamente com visualizações e planilhas pré-construídas. Analise a aquisição do seu site e o comportamento do usuário com pesquisa, detalhamentos e spotIQ.

## Integração

### Etapa 1: conecte a ThoughtSpot 

Registre em sua instância do ThoughtSpot e crie uma conexão Embrace para cada tabela trazida do Braze usando o Braze Currents.

#### Etapa 2: Importar TML

Importe o arquivo compactado para as planilhas e quadros dinâmicos no Thoughtspot e verifique se eles foram importados sem erros. 

Depois de importado, você pode começar a pesquisar e personalizar os liveboards.
