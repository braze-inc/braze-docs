# Geofences

> Saiba como configurar geofences para o SDK do Braze. Uma [geofence]({{site.baseurl}}/user_guide/engagement_tools/locations_and_geofences#about-locations-and-geofences) é uma área geográfica virtual que forma um círculo em torno de uma posição global específica e é representada pela combinação de latitude, longitude e um raio.
