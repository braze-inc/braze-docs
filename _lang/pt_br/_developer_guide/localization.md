# Localização

> Saiba mais sobre localização e idiomas suportados pelo SDK da Braze, para que você possa se conectar com seus usuários em todo o mundo. Para obter orientações sobre como configurar mensagens localizadas, consulte [Localização]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/localization/) em nossa seção Fundamentos do envio de mensagens.

## Sobre a localização

Além do inglês, o Braze oferece suporte a vários idiomas para mensagens SDK exibidas em seu app. 

Quando o idioma do telefone do usuário estiver definido como um dos idiomas suportados, as mensagens SDK incluídas por padrão no canal de envio de mensagens serão traduzidas para esse idioma. Por exemplo, se o seu app exibir uma mensagem sobre problemas de conectividade, ela será traduzida para o idioma escolhido pelo usuário.
