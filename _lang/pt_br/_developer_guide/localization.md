# Localização

> Saiba mais sobre a localização e os idiomas suportados para o Braze SDK, para que você possa se conectar com seus usuários em todo o mundo.

## Sobre localização

Além do inglês, o Braze oferece suporte a vários idiomas para as mensagens integradas do SDK exibidas em seu app.

Quando o idioma do telefone do usuário for definido como um dos idiomas suportados, as mensagens do SDK integrado serão traduzidas para esse idioma. Por exemplo, se o seu app exibir uma mensagem para problemas de conectividade, ela será traduzida para o idioma escolhido.
