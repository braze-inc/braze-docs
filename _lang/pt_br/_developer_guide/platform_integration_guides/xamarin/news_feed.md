# integração do feed de notícias

> Saiba como configurar o feed de notícias do iOS, Android e FireOS para a plataforma Xamarin.

{% alert note %}
O feed de notícias será descontinuado. A Braze recomenda que os clientes que usam nossa ferramenta de feed de notícias migrem para o canal de envio de mensagens Content Cards - é mais flexível, personalizável e confiável. Para saber mais, consulte o [guia de migração]({{site.baseurl}}/user_guide/message_building_by_channel/content_cards/migrating_from_news_feed/).
{% endalert %}

Para saber como integrar o feed de notícias ao seu app Xamarin, consulte as [instruções de integração do Android]({{site.baseurl}}/developer_guide/platform_integration_guides/android/news_feed/integration/) e [instruções de integração do iOS]({{site.baseurl}}/developer_guide/platform_integration_guides/legacy_sdks/ios/news_feed/integration).
