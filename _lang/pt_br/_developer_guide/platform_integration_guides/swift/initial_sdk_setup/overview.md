> Está querendo fazer upgrade para o Swift iOS SDK do nosso antigo Objective C SDK? É fácil. Consulte nosso [guia de migração](https://braze-inc.github.io/braze-swift-sdk/documentation/braze/appboy-migration-guide/) para obter detalhes.

Depois de fazer a integração usando uma das opções listadas, seguir as etapas para [concluir a integração]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/initial_sdk_setup/completing_integration/) e ativar outra personalização de SDK (opcional), você poderá integrar, ativar e personalizar canais e recursos adicionais para atender às necessidades de suas campanhas futuras.  

{% alert note %} O Carthage não é aceito como um gerenciador de pacotes para o Swift Package Manager. {% endalert %}
