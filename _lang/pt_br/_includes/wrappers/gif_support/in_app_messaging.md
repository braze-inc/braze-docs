{% tab Android %}
Você pode adicionar GIFs animados às suas mensagens no app usando o [SDK](https://github.com/braze-inc/braze-android-sdk) nativo [do Braze para Android](https://github.com/braze-inc/braze-android-sdk). Por padrão, o Android SDK usa mensagens no app em HTML para exibir GIFs.

Para todos os outros tipos de mensagens no app, você precisará usar uma biblioteca de imagens personalizada. Para saber mais, consulte [Mensagens no app para Android: GIFs]({{site.baseurl}}/developer_guide/platform_integration_guides/android/in-app_messaging/customization/gifs/).
{% endtab %}

{% tab iOS %}
Você pode adicionar GIFs animados às suas mensagens no app usando o [Swift SDK nativo da Braze](https://github.com/braze-inc/braze-swift-sdk). Por padrão, todas as mensagens no app do Braze suportam GIFs. Para obter um passo a passo completo, consulte [Tutorial: Suporte a GIFs para mensagens no app da Swift](https://braze-inc.github.io/braze-swift-sdk/tutorials/braze/c3-gif-support).
{% endtab %}
