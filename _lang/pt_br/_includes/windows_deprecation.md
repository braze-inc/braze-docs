{% alert warning %}
O SDK da Braze para Windows está obsoleto desde 24 de março de 2022.


* **24 de março de 2022:** Nenhum novo aplicativo do Windows pode ser criado no dashboard da Braze.
* **15 de setembro de 2022:** Nenhuma mensagem nova pode ser enviada para os apps do Windows. As mensagens existentes e a coleta de dados não são afetadas.
* **15 de janeiro de 2023:** A Braze não enviará mais mensagens ou coletará dados de aplicativos do Windows.

{% endalert %}
