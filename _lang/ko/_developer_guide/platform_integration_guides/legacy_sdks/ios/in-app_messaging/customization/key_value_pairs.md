# 키-값 쌍 추가 정보

`ABKInAppMessage` 오브젝트는 키-값 페어를 `extras`로 전달할 수 있습니다. 캠페인 생성 시 대시보드에서 지정할 수 있습니다. 키-값 페어를 사용하여 앱에서 추가 처리를 위해 인앱 메시지와 함께 데이터를 전송할 수 있습니다.
