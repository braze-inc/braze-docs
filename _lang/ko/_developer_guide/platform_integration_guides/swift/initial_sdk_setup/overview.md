> 이전 Objective-C SDK에서 Swift iOS SDK로 업그레이드하고 싶으신가요? 간단합니다. 자세한 내용은 [마이그레이션 가이드를](https://braze-inc.github.io/braze-swift-sdk/documentation/braze/appboy-migration-guide/) 확인하세요.

나열된 옵션 중 하나를 사용하여 통합을 완료하고 [통합 완료 단계]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/initial_sdk_setup/completing_integration/)를 따른 후 다른 SDK 사용자 지정(선택 사항)을 활성화하고 향후 캠페인의 요구 사항에 맞게 추가 채널 및 기능을 통합, 활성화 및 사용자 지정할 수 있습니다.  

{% alert note %} Carthage는 Swift iOS SDK의 패키지 매니저로 지원되지 않습니다. {% endalert %}
