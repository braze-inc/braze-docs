# 뉴스 피드 (사용 중단)

> Braze SDK에 대한 뉴스 피드에 대해 알아보세요. 이는 각 사용자에게 개별적으로 맞춤화할 수 있는 완전한 사용자 정의가 가능한 인앱 콘텐츠 피드입니다. 온보딩 콘텐츠, 광고, 성과, 뉴스 등을 위한 피드를 만드는 데 사용할 수 있습니다.

{% alert warning %}
뉴스 피드는 더 이상 새로운 기능, 버그 수정, 보안 업데이트 또는 기술 지원을 받지 않습니다. 대신, 더 유연하고 사용자 정의 가능하며 신뢰할 수 있는 메시징 채널인 콘텐츠 카드를 사용하세요. 시작하려면 [뉴스 피드에서 마이그레이션]({{site.baseurl}}/user_guide/message_building_by_channel/content_cards/migrating_from_news_feed/)를 확인하세요.
{% endalert %}
