# 현지화

> 전 세계 사용자와 소통할 수 있도록 Braze SDK의 현지화 및 지원 언어에 대해 알아보세요. 현지화된 메시지를 설정하는 방법에 대한 안내는 메시징 기본 사항 섹션의 [현지화를]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/localization/) 참조하세요.

## 현지화 정보

Braze는 영어 외에도 앱에 표시되는 소프트웨어 개발 키트 메시지에 대해 여러 언어를 지원합니다. 

사용자의 휴대폰 언어가 지원되는 언어 중 하나로 설정되어 있으면 메시징 채널에 기본값으로 포함된 소프트웨어 개발 키트 메시지가 해당 언어로 번역됩니다. 예를 들어 앱에 연결 문제에 대한 메시지가 표시되는 경우 사용자가 선택한 언어로 번역됩니다.
