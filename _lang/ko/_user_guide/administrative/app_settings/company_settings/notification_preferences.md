# 알림 환경설정

> 회사 계정의 메시징 및 활동을 모니터링하려면 특정 알림을 설정하고 알림이 전송되는 위치를 선택할 수 있습니다.

**알림 환경설정** 페이지에서는 회사에 대한 알림을 받을 사람(있는 경우)을 구성할 수 있습니다. 캠페인 전달 또는 기술적 오류에 대한 알림을 수신할 대상을 구성할 수 있습니다. 주간 분석 보고서의 수신자를 지정할 수도 있습니다. 대부분의 알림에 대해 Braze는 이메일과 웹훅 채널을 지원합니다.

![Braze 대시보드의 알림 환경설정 페이지]({% image_buster /assets/img_archive/notification_preferences.png %})

이 페이지에 액세스하려면 **설정** > **관리자 설정** > **알림 환경설정**으로 이동합니다.

{% alert tip %}
Slack과 통합하여 알림을 받을 수도 있습니다. 단계는 [수신 웹훅을 사용하여 메시지 보내기](https://api.slack.com/incoming-webhooks)를 참조하세요.
{% endalert %}

## 사용 가능한 알림

다음 표에서는 사용 가능한 알림과 알림을 전달하는 데 사용되는 채널에 대해 설명합니다.

{% alert note %}
기본 **수신자** 값인 **모든 대시보드 사용자**를 삭제한 후 다시 추가하려면 드롭다운 필드에 수동으로 입력하면 됩니다.
{% endalert %}

| 알림 | 설명 | 사용 가능한 알림 채널 |
|--------------|-------------|-----------------|
| API 사용량 알림 | 이 항목을 선택하면 **API 사용량 대시보드**로 이동하며, 여기서 [**API 사용량 알림**]({{site.baseurl}}/user_guide/analytics/dashboard/api_usage_alerts/) 탭으로 이동하여 주요 API 요청 볼륨을 추적하는 알림을 설정할 수 있습니다. | 이메일, 웹훅 |
| AWS 자격 증명 오류 | Braze가 데이터 내보내기를 위해 Amazon Web Services 자격 증명을 사용하는 중에 오류가 발생할 때 수신자에게 알립니다. 여기에는 Google Cloud Services 및 Azure(Microsoft Cloud Services)에 대한 자격 증명 오류 알림도 포함됩니다. | 이메일, 웹훅 |
| 캠페인 자동 중단 | Braze가 캠페인을 중단했을 때 수신자에게 알립니다. | 이메일 |
| 캔버스 자동 중단 | Braze가 캔버스를 중단했을 때 수신자에게 알립니다. | 이메일 |
| 캠페인 상호작용 만료 | 캠페인 상호작용 데이터 만료가 예정된 캠페인과, 리타겟팅 필터에서 해당 캠페인을 참조하며 지난 30일 이내에 메시지 발송에 사용된 세그먼트, 캠페인, 캔버스에 대한 정보를 수신자에게 알립니다. | 이메일 |
| 캠페인/캔버스 업데이트됨 | 활성 캠페인 또는 캔버스가 업데이트되거나 비활성화될 때, 그리고 비활성 캠페인 또는 캔버스가 재활성화되거나 초안이 시작될 때 수신자에게 알립니다. | 이메일 |
| 캠페인/캔버스 볼륨 한도 도달 | 캠페인 또는 캔버스가 볼륨 한도에 도달했을 때 수신자에게 알립니다. | 이메일 | 
| 캔버스 상호작용 만료 | 캔버스 상호작용 데이터 만료가 예정된 캔버스와, 리타겟팅 필터에서 해당 캔버스를 참조하며 지난 30일 이내에 메시지 발송에 사용된 세그먼트, 캠페인, 캔버스에 대한 정보를 수신자에게 알립니다. | 이메일 |
| 캔버스 내 댓글 | 캔버스에 새 댓글이 달렸을 때 수신자에게 알립니다. | 이메일 |
| 연결된 콘텐츠 오류 | 연결된 콘텐츠 엔드포인트에 오류가 발생했을 때 수신자에게 알립니다. | 이메일 |
| 푸시 오류 | 푸시 엔드포인트에 오류가 발생했을 때 수신자에게 알립니다. | 이메일, 웹훅 |
| 스케줄된 캠페인 한도 도달 | 반복 스케줄 캠페인의 한도에 도달했을 때 수신자에게 알립니다. | 이메일, 웹훅 |
| 스케줄된 캠페인 발송 완료 | 스케줄된 캠페인의 발송이 완료되었을 때 수신자에게 알립니다. | 이메일, 웹훅 |
| 웹훅 오류 | 웹훅 엔드포인트에 오류가 발생했을 때 수신자에게 알립니다. | 이메일 |
| 주간 분석 보고서 | 지난 주 워크스페이스 활동 요약을 매주 월요일에 수신자에게 발송합니다. 수신자는 소속된 각 워크스페이스에 대한 요약을 받습니다. | 이메일 |
| 일일 캔버스/캠페인 진입 볼륨 한도 | 발송 한도에 도달할 때마다 알림을 보냅니다. | 이메일 |
| 에이전트 콘솔 오류 | [에이전트 콘솔 에이전트]({{site.baseurl}}/user_guide/brazeai/agents)가 현재 기능으로 실행 한도에 도달했을 때 수신자에게 알립니다. | 이메일 |
{: .reset-td-br-1 .reset-td-br-2 .reset-td-br-3 role="presentation" }

{% alert note %}
[일시 중지된 사용자]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/#suspending-company-users)도 Braze로부터 알림을 계속 받을 수 있습니다.
{% endalert %}

## 주간 분석 보고

Braze는 선택적으로 매주 월요일 오전 5시(EST)에 회사 내에서 지정한 개인에게 이메일을 통해 주간 보고서를 발송합니다. **데이터 설정** > **커스텀 이벤트**에서 주간 보고서에 포함할 커스텀 이벤트를 선택할 수 있습니다.

주간 보고서에 포함할 이벤트를 최대 5개까지 선택할 수 있습니다:

![분석 보고서에 포함할 이벤트 선택하기]({% image_buster /assets/img_archive/company_analytics_report_new.png %})
