# 세분화된 권한으로 마이그레이션

> 누가 귀하의 계정에 접근하고 특정 작업을 수행할 수 있는지를 관리하는 것은 보안과 운영 효율성 모두에 중요합니다. 더 많은 제어를 제공하기 위해 Braze는 사용자 접근을 관리하는 보다 유연하고 정밀한 방법인 세분화된 권한을 도입하고 있습니다.

마이그레이션에는 다음과 같은 이점이 포함됩니다:

- **보다 정밀한 제어:** 세분화된 권한은 더 많은 제어, 더 나은 보안 및 더 명확한 감독을 제공합니다. 사용자는 필요한 접근만을 받습니다.
- **자동 매핑:** 모든 현재 권한은 자동으로 [세분화된 동등물](#legacy-to-granular-permissions-mapping)에 매핑됩니다. 사용자는 변경하지 않는 한 동일한 접근 수준을 유지합니다.

## 검토할 사항

귀사의 마이그레이션이 계획될 때, 귀하의 Braze 관리자에게는 세분화된 권한 마이그레이션을 알리는 이메일과 대시보드 배너가 전송됩니다. 마이그레이션을 준비하기 위해 Braze 관리자가 다음을 수행할 것을 권장합니다.

1. 새 권한 프레임워크로 마이그레이션한 후 보다 맞춤화된 접근을 위해 업데이트가 필요할 수 있는 사용자, 역할 또는 권한 세트를 식별합니다. 
2. 귀사가 SCIM 또는 [권한 문자열]({{site.baseurl}}/scim_api_appendix/)에 의존하는 준수 도구를 사용하여 자동 사용자 프로비저닝을 하고 있다면, 새로운 세분화된 구조에 맞게 업데이트하십시오. 
3. 혼란을 방지하기 위해 Braze 사용자에게 다가오는 변경 사항을 알립니다.
4. 예정된 마이그레이션 날짜와 시간에 귀사는 자동으로 세분화된 권한으로 마이그레이션됩니다. 회사 관리자의 추가 조치는 필요하지 않습니다.

{% alert important %}
권한 업데이트 기능은 예정된 마이그레이션 시간의 15분 이내에 잠금됩니다. 이는 마이그레이션이 완료될 때까지 권한을 변경할 수 없음을 의미하며, 우리는 이 과정이 최대 15분이 걸릴 것으로 예상합니다.
{% endalert %}

## Frequently asked questions

### 마이그레이션을 선택 해제하거나 되돌릴 수 있나요?

Braze는 마이그레이션을 되돌리는 것을 지원하지 않습니다. 우리는 마이그레이션을 지원하고 문제를 신속하게 해결하기 위해 마이그레이션을 면밀히 모니터링할 것입니다.

### 기존 사용자가 마이그레이션 중에 Braze에 접근할 수 없게 되나요?

아니요, 마이그레이션 중에 Braze에 중단 시간이 없습니다. 그러나 마이그레이션 중에는 권한 업데이트가 잠겨 있습니다. 마이그레이션 완료까지 최대 15분이 소요될 것으로 예상합니다.
