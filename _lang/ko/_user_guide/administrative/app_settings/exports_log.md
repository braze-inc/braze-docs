# 내보내기 로그

> **수출 로그** 페이지를 사용하여 수출 작업의 상태를 확인하고 진행 중인 수출을 Braze 플랫폼에서 바로 취소할 수 있습니다. <br><br> 현재 내보내기 로그는 세그먼트 내보내기만 지원합니다. 세그먼트 내보내기 도구에 대한 자세한 내용은 [세그먼트 데이터를 CSV로 내보내기를]({{site.baseurl}}/user_guide/data/export_braze_data/segment_data_to_csv/) 참조하세요.

{% 경고 중요 %} 내보내기 로그가 얼리 액세스 중입니다. 미리 맛보기에 참여하고 싶다면 고객 성공 관리자에게 문의하세요. {% endalert %}

내보내기 로그는 **설정** > **내보내기 로그로** 이동하여 찾을 수 있습니다. 여기에서 이전 내보내기, 내보낸 세그먼트의 이름, 각 내보내기 상태, 각 내보내기의 출처, 각 내보내기 시작 및 완료 시점을 볼 수 있습니다. 

\![완료된 내보내기 목록이 포함된 내보내기 로그]\[1]

## 보류 중인 내보내기 취소하기

내보내기 **로그** 페이지에서 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택한 다음 **내보내기 취소를** 선택하거나 내보내기 **ID를** 선택한 다음 내보내기 페이지에서 내보내기 **취소를** 선택하면 보류 중인 내보내기를 바로 취소할 수 있습니다.

보류 중인 내보내기 로그와 '내보내기 취소' 옵션이 표시됩니다.]\[2]

## 특정 내보내기 로그 공유

내보내기 **ID를** 선택한 다음 **로그 공유를** 선택하여 내보내기 로그를 공유합니다.

페이지 공유 링크가 포함된 드롭다운 창입니다.]\[3]{:: style="max-width:45%;"}

{경고 노트 %} 내보내기 로그에 대한 피드백이 있으면 고객 성공 관리자와 언제든지 공유하세요. {% endalert %}
