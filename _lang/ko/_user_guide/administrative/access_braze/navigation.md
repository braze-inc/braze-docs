# 브레이즈 탐색

브레이즈 탐색은 기기 전반에 걸쳐 기능과 콘텐츠에 효율적으로 접근할 수 있도록 설계되었습니다. 최대한 활용하는 방법을 살펴보겠습니다.

- **간소화된 메뉴:** 도구를 쌓인 하위 메뉴로 구성하는 수직 메뉴 구조로 직관적으로 탐색하세요.
- **강력한 검색:** 전역 검색 바를 사용하여 이름, 유형 또는 이전 이름으로 페이지나 항목을 찾으세요.
- **모바일 친화적인 디자인:** 모든 기기에서 브레이즈 탐색의 전체 기능에 접근하세요.

{% alert note %}
이 업데이트는 모든 사용자를 통합된 탐색으로 전환하여 경험을 간소화합니다. 탐색 버전 간 전환 옵션은 더 이상 제공되지 않습니다.
{% endalert %}

## 메뉴 사용하기

수직 메뉴 구조는 가장 많이 사용하는 도구를 한 번의 클릭으로 제공합니다. 주 메뉴 항목을 선택하여 쌓인 수직 레이아웃으로 옵션을 표시합니다. 

![사용자가 오른쪽에 쌓인 수직 메뉴로 하위 메뉴를 열기 위해 "설정" 페이지를 선택합니다.]({% image_buster /assets/img/navigation/vertical_menu.gif %})

## 전역 검색

헤더에 위치한 전역 검색은 브레이즈에서 무엇이든 찾는 가장 빠른 방법입니다. 검색 바를 선택하거나 Windows에서는 Ctrl + K, Mac에서는 ⌘ + K를 눌러 바로 이동하세요. 

![검색어가 입력되지 않은 상태에서 글로벌 검색이 열리며, 최근에 열린 페이지가 표시됩니다.]({% image_buster /assets/img/navigation/search_recently_opened.png %})

검색은 업데이트된 탐색에서 이름이 변경된 페이지의 동의어를 이해합니다. 예를 들어, "Currents"를 검색하면 "Data Export"를 찾을 수 있습니다. 해당 페이지의 이름이 변경되었기 때문입니다.

![검색 결과 "currents" 표시 중: "Data Export. 이전의 '커런츠'."]({% image_buster /assets/img/navigation/global_search_synonym.png %})

## 측면 탐색 최소화

시각적 혼잡을 줄이고 작업 공간에 집중하기 위해 측면 탐색 패널을 최소화할 수 있습니다. **메뉴 최소화**를 눌러 탐색을 숨깁니다. 이는 사용자 흐름을 설계하기 위해 Canvas에서 작업할 때 특히 유용합니다. 

![사용자가 Braze 캔버스에서 왼쪽 탐색 메뉴를 확장하고 축소하여 전체 텍스트 레이블과 아이콘 전용 보기 간에 전환합니다.]({% image_buster /assets/img/navigation/expand_collapse_menu.gif %})

최소화된 상태에서도 아이콘 위에 마우스를 올리면 메뉴 항목 이름이 포함된 툴팁을 볼 수 있어 빠르게 접근할 수 있습니다.

![Braze 탐색이 최소화되어, 포커스가 오디언스 메뉴 항목으로 이동할 때 "오디언스"에 대한 툴팁이 표시됩니다.]({% image_buster /assets/img/navigation/minimized_menu_tooltip.png %}){:style="max-width:50%"}

## 반응형 탐색

탐색은 장치 크기에 따라 원활하게 조정됩니다. 작은 화면에서는 측면 탐색이 자동으로 축소됩니다. 필요할 때 <i class="fa-solid fa-bars" aria-label="탐색 메뉴 열기"></i>를 눌러 메뉴를 엽니다. 

![작은 화면에서는 측면 탐색이 자동으로 축소됩니다. 메뉴 아이콘을 탭하면 탐색 옵션이 열립니다.]({% image_buster /assets/img/navigation/navigation_small_screens.png %})

## 피드백 공유

탐색, 사용성, 접근성, 시각적 매력 또는 당신에게 중요한 다른 주제에 대한 피드백을 공유할 수 있습니다. 전역 헤더에서 **지원** 메뉴를 열고 **피드백 공유**를 선택하여 의견을 보내주세요. 귀하의 입력은 Braze 개선에 도움이 되며, 귀하의 의견을 소중히 여깁니다!

![지원 메뉴가 열리며, Braze 설명서, Braze 학습, 피드백 공유 및 도움 받기 옵션이 표시됩니다.]({% image_buster /assets/img/navigation/share_feedback.png %}){:style="max-width:50%"}
