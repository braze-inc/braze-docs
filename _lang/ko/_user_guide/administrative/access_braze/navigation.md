# Braze 탐색하기

Braze 탐색은 여러 기기의 기능과 콘텐츠에 효율적으로 액세스할 수 있도록 설계되었습니다. 이를 최대한 활용하는 방법을 알아보세요.

{% alert note %}
이 업데이트는 모든 사용자를 통합 탐색으로 전환하여 사용 환경을 간소화합니다. 탐색 버전 간에 전환하는 옵션은 더 이상 사용할 수 없습니다.
{% endalert %}

## 메뉴 사용

세로형 메뉴 구조로 가장 자주 사용하는 도구를 한 번만 누르면 바로 사용할 수 있습니다. 주 메뉴 항목을 선택하면 해당 옵션이 세로로 쌓인 레이아웃으로 표시됩니다. 

사용자가 '설정' 페이지를 선택하면 오른쪽에 세로로 쌓인 메뉴로 하위 메뉴가 열립니다.]({% image_buster /assets/img/navigation/vertical_menu.gif %})

## 글로벌 검색

헤더에 있는 글로벌 검색은 Braze에서 무엇이든 가장 빠르게 찾을 수 있는 방법입니다. 검색창을 선택하거나 Windows에서는 Ctrl + K를, Mac에서는 ⌘ + K를 눌러 바로 해당 검색창으로 이동합니다. 

\![검색어를 입력하지 않은 상태에서 글로벌 검색이 열리고 최근에 열린 페이지가 표시됩니다.]({% image_buster /assets/img/navigation/search_recently_opened.png %})

검색은 업데이트된 탐색 기능에서 이름이 변경된 페이지의 동의어를 이해합니다. 예를 들어 '커런츠'를 검색하면 해당 페이지의 이름이 변경되었으므로 '데이터 내보내기'를 찾을 수 있습니다.

!"커런츠" 검색 결과가 표시됩니다: "데이터 내보내기. 이전에는 '커런츠'였습니다."]({% image_buster /assets/img/navigation/global_search_synonym.png %})

## 측면 탐색 최소화

시각적 혼란을 줄이고 작업 공간에 집중하기 위해 측면 탐색 패널을 최소화할 수 있습니다. **최소화 메뉴를** 눌러 탐색 **메뉴를** 숨기면 캔버스에서 사용자 플로우를 디자인할 때 특히 유용합니다. 

사용자가 Braze 캔버스에서 왼쪽 탐색 메뉴를 확장 및 축소하여 전체 텍스트 레이블과 아이콘 전용 보기 사이를 토글합니다.]({% image_buster /assets/img/navigation/expand_collapse_menu.gif %})

최소화한 상태에서도 아이콘 위로 마우스를 가져가면 메뉴 항목 이름이 포함된 도구 설명이 표시되어 빠르게 액세스할 수 있습니다.

초점이 오디언스 메뉴 항목으로 이동하면 '오디언스'에 대한 툴팁이 표시되는 Braze 탐색이 최소화되었습니다.]({% image_buster /assets/img/navigation/minimized_menu_tooltip.png %}){:style="max-width:50%"}

## 반응형 탐색

내비게이션은 기기 크기에 관계없이 원활하게 조정됩니다. 작은 화면에서는 측면 탐색이 자동으로 축소됩니다. 언론 <i class="fa-solid fa-bars" aria-label="탐색 메뉴 열기"></i> 을 눌러 필요할 때 메뉴를 엽니다. 

작은 화면에서는 측면 탐색이 자동으로 축소됩니다. 메뉴 아이콘을 탭하면 탐색 옵션이 열립니다.]({% image_buster /assets/img/navigation/navigation_small_screens.png %})

## 피드백 공유

탐색, 사용성, 접근성, 시각적 매력 또는 기타 중요한 주제에 대한 피드백을 공유할 수 있습니다. 글로벌 헤더에서 **지원** 메뉴를 열고 **피드백 공유를** 선택하여 의견을 보내주세요. 여러분의 의견은 Braze를 개선하는 데 도움이 되며, 저희는 여러분의 의견을 소중하게 생각합니다!

지원 메뉴가 열리면 Braze 설명서, Braze 학습, 피드백 공유, 도움말 받기 옵션이 표시됩니다.]({% image_buster /assets/img/navigation/share_feedback.png %}){:style="max-width:50%"}
