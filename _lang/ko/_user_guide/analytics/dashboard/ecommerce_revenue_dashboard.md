# 이커머스 매출 대시보드

> 이커머스 **매출 - 라스트 터치 기여도** 대시보드는 [이커머스 추천 이벤트를]({{site.baseurl}}/ecommerce_events/) 사용하여 캠페인 및 캔버스에 대한 라스트 터치 기여도 매출을 추적합니다. 이 대시보드를 사용하여 어떤 메시징이 매출을 견인하는지 파악하고 시간 경과에 따른 전반적인 이커머스 성능/성과를 모니터링할 수 있습니다.

{% alert note %}
eCommerce 추천 이벤트는 현재 초기 액세스 중입니다. 이 초기 액세스에 참여하고 싶다면 Braze 고객 성공 매니저에게 문의하세요. <br><br>새로운 [Shopify 커넥터를]({{site.baseurl}}/partners/ecommerce/shopify/multiple_stores/?tab=shopify%20connector) 사용하는 경우 통합을 통해 이러한 권장 이벤트를 자동으로 사용할 수 있습니다. 그렇지 않으면 이 대시보드에 데이터가 표시되기 전에 이러한 이벤트를 구현해야 합니다.
{% endalert %}

이커머스 매출 대시보드를 보려면 **분석** > **대시보드 빌더로** 이동한 다음 **이커머스 매출 - 라스트 터치 기여도를** 선택합니다. 이 대시보드는 선택한 전환 기간 내에 사용자가 주문하기 전에 마지막으로 상호작용한 캠페인 또는 캔버스의 매출 기여도에 대해 보고합니다.

## 사용 가능한 측정기준

| Metric | Definition |
| --- | --- |
| 이커머스 매출 | 선택한 날짜 범위와 전환 기간을 기준으로 한 총 라스트 터치 속성 매출입니다. |
| 일일 주문 수 | 하루 평균 고유 주문 건수입니다. |
| 일별 평균 이커머스 매출 | 선택한 기간 동안의 하루 평균 속성별 매출입니다. |
| 시간 경과에 따른 이커머스 매출 | 선택한 날짜 범위의 속성별 매출의 시계열입니다. |
| 캠페인별 이커머스 매출 | 캠페인별로 세분화된 기여도별 매출. | 
| 캔버스별 이커머스 매출 | 캔버스별로 세분화된 기여도 매출입니다. |
{: .reset-td-br-1 .reset-td-br-2 role=“presentation”}

## 기여도 모델

**이커머스 매출 - 라스트 터치 기여** 도 대시보드에서는 라스트 터치 속성을 사용합니다. 즉, 사용자가 주문하기 전에 참여한 가장 최근의 Braze 캠페인 또는 캔버스에 매출 기여도가 부여됩니다.

{% alert important %}
선택한 전환 기간 내에 메시지 상호 작용이 발생했어야 합니다. 전환 기간 내에 적격 메시지 상호 작용이 없는 주문은 기여도 속성이 부여되지 않습니다.
{% endalert %}

## 포함된 데이터

이커머스 **매출 - 라스트 터치 기여도** 대시보드는 이커머스 추천 이벤트에서 데이터를 가져옵니다:

- `ecommerce.product_viewed`
- `ecommerce.cart_updated`
- `ecommerce.checkout_started`
- `ecommerce.order_placed`
- `ecommerce.order_refunded`
- `ecommerce.order_cancelled`

매출 및 주문 수는 Braze의 표준화된 계산을 사용합니다.

| Metric | 계산 |
| --- | --- |
| Total Revenue | 주문한 금액의 합계 - 환불된 금액의 합계 |
| 총 주문 건수 | 고유 주문이 접수됨 - 고유 주문이 취소됨 |
{: .reset-td-br-1 .reset-td-br-2 role=“presentation”}

### 제외된 데이터

기존 구매 이벤트를 사용하여 기록된 구매는 포함되지 않습니다. **이커머스 매출 - 라스트 터치 기여도** 대시보드는 현재 캠페인이나 캔버스 내의 LTV 또는 매출 보고와 같은 기존 구매 이벤트와 연결된 기능을 지원하지 않습니다. 


## 통화 처리

모든 매출은 USD로 표시됩니다. 미화 이외의 통화는 이벤트가 보고된 날짜의 환율을 사용하여 미화로 변환됩니다. 전환을 방지하려면 이벤트를 보낼 때 통화를 `USD` 으로 하드코딩하세요.
