# 쿼리 빌더 템플릿

> 보고서를 만들 때 **쿼리 템플릿**을 선택하여 [쿼리 빌더]({{site.baseurl}}/user_guide/analytics/query_builder/) 템플릿에 액세스합니다. 모든 템플릿은 최대 최근 60일까지의 데이터를 표시하지만 편집기에서 해당 값과 다른 값을 직접 편집할 수 있습니다.<br><br>쿼리 빌더 보고서에 표시될 수 있는 메트릭에 대한 정의는 [보고서 메트릭 용어집을]({{site.baseurl}}/user_guide/data/report_metrics/) 참조하고 각 채널별로 필터링하세요.

## 이메일 템플릿

| 쿼리 이름 | 설명 | 
| --- | --- | 
| 도메인당 이메일 반송 수 | 이메일 도메인당 반송 횟수를 총 반송, 하드 반송, 소프트 반송으로 구분하여 표시합니다. <br> ![]({% image_buster /assets/img_archive/query_builder_q4.png %}){: style="max-width:60%;"} |
| 일별 이메일 전달 측정기준 | 이 보고서에는 매일 전송된 이메일 수, 전달된 이메일 수, 소프트 반송 및 하드바운스와 같은 메시지에 대한 측정기준이 표시됩니다. <br><br> 모든 측정기준은 보고 기간 내내 고유합니다. 예를 들어 환영 이메일 소프트가 11월 21일에 한 번, 11월 22일에 두 번 반송되어 전달되지 않은 경우입니다. {::nomarkdown} <ul><li> 11월 21일의 <i>소프트바운스</i> 측정기준이 1 증가합니다.</li><li> 11월 22일의 <i>소프트 바운스</i> 지표는 영향을 받지 않습니다. </li></ul>{:/} ![]({% image_buster /assets/img_archive/email_delivery_day.png %})|
|  세그먼트별 이메일 참여 측정기준 | 이 보고서에는 각 세그먼트에 전송된 이메일 수, 전달된 이메일 수, 소프트 반송 및 하드바운스와 같은 메시지에 대한 측정기준이 표시됩니다. <br><br> 모든 측정기준은 보고 기간 내내 고유합니다. 예를 들어 환영 이메일 소프트가 11월 21일에 한 번, 11월 22일에 두 번 반송되어 전달되지 않은 경우입니다. {::nomarkdown} <ul><li> 11월 21일의 <i>소프트바운스</i> 측정기준이 1 증가합니다. </li><li> 11월 22일의 <i>소프트 바운스</i> 지표는 영향을 받지 않습니다.</li></ul>{:/} ![]({% image_buster /assets/img_archive/email_engagement_segment.png %}) |
| 세그먼트별 이형 상품 또는 단계에 대한 이메일 참여 지표 | 이 보고서에는 각 세그먼트에 전송된 메시지의 변형 또는 캔버스 단계에 대한 메트릭이 표시됩니다. 이러한 측정기준에는 전송, 전달, 소프트반송 및 하드바운스된 이메일 수가 포함됩니다. <br><br> 모든 측정기준은 보고 기간 내내 고유합니다. 예를 들어 환영 이메일 소프트가 11월 21일에 한 번, 11월 22일에 두 번 반송되어 전달되지 않은 경우입니다. {::nomarkdown} <ul><li> 11월 21일의 <i>소프트바운스</i> 측정기준이 1 증가합니다. </li> <li> 11월 22일의 <i>소프트 바운스</i> 지표는 영향을 받지 않습니다.</li></ul> {:/} |
| 국가별 이메일 성과 | 이 보고서에는 각 국가별로 전송 수, 간접 열람율, 직접 열람율 등의 측정기준이 표시됩니다. 국가는 푸시 전송 시점의 사용자 국가입니다. <br><br> ![]({% image_buster /assets/img_archive/query_builder_q3.png %}) |
| 이메일 구독 변경 로그 | 이 보고서에는 이메일 주소, 구독 상태, 상태가 변경된 시간, 연결된 캔버스 또는 캠페인 등 각 사용자의 구독 변경에 대해 기록된 측정기준이 표시됩니다. |
| 이메일 구독 그룹 옵트인 및 옵트아웃 | 이 보고서에는 매주 이메일 구독 그룹의 고유 사용자 옵트인 및 옵트아웃 수가 표시됩니다. <br><br> ![]({% image_buster /assets/img_archive/query_builder_q2.png %}){: style="max-width:70%;"} |
| 클릭한 이메일 URL | 이 보고서에는 이메일의 각 링크가 클릭된 횟수가 표시됩니다. 이 보고서를 실행하려면 캠페인 또는 캔버스에 대한 API 식별자를 지정해야 합니다. 캠페인의 API 식별자는 해당 캠페인의 세부 정보 페이지 하단에서 찾을 수 있으며, 캔버스 API 식별자는 **배리언트 분석**에서 확인할 수 있습니다. <br><br> 이 보고서에는 개인화되지 않은 링크와 각 링크에 대한 클릭 수가 표시됩니다. CSV 다운로드에는 클릭한 모든 사용자의 사용자 ID, 클릭한 링크, 클릭한 시점의 타임스탬프가 포함됩니다. <br><br> *개인화되지 않은 URL:* Liquid 태그가 제거된 URL. <br><br> ![]({% image_buster /assets/img_archive/query_builder_q5.png %}){: style="max-width:70%;"} |
| 이메일 참여를 위한 상단/하단 메시징 | 이 보고서는 상위 또는 하위 캠페인, 캔버스 또는 캔버스 단계에 대한 이메일 인게이지먼트 측정기준을 보여줍니다. 상위 또는 하위 실적자를 표시할지 여부와 이 분석을 실행할 특정 지표(예: *전송*, *소프트 바운스*, *고유 열기*)를 지정해야 합니다. <br><br> 상위 성과 보고서의 행은 최고 성과에서 최하위 성과로 정렬되고, 하위 성과 보고서의 행은 최하위에서 최고 성과로 정렬됩니다. <br><br> ![]({% image_buster /assets/img_archive/top-bottom-email.png %}) |
