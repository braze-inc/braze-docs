# 참여 보고서

> 참여 보고서를 사용하면 캠페인과 캔버스에서 특정 메시지에 대한 참여 통계를 가져와 원하는 시간에 이메일로 받아볼 수 있습니다.

{% alert note %}
참여 보고서를 실행하려면 '사용자 데이터 내보내기' 권한이 필요합니다.
{% endalert %}

참여 보고서를 사용하면 이메일 보고서에 포함할 캠페인과 캔버스를 수동으로 선택하거나 관련 캠페인과 캔버스를 자동으로 선택하도록 규칙을 지정할 수 있습니다.

선택한 캠페인 또는 캔버스 수에 관계없이 모든 캠페인 데이터에 대해 하나, 모든 캔버스 데이터에 대해 하나씩 최대 2개의 CSV 파일이 생성됩니다. 보고서 이메일에 포함된 링크를 통해 이러한 CSV 파일에 액세스할 수 있습니다. 참여 보고서는 Braze 대시보드에 저장되지 않습니다.

특정 데이터는 캠페인 또는 캔버스 수준에서 집계되는 반면 개별 캠페인 배리언트 또는 캔버스 단계 수준에서 집계됩니다. [실행 후 캔버스 단계를 삭제하면]({{site.baseurl}}/user_guide/engagement_tools/canvas/managing_canvases/change_your_canvas_after_launch/#canvas-details) 참여 보고서의 데이터도 삭제됩니다.

{% alert tip %}
리포트를 다시 실행하여 업데이트된 통계를 생성할 수 있습니다.
{% endalert %}

## 새 보고서 생성하기

### 1단계: 보고서 만들기

대시보드 계정에서 **분석** > 참여 **보고서로 이동**합니다. **새 보고서 생성을** 선택합니다.

### 2단계: 메시지 추가하기

보고서에 컴파일할 캠페인과 캔버스 메시지를 추가합니다. 두 가지 방법으로 메시지를 선택할 수 있습니다:

- 캠페인 및 캔버스 수동 선택하기
- 특정 규칙에 따라 캠페인과 캔버스를 자동으로 선택합니다.

![engagement_reports_message_selection]({% image_buster /assets/img_archive/engagement_report_add_messages.png %})

#### 캠페인 또는 캔버스 수동 선택

이 옵션을 사용하면 이 보고서에서 원하는 캠페인이나 캔버스를 자유롭게 선택할 수 있습니다.

#### 캠페인 또는 캔버스 자동 선택하기

이 옵션을 사용하면 특정 [태그가]({{site.baseurl}}/user_guide/administrative/app_settings/tags/) 포함된 모든 메시지를 자동으로 포함할 수 있습니다. 나열된 태그 중 하나 또는 전체가 포함된 메시지를 타겟팅할 수 있습니다. 이 옵션은 반복 보고서를 설정하고 정기적으로 참여 메시지에 태그를 지정하는 경우에 유용합니다.

### 3단계: 통계 추가 {#add-statistics-to-your-reports}

통계 **추가** 단계에서는 선택한 캠페인 또는 캔버스 유형에 대한 통계를 표시합니다. 예를 들어 이메일 메시지를 선택한 경우 관련 이메일 통계만 볼 수 있습니다. 이메일과 푸시 조합을 선택한 경우 해당 두 채널에 대한 통계를 볼 수 있습니다.

![engagement_report_add_stats]({% image_buster /assets/img_archive/engagement_report_add_stats.png %})

| 채널 | 사용 가능한 통계 |
| ------| --------------|
| 이메일 | 전송, 열람, 고유 열람, 클릭, 고유 클릭, 클릭하여 열람, 탈퇴, 반송, 전달, 신고된 스팸 |
| 푸시  | 전송, 열기, 영향받은 열기, 바운스, 본문 클릭 |
| 웹 푸시 | 보내기, 열기, 반송, 본문 클릭 |
| 인앱 메시지 | 노출 횟수, 클릭 수, 첫 번째 버튼 클릭 수, 두 번째 버튼 클릭 수 |
| 웹훅  |  전송, 오류 |
| SMS | 전송, 배송업체로 전송, 배송 확인, 배송 실패, 거부 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

{% alert note %}
*이동 통신사로 전송은* 더 이상 사용되지 않지만 이미 해당 기능이 있는 사용자에게는 계속 지원됩니다.
{% endalert %}

### 4단계: 보고서 설정 완료

보고서 이름을 지정하고, 보고서 서식을 선택하고, 수신자를 선택합니다. 기본값으로 참여 보고서는 쉼표로 구분된 데이터(각 데이터가 쉼표로 구분됨)가 있는 ZIP 파일로 전송됩니다.

다음 압축 및 구분 기호 옵션 중에서 선택할 수 있습니다:

- **압축:** ZIP, 비압축 또는 gzip
- **구분 기호:** 쉼표(`,`), 콜론(`:`), 세미콜론(`;`) 또는 파이프(`|`)

{% alert note %}
통계는 보고서에서 지정한 날짜 범위에 대해서만 수집됩니다. 정확한 열람율 및 클릭률 통계를 받으려면 캠페인 및 캔버스에 대해 보내기 이벤트가 수행된 시점이 포함된 날짜 범위를 선택하세요.
{% endalert %}

#### 기간 선택

기본적으로 표시되는 데이터 범위는 회사의 표준 시간대를 기준으로 하며, 가장 최근에 선택한 메시지부터 현재 날짜까지 표시됩니다. 날짜 드롭다운을 선택하고 커스텀 범위 선택을 사용하거나 다음 라디오 버튼을 선택하고 사용 가능한 드롭다운 옵션으로 날짜 범위를 정의하여 사용자 지정할 수 있습니다.

#### 데이터 표시 선택

기본적으로 참여 보고서에 표시되는 데이터는 일별(1일)입니다. 이 데이터를 다른 간격으로 보려면 보고서의 데이터를 집계할 명시적인 일수 또는 주 수를 선택합니다. 따라서 일별 측정기준 대신 주, 월, 분기 등으로 참여도를 확인할 수 있습니다. 시간 중심의 집계가 충분하지 않은 경우 캠페인 또는 캔버스 수준에서 데이터를 내보내도록 선택할 수도 있습니다.

![engagement_reports_data_coverage]({% image_buster /assets/img_archive/engagement_report_datacoverage.png %})

#### 보고서 예약하기

보고서를 예약할 때는 두 가지 옵션이 있습니다:

- **즉시 전송하세요:** 보고서가 시작되면 Braze는 이 보고서를 즉시 전송합니다.
- **지정된 시간에 전송합니다:** 이 옵션을 사용하면 이 보고서를 얼마나 자주 받을지 유연하게 선택할 수 있습니다. 이 보고서를 일, 주 또는 월 단위로 설정한 횟수마다 보내도록 선택할 수 있습니다. 보고서 전송을 중지할 시점을 정의할 수도 있습니다.

![engagement_reports_schedule_report]({% image_buster /assets/img_archive/engagement_report_reportschedule.png %}){: style="max-width:65%;" }

### 5단계: 검토 및 실행

보고서 설정의 마지막 단계에서는 구성된 옵션에 대한 보기 전용 개요가 표시됩니다. 보고서를 검토하고 만족스러우면 **보고서 시작을** 선택합니다.

### 6단계: 이메일 확인  

선택한 시간 또는 일정에 보고서 링크가 포함된 이메일을 받게 됩니다. **이러한 링크는 신고가 전송된 후 1시간 후에 만료됩니다.** 제공된 링크를 선택하면 모든 캠페인에 대해 CSV 파일이 포함된 ZIP 파일이 자동으로 다운로드됩니다.

보고서에는 설정 프로세스의 [통계 추가](#add-statistics-to-your-reports) 섹션에서 선택한 모든 통계가 포함됩니다.
