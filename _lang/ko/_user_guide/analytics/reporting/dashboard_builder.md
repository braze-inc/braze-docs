# 대시보드 빌더

> 대시보드 빌더를 사용하여 보고서 빌더 또는 쿼리 빌더에서 만든 보고서를 사용하여 대시보드 및 시각화를 만들 수 있습니다.

대시보드 빌더는 사용자 정의 분석 대시보드를 처음부터 구성하고 시각화할 수 있도록 합니다. Braze에서 제공하는 대시보드에서 시작할 수도 있습니다. 대시보드를 구동하기 위해 코드 없는 데이터 소스(보고서 빌더) 또는 SQL 데이터 소스(쿼리 빌더)를 사용할 수 있으며, Braze에서 제공하는 여러 대시보드 중 하나에서 시작할 수 있습니다.

## 사용자 지정 대시보드 만들기

1. **애널리틱스** > **대시보드 빌더로** 이동합니다.
2. **대시보드 만들기를** 선택합니다.
3. 보고서를 작성할 데이터 소스를 선택합니다:
- 보고서 빌더에서 작성된 **보고서** 
- 쿼리 빌더에서 만든 **사용자 지정 쿼리** <br><br>![대시보드를 위한 데이터 소스를 선택하는 창입니다.]({% image_buster /assets/img/select_data_source.png %})<br><br>

이제 데이터 소스에 대한 각 단계를 따르세요:

4\. **타일 추가를** 선택한 다음 [보고서 작성기(새로]({{site.baseurl}}/user_guide/analytics/reporting/report_builder/) 만들기)에서 만든 보고서 중 하나를 선택합니다.

{% alert important %}
보고서 빌더 보고서가 대시보드 빌더 타일에 추가되면, 해당 타일은 원래 보고서와 연결되지 않습니다. 보고서 빌더에서 원래 보고서를 편집하면, 기존 대시보드 타일을 삭제하고 업데이트된 보고서를 데이터 소스로 사용하여 새로 만들어야 합니다.
{% endalert %}

5\. 연필 아이콘을 선택하여 타일에 제목과 차트 유형이 표시되는 방식을 변경합니다.
    \- 기본 시각화 아래에서 다양한 차트 유형 간에 전환할 수 있습니다. 현재 옵션에는 막대형 차트(가로 또는 세로) 및 꺾은선형 차트(보고서 작성기 설정에서 **날짜를** 드릴다운 옵션으로 선택한 경우에만 사용 가능)가 있습니다.<br><br>![다양한 차트 유형을 위한 토글입니다.]({% image_buster /assets/img/report_builder_types.png %})<br><br>

6\. 원하는 대로 시각화를 변경한 후 **저장을** 선택합니다.

4\. **타일 추가를** 선택한 다음 쿼리 빌더에서 실행한 쿼리를 선택합니다.
5\. 쿼리 결과가 타일에 표시되는 방식을 편집하려면 연필 아이콘을 선택하여 제목과 차트 유형을 변경합니다.
    \- 기본 시각화 아래에서 다양한 차트 유형 간에 전환할 수 있습니다. 현재 옵션에는 표, 막대형 차트(가로 또는 세로), 꺾은선형 차트 등이 있습니다.<br><br>![다양한 차트 유형을 위한 토글입니다.]({% image_buster /assets/img/query_builder_types.png %})<br><br>
        \- 차트 옵션 중 하나를 선택하면, **엑스축** 드롭다운을 사용하여 쿼리 결과에서 사용할 단일 열을 선택합니다.
        - **Y축** 드롭다운을 사용하여 비주얼라이제이션에 포함할 메트릭을 선택합니다. 기본적으로 쿼리 결과의 모든 열이 표시되므로 보고 싶지 않은 열은 선택 해제하세요.<br><br>![다양한 차트 유형을 위한 토글입니다.]({% image_buster /assets/img/query_builder_axis.png %})<br><br>
        \- (선택 사항) **그룹화** 드롭다운을 사용하여 쿼리 결과를 그룹화할 수 있습니다. 예를 들어 열 결과로 캠페인 ID가 있고 해당 값을 가진 모든 행을 합산하려는 경우 **그룹화** 드롭다운을 사용합니다.  
        \- (선택 사항) 표시되는 데이터를 편집하려면 비주얼에 첨부된 쿼리를 선택하고 [쿼리 빌더에서]({{site.baseurl}}/user_guide/analytics/query_builder/) 편집합니다.
6\. 원하는 대로 시각화를 변경한 후 **저장을** 선택합니다.
7\. 나중에 대시보드를 더 쉽게 찾을 수 있도록 이름, 설명 및 태그를 추가하세요.

{% endalert %}

## 대시보드 타일 관리

### 타일 삭제

타일 하단에서 **타일 삭제를** 선택하여 대시보드 타일을 삭제합니다. **이 작업은 되돌릴 수 없습니다.**

### 중복 타일

타일 하단에서 **타일 복제를** 선택하여 타일의 사본을 만듭니다.

### 타일 크기 및 위치 조정

타일의 오른쪽 하단 모서리를 드래그하여 타일 크기를 조정하고, 타일의 오른쪽 상단 모서리에 있는 핸들을 드래그하여 대시보드에서 타일 위치를 조정합니다.

## 대시보드 실행

1. **애널리틱스** > **대시보드 빌더로** 이동합니다. 홈 페이지에는 작업 공간 내의 모든 기존 대시보드가 나열되며, Braze에서 생성한 대시보드가 상단에 표시됩니다. 제목에 "(Braze)"가 표시되어 있습니다.
2. 관심 있는 대시보드를 선택합니다.
3. **대시보드 실행**을 선택하여 해당 대시보드를 로드합니다.

### 사용 가능한 대시보드

Braze는 마지막 터치 기여도를 사용한 수익 분석과 같은 빈번한 사용 사례를 위한 미리 구축된 대시보드를 제공합니다. 대시보드를 편집할 수 있는 기능은 아직 제공되지 않습니다. 미래에 특정 대시보드를 보고 싶다면 고객 성공 매니저에게 문의하세요.

#### 매출 - 마지막 터치 기여도

**수익 - 마지막 터치 기여도** 대시보드는 캠페인, 캔버스 및 채널 전반의 수익을 검토합니다. 모든 매출 데이터는 어트리뷰션 기간 동안 마지막으로 터치한 메시지에 어트리뷰션됩니다.

터치에는 _이메일 클릭_(링크 클릭), _콘텐츠 카드 클릭_, _인앱 메시지 클릭_(닫기 버튼 제외), _푸시 오픈_, _SMS 단축 링크 클릭_, _WhatsApp 읽기_, 및 _웹훅 전송_이 포함됩니다.

| Metric | Definition |
| --- | --- |
| 총 라스트 터치 수익 | 선택한 날짜 범위 및 어트리뷰션 기간 내에 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계입니다. |
| 총 구매 전환 수 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 수입니다. |
| 평균 전환 소요 일수 | 적격 라스트 터치 이벤트가 있는 모든 캠페인과 캔버스 구매 이벤트 사이의 평균 시간입니다. |
| 수신자 당 수익 | 적격 수익 이벤트에서 발생한 수익의 합계를 날짜 범위 내에서 메시지를 받은 순 사용자 수로 나눈 값입니다. |
| 고유 구매자 | 적격 수익 이벤트가 있는 순 사용자 수입니다. |
| 국가별 수익 | 국가별로 그룹화된 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계입니다. |
| 캠페인별 수익 | 캠페인별로 그룹화된 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계입니다. |
| 캠페인 변형별 수익 | 캠페인 변형별로 그룹화된 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계입니다. |
| 캔버스별 수익 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 캔버스별로 그룹화됩니다. |
| 캔버스 변형별 수익 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계로, 캔버스 배리언트별로 그룹화됩니다. |
| 제품별 구매 | 제품별로 그룹화된 모든 구매의 수입니다. |
| 채널별 수익 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계로, 채널별로 그룹화됩니다. | 
| 수익 시계열 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 수익 이벤트의 합계로, UTC 기준 요일별로 그룹화됩니다. |
