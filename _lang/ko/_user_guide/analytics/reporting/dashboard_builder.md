# 대시보드 빌더

> 보고서 빌더 또는 쿼리 빌더에서 만든 보고서를 사용하여 대시보드 및 시각화를 만들려면 대시보드 빌더를 사용하세요.

대시보드 빌더를 사용하면 처음부터 커스텀 분석 대시보드를 구성하고 시각화할 수 있으며, Braze에서 제공하는 대시보드도 사용할 수 있습니다. 코드가 없는 데이터 소스(보고서 빌더) 또는 SQL 데이터 소스(쿼리 빌더)를 사용하여 대시보드를 만들거나, 여러 Braze에서 제공하는 대시보드 중 하나에서 시작할 수 있습니다.

## 커스텀 대시보드 만들기

1. **분석** > **대시보드 빌더로** 이동합니다.
2. **대시보드 만들기를** 선택합니다.
3. 보고서를 작성할 데이터 소스를 선택합니다:
- 보고서 빌더에서 구축된 **보고서** 
- 쿼리 빌더에서 만든 **커스텀 쿼리** <br><br>대시보드의 데이터 소스를 선택할 수 있는 창입니다.]({% image_buster /assets/img/select_data_source.png %})<br><br>

이제 데이터 소스에 대한 각 단계를 따르세요:

## 대시보드 타일 관리하기

### 타일 삭제

타일 하단의 **타일 삭제를** 선택하여 대시보드 타일을 삭제합니다. **이 작업은 되돌릴 수 없습니다.**

### 중복 타일

타일 하단에서 **타일 복제를** 선택하여 타일의 사본을 만듭니다.

### 타일 크기 및 위치 조정

타일의 오른쪽 하단 모서리를 드래그하여 타일 크기를 조정하고, 타일의 오른쪽 상단 모서리에 있는 핸들을 드래그하여 대시보드에서 타일 위치를 조정합니다.

## 대시보드 실행하기

1. **분석** > **대시보드 빌더로** 이동합니다. 홈 페이지에는 워크스페이스 내의 모든 기존 대시보드가 나열되며, 상단에 Braze에서 만든 대시보드가 표시됩니다. 제목에 "(Braze)"가 표시되어 있습니다.
2. 관심 있는 대시보드를 선택합니다.
3. **대시보드 실행을** 선택하면 해당 대시보드를 사용하여 해당 대시보드를 로드할 수 있습니다.

### 사용 가능한 대시보드

Braze는 라스트 터치 기여도를 이용한 매출 분석과 같이 자주 사용되는 사용 사례를 위해 미리 구축된 대시보드를 제공합니다. 대시보드 편집 기능은 아직 사용할 수 없습니다. 향후 특정 대시보드를 보고 싶다면 고객 성공 매니저에게 문의하세요.

#### 매출 - 라스트 터치 기여도

**매출 - 라스트 터치 기여도** 대시보드에서는 캠페인, 캔버스, 채널 전반의 매출을 검토할 수 있습니다. 모든 매출 데이터는 어트리뷰션 기간 동안 마지막으로 터치한 메시지에 기여도 속성이 부여됩니다.

터치에는 _이메일 클릭_ (링크 클릭), _콘텐츠 카드 클릭_, _인앱 메시지 클릭_ (닫기 버튼 제외), _푸시 열기_, _SMS 짧은 링크 클릭_, _WhatsApp 읽기_ 및 _웹훅 전송이_ 포함됩니다.

| 측정기준 | 정의 |
| --- | --- |
| 총 라스트 터치 매출 | 선택한 날짜 범위 및 속성 기간 내에 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계입니다. |
| 총 구매 전환 수 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 수입니다. |
| 평균 전환 소요 일수 | 적격 라스트 터치 이벤트가 있는 모든 캠페인과 캔버스 구매 이벤트 사이의 평균 시간입니다. |
| 수신자당 매출 | 적격 매출 이벤트에서 발생한 매출의 합계를 날짜 범위 내에서 메시지를 수신한 순 사용자 수로 나눈 값입니다. |
| 고유 구매자 | 적격 매출 이벤트가 있는 고유 사용자 수입니다. |
| 국가별 매출 | 라스트 터치 이벤트가 포함된 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 국가별로 그룹화됩니다. |
| 캠페인별 매출 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 캠페인별로 그룹화합니다. |
| 캠페인 배리언트별 매출 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 캠페인 배리언트별로 그룹화합니다. |
| 캔버스별 매출 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 캔버스별로 그룹화합니다. |
| 캔버스 배리언트별 매출 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 캔버스 배리언트별로 그룹화합니다. |
| 제품별 구매 | 제품별로 그룹화된 모든 구매 횟수입니다. |
| 채널별 매출 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, 채널별로 그룹화됩니다. | 
| 매출 시계열 | 적격 라스트 터치 이벤트가 있는 모든 캠페인 및 캔버스 매출 이벤트의 합계로, UTC 기준으로 요일별로 그룹화됩니다. |
