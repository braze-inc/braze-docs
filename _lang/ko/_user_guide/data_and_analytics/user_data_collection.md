{% alert important %}
Braze는 일반적으로 잘못된 통합의 결과이므로 5백만 세션이 넘는 사용자("더미 사용자")를 금지 또는 차단하고 더 이상 해당 사용자의 SDK 이벤트를 수집하지 않습니다. 합법적인 사용자에게 이런 일이 발생한 경우 Braze 계정 관리자에게 문의하세요.
{% endalert %}
