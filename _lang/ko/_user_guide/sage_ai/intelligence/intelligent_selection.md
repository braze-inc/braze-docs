# 지능형 선택 {#intelligent-selection}

> 지능형 선택은 하루에 두 번 반복되는 캠페인 또는 캔버스의 성과를 분석하여 각 메시지 변형을 수신하는 사용자의 비율을 자동으로 조정하는 기능입니다. 

다른 변형보다 실적이 더 좋은 것으로 보이는 변형은 더 많은 사용자에게 전송되고, 실적이 저조한 변형은 더 적은 수의 사용자에게 타겟팅됩니다. 각 조정은 \[통계 알고리즘][227] ]을 사용하여 무작위적인 우연이 아닌 실제 성능 차이에 맞게 조정합니다.

![지능형 선택이 활성화된 캠페인의 A/B 테스트 섹션입니다.][3]

지능형 선택이 가능합니다:
- 성과 데이터를 반복적으로 살펴보고 캠페인 트래픽을 점차적으로 우승 변형으로 전환하세요.
- 통계적 신뢰도를 유지하면서 더 많은 사용자가 가장 성능이 좋은 변형을 받을 수 있는지 확인하세요.
- 성과가 저조한 변형을 배제하고 성과가 우수한 변형을 [기존 A/B 테스트보다][1] 빠르게 식별할 수 있습니다.
- 사용자가 최상의 메시지를 볼 수 있도록 더 자주, 더 자신 있게 테스트하세요. 

지능형 선택은 여러 번 전송하도록 예약된 캠페인에 이상적입니다. 캠페인 조정을 시작하려면 초기 결과가 필요하므로 한 번만 전송하는 캠페인은 혜택을 받지 못합니다. 이러한 캠페인의 경우 [A/B 테스트가][1] 더 효과적일 수 있습니다.

## 내 캠페인에 지능형 선택을 추가하려면 어떻게 해야 하나요?

### 캠페인 인텔리전트 선택
지능형 선택은 Braze 캠페인 작성기의 **타겟 오디언스** 단계에서 모든 다중 전송 캠페인에 추가할 수 있습니다. 한 번만 전송하는 캠페인은 이 기능을 활용할 수 없습니다.

### 캔버스 인텔리전트 셀렉션
캔버스에 이형 상품을 추가할 때 이형 상품 비율 중 하나를 클릭합니다. 이를 통해 이형 상품 배포를 편집하고 지능형 선택을 켤 수 있습니다.

![두 개의 이형 상품이 있는 캔버스에서 각각 50%의 이형 상품 분포로 설정하여 지능형 선택을 활성화할 수 있습니다.][2]

아직 캔버스에 전환 이벤트를 추가하지 않았거나 캠페인이 단독 변형으로 구성된 경우에는 인텔리전트 셀렉션을 사용할 수 없습니다.

{% alert note %}
24시간 이내에 재적격성이 활성화된 캠페인에 지능형 선택을 사용하면 제어 변형의 무결성에 영향을 미칠 수 있으므로 허용되지 않습니다. 자세한 내용은 [인텔리전스 FAQ를]({{site.baseurl}}/user_guide/intelligence/faqs/#why-is-re-eligibility-not-available-when-combined-with-intelligent-selection) 참조하세요.
{% endalert %}

## 얼마나 오래 운영되나요?

캠페인 및 캔버스의 경우, 지능형 선택은 이형 상품의 '실제' 전환율에 대한 충분한 증거를 수집할 때까지 실행됩니다. "충분함"은 "후회"라는 특별한 지표에 의해 결정됩니다. 어떤 변형이 가장 적합한지 알 수 있는 충분한 데이터가 확보되면 지능형 선택 기능이 저절로 꺼질 것이라는 확신과 비슷하다고 생각하면 됩니다. 

대부분의 경우 지능형 선택은 이형 상품 중 하나를 우승 이형 상품으로 선택합니다. 이 변형은 향후 전송 시 100%의 오디언스에게 제공됩니다.

{% alert note %}
지능형 선택은 하나의 확실한 승자를 선택하지 않고 최적화를 중단할 수 있습니다. 지능형 선택은 실험을 계속해도 전환율이 현재 전환율의 1% 이상 개선되지 않을 것이라는 95%의 확신이 들면 최적화를 중지합니다.
{% endalert %}
