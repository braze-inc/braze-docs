  - name: Braze Teams

    description: "<a href=\"https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/connected_content/\">연결된 콘텐츠는</a> 마케팅 개인화를 확장하여 고객 참여와 전환을 촉진합니다. API를 사용하여 액세스할 수 있는 모든 정보를 사용자에게 보내는 메시징에 직접 삽입할 수 있습니다. 연결된 콘텐츠를 사용하면 웹 서버 또는 공개적으로 액세스 가능한 API에서 직접 콘텐츠를 가져올 수 있습니다."

Liquid에서는 오브젝트, 태그, 필터를 사용해 <a href=\"http://www.braze.com/docs/user_guide/personalization_and_dynamic_content/liquid/supported_personalization_tags/\">개인화된 커스텀을 추가할</a> 수 있습니다."
