  - name: Braze 팀

    description: "<a href=\"https://www.braze.com/docs/user_guide/personalization_and_dynamic_content/connected_content/\">연결된 콘텐츠</a>는 마케팅 개인화를 확장하여 고객 참여와 전환을 촉진합니다. 이 기능을 사용하면 API를 통해 액세스할 수 있는 모든 정보를 사용자에게 보내는 메시지에 직접 삽입할 수 있습니다. 연결된 콘텐츠를 사용하면 웹 서버 또는 공개적으로 액세스할 수 있는 API에서 직접 콘텐츠를 가져올 수 있습니다."

  - name: 커런츠

Liquid에서는 개체, 태그 및 필터를 사용하여 <a href=\"http://www.braze.com/docs/user_guide/personalization_and_dynamic_content/liquid/supported_personalization_tags/\">개인화 설정을 추가할</a> 수 있습니다."
