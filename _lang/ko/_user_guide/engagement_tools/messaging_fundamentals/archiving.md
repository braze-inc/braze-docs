# 캠페인 및 캔버스 아카이빙

> 캠페인과 캔버스를 보관하는 것은 작업 공간을 간단하고 효과적으로 정리하는 방법이 될 수 있습니다. 계속해서 더 많은 메시지를 작성할 때 아카이빙을 사용하면 적극적이고 관련성 있는 메시징 활동에 집중할 수 있습니다.

## 아카이빙

캠페인 또는 캔버스를 보관하면 더 이상 메시지가 전달되지 않으며 인앱 메시지가 표시되지 않습니다. 결과가 보관된 후에도 결과를 볼 수는 있지만 캠페인이나 캔버스를 편집할 수는 없습니다. 그렇게 하려면 먼저 [아카이브를 해제해야](#unarchiving-campaigns-and-canvases) 합니다.

{% tabs local %}
{% tab 캠페인 %}
캠페인을 보관하려면:

1. **메시징** > **캠페인으로** 이동합니다.
2. 캠페인 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택합니다.
3. **아카이브를** 선택합니다.
{% endtab %}

{% tab 캔버스 %}
캔버스를 보관하려면

1. **메시징** > **캔버스로** 이동합니다.
2. 캔버스의 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택합니다.
3. **아카이브를** 선택합니다.
{% endtab %}
{% endtabs %}

## 보관 해제

캠페인이나 캔버스의 보관을 취소하면 해당 캠페인이나 캔버스의 상태가 **활성으로만** 변경될 뿐 라이브 상태가 되지는 않습니다. 보관 해제 후에는 메시지가 전송되지 않은 상태에서 메시지 설정 방식을 편집하고 검토할 수 있습니다.

캠페인 또는 캔버스를 재개하고 메시지 전송을 시작하려면 해당 행의 g<i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택하고 **재개를** 선택합니다.
