# 캠페인 및 캔버스 복제하기

> 기존 캠페인과 캔버스를 복제하여 성공적인 전략을 지속적으로 반복하고 구축할 수 있습니다. 캠페인과 캔버스를 복제하면 전달 세부 정보 및 오디언스 세그먼트를 재구성하는 데 소요되는 시간을 최소화할 수 있습니다.

## 복제

캠페인 또는 캔버스를 복제하면 새 메시지의 제목이 `"Copy of [Original Name]"` 으로 자동 변경되고 해당 메시지의 편집 페이지로 이동합니다. 이러한 복제본은 편집하고 실행할 때까지 초안으로 유지됩니다.

{% alert tip %}
워크스페이스 간에 [캠페인을 복사할]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/copying_across_workspaces/) 수도 있습니다. 사본은 편집하고 실행할 때까지 초안으로 유지됩니다.
{% endalert %}

{% tabs %}
{% tab campaign %}
캠페인을 복제하려면:

1. **메시징** > 캠페인으로 이동합니다.
2. 캠페인 위로 마우스를 가져가 <i class="fas fa-gear"></i> **추가 작업을** 선택합니다.
3. **복제를** 선택합니다.
{% endtab %}

{% tab canvas %}
캔버스를 복제하려면:

1. **메시징** > **캔버스로** 이동합니다.
2. 캔버스 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> **추가 작업을** 선택합니다.
3. **복제를** 선택합니다.

{% endtab %}
{% endtabs %}
