# 캠페인 및 캔버스 복제

> 기존 캠페인과 캔버스를 복제하여 성공적인 전략을 지속적으로 반복하고 구축할 수 있습니다. 캠페인과 캔버스를 복제하면 전달 세부 정보 및 대상 세그먼트를 재구성하는 데 소요되는 시간을 최소화할 수 있습니다.

## 복제

캠페인 또는 캔버스를 복제하면 새 메시지의 제목이 `"Copy of [Original Name]"` 으로 자동 변경되고 해당 메시지 편집 페이지로 이동합니다. 이러한 복제본은 편집하고 실행할 때까지 초안으로 유지됩니다.

{% tabs %}
{% tab 캠페인 %}
캠페인을 복제하려면:

1. **메시징** > **캠페인으로** 이동합니다.
2. 캠페인 위로 마우스를 가져가 <i class="fas fa-gear"></i> **추가 작업을** 선택합니다.
3. **중복** 선택.
{% endtab %}

{% tab 캔버스 %}
캔버스를 복제하려면

1. **메시징** > **캔버스로** 이동합니다.
2. 캔버스 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> **추가 작업을** 선택합니다.
3. **중복** 선택.

![캔버스 목록 페이지에 "api_triggered_canvas 3"이라는 이름의 캔버스를 복제하는 옵션이 표시됩니다.]({% image_buster /assets/img_archive/duplicate_canvas.png %})

{% details 원래 캔버스 편집기 %}

2023년 2월 28일부터는 더 이상 기존 캔버스 환경을 사용하여 캔버스를 만들거나 복제할 수 없습니다. 

Braze는 기존 캔버스 환경을 사용하는 고객은 캔버스 흐름으로 전환할 것을 권장합니다. 향상된 편집 경험을 통해 캔버스를 더 잘 구축하고 관리할 수 있습니다. [캔버스를 캔버스 플로우에 복제하는 방법]({{site.baseurl}}/user_guide/engagement_tools/canvas/managing_canvases/cloning_canvases/)에 대해 자세히 알아보세요.
