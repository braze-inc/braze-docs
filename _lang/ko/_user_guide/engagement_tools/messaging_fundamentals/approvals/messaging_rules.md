# 메시징 규칙

> 승인 워크플로에서 메시징 규칙을 사용하여 추가 승인이 필요하기 전에 도달 가능한 사용자 수를 제한하면 더 많은 오디언스를 타겟팅하기 전에 캠페인과 캔버스를 검토할 수 있습니다.

## How it works

메시징 규칙은 워크스페이스에 적용되며 메시지 유형과 최대 도달 가능한 사용자 수로 구성됩니다.

- **메시지 유형:** 규칙을 적용할 메시지 유형(캠페인, 캔버스 또는 캔버스와 캠페인 모두)을 정의합니다.
- **도달 가능한 최대 사용자 수:** 추가 승인이 필요한 대상자 규모를 결정합니다.

### 공유 메시지 유형 및 최대 도달 가능한 사용자

동일한 메시지 유형에 대해 도달 가능한 사용자 수가 동일한 두 개의 규칙이 존재할 수 있습니다. 예를 들어, 캔버스에 최대 10,000명의 사용자를 설정하고 캔버스와 캠페인 모두에 대해 10,000명의 사용자를 설정할 수 있습니다. 

### 별도의 승인자

최대 두 개의 규칙이 동일한 사용자를 공유할 수 있으므로 승인자별로 규칙을 구성하고 분리할 수 있습니다. 예를 들어 다음 두 가지 규칙을 만듭니다:

- 법무팀에 승인자가 있는 최대 100,000명의 사용자가 있는 캔버스에 대한 규칙 A
- 마케팅 팀에 승인자가 있는 최대 100,000명의 사용자가 있는 캔버스에 대한 규칙 B 

### 도달 가능한 사용자 중복 없음

동일한 메시지 유형에 대해 사용자 수가 겹치는 규칙은 설정할 수 없습니다. 예를 들어 다음과 같은 메시징 규칙은 설정할 **수 없습니다**: 

- 최대 10,000명의 사용자가 있는 캔버스에 대한 규칙 C 
- 최대 1,000,000명의 사용자가 있는 캔버스에 대한 규칙 D

## 메시징 규칙 만들기

### Prerequisites

브레이즈 관리자만 메시징 규칙을 설정할 수 있지만, 모든 브레이즈 사용자는 메시징 규칙 승인자가 될 수 있습니다(일반 승인 권한이 없는 사용자 포함).

### 1단계: 규칙 추가하기

{% alert note %}
메시징 규칙은 최대 5개까지 만들 수 있습니다.
{% endalert %}

1. **설정** > **승인 워크플로** > **메시징 규칙으로** 이동합니다.
2. **규칙 만들기를** 선택합니다.
3. 이 규칙에 이름을 지정합니다(예: "모든 사용자 구독").
4. **메시지 유형에서** **캠페인**, **캔버스** 또는 **캔버스와 캠페인 모두를** 선택하여 승인 규칙을 적용합니다.
5. **최대 도달 가능한 사용자** 수에 대한 숫자를 입력합니다. 자세한 내용은 [오디언스 통계를]({{site.baseurl}}/user_guide/engagement_tools/campaigns/building_campaigns/targeting_users#audience-statistics) 참조하세요.
6. Select **Save**.

![최대 사용자 수가 100,000명인 캠페인에 대한 메시징 규칙 '규칙 1'의 예시입니다. 캔버스와 캠페인 시작을 승인할 수 있는 사용자는 한 명입니다.]({% image_buster /assets/img/target_population_approval_example.png %}){: style="max-width:90%;"}

### 2단계: 승인을 받아 출시 결정(선택 사항)

**승인 후 실행 허용을** 선택합니다. 다음으로, **승인 대상에서** 캔버스 또는 캠페인을 승인할 수 있는 권한이 있는 승인자를 선택합니다(최대치가 충족되는 경우).

승인을 받은 메시지를 실행할 때는 다음 세부 사항을 참고하세요:

- 최대치를 충족하고 승인자가 선택되면 승인 권한이 있는 Braze 사용자는 **대상 고객** 승인 드롭다운에서 **승인됨을** 선택할 수 있습니다.
- 최대치를 충족하고 승인을 선택하지 않으면 캔버스 또는 캠페인이 시작되지 않습니다.

## Frequently asked questions

### 메시징 규칙을 사용하려면 권한을 다시 구성해야 하나요?

아니요. 현재 권한에 관계없이 모든 사용자를 대상 집단 승인자로 선택할 수 있습니다.

### 메시징 규칙은 타겟 오디언스 단계와 어떤 관련이 있나요?

메시징 규칙은 이벤트 트리거와 같은 세부 사항을 고려하지 않습니다. 예를 들어, 캠페인은 모든 사용자를 대상으로 할 수 있습니다. 하지만 이 캠페인은 이벤트 트리거 방식이므로 실제 혜택을 받는 사용자는 더 적습니다.

### 메시징 규칙을 켜면 자동으로 변경되는 사항이 있나요?

아니요. 이 기능을 사용하도록 설정한 후에는 최대 사용자 수를 수동으로 입력하고 승인자를 선택해야 기능을 사용할 수 있습니다.
