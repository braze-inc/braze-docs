# 우승 배리언트 또는 개인화된 배리언트로 A/B 테스트 최적화하기

한 번 전송하도록 예약된 이메일, 푸시, 웹훅, SMS 및 WhatsApp 캠페인에 대한 [A/B 테스트를 만들]({{site.baseurl}}/user_guide/engagement_tools/testing/multivariant_testing/create_multivariate_campaign/) 때 최적화를 선택할 수 있습니다. 두 가지 최적화 옵션이 있습니다: **우승 배리언트** 및 **개인화된 배리언트**.

타겟팅 오디언스를 선택할 때 A/B 테스트 섹션에 나열된 최적화 옵션. 세 가지 옵션이 나열됩니다: 최적화 없음, 우승 배리언트 및 개인화된 배리언트. 개인화된 배리언트가 선택되었습니다.]({% image_buster /assets/img_archive/ab_personalized_variant.png %})

두 옵션 모두 타겟 세그먼트의 일정 비율에 초기 테스트를 전송하는 방식으로 작동합니다. 테스트가 종료된 후 오디언스의 나머지 사용자에게는 성능/성과가 가장 우수한 배리언트(우승 배리언트) 또는 참여 가능성이 가장 높은 배리언트(개인화된 배리언트)가 전송됩니다.

{% alert tip %}
최적화는 캠페인 생성의 **타겟 오디언스** 단계의 **A/B 테스트** 아래에 있습니다.
{% endalert %}

## 우승 배리언트

우승 배리언트를 보내는 것은 표준 A/B 테스트와 유사합니다. 이 그룹의 사용자는 초기 테스트가 완료되면 우승 배리언트를 받게 됩니다.

1. 캠페인 **배리언트를** 선택한 다음, 캠페인 오디언스 중 몇 퍼센트를 배리언트 그룹에 할당할지 지정합니다.
2. 다음 추가 설정을 구성합니다.

| 필드 | 설명 |
| --- | --- | 
| 우승 배리언트 결정하기 | 최적화할 측정기준입니다. 이메일의 경우 *고유 열람* 또는 *클릭* 수, 푸시의 경우 *열람* 수 또는 모든 채널의 *기본 전환율* 중에서 선택합니다. *열기* 또는 *클릭을* 선택하여 당첨자를 결정하는 것은 캠페인의 전환 이벤트에 대해 선택한 항목에 영향을 미치지 않습니다. <br><br>대조군을 사용하는 경우 대조군의 사용자는 *열기* 또는 *클릭을* 수행할 수 없으므로 대조군의 성능/성과는 `0` 으로 보장된다는 점에 유의하세요. 결과적으로 대조군은 A/B 테스트에서 이길 수 없습니다. 그러나 메시지를 받지 않는 사용자에 대한 다른 측정기준을 추적하기 위해 대조군을 사용할 수도 있습니다. |
| 승리한 배리언트 전송 시간 | 당첨된 배리언트가 전송된 날짜와 시간입니다. |
| 우승 배리언트를 결정할 수 없는 경우 | 통계적으로 유의미한 차이로 승리한 배리언트가 없는 경우 어떻게 될까요? 어쨌든 가장 성능/성과가 좋은 배리언트를 보내거나 테스트를 종료하고 더 이상 메시지를 보내지 않는 것 중에서 선택합니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 개인화된 배리언트

개인화된 배리언트를 사용하여 타겟팅 세그먼트의 각 사용자에게 참여 가능성이 가장 높은 배리언트를 보낼 수 있습니다.

각 사용자에게 가장 적합한 배리언트를 결정하기 위해 Braze는 일부 타겟 오디언스에게 초기 테스트를 전송하여 사용자 특성과 메시지 선호도 간의 연관성을 찾습니다. 초기 테스트에서 각 배리언트에 대한 사용자의 반응에 따라 이러한 특성을 바탕으로 나머지 사용자에게 각 배리언트를 제공할지 결정합니다. 연결이 발견되지 않고 개인화를 수행할 수 없는 경우, 우승 배리언트는 자동으로 나머지 사용자에게 전송됩니다. 개인화된 배리언트가 결정되는 방법에 대해 자세히 알아보려면 [다변량 및 A/B 테스트 분석을]({{site.baseurl}}/user_guide/engagement_tools/testing/multivariant_testing/multivariate_analytics/#personalized-variant) 참조하세요.

1. **개인화된 배리언트를** 선택한 다음 캠페인 오디언스 중 몇 퍼센트를 개인화된 배리언트 그룹에 할당할지 지정합니다.
2. 다음 추가 설정을 구성합니다.

| 필드 | 설명 |
| --- | --- | 
| 개인화된 배리언트 결정하기 | 최적화할 측정기준입니다. 이메일의 경우 *고유 열람* 또는 *클릭* 수, 푸시의 경우 *열람* 수 또는 모든 채널의 *기본 전환율* 중에서 선택합니다. *열기* 또는 *클릭을* 선택하여 당첨자를 결정하는 것은 캠페인의 전환 이벤트에 대해 선택한 항목에 영향을 미치지 않습니다. <br><br>대조군을 사용하는 경우 대조군의 사용자는 *열기* 또는 *클릭을* 수행할 수 없으므로 대조군의 성능/성과는 `0` 으로 보장된다는 점에 유의하세요. 결과적으로 대조군은 A/B 테스트에서 이길 수 없습니다. 그러나 메시지를 받지 않는 사용자에 대한 다른 측정기준을 추적하기 위해 대조군을 사용할 수도 있습니다. |
| 개인화된 배리언트 전송 시간 | 개인화된 배리언트가 전송되는 날짜와 시간입니다. |
| 개인화된 배리언트를 결정할 수 없는 경우 | 개인화된 배리언트를 찾지 못하면 어떻게 되나요? 대신 우승 배리언트를 보내거나 테스트를 종료하고 더 이상 메시지를 보내지 않는 방법 중에서 선택합니다. |
