# 위치 타겟팅

> 이 문서에서는 위치 타겟팅을 설정하는 방법을 안내하여 가장 최근 위치를 기준으로 사용자를 세분화할 수 있도록 합니다. 위치 기반 캠페인 및 전략을 검토하는 경우에 적합합니다.

## 1단계: 세그먼트 만들기

**오디언스** 아래의 **세그먼트** 페이지로 이동하여 현재 모든 사용자 세그먼트를 확인합니다. 이 페이지에서 새 세그먼트를 만들고 이름을 지정할 수 있습니다. 시작하려면 **세그먼트 만들기를** 선택하고 세그먼트에 이름을 지정합니다.

세그먼트를 만들려면 모달을 클릭합니다.]({% image_buster /assets/img_archive/createsegment2.png %}){: style="max-width:70%;"}

## 2단계: 위치 커스텀하기

세그먼트를 생성한 후 **가장 최근 위치** 필터를 추가하여 사용자가 마지막으로 앱을 사용한 장소를 기준으로 사용자를 타겟팅합니다. 표준 원형 영역 또는 사용자 지정 가능한 다각형 영역 안팎의 사용자를 강조 표시하는 옵션이 있습니다.

원 안의 가장 최근 위치로 필터링합니다.]({% image_buster /assets/img_archive/filter_recent_location.png %})

뉴저지와 뉴욕 사이의 도시를 원형으로 표시한 윤곽선.]({% image_buster /assets/img_archive/location_circle.png %}){: style="max-width:70%;"}

선택한 다각형 영역으로 뉴욕주의 윤곽선입니다.]({% image_buster /assets/img_archive/create_polygon.png %}){: style="max-width:70%;"}
