# 세그먼트 관리하기

> 세그먼트 섹션에서는 기존 세그먼트의 전체 목록을 보고, 새 세그먼트를 만들고, 기존 세그먼트를 편집할 수 있습니다. 다양한 필터와 열을 선택하여 세그먼트 목록을 세분화하여 나에게 가장 관련성이 높은 정보만 표시되도록 할 수 있습니다.

활성 세그먼트 목록을 표시하는 세그먼트 섹션입니다.]({% image_buster /assets/img/segment/segments_page.png %})

## 뷰 커스텀하기

필터를 사용하고 표시할 열을 변경하여 세그먼트 목록 보기를 맞춤 설정할 수 있습니다. **세그먼트** 섹션에서 나갔다가 돌아오면 목록이 기본값 보기로 되돌아가 이전에 선택한 모든 필터가 지워집니다.

### 상태 필터

목록을 세분화하여 활성 또는 보관된 세그먼트만 표시할 수 있습니다. 보관되지 않은 세그먼트는 모두 활성 상태로 간주됩니다.

### 필터

다음 필터를 조정하여 목록의 세그먼트를 세분화할 수 있습니다:
- **마지막 편집자:** 세그먼트를 마지막으로 편집한 사용자
- **마지막 편집되었습니다:** 세그먼트가 마지막으로 편집된 시간 범위
- **예상 크기:** 세그먼트 내 사용자 수에 대한 대략적인 범위
- **태그:** 세그먼트와 관련된 태그
- **Teams:** 세그먼트와 연결된 팀(Teams)
- **고급 추적 세그먼트만 진행:** [분석 추적이]({{site.baseurl}}/user_guide/data_and_analytics/tracking/segment_analytics_tracking#segment-analytics-tracking) 활성화된 세그먼트만 볼 수 있습니다.

### 열

다음은 세그먼트 목록에 표시하도록 선택할 수 있는 정보 열입니다:
- **필터:** 세그먼트의 필터 수
- **마지막 편집되었습니다:** 세그먼트가 마지막으로 편집된 날짜
- **마지막 편집자는 다음과 같습니다:** 세그먼트를 마지막으로 편집한 사용자
- **태그:** 세그먼트와 관련된 태그
- **Teams:** 세그먼트와 연관된 Teams
- **예상 크기:** 세그먼트의 예상 사용자 수
- **캔버스:** 세그먼트를 사용하는 캔버스 수
- **캠페인:** 세그먼트를 사용하는 캠페인 수

### 별표만 표시

별표 **표시된 항목만 표시** 를 선택하면 내가 별표 표시한 세그먼트로만 보기가 좁혀집니다.

## 세그먼트의 메시징 사용량 보기

세그먼트의 **메시징 사용** 섹션으로 이동하여 다른 세그먼트, 캠페인, 캔버스 등 해당 세그먼트가 사용되는 위치에 대한 개요를 확인할 수 있습니다.

{% alert note %}
세그먼트가 서로를 참조하는 루프를 방지하기 위해 **세그먼트 멤버십** 필터를 사용하는 세그먼트는 다른 세그먼트에서 참조할 수 없습니다. 자세한 내용은 [세분화 필터를]({{site.baseurl}}/user_guide/engagement_tools/segments/segmentation_filters/) 참조하세요.
{% endalert %}

## 특정 세그먼트 관리하기

"편집", "복제", "보관", "별표에 추가" 옵션이 표시된 세그먼트의 편집 메뉴입니다.]({% image_buster /assets/img/segment/segments_page_edit_menu.png %}){: style="float:right;max-width:25%;"}

특정 세그먼트를 관리하려면 해당 세그먼트에 마우스를 갖다 대고 행 끝에 있는 메뉴 아이콘을 선택하면 다음과 같은 옵션이 표시됩니다:
- **편집:** 세그먼트의 필터를 편집합니다.
- **복제:** 세그먼트의 사본을 만듭니다.
- **아카이브:** 세그먼트를 보관합니다. 이렇게 하면 해당 세그먼트를 사용하는 모든 캠페인이나 캔버스도 아카이브됩니다.
- **별표에 추가합니다:** 세그먼트에 별표를 표시하면 세그먼트 섹션에서 별표만 표시 상자를 선택하여 해당 세그먼트에 빠르게 액세스할 수 있습니다.
 
여러 세그먼트 이름 옆의 확인란을 선택하여 대량 작업(특히 대량 보관 및 대량 태그 지정)을 수행할 수도 있습니다.

!"[다른 이름으로 태그" 드롭다운 필드에서 "CRM"을 선택하여 여러 세그먼트를 선택합니다.]({% image_buster /assets/img/segment/segments_bulk_action.png %}){: style="max-width:45%;"}

### 마지막으로 본 이후 변경 사항

다른 팀원이 세그먼트를 업데이트한 횟수는 세그먼트 개요 페이지의 *마지막 본 이후 변경* 사항 측정기준을 통해 추적할 수 있습니다. **마지막으로 본 이후 변경사항을** 선택하여 세그먼트의 이름, 설명 및 타겟팅 오디언스에 대한 업데이트 체인지로그를 확인합니다. 각 업데이트에 대해 누가 언제 업데이트를 수행했는지 확인할 수 있습니다. 이 체인지로그를 사용하여 세그먼트의 변경 사항을 감사할 수 있습니다.

## 세그먼트 검색하기
검색 필드에 단어를 입력하여 세그먼트 이름을 검색합니다. 

이 필드에 입력한 모든 용어와 문자열이 검색됩니다. 예를 들어 '테스트 세그먼트 1'을 검색하면 이름에 '테스트', '세그먼트' 또는 '1'이 포함된 세그먼트가 반환됩니다. 정확한 문자열을 검색하려면 검색어 주위에 따옴표를 넣으세요. "테스트 세그먼트 1"]을 검색하면 이름에 "테스트 세그먼트 1"이라는 정확한 문구가 포함된 모든 세그먼트가 반환됩니다.

검색 필드에 '모든 사용자'를 입력한 검색 결과에는 '모든 사용자(테스트)', '모든 사용자', '모든 사용자 15'가 표시됩니다.]({% image_buster /assets/img/segment/segments_search.png %})
