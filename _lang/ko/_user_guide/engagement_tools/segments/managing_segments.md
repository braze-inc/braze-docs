# 세그먼트 관리하기

> 세그먼트 섹션에서는 기존 세그먼트의 종합적인 목록을 보고, 새 세그먼트를 만들고, 기존 세그먼트를 편집할 수 있습니다. 다양한 필터와 열을 선택하여 세그먼트 목록을 세분화하면 가장 관련성이 높은 정보만 표시할 수 있습니다.

![활성 세그먼트 목록이 표시된 세그먼트 섹션.]({% image_buster /assets/img/segment/segments_page.png %})

## 보기 커스텀하기

필터를 사용하고 표시할 열을 변경하여 세그먼트 목록 보기를 맞춤 설정할 수 있습니다. **세그먼트** 섹션에서 나갔다가 돌아오면 목록이 기본값 보기로 되돌아가며 이전에 선택한 모든 필터가 초기화됩니다.

### 상태 필터

활성 또는 아카이브된 세그먼트만 표시하도록 목록의 범위를 좁힐 수 있습니다. 아카이브되지 않은 세그먼트는 모두 활성 세그먼트로 간주됩니다.

### 필터

다음 필터를 조정하여 목록의 세그먼트를 정렬합니다:
- **마지막 편집자:** 세그먼트를 마지막으로 편집한 사용자
- **마지막 편집:** 세그먼트가 마지막으로 편집된 시간 범위
- **예상 크기:** 세그먼트 내 사용자 수의 대략적인 범위
- **태그:** 세그먼트와 연결된 태그
- **Teams:** 세그먼트와 연결된 Teams
- **고급 추적 세그먼트만:** [분석 추적]({{site.baseurl}}/user_guide/data_and_analytics/tracking/segment_analytics_tracking#segment-analytics-tracking)이 활성화된 세그먼트만 볼 수 있습니다.

### 열

다음은 세그먼트 목록에 표시하도록 선택할 수 있는 정보 열입니다:
- **필터:** 세그먼트의 필터 개수
- **마지막 수정일:** 세그먼트가 마지막으로 편집된 날짜
- **마지막 수정자:** 세그먼트를 마지막으로 편집한 사용자
- **태그:** 세그먼트와 연결된 태그
- **Teams:** 세그먼트와 연결된 Teams
- **예상 크기:** 세그먼트의 예상 사용자 수
- **캔버스:** 세그먼트를 사용하는 캔버스 수
- **캠페인:** 세그먼트를 사용하는 캠페인 수

### 별표만 표시

**별표 표시된 항목만 표시**를 선택하면 내가 별표 표시한 세그먼트만 보이도록 보기가 좁혀집니다.

## 세그먼트의 메시징 사용 현황 보기 {#messaging-use}

세그먼트의 **메시징 사용** 섹션으로 이동하면 다른 세그먼트, 캠페인, 캔버스 등 해당 세그먼트가 사용되는 위치에 대한 개요를 확인할 수 있습니다.

{% alert note %}
세그먼트가 서로를 참조하는 루프를 방지하기 위해 **세그먼트 멤버십** 필터를 사용하는 세그먼트는 다른 세그먼트에서 참조할 수 없습니다. 자세한 내용은 [세분화 필터]({{site.baseurl}}/user_guide/engagement_tools/segments/segmentation_filters/)를 참조하세요.
{% endalert %}

## 특정 세그먼트 관리

![세그먼트의 편집 메뉴에 "편집", "복제", "아카이브" 및 "별표에 추가" 옵션이 표시됩니다.]({% image_buster /assets/img/segment/segments_page_edit_menu.png %}){: style="float:right;max-width:25%;"}

특정 세그먼트를 관리하려면 해당 세그먼트 위로 마우스를 가져간 다음 행 끝에 있는 메뉴 아이콘을 선택하면 다음 옵션이 표시됩니다:
- **편집:** 세그먼트의 필터를 편집합니다.
- **복제:** 세그먼트의 사본을 만듭니다.
- **아카이브:** 세그먼트를 아카이브합니다. 이렇게 하면 해당 세그먼트를 사용하는 캠페인이나 캔버스도 함께 아카이브됩니다.
- **별표에 추가:** 세그먼트에 별표를 표시하면 세그먼트 섹션에서 별표 표시된 항목만 표시 상자를 선택하여 빠르게 액세스할 수 있습니다.
 
여러 세그먼트 이름 옆의 확인란을 선택하여 대량 작업(특히 대량 아카이브 및 대량 태그 지정)을 수행할 수도 있습니다.

!["태그 지정" 드롭다운 필드에서 "CRM"이 선택된 상태로 여러 세그먼트가 선택되어 있습니다.]({% image_buster /assets/img/segment/segments_bulk_action.png %}){: style="max-width:45%;"}

### 마지막 확인 이후 변경 사항

다른 팀원의 세그먼트 업데이트 횟수는 세그먼트 개요 페이지의 *마지막으로 본 이후 변경 사항* 측정기준을 통해 추적됩니다. **마지막으로 본 이후 변경 사항**을 선택하면 세그먼트의 이름, 설명 및 타겟 오디언스에 대한 업데이트 체인지로그를 확인할 수 있습니다. 각 업데이트마다 누가 언제 수행했는지 확인할 수 있습니다. 이 체인지로그를 사용하여 세그먼트의 변경 사항을 감사할 수 있습니다.

## 세그먼트 검색

검색 필드에 단어를 입력하여 세그먼트 이름을 검색합니다. 

이 필드에 입력한 모든 용어와 문자열이 검색됩니다. 예를 들어 "test segment 1"을 검색하면 이름에 "test", "segment" 또는 "1"이 포함된 세그먼트가 반환됩니다. 정확한 문자열을 검색하려면 검색어를 따옴표로 감싸세요. ["test segment 1"]을 검색하면 이름에 "test segment 1"이라는 정확한 문구가 포함된 모든 세그먼트가 반환됩니다.

![검색 필드에 "all users"를 입력한 검색 결과에 "All Users (Test)", "All Users", "All Users 15"가 포함되어 있습니다.]({% image_buster /assets/img/segment/segments_search.png %})

### 캔버스 내 세그먼트

다른 세그먼트, 캠페인 또는 캔버스에 포함된 모든 세그먼트 참조를 검색하려면 세그먼트의 [메시징 사용](#messaging-use) 섹션으로 이동하세요. **캔버스** 페이지의 **타겟 세그먼트** 필터는 캔버스 오디언스 세그먼트만 검색합니다.

![캔버스 페이지의 타겟 세그먼트 필터.]({% image_buster /assets/img/segment/target_segment.png %}){: style="max-width:45%;"}
