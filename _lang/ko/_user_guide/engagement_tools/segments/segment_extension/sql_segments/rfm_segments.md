# RFM SQL 세그먼트

> RFM(최근, 빈도, 금액) 세그먼트 확장을 생성하여 구매 습관을 측정하여 우수 사용자를 타겟팅할 수 있습니다.

RFM 분석은 각 카테고리(최근성, 빈도, 금액)별로 사용자에게 0~3점(3점이 최고점, 0점이 최저점)의 점수를 부여하여 최고의 사용자를 식별하는 마케팅 기법입니다. 최근성, 빈도 및 금전적 가치는 모두 사용자가 선택한 특정 시간 범위의 데이터를 기반으로 합니다.

## RFM 카테고리

| 카테고리 | Definition |
| --- | --- |
| 최근 시점 | 고객이 최근에 구매한 내역입니다. 점수가 높을수록 최근 구매 횟수가 많다는 뜻입니다. |
| 빈도 | 고객의 구매 빈도. 점수가 높을수록 빈도가 높다는 뜻입니다. |
| 금전적 가치 | 고객이 지출한 총 금액입니다. 점수가 높을수록 지출이 많다는 뜻입니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation"}

{% alert note %}
사용자의 금전적 가치는 Braze 구매 이벤트를 통해 발생한 매출에 따라 결정되므로 구매 이벤트를 사용하려면 RFM SQL 세그먼트를 사용하도록 설정해야 합니다.
{% endalert %}

## RFM 세그먼트 만들기

1. **대상** > **세그먼트 확장으로** 이동합니다.
2. **새 확장을** 선택한 다음 **최근성, 빈도 및 금전적 가치(RFM) 세그먼트를** 선택합니다.

![이벤트, 구매 또는 RFM 세그먼트에 대한 카탈로그 세그먼트를 생성하는 옵션이 있는 모달입니다.]({% image_buster /assets/img/segment/select_rfm_segment.png %}){: style="max-width:80%" }

{: start="3"}
3\. **변수** 패널에서 **시간 범위를** 선택하여 분석할 구매 데이터의 기간을 지정합니다. 최대 지난 60일까지 지정할 수 있습니다. 선택한 시간 범위는 사용자 행동 데이터를 가져오는 시간 범위이며 캠페인 목표에 따라 달라집니다.

| 시간 범위 필드 | 설명 | Use case |
| --- | --- | --- |
| 상대 | 지난 X일 이내의 활동 지정 | 롤링 윈도우로 가장 최근의 사용자 행동을 분석합니다. | 
| 시작일 | 분석의 고정된 시작점 지정 | 특정 날짜 이후의 사용자 활동(예: 캠페인 시작 후)을 분석합니다. |
| 종료일 | 분석의 고정 종료 지점 지정 | 제품 업데이트 전과 같은 특정 날짜까지의 사용자 활동을 분석하세요. |
| 날짜 범위 | 커스텀 기간의 시작 날짜와 종료 날짜를 모두 지정합니다. | 프로모션 이벤트와 같이 정해진 기간 동안 사용자 행동을 분석합니다. |
{: .reset-td-br-1 .reset-td-br-2 .reset-td-br-3 role="presentation"}

{: start="4"}
4\. 세그먼트에 포함할 생성된 [RFM 그룹을](#rfm-groups) 선택합니다. 여러 그룹을 선택하는 경우 세그먼트에는 선택한 그룹에 속한 사용자가 포함됩니다.

![변수 패널에서 '챔피언' 및 '충성도 높은 사용자' RFM 그룹이 선택되어 있습니다.]({% image_buster /assets/img/segment/rfm_groups.png %})

{: start="5"}
5\. 미리 보기를 실행한 다음 세그먼트 저장하기

{% alert note %}
RFM 세그먼트를 만들기 위해 템플릿의 SQL 코드를 편집할 필요는 없습니다. **변수** 패널만 사용하여 세그먼트를 커스텀할 수 있습니다.
{% endalert %}

### RFM 그룹

RFM 세그먼트는 특정 순서로 평가됩니다. 사용자는 우선순위 목록의 맨 위에서 아래로 기준을 충족하는 첫 번째 세그먼트에 배정됩니다. 예를 들어 '챔피언'과 '충성도 높은 사용자' 자격을 모두 갖춘 사용자는 우선순위가 더 높기 때문에 '챔피언' 세그먼트에 배정됩니다.

| RFM 그룹          | 세그먼트 설명                                                                 | 최근(R) 순위 | 빈도(F) 순위 | 금전적(M) 순위 |
|--------------------|-------------------------------------------------------------------------------------|------------------|--------------------|-------------------|
| 챔피언          | 모든 카테고리에서 최고 점수를 받은 가장 가치 있는 사용자 세그먼트입니다.                   | 3                | 2-3                | 2-3               |
| 충성 사용자        | 최근 사용 빈도가 높은 사용자. 챔피언보다 금전적 가치가 낮을 수 있습니다. | 2-3              | 2-3                | 1-3               |
| 잠재적 충성도 고객| 최근 구매 빈도가 중간 정도이고 금액이 중간 정도인 사용자.   | 3                | 1-3                | 1-3               |
| 유망한          | 최근 고가의 초기 구매를 했지만 아직 구매 빈도가 높지 않은 사용자. | 3                | 0-3                | 1-3               |
| 신규 고객       | 아주 최근에 첫 구매를 한 사용자.                             | 3                | 0-3                | 0-3               |
| 주의 필요  | 최근 사용 기간이 평균 이상이지만 구매 빈도나 금전적 가치가 평균 이하인 사용자입니다. | 2-3              | 0-3                | 0-3               |
| 잃어버릴 수 없음   | 이전에 구매 빈도와 금액 점수가 높았지만 오랫동안 구매하지 않았던 사용자. | 0-1              | 2-3                | 2-3               |
| 위험            | 역사적으로 구매 빈도와 금액 점수가 적당했지만 오랫동안 구매하지 않은 사용자. | 0-1              | 1-3                | 1-3               |
| 잠자기 직전     | 모든 측정기준에서 점수가 낮은 사용자.                                       | 1                | 0-3                | 0-3               |
| 최대 절전 모드        | 활동 빈도는 보통이지만 장기간 활동하지 않은 사용자.    | 0                | 0-2                | 0-3               |
