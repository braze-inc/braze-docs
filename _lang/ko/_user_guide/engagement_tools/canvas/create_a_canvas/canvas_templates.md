# 캔버스 템플릿 만들기

> 이 참조 문서에서는 캔버스용 템플릿을 만들고 관리하는 방법에 대해 설명합니다. 템플릿을 사용하면 캔버스 전반에서 특정 목표에 맞게 쉽게 사용자 지정할 수 있는 일관된 프레임워크를 만들어 메시지를 구체화할 수 있습니다.

{% alert tip %}
[브레이즈 캔버스 템플릿을](#available-braze-templates) 사용하여 시간을 절약하고 캔버스 제작을 간소화하세요! 사전 구축된 템플릿 라이브러리를 검색하여 사용 사례에 맞는 템플릿을 찾고 특정 요구 사항에 맞게 사용자 지정하세요.
{% endalert %}

## 방법 1: 기존 캔버스에서 만들기

### 1단계: 기존 캔버스 선택

Braze 대시보드에서 **메시징** > **캔버스로** 이동하여 템플릿으로 사용하려는 기존 캔버스를 선택합니다.

### 2단계: 템플릿 만들기

캔버스 편집기에서 캔버스가 활성 상태인지 초안 상태인지에 따라 **캔버스 편집** 또는 **초안 편집**을 선택합니다. 바닥글에서 **초안으로 저장** 드롭다운을 펼치고 **템플릿으로 저장을** 선택합니다.

![][2]

### 3단계: 템플릿 저장

그런 다음 템플릿에 이름을 지정하고 관련 태그를 추가합니다. 그런 다음 **저장**을 선택합니다. 이제 캔버스 제작에 템플릿을 사용할 준비가 되었으므로 기본 설정과 단계를 이미 완료한 상태에서 시작할 수 있습니다.

## 방법 2: 캔버스 템플릿 편집기를 통해 생성

### 1단계: 캔버스 템플릿 편집기로 이동합니다.

Braze 대시보드에서 **템플릿** > **캔버스 템플릿으로** 이동합니다.

{% alert note %}
이전 탐색을 사용하는 경우 이 페이지는 **참여** > **템플릿 및 미디어** > **캔버스 템플릿에서** 찾을 수 있습니다.
{% endalert %}

### 2단계: 새 템플릿 만들기

**템플릿 만들기를** 선택하고 캔버스 세부 정보 설정을 시작합니다. 캔버스 템플릿에 이름을 지정하는 것으로 시작할 수 있습니다.

!["연간 봄 프로모션에 사용"이라는 설명이 포함된 "연간 세일 캔버스 템플릿"이라는 이름의 캔버스 템플릿 예시입니다.][1]

### 3단계: 템플릿 사용자 지정

그런 다음 [캔버스를 설정하여]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/#step-2-set-up-your-canvas) 템플릿을 사용자 지정합니다. 사용자가 언제 캔버스에 입장해야 하는지 결정하고, 어떤 사용자가 이 캔버스에 입장할 수 있는지 결정하고, 전송 설정을 조정하고, 템플릿에 대한 사용자 여정을 구축할 수 있습니다.

### 4단계: 템플릿 저장

템플릿 사용자 지정을 완료한 후 **템플릿 저장** 버튼을 선택합니다. **캔버스 템플릿** 페이지에서 <i class="fas fa-list"></i> **템플릿 세부** 정보를 선택하여 캔버스 템플릿 세부 정보를 볼 수 있습니다. 

## 캔버스 템플릿 사용

캔버스를 작성할 때 템플릿을 사용하는 방법에는 두 가지가 있습니다:

- **메시징에서** **메시징** > **캔버스로** 이동합니다. **캔버스 만들기** 버튼을 선택하고 **캔버스 템플릿을 사용합니다**.
- **템플릿에서** **템플릿** > **캔버스 템플릿으로** 이동하여 원하는 템플릿을 찾습니다. 그런 다음 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택한 다음 **템플릿 적용**을 선택합니다. 그러면 캔버스 작성기에 템플릿이 적용된 새 캔버스로 이동합니다.

### 사용 가능한 Braze 템플릿

Braze에는 일반적인 사용 사례에 대한 모범 사례로 참조하고 사용할 수 있는 다양한 캔버스 템플릿이 있습니다. 이러한 템플릿은 편집할 수 없지만 **템플릿** > **브레이즈 템플릿에서** 보거나 캔버스에서 사용할 수 있습니다.

![캔버스 템플릿 섹션에서 6가지 템플릿을 사용할 수 있는 브레이즈 템플릿을 만들 수 있습니다.][3]

다음 템플릿 중에서 선택하여 캔버스로 참조하거나 사용할 수 있습니다.

{% tabs %}
{% tab 버려진 의도 %}

실시간으로 사용자와 소통하여 구매를 완료하도록 유도하세요.

이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 특정 대상을 추가합니다. 현재 잠재고객 경로는 '모든 구매'를 기준으로 트리거되지만, 타겟팅하려는 특정 제품에 맞게 조정할 수 있습니다.
- 이 템플릿은 별도의 구매 후 여정이 있다고 가정하므로 구매를 하면 사용자가 캔버스를 종료하게 됩니다.
- 대상 동기화 단계에서 세부 정보를 입력합니다.

{% endtab %}
{% tab 재고 있음 %}

상품 재입고 시 사용자에게 개인화된 메시지로 알림을 보내 구매를 유도하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- **입력 일정에서** 사용할 카탈로그를 선택합니다. 이를 통해 제품, 할인, 프로모션 등의 데이터에 액세스하여 사용자를 더욱 세분화하여 타겟팅할 수 있습니다.
- **타겟 오디언스에서** 특정 항목에 관심을 표시한 사용자를 타겟팅할 세그먼트를 추가합니다.
- 캔버스의 메시지 단계에서 리퀴드가 카탈로그를 참조하도록 업데이트합니다.

{% endtab %}
{% tab 기능 채택 %}

시의적절하게 개인화된 메시지를 전달하여 혜택과 사용 팁을 강조하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 이미 제품을 사용한 적이 있는 사용자는 제외합니다. 예를 들어, **타겟 대상**에서 필터를 추가합니다. 
-  실험 경로 단계를 사용하려면 전환 이벤트를 정의합니다. 이 이벤트는 기능 채택을 알리는 이벤트가 되어야 합니다.
- 템플릿에서 '활성화된 기능' 및 '둘러보기'에 대한 사용자 지정 이벤트를 사용하여 작업 경로 단계를 설정합니다.
- 메시지 단계에서 '피드백 설문조사'라는 이름의 사용자 지정 속성을 설정하여 피드백의 감성을 포착합니다.

{% endtab %}
{% tab 휴면 사용자 %}

과거 참여에 기반한 인센티브를 통해 사용자를 앱으로 다시 돌아오게 하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- **기본** 사항에서 전환을 추적할 특정 앱을 선택합니다.
- 캔버스 편집기에서 작업 경로 단계에 특정 앱을 추가합니다.
- 사용 사례에 맞는 파트너 및 대상과 대상 동기화 단계를 구성합니다.

{% endtab %}
{% tab 온보딩 %}

강력한 초기 채택을 촉진하고 사용자와의 지속적인 관계를 장려하는 온보딩 여정을 만드세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- "잠재 고객 분할"이라는 잠재 고객 경로 단계에서 참여 사용자에 대한 주요 작업을 사용자 지정하는 것을 고려합니다. 템플릿에서 세그먼트 필터는 "환영 이메일 단계의 이메일을 클릭함"입니다.

{% endtab %}
{% tab 구매 후 피드백 %}

피드백에 응답하고 사용자와의 관계를 구축할 수 있는 개인화된 경험을 오케스트레이션하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 캔버스 편집기의 첫 번째 단계입니다:
    - 인앱 메시지에서 사용자 지정 속성을 지정하여 선택한 설문조사 옵션에 따라 피드백의 감성을 표시합니다. 
    - 각 콜투액션의 링크에 속성을 지정하여 어떤 옵션이 선택되었는지 캡처합니다. 이러한 속성은 후속 대상 경로에서 참조됩니다.
- 이 템플릿의 첫 번째 단계의 속성을 사용하여 대상 경로를 사용자 지정합니다.
- '광고 리타겟팅'이라는 오디언스 동기화 단계를 설정합니다.

{% endtab %}
{% endtabs %}

{% alert tip %}
이러한 Braze 템플릿을 사용하여 예제 캔버스를 만드는 단계별 가이드는 [Braze 템플릿 사용을]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates) 참조하세요.
{% endalert %}

## 캔버스 템플릿 관리

캔버스 템플릿은 실제 캔버스와 유사하게 복제하고 보관할 수 있습니다. 캔버스 템플릿을 편집하려면 템플릿을 선택한 다음 **<i class="fas fa-pencil-alt"></i>편집을** 클릭합니다.

작업 공간 수준에서 사용자 권한을 업데이트하여 캔버스 템플릿을 만들거나, 편집하거나, 보거나, 보관할 수 있는 액세스를 허용하거나 제한할 수 있습니다.

### 팀 및 워크스페이스에 대한 권한

특정 사용자만 특정 캔버스 템플릿에 액세스하고 사용하도록 허용하려면 템플릿에 [팀을 추가한]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/teams/) 다음 팀 수준의 "캠페인, 캔버스, 콘텐츠 카드, 콘텐츠 블록, 기능 플래그, 세그먼트, 미디어 라이브러리 및 환경설정 센터 액세스" 권한을 할당합니다.

워크스페이스 수준이 아닌 팀 수준에서 다음 권한 중 하나를 할당하는 경우 팀에 할당된 다음 작업만 수행할 수 있습니다:

- 캔버스 템플릿 만들기 및 편집
- 캔버스 템플릿 보기
- 아카이브 캔버스 템플릿

권한이 워크스페이스와 팀 수준 모두에서 부여된 경우, 워크스페이스 수준 권한이 우선적으로 적용됩니다.

## 자주 묻는 질문

### 캔버스 템플릿에서 완료되지 않은 단계를 저장할 수 있나요?

예, 완료되지 않은 단계를 캔버스 템플릿으로 저장할 수 있습니다. 그러나 템플릿을 사용할 때 캔버스를 시작하는 데 필요한 사항을 나타내는 **템플릿 저장** 버튼에 오류가 발생합니다.

### 캔버스 빌더 설정을 템플릿으로 저장할 수 있나요, 아니면 단계만 저장할 수 있나요? 

예, 캔버스 템플릿 내의 캔버스 빌더에서 설정을 저장할 수 있습니다. 예를 들어 세그먼트와 필터의 조합을 자주 사용하려는 경우 이러한 **타겟 오디언스** 설정을 캔버스 템플릿의 일부로 저장할 수 있습니다.
