# 종료 기준

>  이를 통해 오디언스에게 캔버스 메시지를 보다 타겟팅된 방식으로 전달할 수 있습니다.

## 

캔버스 빌더의 **대상 고객** 단계에서 종료 기준을 설정하여 캔버스를 종료하려는 사용자를 식별할 수 있습니다. 





### 

 

   

추가 예외 이벤트는 다음과 같습니다:

- 구매하기
- 세션 시작하기
- 사용자 지정 이벤트 수행
- 전환 이벤트 수행
- 이메일 주소 추가하기
- 사용자 지정 속성 값 변경하기
- 구독 상태 업데이트하기
- 구독 그룹 상태 업데이트하기
- 캠페인과 상호 작용하기
- 위치 입력
- 지오펜스 트리거
- SMS 인바운드 메시지 보내기
- WhatsApp 인바운드 메시지 보내기
- LINE 인바운드 메시지 보내기
- 카트 업데이트 이벤트 수행
- 결제 완료 이벤트 수행
- 결제 시작 이벤트 수행

### 

  

 따라서 사용자가 퇴출 기준을 충족하면 5일이 지나면 퇴출됩니다.

{% alert note %}
배열 속성은 현재 예외 이벤트의 종료 조건으로 지원되지 않습니다.
{% endalert %}
