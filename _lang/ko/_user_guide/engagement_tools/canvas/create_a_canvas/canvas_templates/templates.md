# 캔버스 템플릿

> Braze에는 일반적인 사용 사례에 대한 모범 사례로 참조하고 사용할 수 있는 다양한 캔버스 템플릿이 있습니다. 이러한 템플릿은 편집할 수 없지만, **템플릿** > **Braze 템플릿에서** 보거나 캔버스에서 사용할 수 있습니다.

![캔버스 템플릿 섹션에서 13개의 템플릿을 사용할 수 있는 Braze 템플릿을 사용하세요.]({% image_buster /assets/img/braze_canvas_templates.png %})

다음 템플릿 중에서 선택하여 캔버스로 참조하거나 사용할 수 있습니다.

## 표준 캔버스 템플릿

{% tabs %}
{% tab Abandoned Intent %}

### 포기된 의도

실시간으로 사용자 참여를 유도하여 구매를 완료하도록 유도하세요.

이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 특정 오디언스를 추가합니다. 현재 오디언스 경로는 '모든 구매'를 기준으로 트리거되지만, 이를 타겟팅하려는 특정 제품에 맞게 조정할 수 있습니다.
- 이 템플릿은 별도의 구매 후 여정이 있다고 가정하므로 구매를 하면 사용자가 캔버스에서 나가게 됩니다.
- 오디언스 동기화 단계에서 세부 정보를 입력합니다.

{% endtab %}
{% tab Back In Stock %}

### 재입고

개인화된 메시징을 통해 상품 재입고 시 사용자에게 알림을 보내 구매를 유도하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- **입력 일정에서** 사용할 카탈로그를 선택합니다. 이를 통해 제품, 할인, 프로모션 등의 데이터에 액세스하여 사용자를 더욱 타겟팅할 수 있습니다.
- **타겟 오디언스에서** 세그먼트를 추가하여 특정 항목에 관심을 표시한 사용자를 타겟팅할 수 있습니다.
- 캔버스 전체의 메시지 단계에서 Liquid를 업데이트하여 카탈로그를 참조합니다.

{% endtab %}
{% tab Feature Adoption %}

### 기능 채택

시의적절하게 개인화된 메시지를 전달하여 혜택과 사용 팁을 강조하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 이미 해당 기능을 채택한 사용자는 제외합니다. 예를 들어, **타겟 오디언스에서** 이미 발생한 '활성화된 기능'과 같은 커스텀 이벤트에 대한 필터를 추가합니다.
- 실험 경로 단계를 사용하려면 전환 이벤트를 정의합니다. 이 이벤트는 기능 채택을 알리는 이벤트가 되어야 합니다.
- 템플릿에서 '활성화된 기능' 및 '둘러보기'에 대한 커스텀 이벤트를 사용하여 행동 경로 단계를 설정합니다.
- 메시지 단계에서 '피드백 설문조사'라는 이름의 커스텀 속성을 설정하여 피드백의 감성을 포착합니다.

{% endtab %}
{% tab Lapsed User %}

### 휴면 사용자

과거 참여에 기반한 인센티브를 통해 사용자를 앱으로 다시 유도하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- **기본** 설정에서 전환을 추적할 특정 앱을 선택합니다.
- 캔버스 편집기에서 행동 경로 단계에 대한 특정 앱을 추가합니다.
- 사용 사례에 맞는 파트너 및 오디언스와 함께 오디언스 동기화 단계를 구성합니다.

{% endtab %}
{% tab Onboarding %}

### 온보딩

강력한 초기 채택을 촉진하고 사용자와의 지속적인 관계를 장려하는 온보딩 여정을 만드세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- "오디언스 경로"라는 이름의 오디언스 경로 단계에서 참여 사용자에 대한 주요 작업을 커스텀하는 것을 고려하세요. 템플릿에서 세그먼트 필터는 "환영 이메일 단계에 대해 이메일을 클릭함"입니다.

{% endtab %}
{% tab Post-Purchase Feedback %}

### 구매 후 피드백

피드백에 대응하고 사용자와의 관계를 구축할 수 있는 개인화된 경험을 오케스트레이션하세요. 이 템플릿을 사용할 때는 다음 사항을 고려하세요:

- 캔버스 편집기의 첫 번째 단계입니다:
    - 인앱 메시지에서 커스텀 속성을 지정하여 선택한 설문조사 옵션에 따라 피드백의 감성을 표시할 수 있습니다. 
    - 각 콜투액션의 링크에 속성을 지정하여 어떤 옵션이 선택되었는지 캡처합니다. 이러한 속성은 후속 오디언스 경로에서 참조됩니다.
- 이 템플릿의 첫 번째 단계의 속성을 사용하여 오디언스 경로를 사용자 지정합니다.
- '광고 리타겟팅'이라는 이름의 오디언스 동기화 단계를 설정합니다.

{% endtab %}
