# 행동 경로 

> 캔버스의 행동 경로를 사용하면 사용자의 행동을 기반으로 사용자를 정렬할 수 있습니다. 

캔버스 사용자 여정의 행동 경로 단계입니다.]({% image_buster /assets/img/canvas_actionpath.png %}){: style="float:right;max-width:40%;margin-left:15px;"}

행동 경로를 사용하면 가능합니다:

* 고객 참여 이벤트 및 커스텀 이벤트를 포함하여 특정 행동을 기반으로 사용자 경로를 사용자 지정합니다.
* 이 평가 기간 동안 사용자의 행동에 따라 다음 경로의 우선 순위를 정하기 위해 주어진 기간 동안 사용자를 보류합니다.

## 행동 경로 만들기

행동 경로를 만들려면 캔버스에 컴포넌트를 추가합니다. 사이드바에서 컴포넌트를 드래그 앤 드롭하거나 단계 하단에 있는 <i class="fas fa-plus-circle"></i> 더하기 버튼을 선택하고 **행동 경로를** 선택합니다. 

### 작업 설정

**작업 설정에서** **평가 기간을** 설정하여 사용자가 단계에 머무르는 시간을 결정합니다. 기본값은 하루 이내에 사용자를 평가하지만 캔버스에 따라 이 기간을 초, 분, 시간, 일, 주 단위로 조정할 수 있습니다. 행동 경로의 최대 평가 기간은 31일입니다.

**작업 설정에서** **순위 순서에 따라 사용자 기반 진행** 토글을 켜서 컴포넌트의 순위 순서를 설정할 수도 있습니다.

평가 기간이 1일인 작업 설정입니다.]({% image_buster /assets/img/actionpath_settings.png %})

기본값은 **순위가** 꺼져 있습니다. 사용자가 행동 경로에 진입하여 행동 그룹에 연결된 트리거 이벤트를 수행하면 즉시 해당 행동 그룹을 통해 진행됩니다. 사용자가 트리거 이벤트를 수행하지 않으면 평가 기간이 끝나면 기본값인 **모두 기타** 그룹으로 진행됩니다.

**순위 순서에 따른 사용자 기반 고급이** 켜져 있으면 **순위가** 켜져 있음을 의미합니다. 따라서 모든 사용자는 평가 기간이 끝날 때까지 보류됩니다. 평가 기간이 끝나면 사용자는 평가 기간이 끝날 때 가장 우선 순위가 높은 활동 그룹으로 진행됩니다. 평가 기간 동안 어떤 작업도 수행하지 않은 사용자는 기본값인 **다른 모든** 사용자 그룹으로 진행됩니다.

#### 인앱 메시지

액션 그룹 트리거가 세션을 시작하고 다음 단계가 인앱 메시지인 경우, 사용자는 인앱 메시지를 수신하기 위해 두 번의 세션 시작을 수행해야 한다는 점에 유의하세요. 첫 번째 세션은 사용자를 행동 경로 내의 행동 그룹에 할당하고, 두 번째 세션은 인앱 메시지를 트리거합니다.

#### 랭킹 상태 예시

두 개의 작업 그룹이 있는 평가 기간이 하루인 작업 경로가 있다고 가정해 보겠습니다: 그룹 1 및 그룹 2. 그룹 1에는 "세션 시작" 트리거 이벤트가 있고 그룹 2에는 "구매하기" 트리거 이벤트가 있습니다. **랭킹이** 켜져 있으면 행동 경로에 있는 모든 사용자가 하루 동안 '보류'됩니다. 최종 사용자가 세션을 시작하고 구매를 완료하면 가장 높은 등급 경로로 진행됩니다. 이 경우 사용자는 그룹 1로 진행됩니다. 

앞의 예에서 **랭킹이** 꺼져 있고 사용자가 트리거 이벤트 중 하나("세션 시작" 또는 "구매하기")를 수행하면 해당 사용자는 트리거 동작에 따라 관련 액션 그룹에서 진행됩니다.

캔버스 항목 속성은 이벤트 속성정보와 다르다는 점에 유의하세요. 캔버스 항목 속성은 캔버스를 트리거한 이벤트의 속성정보입니다. 이러한 속성은 원래 캔버스 워크플로우를 사용할 때 캔버스의 첫 번째 전체 단계에서만 사용할 수 있습니다. 캔버스를 사용할 때 지속성 항목 속성정보가 인에이블먼트되어 전체 캔버스에서 항목 속성을 재사용할 수 있습니다. 반대로 이벤트 속성정보는 사용자가 워크플로를 진행하면서 발생하는 이벤트나 작업에서 비롯됩니다.

### 액션 그룹

트리거 하나 또는 여러 개의 트리거를 추가하여 동작 그룹을 정의하세요. 여기에서 사용자가 있는지 여부와 같은 다양한 트리거를 선택할 수 있습니다:

- 구매하기
- 세션 시작
- [커스텀 이벤트]({{site.baseurl}}/user_guide/data/custom_data/custom_events/) 수행하기
- 전환 이벤트 수행하기
- 이메일 주소 추가
- 커스텀 속성 값을 변경합니다(배열 포함, 중첩된 고객 속성은 제외). 여기에는 고객 프로필에 처음으로 값이 있는 새 속성을 추가하는 경우(해당 속성이 이전에 없던 경우)가 포함됩니다.
- 구독 상태 또는 구독 그룹 상태 업데이트하기
- 캠페인 또는 콘텐츠 카드와 상호 작용하기
- 위치 입력
- 지오펜스 트리거하기
- SMS 또는 WhatsApp 인바운드 메시지 보내기

구매를 한 사용자에 대해 '그룹 1'이라는 액션 그룹을 생성합니다.]({% image_buster /assets/img/actionpath_group.png %})

각 작업 그룹 설정에서 **이 그룹이 캔버스에서 나가기를 원함** 확인란을 선택하여 이 그룹에 속한 사용자가 평가 기간이 끝나면 캔버스에서 나가기를 **원하는** 옵션도 있습니다.

### 재자격이 있는 캔버스

사용자가 행동 경로를 여러 번 입력하고 동시에 행동 경로에 여러 항목이 있는 경우 **순위** 상태에 따라 예상되는 동작이 달라집니다.

| 랭킹 상태 | 행동 경로 동작 |
|---|--------------|
| **꺼짐** | 관련 작업이 수행되면 Braze는 중복된 항목을 제거하고 관련 작업 그룹을 통해 가장 빠른 항목을 즉시 진행합니다. <br><br/> 관련 작업이 수행되지 않으면 모든 항목은 해당 평가 기간이 끝날 때 진행됩니다. 중복이 발생하지 않습니다. |
| **켜짐** | 모든 응모는 해당 평가 기간이 끝나면 진행됩니다. 중복이 발생하지 않습니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

순위는 [출시 후에는 편집할]({{site.baseurl}}/post-launch_edits/) 수 없습니다.
