# 캔버스용 초안 저장하기

> 캔버스를 만들고 실행할 때 활성 캔버스를 편집하고 초안으로 저장하여 다른 실행 전에 변경 사항을 시험해 볼 수 있습니다. 

대규모 변경이 필요한 활성 캔버스가 있는 경우 이 기능을 사용하여 활성 캔버스에서 이러한 변경 사항을 실행하기 **전에** 구축, 저장 및 품질 검사를 수행할 수 있습니다. 

다른 캔버스와 마찬가지로 한 번에 한 명의 사용자만 초안을 편집할 수 있으며 캔버스에는 한 번에 하나의 초안만 있을 수 있습니다. 아직 초안 변경이 시작되지 않았기 때문에 이 초안에는 분석이 없습니다.

활성 캔버스를 볼 수 있는 옵션이 있는 초안 캔버스를 편집 중임을 사용자에게 알려주는 배너가 있는 초안 캔버스 예시입니다. 바닥글에는 분석 보기로 돌아가거나 초안으로 저장하거나 초안을 실행할 수 있는 옵션이 있습니다.]({% image_buster /assets/img_archive/canvas_draft1.png %})

## 초안 만들기

초안을 만들려면 다음과 같이 하세요:

1. 활성 캔버스로 이동합니다.
2. 캔버스 바닥글에서 **초안으로 저장** 버튼을 선택합니다. 

캔버스의 초안이 있는 동안에는 활성 캔버스를 편집할 수 없습니다. 캔버스를 업데이트하여 변경 사항을 적용하거나 초안을 삭제할 수 있습니다.

## 활성 초안 참조

활성 캔버스를 참조하려면 분석 보기의 바닥글에서 **활성 캔버스 보기를** 선택하거나 초안에서 캔버스 헤더를 선택합니다. 활성 캔버스로 돌아가려면 분석 보기 또는 활성 캔버스 보기에서 **초안 편집을** 선택합니다.

초안이 생성되기 전에 이미 시작된 단계만 참조할 수 있습니다. 즉, 초안을 만든 **후에** 단계나 채널을 만든 경우에는 초안에서 참조할 수 없습니다.

{% alert note %}
콘텐츠 블록이 캔버스 초안에서 참조되는 경우 해당 캔버스는 콘텐츠 블록 포함 횟수에 나열됩니다. 그러나 콘텐츠 블록이 **활성** 캔버스의 초안에서 참조되는 경우 해당 캔버스는 콘텐츠 블록 포함 수에 표시되지 않습니다.
{% endalert %}

### 인앱 메시지 우선순위 지정

활성 캔버스의 초안의 경우, 사용자가 우선순위를 변경하면 캔버스 빌더 내의 인앱 메시지 우선순위가 즉시 업데이트됩니다. 즉, 초안이 존재하는 경우에도 캔버스 수준의 인앱 메시지 우선순위가 활성 캔버스에 즉시 적용됩니다. 

그러나 단계별 인앱 메시지 우선순위 변경은 초안으로 저장되며 캔버스가 업데이트될 때 적용됩니다. 예를 들어 메시지 단계에서는 단계 설정이 단계 수준에서 적용되므로 사용자가 초안을 실행하면 우선순위 분류기가 업데이트됩니다.
