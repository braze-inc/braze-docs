# 랜딩 페이지 추적 데이터 정보

> Braze에서 랜딩 페이지의 추적 및 익명화된 데이터에 대해 알아보세요.

## 추적 방법

### 웹 소프트웨어 개발 키트

Braze 웹 소프트웨어 개발 키트는 사용자가 랜딩 페이지에서 양식을 제출할 때만 초기화됩니다. 양식을 제출하기 전에는 개인 데이터가 수집되지 않으며 소프트웨어 개발 키트에서 사용자를 적극적으로 추적하지 않습니다. 초기화가 완료된 후 소프트웨어 개발 키트는 쿠키, 로컬 저장소 등의 데이터를 브라우저에 저장하지 않습니다.

양식이 제출되면 소프트웨어 개발 키트에서 다음 데이터를 수집합니다:

- 양식 제출 이벤트(이벤트 이름 및 제출 시간)
- 팀에서 양식에 지정한 데이터(예: 이름, 이메일, 전화번호)
- 세션 시작 시간
- 기기 ID(기기에 대해 생성되지만 저장되지는 않는 고유 ID)
- IP 주소로 확인된 국가

### 익명화된 데이터

사용자가 양식을 제출하기 전에 랜딩 페이지에서 추적되는 사용자 데이터는 익명화되고 식별할 수 없는 정보로만 구성됩니다. 이는 랜딩 페이지의 페이지 조회 수(노출 횟수) 및 클릭 수와 같은 표준 웹사이트 집계 측정기준으로 구성됩니다.

이 데이터는 식별 가능한 사용자와 연결되지 않으므로 개별 사용자 행동을 리타겟팅하거나 추적하는 데 사용할 수 없습니다.
