# 템플릿 관리

> 템플릿을 보관하거나 복제하면 템플릿을 더 잘 정리하고 관리할 수 있습니다. 이 참고 문서에서는 Braze 대시보드의 **템플릿** 섹션에서 템플릿을 보관하고 복제하는 방법에 대해 설명합니다.

## 템플릿 복제

### 개별 템플릿

중복 옵션이 있는 드롭다운 메뉴.]({% image_buster /assets/img/template_duplicate_cog.png %}){: style="float:right;max-width:15%;margin-left:15px;"}

개별 템플릿을 복제하려면 개별 템플릿의 톱니바퀴 아이콘( <i class="fas fa-cog"></i> )을 선택하고 드롭다운 메뉴에서 **복제를** 선택합니다.
<br><br>

{% alert note %}
[콘텐츠 블록]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/content_blocks/) 템플릿의 경우 초안 사본이 생성됩니다. 다른 모든 템플릿의 경우 새 복제본이 자동으로 생성됩니다.
{% endalert %}

### 여러 템플릿

{% raw %}

템플릿 이름 옆의 확인란을 선택하면 여러 템플릿을 복제할 수 있습니다. 먼저 템플릿을 선택한 다음 **복제를** 선택합니다.

중복된 템플릿은 **마지막으로 편집한** 열을 정렬하여 찾을 수 있습니다. 기본값으로 새 템플릿의 이름은 `Copy of ORIGINAL_TEMPLATE_NAME` 입니다.

{% endraw %}

템플릿이 마지막으로 편집된 시간별로 정렬된 세 개의 템플릿과 복사된 템플릿이 목록 맨 위에 표시됩니다.]({% image_buster /assets/img/duplicate_multiple_template.gif %})

## 템플릿 보관

세 가지 옵션이 표시되는 확장된 설정 드롭다운 메뉴입니다: "보관", "복제", "작업 공간에 복사"를 클릭하고 "보관" 옵션을 강조 표시합니다.]({% image_buster /assets/img/template_archive_cog.png %}){: style="float:right;max-width:20%;margin-left:15px;"}

개별 템플릿을 보관하려면 템플릿 그리드 화면에서 설정 아이콘을 선택하고 **보관을** 선택합니다. 템플릿을 보관할 때는 다음과 같은 다양한 시나리오에 유의하세요:

- 활성 캠페인은 중단 없이 보관된 템플릿을 계속 사용할 수 있습니다.
- 초안 캠페인에는 보관된 템플릿의 콘텐츠가 유지되며 편집 및 실행할 수 있습니다.
- 보관된 템플릿을 편집하려면 먼저 보관된 템플릿의 보관을 해제해야 합니다. 마찬가지로 아카이브된 템플릿을 캠페인에 사용하려면 먼저 템플릿의 아카이브를 해제해야 합니다.

여러 템플릿을 보관하려면 보관하려는 각 템플릿 옆의 확인란을 선택합니다. 여러 템플릿을 선택한 후 **보관을** 선택합니다. 템플릿 그리드의 **표시** 아래에서 **보관됨을** 선택하면 보관된 템플릿을 찾을 수 있습니다.

저장됨 & 이메일 템플릿 드롭 섹션에 선택한 두 개의 템플릿과 보관 옵션이 있는 도구 모음이 표시됩니다.]({% image_buster /assets/img/archive_multiple_template.png %}){: style="max-width:60%;"}
