# 속성 트리거

> 속성 트리거를 사용하면 사용자의 구독 상태 또는 커스텀 속성 값이 변경될 때 동작 기반 메시지를 보낼 수 있습니다. 

다음 시나리오에서 속성 트리거를 사용할 수 있습니다:

- 구독 상태 업데이트.
- 부울, 정수, 문자열 또는 날짜 커스텀 속성 값이 임의의 값으로 변경됩니다.
- 부울, 정수 또는 문자열 커스텀 속성 값이 특정 값으로 변경됩니다.

속성 트리거 사용을 시작하려면 캠페인 또는 캔버스 컴포넌트를 만들고 전달 방법으로 **실행 기반 전달을** 선택합니다. 그런 다음 사용하려는 속성 트리거를 선택합니다.

트리거를 선택할 수 있는 드롭다운이 있는 "실행 기반 전달" 섹션을 클릭합니다.]({% image_buster /assets/img_archive/trigger_attribute.png %})

### 구독 상태 업데이트

`Update Subscription Status` 트리거를 사용하여 구독 상태가 업데이트될 때 사용자를 타겟팅하세요. 

예를 들어 이메일 또는 푸시 구독 상태가 옵트인으로 변경되면 사용자를 타겟팅하고 옵트인해 주셔서 감사하다는 메시지를 보낼 수 있습니다. 또한 사용자가 이메일을 탈퇴할 때마다 시스템에 웹훅을 전송하여 내부 시스템에 최신 구독 상태 정보가 업데이트되도록 할 수 있습니다.

{% alert important %}
이 트리거는 이메일 글로벌 기본값이 `subscribed` 인 새 사용자가 생성되고 이후 구독 상태가 변경되지 않았으므로 `subscribed` 로 상태를 업데이트하라는 요청이 있는 경우에는 적용되지 않습니다.
{% endalert %}

### 구독 그룹 상태 업데이트하기

이메일, SMS 또는 WhatsApp의 구독 그룹 상태가 업데이트되면 `Update Subscription Group Status` 트리거를 사용하여 사용자를 타겟팅하세요. 

예를 들어, 사용자가 프로그램에 옵트인할 때 환영 SMS 메시지로 사용자를 타겟팅할 수 있습니다. 또한 업데이트 소스를 지정하여 메시징이 실행되는 시기를 보다 세밀하게 제어할 수도 있습니다. 

사용 가능한 업데이트 소스는 채널별로 다릅니다:
- 캔버스 사용자 업데이트 단계
- CSV 가져오기
- 목록-탈퇴하기
- 환경 설정 센터
- REST API
- 소프트웨어 개발 키트
- Shopify(이메일, SMS)
- 인바운드 메시지(SMS)

예를 들어, Braze는 이미 특정 인바운드 SMS에 자동으로 응답하므로 인바운드 메시징이 아닌 REST API에서 업데이트가 올 때만 환영 SMS를 보내려고 할 수 있습니다.

### 커스텀 속성 값 변경하기

변경 속성의 경우 트리거를 먼저 평가한 다음 오디언스 기준을 평가합니다. 이는 오디언스 기준이 먼저 평가된 후 트리거되는 기본값과 다릅니다. 경합 조건을 피하려면 트리거로 사용되는 속성이 오디언스 자격을 부여하는 데 사용되는 속성과 동일하지 않은지 확인하세요.

#### 새로운 값 옵션

`any new value` 옵션과 함께 `Change Custom Attribute Value` 트리거를 사용하여 부울, 정수, 문자열 또는 날짜 값이 새 값으로 변경될 때 사용자를 타겟팅하세요.

예를 들어, 보상 포인트 수가 변경되면 사용자를 타겟팅하여 현재 보유한 포인트 수를 알려줍니다. 이 예제에서는 사용자에게 85개의 보상 포인트가 있고 보상 포인트 속성이 새로운 값으로 변경될 때 트리거되도록 캠페인을 설정했다고 가정해 보겠습니다. 이 사용자의 보상 포인트 속성 값이 새로운 값(예: 83, 84, 86 등)으로 변경되면 캠페인이 트리거됩니다.

다음 티어 업데이트 알림의 사용 사례 예시를 살펴보세요. 보상 등급이 변경되면 사용자에게 알림을 보낼 수 있습니다. 이 사용 사례를 달성하려면 `Change Custom Attribute Value` 에서 트리거되는 캠페인을 설정하고 커스텀 속성 보상 티어가 새로운 값으로 변경될 때 트리거되도록 설정하세요.

{% alert important %}
현재 배열 속성에는 속성 트리거를 사용할 수 없습니다.
{% endalert %}

"AA_current_rewards_tier" 을 임의의 값으로 변경하는 "커스텀 속성 값 변경" 트리거입니다.]({% image_buster /assets/img_archive/any_value.png %})

또한 Liquid를 사용하여 고객의 새로운 보상 등급으로 메시지 본문을 개인화하고 고객에게 변경 사항에 대한 자세한 정보를 제공할 수 있습니다.

{% raw %}
```liquid
Your rewards tier was just changed to {{custom_attribute.${AA_current_rewards_tier}}}
```
{% endraw %}

#### 특정 값

`specific value` 옵션과 함께 `Change Custom Attribute Value` 트리거를 사용하여 부울, 정수 또는 문자열 커스텀 속성이 특정 값으로 변경될 때 사용자를 타겟팅할 수 있습니다. 

예를 들어, 보상 등급이 최고 등급으로 변경될 때 사용자를 타겟팅할 수 있습니다. 이 예제에서는 최고 보상 등급이 슈퍼 VIP라고 가정합니다. 사용자의 보상 티어 커스텀 속성이 `Super VIP` 로 변경될 때 트리거되도록 캠페인을 설정하여 사용자가 슈퍼 VIP가 된 것을 축하할 수 있습니다.

"AA_current_rewards_tier" 이 특정 값인 "super vip"로 변경되는 "커스텀 속성 값 변경" 트리거입니다.]({% image_buster /assets/img_archive/super_vip.png %})

{% alert important %}
- 배열 및 날짜 커스텀 속성에는 특정 커스텀 속성 값에 대한 속성 트리거를 사용할 수 없습니다.
- 커스텀 속성 값이 null로 업데이트되면 커스텀 속성 값 변경 트리거가 트리거되지 않습니다.  
- 커스텀 속성 값 변경 트리거는 커스텀 속성의 값이 변경될 때만 트리거됩니다. e.g 커스텀 속성의 현재 값을 Braze로 다시 전송하는 경우(즐겨 찾는 색상 속성의 값이 빨간색인 경우 빨간색 값을 Braze로 다시 전송하는 경우) 커스텀 속성 값 변경 트리거가 발생하지 않습니다.
- 커스텀 속성 값 변경 트리거는 새로 생성된 사용자에게도 적용됩니다.
{% endalert %}
