# 콘텐츠 캘린더

> 콘텐츠 캘린더는 예정된 캠페인에 대한 높은 수준의 개요를 제공합니다.

메시지 데이터를 분석하고 예정된 캠페인을 보고 관리할 수 있습니다. 캘린더를 열려면 **메시징** > **콘텐츠 캘린더**를 선택하세요.

## 메시징 분석

메시징 분석 섹션에는 최근 및 예정된 캠페인에 대한 두 개의 보고서가 포함되어 있습니다:

- **예정된 메시지 유형:** 향후 14일 이내에 예정된 푸시, 이메일 및 인앱 메시징 캠페인의 수입니다.
- **전송된 메시지:** 지난 14일 동안 전송되고 열린 메시지의 수입니다.

보고서에서 특정 날짜에 커서를 올려 놓으면 해당 날짜의 메시징 분석을 볼 수 있습니다.

\![메시징 분석 내의 '예정된 메시지 유형' 및 '전송된 메시지' 그래프.]({% image_buster /assets/img/campaign_calendar/content_calendar_messaging_analytics.png %})

데이터 사본을 다운로드하려면 <i class="fa-solid fa-bars" style="color: #2e7487;"></i> **차트 컨텍스트 메뉴**를 선택한 다음 선호하는 파일 형식을 선택하세요.

## 캘린더 전송

전송 캘린더는 현재 월의 모든 예정된 캠페인을 표시합니다.

{% alert note %}
인앱 메시지는 항상 행동 기반이므로 전송 캘린더에서 인앱 메시지 캠페인을 볼 수 없습니다.
{% endalert %}

월을 변경하려면 <i class="fa-solid fa-chevron-left" style="color: #2e7487;"></i> **이전 달 보기** 또는 <i class="fa-solid fa-chevron-right" style="color: #2e7487;"></i> **다음 달 보기**를 선택하세요. 현재 월로 돌아가려면 **오늘**를 선택하세요.

\![현재 월에 예정된 모든 캠페인을 표시하는 전송 캘린더.]({% image_buster /assets/img/campaign_calendar/content_calendar_sends.png %})

캘린더의 각 색상은 다른 예정된 캠페인 유형을 나타냅니다:

- **빨간색:** 단일 채널 예정 캠페인입니다.
- **파란색:** 다중 채널 예정 캠페인입니다.
- **금색:** 중단된 캠페인으로, 역사적 참조를 위해 캘린더에 포함됩니다.

캘린더에서 캠페인을 선택하여 캠페인을 보거나 수정할 수 있습니다. 자세한 정보는 [캠페인 관리]({{site.baseurl}}/user_guide/engagement_tools/campaigns/managing_campaigns)을 참조하세요.
