# 세그먼트, 캠페인 및 캔버스 복제하기

> 기존 세그먼트, 캠페인 및 캔버스를 복제하여 성공적인 전략을 지속적으로 반복하고 구축할 수 있습니다. 대시보드의 해당 페이지**(세그먼트**, **캠페인** 또는 **캔버스**)에서 해당 항목을 목록으로 볼 때 이러한 항목을 복제할 수 있습니다.

{% alert tip %}
세그먼트와 캠페인에 대한 자세한 내용은 [Braze 학습 과정을](https://learning.braze.com/quick-overview-segment-and-campaign-setup) 확인하세요.
{% endalert %}

## 캠페인 및 세그먼트 복제

세그먼트 또는 캠페인을 복제하려면 다음을 수행합니다: 
1. **메시징** > **캠페인** 또는 **대상** > **세그먼트로** 이동합니다.
2. 캠페인 또는 세그먼트 위로 마우스를 가져가 <i class="fas fa-gear"></i> **추가 작업을** 선택합니다.
3. **복제를** 선택합니다. 

새 세그먼트 또는 캠페인의 제목은 자동으로 '[원래 이름]]의 사본'으로 지정되며 해당 편집 페이지로 이동합니다. 중복된 캠페인은 편집하고 실행할 때까지 초안 상태로 유지됩니다.

## 캔버스 복제

캔버스를 복제하려면 다음과 같이 하세요:

1. **메시징** > **캔버스로** 이동합니다.
2. 캔버스 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> **추가 작업을** 선택합니다.
3. **복제를** 선택합니다. 

새 캔버스의 제목은 자동으로 '[원본 이름]의 사본'으로 바뀌고 해당 편집 페이지로 이동합니다. 캔버스의 복제본은 편집하고 실행할 때까지 초안으로 남아 있습니다.
