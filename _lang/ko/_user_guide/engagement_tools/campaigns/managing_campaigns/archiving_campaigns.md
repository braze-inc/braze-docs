# 캠페인 아카이빙

> 아카이빙 캠페인은 작업 공간을 정리하는 간단하고 효과적인 방법이 될 수 있습니다. 더 많은 캠페인을 계속 구축할수록 적극적이고 관련성 높은 메시징 활동에 집중할 수 있습니다.

캠페인의 전송을 중지하거나 대시보드에서 지우려면 캠페인을 보관하면 됩니다:

1. **메시징** > **캠페인으로** 이동합니다.
2. 캠페인 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택합니다. 
3. **아카이브를** 선택합니다.

캠페인을 보관하면 해당 캠페인에서 더 이상 메시지가 전달되지 않습니다. 인앱 메시지의 경우, 해당 캠페인의 인앱 메시지는 더 이상 사용자에게 표시되지 않습니다.

캠페인 옆의 확인란을 선택하고 관련 옵션을 선택하여 사용할 수 있는 일괄 작업도 있습니다.

## 캠페인 보관 해제

보관된 캠페인을 선택하면 과거 결과를 볼 수 있지만 캠페인을 수정할 수는 없습니다. 캠페인을 수정하려면 캠페인의 보관을 해제해야 합니다. 캠페인 아카이브 해제하기:

2. 보관된 캠페인 행 위로 마우스를 가져가 <i class="fas fa-ellipsis-vertical"></i> 메뉴를 선택합니다. 
3. **보관 해제를** 선택합니다.

캠페인을 보관 해제한다고 해서 캠페인이 라이브 상태가 되는 것은 아닙니다. 이렇게 하면 캠페인 상태가 **활성으로** 업데이트됩니다. 이 시점에서 캠페인 설정 방법을 편집하고 검토할 수 있으며, 캠페인이 중지되고 메시지를 보내지 않습니다.

캠페인을 재개하고 메시지 전송을 시작하려면 캠페인 옆의 톱니바퀴 아이콘을 선택하고 **재개를** 선택합니다.

{% alert warning %}
세그먼트를 아카이브하면 해당 세그먼트를 사용하는 모든 캠페인도 아카이브됩니다.
{% endalert %}
