# 출시 후 캠페인 편집하기

> 이 문서에서는 캠페인 출시 후 캠페인의 특정 측면을 편집한 결과에 대한 개요를 설명합니다.

## 캠페인 중지하기

캠페인을 중지하려면 **캠페인 세부정보** 페이지를 열고 **캠페인 중지를** 선택합니다. 캠페인이 중지된 경우:

- 전송 예정 메시징이 취소됩니다.
- 초기 테스트가 이미 전송된 A/B 테스트는 영구적으로 취소됩니다.
- 이미 전송된 메시징에 대한 이벤트(예: 열린 클릭)는 계속 추적됩니다.

캠페인을 다시 시작하려면 **재시작을** 선택합니다. 캠페인은 메시지 전송 및 A/B 테스트를 계속 진행하지만 부재중 메시지가 다시 전송되거나 다시 예약되지 않습니다.

## 트리거된 캠페인

실행 기반 전달 캠페인 및 API 트리거 전달 캠페인에 대한 모든 변경 사항은 앞으로 보내기 전송에 즉시 적용됩니다. 

이러한 캠페인이 트리거되었지만 아직 전송되지 않은 경우(예: 1일 지연이 있는 실행 기반 전달 캠페인이 1일 지연 기간 동안 편집되는 경우) 예약된 캠페인에 대한 다음 지침을 참조하세요.

### 예약된 캠페인

캠페인 출시 후 캠페인을 변경해야 하는 경우 캠페인을 편집할 때 다음 항목을 참고하여 변경 사항이 원하는 효과를 가져오는지 확인하세요.

### 메시지 내용

메시지 내용(제목, 본문, 이미지 포함)이 변경되면 앞으로 전송되는 모든 메시징에 대해 저장 즉시 적용됩니다. 이미 발송된 메시징의 내용은 변경할 수 없습니다.

### 스케줄링 및 오디언스

캠페인의 예약된 전송 시간이나 오디언스를 편집하면 해당 변경 사항이 실제 캠페인에 즉시 반영됩니다.

### 전송 속도

전송 속도 제한을 사용할 때 Braze는 분 단위로 메시지를 '예약'하므로 메시지 전송 속도를 변경하려면 다음 프로세스를 준수하여 즉시 변경하세요.

## 즉시 변경하기

변경 사항을 즉시 적용하려면 다음을 수행하세요:

1. 영향을 받는 캠페인을 중지합니다.
2. 캠페인을 복제합니다.
3. 중복 캠페인을 편집합니다.

{% alert important %}
이렇게 하면 원래 캠페인을 이미 받은 사람들의 자격이 초기화되므로 원래 캠페인을 받지 못한 사람들을 위해 중복 캠페인을 필터링해야 할 수 있습니다.
{% endalert %}

## 활성 캠페인 초안 저장하기 {#campaign-drafts}

초안은 활성 캠페인을 대규모로 변경할 때 유용합니다. 초안을 작성하면 다음 출시 전에 계획된 변경 사항을 시험해 볼 수 있습니다.

{% alert note %}
캠페인은 한 번에 하나의 초안만 만들 수 있습니다. 또한 초안 변경 사항이 아직 출시되지 않았기 때문에 분석 기능을 사용할 수 없습니다.
{% endalert %}

초안을 만들려면 다음과 같이 하세요:

1. 활성 캠페인으로 이동합니다.
2. 변경하세요.
3. **초안으로 저장을** 선택합니다. 초안을 만든 후에는 초안을 실행하거나 삭제할 때까지 활성 캠페인을 편집할 수 없습니다.

활성 캠페인을 볼 수 있는 옵션이 있는 활성 캠페인의 초안입니다.]({% image_buster /assets/img/campaign_draft.png %})

초안을 편집할 때 캠페인 초안의 헤더 또는 캠페인 분석의 바닥글에서 활성 캠페인을 참조할 수도 있습니다. 

활성 캠페인으로 돌아가려면 분석 보기 또는 활성 캠페인 보기에서 **초안 편집을** 선택합니다.

### 인앱 메시지 우선순위 지정

**정확한 우선순위 설정을** 선택하고 다른 캠페인 또는 캔버스와 관련하여 우선순위를 지정하면 인앱 메시지 우선순위가 즉시(초안이 시작되기 전에) 업데이트됩니다.
