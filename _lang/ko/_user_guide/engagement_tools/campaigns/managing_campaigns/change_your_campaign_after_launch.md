# 론칭 후 캠페인 편집하기

> 이 문서에서는 출시 후 캠페인의 특정 측면을 편집한 결과에 대한 개요를 제공합니다.

## 캠페인 중지하기

캠페인을 중지하려면 **캠페인 세부정보** 페이지를 열고 **캠페인 중지**를 선택하세요. 캠페인이 중지된 경우:

- 전송 예정인 메시지는 취소됩니다.
- 초기 테스트가 이미 전송된 A/B 테스트는 영구적으로 취소됩니다.
- 이미 전송된 메시지에 대한 이벤트(예: 열기 클릭)는 여전히 추적됩니다.

캠페인을 재시작하려면 **재개**를 선택하세요. 캠페인은 메시지와 A/B 테스트를 계속 전송하지만, 놓친 메시지는 재전송되거나 재일정이 잡히지 않습니다.

## 트리거된 캠페인

실행 기반 전달 캠페인과 API 트리거 전달 캠페인에 대한 모든 변경 사항은 즉시 적용됩니다. 

이 캠페인이 트리거되었지만 아직 전송되지 않은 경우(예: 1일 지연이 있는 실행 기반 전달 캠페인이 1일 지연 기간 동안 편집된 경우), 예정된 캠페인에 대한 다음 지침을 참조하세요.

### 예약된 캠페인

캠페인 출시 후 캠페인을 변경해야 하는 경우, 캠페인을 편집할 때 다음 항목을 참고하여 변경 사항이 원하는 효과를 가져오는지 확인하세요.

### 메시지 내용

메시지 내용 변경(제목, 본문 및 이미지 포함)은 저장 시 즉시 적용되어 모든 메시지 전송에 반영됩니다. 이미 발송된 메시지의 내용을 변경하는 것은 불가능합니다.

### 예약 및 오디언스

캠페인의 예약된 전송 시간 또는 대상 그룹을 수정하면 해당 변경 사항이 실제 캠페인에 즉시 반영됩니다.

### 전송률

전송률 제한을 사용하는 경우, Braze는 메시지를 분 단위로 한"예약"하므로 메시지 전송률을 변경하려면 다음 프로세스를 준수하여 즉시 변경하세요.

## 즉시 변경하기

변경 사항을 즉시 적용하려면 다음을 수행하세요.

1. 영향을 받는 캠페인을 중지합니다.
2. 캠페인을 복제합니다.
3. 중복 캠페인을 수정합니다.

{% alert important %}
이렇게 하면 원본 캠페인을 이미 받은 사람들의 자격이 초기화되므로 원본 캠페인을 받지 않은 사람들을 위해 중복 캠페인을 필터링해야 할 수 있습니다.
{% endalert %}

## 활성 캠페인의 초안을 저장합니다 {#campaign-drafts}

초안은 활성 캠페인에 대한 대규모 변경을 만드는 데 유용합니다. 초안을 생성함으로써 다음 출시 전에 계획된 변경 사항을 시험해 볼 수 있습니다.

{% alert note %}
캠페인은 한 번에 하나의 초안만 가질 수 있습니다. 또한, 초안 변경 사항이 아직 출시되지 않았기 때문에 분석이 제공되지 않습니다.
{% endalert %}

초안을 작성하려면 다음을 수행하십시오:

1. 활성 캠페인으로 이동하십시오.
2. 변경 사항을 만드세요.
3. **초안으로 저장**을 선택하세요. 초안을 생성한 후에는 초안을 출시하거나 폐기할 때까지 활성 캠페인을 편집할 수 없습니다.

![활성 캠페인의 초안으로 활성 캠페인을 볼 수 있는 옵션이 있습니다.]({% image_buster /assets/img/campaign_draft.png %})

초안을 편집하는 동안 캠페인 초안의 헤더 또는 캠페인 분석의 바닥글에서 활성 캠페인을 참조할 수 있습니다. 

활성 캠페인으로 돌아가려면 분석 보기 또는 활성 캠페인 보기에서 **편집 초안**을 선택하십시오.

### 인앱 메시지 우선순위 지정

인앱 메시지 우선순위는 **정확한 우선순위 설정**을 선택하고 다른 캠페인 또는 캔버스와의 우선순위를 지정할 때 즉시 업데이트됩니다(초안이 시작되기 전).
