# 알림 채널

> [알림 채널은](https://www.braze.com/blog/android-o-push-notifications-channels/) Android O에서 추가된 푸시 알림을 정리하는 방법입니다. O부터는 모든 푸시 알림에 메시지 유형(예: '채팅 알림' 또는 '팔로우 알림')을 나타내는 알림 채널이 있어야 합니다. 그러면 사용자는 개별 채널에 따라 알림의 일부 기능(예: 스누즈, 소음/진동 설정 또는 옵트아웃 등)을 제어할 수 있습니다.

## Android O로 전환하기

알림 채널은 애플리케이션의 코드에서만 생성할 수 있으며 Braze 대시보드에서 프로그래밍 방식으로 생성할 수 없습니다. 엔지니어링 팀이 마케터와 협력하여 원하는 알림 채널이 대시보드에 제대로 추가되도록 하는 것이 좋습니다.

Android O부터는 푸시 알림을 표시하려면 유효한 채널이 필요합니다. 앱이 Android O 이상을 타겟팅하는 경우 Braze 소프트웨어 개발 키트 버전 2.1.0 이상을 사용해야 합니다. 개발팀은 애플리케이션 코드에서 사용하려는 채널과 각 채널에 대한 권장 알림 설정(예: 중요도, 소리, 조명)을 정의해야 합니다. Android의 개발자 설명서는 [여기에서](https://developer.android.com/preview/features/notification-channels.html), Braze의 개발자 설명서는 [여기에서]({{site.baseurl}}/developer_guide/platform_integration_guides/android/push_notifications/integration/standard_integration/#step-5-define-notification-channels) 확인할 수 있습니다.

{% alert note %}
Android는 채널 이름에 대한 현지화를 지원하므로 애플리케이션 코드에서 하나의 채널 ID를 여러 번역된 채널 이름과 연결할 수 있습니다.
{% endalert %}

이러한 채널이 생성되면 엔지니어링 담당자가 관련 채널 ID를 마케팅 팀에 전달해야 합니다. 팀은 캠페인과 캔버스에서 사용할 수 있도록 Braze 대시보드에 채널 이름과 채널 ID를 입력해야 합니다.

Braze 대시보드에 채널을 추가하려면 Android 푸시 작성기로 이동하여 알림 채널 필드를 선택한 다음 "채널 관리"를 선택합니다.
{% alert important %}
'앱 관리'가 포함된 권한을 가진 사용자만 채널을 관리할 수 있습니다.
{% endalert %}

## 소프트웨어 개발 키트 기본값 채널

Android에서는 API 레벨 26(Android O) 이상에서 푸시 알림을 표시하려면 유효한 채널이 필요합니다. 대시보드에서 추가 채널을 지정하지 않거나 잘못된 채널로 전송을 시도하는 경우 기본값으로 '일반'이라는 채널이 생성되어 사용되며, 이 채널은 Android SDK 2.1.0에 포함되어 있습니다. 소프트웨어 개발 키트에서 이 레이블의 이름을 바꾸고 채널에 대한 설명을 제공할 수 있습니다. 더 나은 사용자 경험을 제공하기 위해 이를 고려하는 것이 좋습니다.  

애플리케이션에 채널을 추가한 후에는 채널을 제거하도록 옵트인할 수 있습니다. 그러나 소비자는 항상 [제거한] 채널 수를 볼 수 있습니다.[3] Braze 대시보드에는 프로그래밍 방식으로 채널을 만드는 기능이 포함되어 있지 않으며, 원활한 경험을 제공하려면 애플리케이션의 코드에서 채널을 만들고 정의해야 합니다.

다시 한 번 강조하지만, 엔지니어링 팀과 협력하여 Android O 타겟팅으로 원활하게 전환할 수 있도록 하는 것이 좋습니다.

## 대시보드 대체 채널

Braze를 사용하면 대시보드 대체 채널을 지정할 수 있습니다. 대시보드 대체 채널의 목적은 명시적인 채널 선택이 없는 레거시 푸시 메시징에 대한 채널 ID를 제공하는 것입니다. 채널 선택은 Android 푸시 작성기에서 채널을 선택하는 것으로 정의합니다.

채널을 선택하지 않은 메시지는 대시보드 대체 채널 ID로 전송됩니다. 대시보드 대체 채널을 변경하면 명시적으로 선택한 채널이 없는 모든 메시지는 새 대체 채널의 ID로 전송됩니다.

다음은 대시보드 대체 채널 예상 동작의 예입니다:

대시보드 대체 채널이 '마케팅'이고 채널을 선택하지 않은 10개의 Android 푸시 메시지가 있습니다. '마케팅' 채널이 대시보드 대체 채널이기 때문에 이러한 캠페인은 '마케팅' 채널을 통해 전송됩니다.

또한 '소셜 알림' 채널을 통해 보내도록 선택한 15개의 메시지와 '마케팅' 채널을 통해 보내도록 선택한 5개의 메시지가 있습니다.

그런 다음 대시보드 기본값 채널을 "마케팅"에서 "업데이트"로 변경합니다.

이 경우 이전에 '마케팅' 채널을 통해 전송되던 채널 선택이 없는 10개의 캠페인은 모두 '업데이트' 채널을 통해 전송되므로 해당 메시지는 대체 채널을 통해 전송됩니다. '소셜 알림' 채널을 통해 전송되던 15개의 메시지는 '소셜 알림' 채널을 통해 계속 전송됩니다. '마케팅' 채널을 통해 발송되었던 5가지 메시지는 앞으로도 '마케팅' 채널을 통해 계속 발송될 예정입니다.

잘못된 채널 ID가 Braze에 제공된 경우(예: 개발자가 SDK에서 생성하지 않은 채널 ID를 제공한 경우), 당사는 귀하의 SDK 기본값 채널을 통해 알림을 전달합니다. 따라서 개발 중에 Braze 대시보드를 통해 알림 채널을 테스트하는 것을 적극 권장합니다.

채널에 대한 예상 동작을 더 잘 이해하려면 다음 표를 참조하세요:

## Braze 대시보드에 채널 추가하기

1. Android 푸시가 포함된 캠페인 또는 캔버스를 열고 **캠페인 편집을** 클릭합니다.
2. Android 푸시 메시지 작성기로 이동합니다.
3. **알림 채널 관리를** 클릭합니다. 여기에 추가된 모든 채널은 모든 캠페인과 캔버스에서 전 세계적으로 사용할 수 있습니다. 워크스페이스에 대한 '앱 관리' [권한이]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/user_permissions/#limited-and-team-role-permissions) 있어야 채널을 관리할 수 있습니다.

특정 캠페인 또는 캔버스 단계에 알림 채널을 적용하면 Android 푸시에 대한 **도달 가능한 사용자** 집계(타겟 오디언스 단계에 위치)는 변경되지 않습니다. 단, 선택한 알림 채널에 가입한 사용자만 메시지를 볼 수 있으며, 이 오디언스를 기준으로 캠페인 분석(좋아요 클릭 수)이 측정됩니다.

\![]({% image_buster /assets/img_archive/Click_Here.png %})

{:start="4"}
4\. **알림 채널 추가를** 클릭합니다.
5\. 추가하려는 알림 채널의 이름과 ID를 입력합니다.<br><br>\![]({% image_buster /assets/img_archive/Enter_Channel.png %})<br><br>
6\. 추가하려는 각 알림 채널에 대해 4단계와 5단계를 반복합니다.
7\. **저장을** 눌러 변경 사항을 저장합니다.

## 대체 채널 지정하기

대체 채널은 메시지에 대한 채널을 선택하지 않은 경우 Braze가 Android 메시지를 보내려고 시도하는 채널입니다. 채널 선택 없이 Android 메시징 메시지를 표시하는 유일한 캠페인과 캔버스는 팀이 Braze 대시보드에 채널을 추가하기 전에 만든 캠페인과 캔버스입니다. 대체 채널을 변경하면 명시적으로 채널을 선택하지 않아도 변경 사항이 모든 캠페인과 캔버스에 전역적으로 적용됩니다.

1. 기존 캠페인 또는 캔버스를 엽니다.
2. Android 푸시 작성기로 이동합니다.
3. 알림 채널 옵션을 확장한 후 **알림 채널 관리를** 선택합니다. <br><br>\![]({% image_buster /assets/img_archive/Change_Fallback.png %}){: style="max-width:80%;"}<br><br>
4. 대시보드에 채널을 추가합니다(아직 추가하지 않은 경우).
5. 대체 채널로 지정하려는 채널 옆의 라디오 다이얼을 선택합니다.
6. 변경 사항을 저장합니다. 변경 사항은 전 세계에 적용됩니다.

## Android 푸시 메시지에 채널 추가하기

1. 캠페인 또는 캔버스에서 Android 푸시 작성기로 이동합니다.
2. 드롭다운에서 사용하려는 채널을 선택합니다. 드롭다운이 없는 대신 다음과 같은 보기가 있는 경우 캠페인을 위해 채널을 추가한 후 선택해야 합니다.
