## 훌륭한 사용 사례

여기에서 영감을 받을 수 있는 훌륭한 사용 사례를 소개합니다!

| 사용 사례 | 설명 |
| --- | --- |
| 재참여 | 앱을 설치하지 않은 사용자 포함하여 앱 외부의 사용자에게 도달하세요. |
| 온보딩 | 신규 사용자가 푸시 알림을 켜거나 소셜 네트워크에서 앱을 공유하도록 onboard하고 격려하세요. |
| 풍부한 메시지 | 풍부하고 동적인 HTML 메시지를 허용합니다. |
| 멀티미디어 콘텐츠 | 비디오 및 이미지와 같은 사용자를 참여시키는 멀티미디어 콘텐츠 배치의 용이성. |
| 뉴스레터 | 사용자 참여를 유지하기 위해 매월 또는 매주 뉴스레터를 편리하게 보냅니다. |
| 거래 | 최근 구매에 대해 사용자에게 알리고 [거래 이메일]({{site.baseurl}}/user_guide/message_building_by_channel/email/transactional_message_api_campaign/)을 통해 중요한 제품 및 배송 정보를 전달합니다.
