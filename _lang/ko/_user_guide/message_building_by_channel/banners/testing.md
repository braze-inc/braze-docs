# 배너 테스트

> 캠페인을 시작하기 전에 배너 메시지를 테스트하여 모든 미디어, 카피, 개인화 및 커스텀 속성이 올바르게 렌더링되는지 확인하는 방법을 알아보세요. 자세한 내용은 [배너 정보를]({{site.baseurl}}/user_guide/message_building_by_channel/banners) 참조하세요.

## 전제 조건

Braze에서 배너 메시지를 테스트하려면 먼저 [Braze에서 배너 캠페인을]({{site.baseurl}}/user_guide/message_building_by_channel/banners/creating_campaigns/) 만들어야 합니다. 또한 테스트하려는 게재 위치가 [앱이나 웹사이트에]({{site.baseurl}}/developer_guide/banners/placements) 이미 배치되어 있는지 확인합니다.
