# 배너 테스트

> 캠페인을 시작하기 전에 배너 메시지를 테스트하여 모든 미디어, 카피, 개인화 및 사용자 지정 속성이 올바르게 렌더링되는지 확인하는 방법을 알아보세요. 더 일반적인 정보는 [배너에 대하여]({{site.baseurl}}/user_guide/message_building_by_channel/banners)를 참조하십시오.

## Prerequisites

Braze에서 배너 메시지를 테스트하려면 먼저 [Braze에서 배너 캠페인을]({{site.baseurl}}/user_guide/message_building_by_channel/banners/creating_campaigns/) 생성해야 합니다. 또한 테스트하려는 게재 위치가 [앱이나 웹사이트에]({{site.baseurl}}/developer_guide/banners/creating_placements) 이미 배치되어 있는지 확인합니다.
