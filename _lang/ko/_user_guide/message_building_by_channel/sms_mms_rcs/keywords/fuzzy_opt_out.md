# 퍼지 옵트아웃

\![iOS 메시지 채팅은 수신된 퍼지 옵트아웃 "제발 멈춰줘"에 대한 아웃바운드 옵트아웃 메시지를 보여줍니다.]({% image_buster /assets/img/sms/fuzzy1.jpg %}){: style="float:right;max-width:30%;margin-left:15px;"}

> Braze로 SMS, MMS 및 RCS를 보내는 사용자는 정의된 적용 가능한 법률, 규정 및 산업 표준을 준수해야 합니다. 옵트아웃의 경우, 법률에 따르면 사용자가 "STOP"이라고 문자 메시지를 보내면 해당 메시징 프로그램과 관련된 모든 후속 메시지가 중단됩니다. Braze는 이러한 메시지를 자동으로 처리하고 사용자를 구독 해지합니다.<br><br>퍼지 옵트아웃은 수신 메시지가 [옵트아웃 키워드]({{site.baseurl}}/user_guide/message_building_by_channel/sms_mms_rcs/keywords/optin_optout/)와 일치하지 않을 때를 인식하려고 시도하지만, 옵트아웃 의도를 나타냅니다. 퍼지 옵트아웃이 활성화되고 수신된 키워드 응답이 "퍼지"로 간주되면, Braze는 사용자가 옵트아웃하도록 지시하는 응답 메시지로 자동 응답합니다.

현재, [지역 언어]({{site.baseurl}}/user_guide/message_building_by_channel/sms/keywords/keyword_handling/#multi-language-support)로 영어를 사용하여 생성된 옵트아웃 키워드만 지원됩니다.

## 퍼지로 간주되는 것은 무엇입니까?

수신 응답이 "퍼지"로 간주되기 위한 기준은 다음과 같습니다:
- QWERTY 키보드에서 왼쪽 또는 오른쪽으로 한 글자를 바꾸면 일치하는 옵트아웃 키워드가 생성됩니다.
- 메시지의 부분 문자열이 옵트아웃 키워드와 일치합니다.

예를 들어, "Stpo" 또는 "제발 멈춰줘"는 퍼지로 간주되며, 퍼지 옵트아웃 응답이 전송됩니다. 사용자가 옵트아웃 키워드로 응답하면 구독 해지 이벤트가 발생합니다.

## 퍼지 옵트아웃 구성하기

퍼지 옵트아웃을 구성하려면 구독 그룹 키워드 관리 페이지로 이동합니다.

1. **청중** > **구독 그룹 관리**로 이동하여 **SMS/MMS/RCS** 구독 그룹을 선택합니다.
2. **전역 키워드**에서 **옵트아웃** 카테고리를 찾아 연필 아이콘을 선택합니다.
3. **퍼지 옵트 아웃**을 켜서 활성화하세요.
4. 원하는 대로 퍼지 옵트 아웃 응답을 수정하세요. 

\![옵트 아웃 키워드를 편집할 섹션.]({% image_buster /assets/img/sms/fuzzy2.png %})

## 퍼지 옵트 아웃 메시지에 대한 모범 사례

구독자에게 명확하고 준수하며 긍정적인 경험을 보장하기 위해, 퍼지 옵트 아웃 메시지를 신중하게 구성하는 것이 중요합니다. 퍼지 옵트 아웃 메시지의 주요 목적은 **지정된 옵트 아웃 키워드와 유사하지만 정확히 일치하지 않는 메시지를 보내는 사용자를 안내하는 것입니다.** 메시지는 사용자가 성공적으로 구독 취소하는 방법을 안내합니다.

### 중요한 고려 사항

{% alert warning %}
**절대** 퍼지 옵트 아웃 메시지를 구독 취소를 확인하도록 구성하지 마세요. 퍼지 옵트 아웃 메시지에는 사용자가 이미 성공적으로 구독 취소했음을 암시하는 언어가 포함되어서는 안 됩니다. 예를 들어, **사용하지 마세요** "당신은 구독 취소되었습니다," "이 번호로부터 더 이상 메시지를 받지 않을 것입니다," 또는 "당신은 이제 옵트 아웃되었습니다"와 같은 문구를 사용하지 마세요.
{% endalert %}

퍼지 옵트 아웃 메시지는 사용자가 성공적으로 옵트 아웃하기 전에 전송됩니다. 확인 언어를 사용하면 구독자가 구독 취소되었다고 믿게 되어 원치 않는 메시지가 계속 오고, 구독자가 불만을 느끼며, 상당한 준수 위험이 발생합니다.

{% alert warning %}
**절대** 퍼지 옵트 아웃 메시지를 정확한 옵트 아웃 키워드와 동일하거나 유사하게 구성하지 마세요.
{% endalert %}

퍼지 메시지가 정확한 옵트 아웃 키워드와 동일하거나 너무 가까운 경우(예를 들어, "STOP"이 정확한 키워드이고 퍼지 메시지가 "구독 취소를 위해 STOP을 문자로 보내세요"인 경우), 사용자의 초기 메시지가 실제로 구독 취소로 이어졌는지 아니면 다른 조치를 취해야 하는지에 대한 혼란을 초래할 수 있습니다. 퍼지 메시지는 항상 사용자가 취해야 할 조치를 명확히 해야 합니다.
