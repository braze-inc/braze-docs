# MMS 캠페인 만들기

> 이 문서에는 SMS 작곡가의 일부인 MMS 구성에 대한 특정 정보가 포함되어 있습니다. SMS/MMS 작성기에 대한 자세한 정보는 SMS 작성기를 참조하십시오.

## MMS 전송 기본 사항

### 구독 그룹 선택

타겟팅할 MMS 사용 전화 번호로 구독 그룹을 지정해야 합니다(짧은 코드 또는 긴 코드 가능).

### 메시지 본문 입력

미디어 라이브러리에서 PNG, JPEG, GIF 및 VCF 이미지 유형을 입력하거나 URL을 지정합니다. 단 하나의 이미지만 지원됩니다.

### MMS 전송에 대한 이해

MMS는 텍스트 전용 SMS와 다른 요금으로 청구되며, 모든 통신사가 MMS를 수신할 수 있는 것은 아닙니다. 이러한 경우 Twilio는 자동으로 MMS를 사용자가 클릭할 수 있는 이미지 링크로 변환합니다.

### 연락처 카드 사용

연락처 카드(때때로 vCard 또는 가상 연락처 파일(vcf)로 알려짐)는 주소록이나 연락처 책에 쉽게 가져올 수 있는 비즈니스 및 연락처 정보를 전송하기 위한 표준화된 파일 형식입니다. 이 카드들은 프로그래밍 방식으로 생성되어 Braze 미디어 라이브러리에 업로드되거나 내장된 연락처 카드 생성기를 통해 생성될 수 있습니다.

## MMS 메시지 만들기

MMS 메시지를 작성하려면 MMS 전송을 위해 구독 그룹을 구성해야 합니다. 이는 구독 그룹을 선택할 때 MMS 태그가 표시되는 것으로 알 수 있습니다. MMS-활성화된 구독 그룹을 선택하면 이미지를 업로드하거나 이미지 URL을 참조하거나 연락처 카드를 포함할 수 있는 기능이 제공됩니다.

"작성" 탭에서 메시지를 작성하세요.

### 이미지 사양

| **이미지 사양** | **권장 속성** |
|--------------------------|----------------------------|
| 크기                     | 최대 600 KB        |
| 파일 유형               | PNG, JPEG, GIF             |

## MMS 메시지 미리보기

Braze는 메시지 작성기의 **미리보기** 패널에서 업로드한 이미지의 미리보기를 제공합니다. 

{% alert note %}
SMS/MMS 자산의 순서는 사용자 지정할 수 없습니다. 주문은 이 메시지를 수신하는 휴대폰에 따라 달라집니다.
{% endalert %}

다음 여행을 준비하세요!!! 미리보기는 텍스트로 전송된 메시지와 이미지를 보여줍니다.
