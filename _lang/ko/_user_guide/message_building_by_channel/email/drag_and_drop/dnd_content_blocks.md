# 드래그 앤 드롭 편집기 콘텐츠 블록

> 드래그 앤 드롭 편집기 전용으로 사용되는 콘텐츠 블록은 다른 채널에서 사용되는 [콘텐츠 블록과]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/content_blocks/) 기능이 유사합니다. 다양한 이메일 캠페인에서 참조할 수 있는 정보를 보관할 수 있는 중앙 집중식 위치입니다. 여기에는 이메일 헤더, 프로모션 콜아웃 등을 재사용 가능한 하나의 행에 그룹화하는 것이 포함됩니다.

{% alert note %}
드래그 앤 드롭 콘텐츠 블록은 캔버스의 이메일 캠페인과 이메일 메시지에서만 사용할 수 있습니다.
{% endalert %}

## 콘텐츠 블록 생성하기

콘텐츠 블록을 만들려면 다음과 같이 하세요:

{% multi_lang_include create_content_block.md location="dnd" %}

{% alert important %}
각 드래그 앤 드롭 콘텐츠 블록은 한 행으로 제한됩니다. 그러나 드래그 앤 드롭 편집기 블록을 사용하여 이메일 메시징에 맞게 콘텐츠 블록을 구축하고 커스텀할 수 있습니다.
{% endalert %}

## 콘텐츠 블록 사용

이메일에 콘텐츠 블록을 추가하는 방법에는 편집기를 사용하거나 Liquid를 사용하는 두 가지 방법이 있습니다.

### 편집기를 사용하여 콘텐츠 블록 추가하기

편집기에서 콘텐츠 블록을 추가하려면 다음과 같이 하세요:

1. 편집기의 **행** 탭으로 이동하여 **콘텐츠 블록을** 선택합니다. 
2. 콘텐츠 블록을 이메일 편집기로 드래그 앤 드롭합니다. 
3. (선택 사항) 탐색 메뉴에서 버튼을 선택하여 콘텐츠 블록의 너비를 조정합니다. 이메일 글로벌 스타일 설정에서 지정하지 않은 경우 기본값은 100%이며, 그렇지 않은 경우 글로벌 설정이 적용됩니다. <br><br>너비를 편집할 수 있는 옵션이 있는 양면 화살표입니다.]({% image_buster /assets/img_archive/content_block_width_updated.png %}){: style="max-width:30%;" }<br><br>

이메일 편집기에 콘텐츠 블록을 추가한 후에는 **템플릿 & 미디어에서** 만든 원본 콘텐츠 블록에 영향을 주지 않는 콘텐츠 블록을 편집할 수 있습니다. 드래그 앤 드롭으로 추가한 콘텐츠 블록은 원래 콘텐츠 블록에 연결되지 않기 때문입니다. 원래 콘텐츠 블록의 변경 내용을 보려면 콘텐츠 블록을 다시 이메일 편집기로 끌어다 놓습니다. 

하나의 행 블록에 여러 개의 콘텐츠 블록을 추가하면 드래그 앤 드롭 편집기에서 정렬이 잘못될 수 있습니다. 별도의 행 블록을 사용하여 행 수준에서 콘텐츠 전체의 정렬을 유지하세요.

### Liquid를 사용하여 콘텐츠 블록 추가하기

Liquid를 사용하여 콘텐츠 블록을 추가하려면 다음과 같이 하세요:

1. 이메일 캠페인으로 이동하여 **이메일 본문 편집을** 선택합니다. 
2. <i class="fas fa-plus"></i> **개인화를** 클릭합니다.
3. 개인화 **추가** 탭을 찾아 **개인화 유형** 드롭다운에서 **콘텐츠 블록을** 선택합니다.
4. **속성** 필드에서 콘텐츠 블록의 이름을 선택합니다. Liquid 스니펫 필드가 콘텐츠 블록 Liquid 태그로 채워집니다. 
5. Liquid 스니펫을 복사하여 텍스트 편집기 블록에 붙여넣습니다. <br>옵션이 있는 개인화 추가 탭을 클릭합니다.]({% image_buster /assets/img_archive/dnd_content_block_personalization.png %}){: style="max-width:30%;"}

이메일 메시징을 미리 볼 때 Liquid 스니펫이 드래그 앤 드롭 편집기 콘텐츠 블록으로 표시됩니다. 

{% alert important %}
Liquid로 이메일 편집기에 콘텐츠 블록을 추가하면 이 콘텐츠 블록은 **템플릿 & 미디어에서** 만든 원본 콘텐츠 블록에 연결됩니다. 즉, 콘텐츠 블록이 원래 콘텐츠 블록 템플릿의 변경 사항을 반영하도록 업데이트됩니다.
{% endalert %}

## 콘텐츠 블록 업데이트하기

기존 콘텐츠 블록을 업데이트하려면 **콘텐츠 블록** 페이지에서 원본 콘텐츠 블록을 편집하거나 원본 메시지의 HTML을 새 콘텐츠 블록에 복사하면 됩니다. 콘텐츠 블록 템플릿에 대한 업데이트는 Liquid를 통해 콘텐츠 블록이 추가된 모든 이메일 메시징에서 업데이트됩니다.

콘텐츠 블록을 보관하려면 **템플릿** > **콘텐츠 블록으로** 이동하여 콘텐츠 블록의 세로 줄임표 아이콘( <i class="fas fa-ellipsis-vertical"></i> )을 선택한 다음 **보관을** 클릭합니다. 콘텐츠 블록을 보관하면 메시징에 보관된 블록의 콘텐츠가 계속 포함됩니다. 그러나 보관된 콘텐츠 블록은 읽기 전용이므로 편집하기 전에 콘텐츠 블록의 보관을 해제하세요.
