# 자주 묻는 질문

> 이 페이지에서는 이메일용 드래그 앤 드롭 편집기와 관련하여 자주 묻는 질문에 대한 답변을 제공합니다.

### 다크 모드에서 이메일이 어떻게 표시되는지 미리 볼 수 있나요?

예. 드래그 앤 드롭 편집기의 **미리보기 및 테스트** 섹션으로 이동하여 **다크 모드를** 켭니다. 또한 다양한 사용자 플랫폼에서 이메일을 미리 보고 테스트하고 가능한 경우 행 배경 이미지에 투명한 이미지를 사용하는 것이 좋습니다. 

### 웹 보기에서 패딩을 업데이트하지 않고 모바일에서 이메일 패딩을 변경하려면 어떻게 해야 하나요?

모바일 및 웹 보기의 패딩을 단독으로 편집할 수 없으므로 편집한 내용은 두 보기 모두에 반영됩니다. 해결 방법으로 다양한 화면 크기에 따라 패딩을 설정하는 CSS 로직을 추가할 수 있습니다. 그러나 드래그 앤 드롭 편집기에서는 이 작업을 수행할 수 없습니다. 따라서 HTML 파일을 내보내고 대신 HTML 편집기를 사용하는 것이 좋습니다.

### 드래그 앤 드롭 편집기에서 행 높이를 조정할 수 있나요?

행 높이가 콘텐츠에 맞게 자동으로 조정됩니다. 대안으로 다음과 같은 방법을 권장합니다:
1. 디바이더 블록을 추가합니다.
2. 투명도를 켜려면 토글을 클릭합니다.
3. 높이를 조정합니다.

### 에디터에서 레이어를 구축할 수 있나요? 배경 이미지를 추가하고 이미지 위에 레이어를 추가한 다음 그 위에 텍스트 레이어를 추가할 수 있나요?

드래그 앤 드롭 편집기는 현재 두 개의 레이어를 지원합니다. 행 배경 이미지를 설정하고 배경 색상을 커스텀할 수 있습니다.

### 드래그 앤 드롭 이메일을 캠페인이나 캔버스 내에서 구축한 후 템플릿으로 저장할 수 있나요?

아니요, 저장하려면 **이메일 템플릿에서** 이메일을 다시 만들어야 합니다.

### 드래그 앤 드롭 편집기에 이메일 첨부파일을 추가할 수 있나요?

아니요, 드래그 앤 드롭 편집기는 현재 이메일에 첨부파일을 추가하는 기능을 지원하지 않습니다.
