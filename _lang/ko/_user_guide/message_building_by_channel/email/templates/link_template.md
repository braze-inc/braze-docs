# 링크 템플릿

> 링크 템플릿을 사용하면 매개변수를 추가하거나 URL을 미리 추가하여 이메일 캠페인에 사용할 수 있는 동적이고 재사용 가능한 링크를 만들 수 있습니다. 이렇게 하면 캠페인과 메시지 전반에 걸쳐 URL의 일관성을 유지할 수 있습니다. 

{% alert note %}
링크 템플릿은 선택적 기능입니다. **템플릿** 섹션에 **이메일 링크 템플릿이** 없는 경우 계정 매니저에게 문의하여 기능을 사용 설정하세요.
{% endalert %}

## 작동 방식

링크 템플릿은 다음과 같은 사용 사례에서 가장 자주 사용됩니다:

- 지정된 이메일 메시징의 모든 링크에 Google 애널리틱스 쿼리 매개변수 추가하기
- 지정된 이메일 메시지에 있는 모든 링크에 대한 URL 준비하기

신제품 출시를 위한 프로모션 이메일 캠페인을 운영한다고 가정해 보겠습니다. 사용자를 제품 페이지로 안내하는 링크 템플릿을 사용하고 사용자 이름이나 특정 프로모션 코드를 포함하도록 링크를 개인화할 수 있습니다. 이를 통해 얼마나 많은 사용자가 링크를 클릭하고 구매했는지 추적할 수 있습니다. 이렇게 하면 링크 전반에 걸쳐 일관성을 유지하고 분석을 더 잘 추적할 수 있습니다.

## 링크 템플릿 만들기

다양한 요구 사항을 지원하기 위해 링크 템플릿을 무제한으로 만들 수 있습니다. 링크 템플릿을 만들려면 다음과 같이 하세요:

1. **템플릿** > **이메일 링크 템플릿으로** 이동합니다. 
2. **이메일 링크 템플릿 생성을** 선택합니다.
3. 링크 템플릿에 이름을 지정합니다.
4. (선택 사항) 설명, 팀 또는 태그를 추가하여 링크 템플릿에 대한 세부 정보를 추가합니다.
5. (선택 사항) 토글을 선택하면 이메일 캠페인과 캔버스의 링크에 링크 템플릿을 자동으로 추가할 수 있습니다. 이는 새 이메일 또는 기존 이메일에 새 링크를 추가할 때 적용됩니다.

만들 수 있는 링크 템플릿에는 두 가지 유형이 있습니다:

- [URL 앞에 삽입되는 링크 템플릿](#prepend-link-template)
- [URL 뒤에 삽입되는 링크 템플릿](#append-link-template)

링크 템플릿과 [Liquid를]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/liquid/) 사용하는 경우 일관된 렌더링을 위해 Liquid는 본문 태그 내에만 추가해야 합니다.

### 준비합니다: URL 앞에 삽입되는 링크 템플릿 만들기 {#prepend-link-template}

이메일 메시징의 링크 앞에 문자열 또는 URL을 추가하려면 다음과 같이 하세요:

1. 새 링크 템플릿을 만듭니다.
2. **템플릿 위치를** **URL 이전으로** 설정합니다. 
3. URL에 항상 앞에 붙는 문자열을 입력합니다. 

**템플릿 미리보기는** 링크 템플릿이 URL 앞에 어떻게 삽입되는지 예시를 보여주기 위해 제공됩니다.

템플릿 위치, URL 추가, 템플릿 미리보기 필드에 URL 앞에 링크 템플릿을 삽입하는 과정을 입력합니다.]({% image_buster /assets/img_archive/link_template_preappend.png %}){: style="max-width:90%;"}

### 추가: URL 뒤에 삽입되는 링크 템플릿 만들기 {#append-link-template}

이메일 메시징의 URL 뒤에 쿼리 매개변수를 추가하려는 경우:

1. 새 링크 템플릿을 만듭니다.
2. **템플릿 위치를** **URL 뒤로** 설정합니다. 
3. 각 URL 끝에 쿼리 매개변수(`value=example`)를 입력합니다. URL 끝에 여러 개의 매개변수를 추가할 수 있습니다.

템플릿 위치, 쿼리 매개변수, 템플릿 미리보기 필드는 URL 뒤에 링크 템플릿 삽입 프로세스를 위한 필드입니다.]({% image_buster /assets/img_archive/link_template_postappend.png %}){: style="max-width:90%;"}

## 이메일 캠페인에서 링크 템플릿 사용

링크 템플릿을 설정한 후에는 이메일에 사용할 템플릿을 선택할 수 있습니다.

- **HTML 편집기:** **콘텐츠** 섹션 아래의 **링크 관리** 탭으로 이동합니다. 링크 템플릿 **추가를** 선택하고 링크 템플릿을 선택한 다음 **추가를** 선택합니다.

{% alert important %}
업데이트된 HTML 이메일 편집기에서 **링크 관리** 탭에 액세스하려면 링크 별칭 지정이 켜져 있어야 합니다. 링크 별칭 지정을 사용 설정하려면 계정 매니저에게 문의하세요.
{% endalert %}

- **드래그 앤 드롭 편집기:** **콘텐츠** > **링크 관리** 탭을 선택합니다. 그런 다음 **링크 템플릿 추가를** 선택합니다. 드래그 앤 드롭 편집기에서 링크 템플릿에 액세스하려면 [링크 별칭]({{site.baseurl}}/user_guide/message_building_by_channel/email/templates/link_aliasing/) 지정이 켜져 있어야 합니다.

드래그 앤 드롭 편집기의 링크 관리 탭에 링크 템플릿 예시 목록이 표시됩니다.]({% image_buster /assets/img_archive/link_template_messagecomposer2.png %})

{% alert note %}
링크 템플릿은 일반 텍스트에는 적용되지 않습니다. 즉, 링크 템플릿의 매개변수가 포함되지 않은 클릭은 이메일의 일반 텍스트 버전에서 발생한 클릭일 수 있으므로 커런츠에 표시될 수 있습니다.
{% endalert %}

**링크 관리** 탭에서 링크 템플릿을 추가할 때 오른쪽으로 스크롤하여 추가한 템플릿을 확인합니다. 이메일 내의 기존 링크에 이미 링크 템플릿이 추가된 경우 새로 추가되는 링크에도 기본값으로 링크 템플릿이 추가됩니다.

## 링크 템플릿 관리하기

링크 템플릿을 [복제할]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/managing_templates/) 수도 있습니다. 템플릿에서 템플릿 및 크리에이티브 콘텐츠를 만들고 관리하는 방법에 대해 자세히 알아보세요 [ & 미디어]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/).

{% alert important %}
현재 링크 템플릿에는 템플릿 보관 기능을 사용할 수 없습니다.
{% endalert %}
