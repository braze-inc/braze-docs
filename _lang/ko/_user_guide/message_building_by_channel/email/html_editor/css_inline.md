# CSS

> CSS 인라이닝은 CSS 스타일 시트의 스타일을 HTML 이메일 본문으로 이동하는 이메일 전처리의 한 형태입니다. "인라인"이라는 용어는 스타일이 개별 HTML 요소에 "인라인"으로 적용된다는 것을 의미합니다.

일부 이메일 클라이언트의 경우 CSS 인라이닝을 사용하면 이메일이 렌더링되는 방식을 개선하고 이메일이 예상한 대로 표시되는지 확인할 수 있습니다. 이미 CSS의 대부분을 인라인 처리했거나 HTML 및 CSS가 대부분의 메일 클라이언트의 요구 사항과 호환된다고 확신하는 경우에는 이 기능을 설정하지 않아도 될 수도 있습니다. 동적으로 임베드된 스타일이 기존 인라인 스타일과 충돌할 수 있으며 예상되는 미리보기 및 이메일 렌더링이 변경될 수 있습니다.

## CSS 인라이닝 사용

HTML 편집기의 **정보 보내기** 탭에서 **인라인 CSS** 사용 토글을 사용하여 이메일 메시징에 대한 CSS 인라이닝을 설정할지 여부를 제어할 수 있습니다.

!!! HTML 작성기에서 CSS 인라이닝을 관리하려면 체크박스를 선택합니다.]({% image_buster /assets/img_archive/css-inline2.png %}){: style="max-width:40%;"}

### 기본값 인라이닝 상태

**설정** > **이메일 환경설정에서** 기본값을 전역적으로 켜거나 끌 상태로 설정할 수 있습니다. **CSS 인라이닝** 설정을 찾습니다. 이 설정은 모든 새 이메일 메시징이 시작할 기본값을 결정합니다. 이 설정을 변경해도 기존 이메일 메시지에는 영향을 미치지 않습니다. 이메일 메시지를 작성하는 동안 언제든지 이 기본값을 재정의할 수도 있습니다.

이메일 설정에 있는 기본값으로 새 이메일에 CSS를 인라인으로 설정합니다.]({% image_buster /assets/img_archive/css-inline1.png %})
