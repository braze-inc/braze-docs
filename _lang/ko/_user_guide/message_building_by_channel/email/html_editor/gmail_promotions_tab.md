# Gmail 프로모션 설정

> [Gmail 모바일 프로모션 탭](https://developers.google.com/gmail/promotab/)은 마케터가 제목 줄이나 프리헤더 정보뿐만 아니라 "카드"의 주석을 통해 더 많은 정보를 보낼 수 있도록 합니다. Braze에는 이메일 캠페인에서 카드를 만드는 데 도움이 되는 내장 도구가 있습니다.

## 전제 조건

먼저, 도메인과 하위 도메인을 <a href="mailto:p-promo-outreach@google.com">p-promo-outreach@google.com</a>의 Google 프로모션 탭 아웃리치 팀에 전달하여 Gmail의 허용 목록에 추가합니다. 이렇게 하면 Gmail 프로모션 탭의 제품 캐러셀과 같은 풍부한 이미지를 표시하는 모든 기능을 사용할 수 있습니다.

## Braze로 카드 만들기

이메일 캠페인을 위한 Gmail 프로모션 카드를 만들려면 다음 단계를 따르세요. 편집기에서 **내용** 섹션에서 벗어나면 **Gmail 프로모션** 탭의 필드와 정보가 초기화됩니다. 프로모션 카드 설정을 완료하고 HTML 코드가 손실되지 않도록 생성된 HTML을 복사하세요.

### 1단계: 이메일 캠페인 만들기

먼저, [이메일 캠페인 만들기]({{site.baseurl}}/user_guide/message_building_by_channel/email/html_editor/creating_an_email_campaign/)를 선택하고 **HTML 코드 편집기**를 편집 경험으로 선택합니다.

### 2단계: Gmail 프로모션 카드에 세부정보 추가

다음으로, HTML 편집기의 **내용** 섹션으로 이동하여 **Gmail 프로모션** 탭을 선택합니다. **기본 정보** 아래의 필드를 작성한 후 **HTML 코드 생성**를 선택합니다. 이것은 **HTML 코드를 복사하여 붙여넣기`<Head>` 섹션 아래에 있는 Gmail 프로모션 탭 카드의 스크립트를 생성하는 데 도움이 됩니다.

\![카드를 만드는 방법의 예.]({% image_buster /assets/img/create-gmail-promo.png %})

### 3단계: Gmail 프로모션 카드를 사용자 정의하세요.

할인 제안, 거래 카드, 프로모션 카드 또는 Gmail 프로모션 카드에 대한 모든 옵션을 포함할지 선택하세요.

{% tabs %}
{% tab Discount offer %}

할인 제안을 설정하면 할인에 대한 유효 날짜를 지정할 수 있습니다. 

1. **할인 제안** 토글을 선택하세요.
2. **제안**에 대해 할인에 대한 간단한 요약을 입력하세요. 예를 들어 "20% 할인"입니다.
3. **코드**에 대해 사용자가 결제 시 적용해야 하는 프로모션 코드를 추가하세요.
4. 그런 다음 할인 제안의 시작 날짜와 시간을 선택하세요.
5. 할인 제안이 특정 시간에 종료되어야 하는지 또는 영원히 종료되지 않아야 하는지 결정하세요.

\![할인 제안에 대한 제안 값, 코드 및 시작 날짜와 시간을 지정하는 옵션.]({% image_buster /assets/img/gmail_promo_discount_details.png %}){: style="max-width:70%;"}

{% endtab %}
{% tab Deal Cards %}

거래 카드를 사용하여 이메일 본문의 맨 위에 주요 거래 정보를 직접 제공합니다. 이것은 수신자가 제안 세부정보를 빠르게 이해하고 조치를 취할 수 있도록 합니다. 예를 들어, 거래 카드를 사용하여 한정 시간 제안을 홍보하고 사용자가 이메일 내에서 세부정보를 검색할 필요성을 줄일 수 있습니다.

1. **거래 카드** 토글을 선택하세요.
2. **제안**에 대해 할인에 대한 간단한 요약을 입력하세요. 예를 들어 "모든 신발 20% 할인"입니다.
3. (선택 사항) **코드**에 대해 사용자가 결제 시 적용해야 하는 프로모션 코드를 추가하세요.
4. 다음 URL 중 하나 이상을 입력하세요. 
-  **제안 페이지 URL:** 특정 제안 랜딩 페이지의 URL입니다. 이것은 "지금 쇼핑하기" (또는 유사한) 버튼을 생성합니다. Deal Card에 이 URL을 제공하는 것을 권장합니다. 
- **상인 홈페이지 URL:** 귀하의 주요 홈페이지에 대한 URL입니다. 특정 제안 페이지 URL이 없는 경우에만 이 필드를 사용하세요.
5. (선택 사항) 제안의 시작 날짜를 추가하세요.
6. 제안이 특정 시간에 종료되어야 하는지 또는 결코 종료되지 않아야 하는지 결정하세요.

\![Deal Card에 대한 제안 값, 코드 및 시작 날짜 및 시간을 지정하는 옵션.]({% image_buster /assets/img/gmail_promo_deal_cards.png %}){: style="max-width:70%;"}

{% endtab %}
{% tab Promotion cards %}

제품 캐러셀의 프로모션 카드는 제안에 대한 이미지를 제공하는 데 유용합니다. 제품 캐러셀의 변수를 사용자 정의하고 각 이미지가 고유한 최대 10개의 이미지 미리보기를 포함할 수 있습니다.

1. **프로모션 카드** 토글을 선택하세요.
2. **프로모션 카드 추가**를 선택하세요. 제품 캐러셀의 각 이미지는 고유한 URL을 가져야 하며 동일한 종횡비(4:5, 1:1, 1.91:1)를 사용해야 합니다.
3. 이미지 URL을 포함하세요.
4. **대상 URL**에 대해 프로모션 링크를 추가하세요.

{% alert tip %}
제품 이미지를 미디어 라이브러리에 업로드한 후, 적절한 필드에 URL을 복사하여 붙여넣는 것을 권장합니다. 정적 이미지 형식(PNG 및 JPEG)만 허용됩니다. 일부 이미지 형식(GIF)은 업로드되지만 예상대로 표시되지 않습니다.
{% endalert %}

### 4단계: HTML 코드 생성 및 붙여넣기

Gmail 프로모션 카드를 만든 후 **HTML 코드 생성**을 선택하세요. 스크립트를 이메일 HTML의 `<head>` 요소에 복사하여 붙여넣기.

{% alert warning %}
프로모션 스크립트는 이메일이 Gmail 프로모션 탭에 도착할 때만 나타납니다. 현재 Gmail은 이메일이 어디에 도착할지를 결정하기 위해 알고리즘을 사용합니다. 그러나 사용자가 귀하의 이메일을 프로모션으로 표시하면 Gmail의 알고리즘은 무시되며, 귀하의 이메일은 앞으로 자동으로 프로모션 탭에 도착합니다.
{% endalert %}

### 이미지 포함

Gmail은 이메일 메시지와 관련된 강력한 이미지로 더 나은 결과를 보았습니다. Gmail은 텍스트 전용 디자인 사용을 권장하지 않으며, 이 공간은 이메일 마케팅에 필수적인 시각적 언어를 미리보기로 가져오기 위해 설계되었습니다. 잘린 텍스트가 있는 이미지를 사용하거나 여러 캠페인에서 이미지를 반복하지 마십시오.

### 제안 설명

Gmail은 "1개 구매 시 1개 무료" 또는 "모든 반바지 및 셔츠 할인"과 같은 문장이나 구문 사용을 권장하지 않으며, 이는 잘리거나 더 이상 눈길을 끌지 않으며 제목 줄과 경쟁할 수 있습니다. 이 공간은 고객과의 메시지 소통을 위해서만 사용해야 하므로 "지금 이 이메일을 열어보세요" 또는 "여기에서 거래를 클릭하세요"와 유사한 언어는 피하십시오. 제목 줄을 반복하는 것은 피하는 것이 좋습니다.

## 자주 묻는 질문

### 내 프로모션 메시지가 최종 사용자의 받은 편지함에 프로모션 카드나 제품 캐러셀을 표시하지 않는 이유는 무엇인가요?

제품 캐러셀이 Gmail 프로모션 탭에 표시될지 여부를 결정하는 많은 요소가 있습니다.

주석의 모든 이미지는 여전히 품질 필터를 통과해야 합니다. 제품 캐러셀이 채워지기 위해서는 주석의 모든 이미지가 권장 이미지 비율, 고품질 또는 고해상도 클로즈업 제품 이미지여야 합니다. 이미지에는 텍스트가 거의 없거나 전혀 포함되지 않아야 합니다(바람직함). 품질 필터는 부적절한 콘텐츠도 필터링하므로 이미지가 가족, 사용자 및 아동 친화적이어야 합니다.

또한, Gmail은 사용자의 Gmail 프로모션 탭에 표시되는 제품 캐러셀의 수에 밀도 제한을 두고 있습니다. 예를 들어, 사용자가 프로모션 이메일에서 제품 캐러셀을 사용하는 많은 브랜드에 가입하면, Gmail은 결국 표시되는 제품 캐러셀의 수에 제한을 둡니다.

Google의 개인 정보 보호 및 안전 규정으로 인해 주석이 있는 이메일은 주석이 작동하기 위해 널리 전송되어야 합니다. Google의 시스템이 이를 "대량 전송"으로 감지할 수 있도록 캠페인을 시작하고 최소 100명의 수신자에게 전송하는 것이 좋습니다. 이미지 URL은 수신자마다 다를 수 없습니다.

### 프로모션 카드나 제품 캐러셀의 클릭은 어떻게 추적되나요?

Braze 또는 기타 ESP는 헤더 섹션의 링크에 링크 추적을 삽입할 수 없습니다. 이는 프로모션 카드나 제품 캐러셀에서 클릭을 추적할 수 없음을 의미합니다.

### 제품 캐러셀을 받은 사용자가 몇 명인지 확인할 방법이 있나요?

Gmail은 카드를 표시할 시기와 대상을 결정하므로 모든 수신자가 제품 캐러셀을 볼 수 있다는 보장은 없습니다.
