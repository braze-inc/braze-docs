# LINE 메시지 유형

> 이 문서에서는 작성할 수 있는 LINE 메시지 유형과 그 특징 및 제한 사항에 대해 설명합니다.

LINE 메시지를 작성할 때는 메시지 유형을 작성기에 드래그 앤 드롭하여 커스텀할 수 있습니다.

텍스트, 이미지, 리치 메시지, 카드 기반 메시지 등 작성기 에디터로 드래그할 수 있는 메시지 유형이 있는 메시지 유형 패널입니다.]({% image_buster /assets/img/line/line_message_types.png %}){: style="max-width:40%;"}

## 텍스트

LINE 문자 메시지는 최대 5,000자까지 작성할 수 있으며, 이모티콘과 Liquid 개인화 기능이 포함되어 있습니다.

사용 사례는 다음과 같습니다:
- 클리어런스 재고에 대한 기간 한정 프로모션 발표
- 독특한 프로모션 카드로 개인화된 생일 축하 인사 보내기
- 예정된 이벤트에 대한 빠른 업데이트 공유

블랙 프라이데이 파티를 잊지 말고 자정 전까지 최대 80%까지 저장할 수 있다는 문자 메시지입니다.]({% image_buster /assets/img/line/line_text_message.png %}){: style="max-width:40%;"}

## 이미지

LINE 이미지 메시지는 [미디어 라이브러리]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/), URL 또는 Liquid를 통해 추가할 수 있습니다. 이러한 이미지는 독립형 이미지이며 클릭 가능한 링크를 포함하지 않습니다.

사용 사례는 다음과 같습니다:
- 사용자가 항공권 구매를 고려하도록 유도하는 휴가 대상지를 소개합니다.
- 시즌 종료 프로모션을 강조하여 최종 사용자가 내년 겨울 의류를 저렴하게 구매할 수 있도록 유도합니다.
- 스토어 전체 연간 세일을 위한 시각적 카운트다운 시작

!"[토스터 판매를 홍보하는 이미지 메시지.]({% image_buster /assets/img/line/line_image_message.png %}){: style="max-width:40%;"}

### URL 이미지

통합하는 사용 사례에는 URL 이미지를 사용합니다:
- 이미지 소스 속성에 Liquid를 포함시켜 동적 이미지를 생성합니다. 예를 들어 {% raw %} `https://example.com/images/?imageBanner={{first_name}}` {% endraw %} 을 이미지 URL로 삽입하여 이미지에 사용자의 이름을 포함할 수 있습니다.
- 웹 서버 또는 공개적으로 액세스 가능한 API에서 직접 이미지를 가져와서 [연결된 콘텐츠]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/connected_content/) 
- 가져온 CSV 파일 및 API 엔드포인트에서 이미지에 액세스하여 [Braze 카탈로그 만들기]({{site.baseurl}}/user_guide/data/activation/catalogs/) 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 이미지 파일 URL 길이 | 최대 2,000자  |
| 이미지 형식          | PNG, JPEG             |
| 파일 크기     |  최대 10MB |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 리치 메시지(이미지 매핑)

LINE 리치 메시지는 이미지의 특정 영역을 선택하면 열리는 하나 이상의 링크가 포함된 이미지입니다. 리치 메시지 템플릿을 선택하여 링크가 이미지에 매핑되는 방식을 선택합니다.

사용 사례는 다음과 같습니다:
- 각 가방의 제품 페이지로 연결되는 링크와 함께 새로 도착한 핸드백 그리드를 표시합니다.
- 항목을 선택하여 콤보 주문을 시작하는 대화형 메뉴를 표시합니다.
- 사용자가 그리드 사각형을 선택하여 선택할 수 있도록 여러 프로모션을 배치합니다.

흑백 격자 사진이 포함된 6각형의 리치 메시지로, 사용자가 탭하면 무작위 제안을 받을 수 있습니다.]({% image_buster /assets/img/line/line_rich_message.png %})

### 이미지 매핑 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 이미지 파일 URL 길이 | 최대 2,000자  |
| 이미지 형식          | PNG(투명 가능), JPEG             |
| 종횡비          | 1:1(너비:높이)
| 파일 크기     |  최대 10MB |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### URI 링크 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 문자 수      | 최대 1,000개 |
| 계획              | HTTP, HTTPS, LINE, 전화 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### 텍스트 

텍스트 리치 메시지는 최대 400자까지 포함할 수 있습니다.

## 카드 기반(캐러셀)

LINE 카드 기반 메시지는 사용자가 여러 개의 메시지를 캐러셀처럼 스크롤하여 가장 관련성이 높은 메시지 또는 카드의 버튼을 선택해 조치를 취할 수 있는 기능입니다.

사용 사례는 다음과 같습니다:
- 특정 메뉴 항목에 대한 프로모션 표시
- 이번 시즌 베스트셀러 재킷 살펴보기
- 키트에 포함된 요리 도구 및 가제트 샘플을 제시합니다.

작성기 에디터에서 샌드위치를 홍보하는 카드가 2개 이상 포함된 카드 기반 메시지입니다.]({% image_buster /assets/img/line/line_card_message.png %})

| 종횡비             | 직사각형: 1.51:1 <br> Square: 1:1  |
| 제목                    | 최대 40자
