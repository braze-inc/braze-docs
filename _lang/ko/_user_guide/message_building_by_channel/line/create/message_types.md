# LINE 메시지 유형

> 이 문서에서는 작성할 수 있는 LINE 메시지 유형과 그 특징 및 제한 사항에 대해 설명합니다.

LINE 메시지를 작성할 때, 메시지 유형을 작성기에 끌어다 놓은 다음 사용자 지정할 수 있습니다.

![메시지 유형 패널에는 텍스트, 이미지, 리치 메시지, 카드 기반 메시지 등 작성기 에디터로 드래그할 수 있는 메시지 유형이 있습니다.][5]{: style="max-width:40%;"}

## 텍스트

LINE 문자 메시지는 최대 5,000자까지 포함할 수 있으며 이모티콘과 리퀴드 맞춤 설정이 가능합니다.

사용 사례에는 다음이 포함됩니다:
- 클리어런스 재고에 대한 기간 한정 프로모션 발표
- 독특한 프로모션 카드로 개인화된 생일 축하 인사 보내기
- 예정된 이벤트에 대한 빠른 업데이트 공유

![사용자에게 블랙 프라이데이 파티와 자정 전까지 최대 80%까지 할인받을 수 있는 기회를 잊지 말라는 문자 메시지를 보냅니다.][1]{: style="max-width:40%;"}

## 이미지

LINE 이미지 메시지는 [미디어 라이브러리]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/media_library/), URL 또는 리퀴드를 통해 추가할 수 있습니다. 이러한 이미지는 독립형 이미지이며 클릭 가능한 링크를 포함하지 않습니다.

사용 사례에는 다음이 포함됩니다:
- 사용자가 비행기 티켓 구매를 고려하도록 유도하는 휴가 목적지 소개
- 시즌 종료 프로모션을 강조하여 사용자들이 내년 겨울 의류를 저렴한 가격으로 미리 준비할 수 있도록 유도하세요.
- 스토어 전체 연간 세일을 위한 시각적 카운트다운 시작

![토스터 판매를 홍보하는 이미지 메시지입니다.][2]{: style="max-width:40%;"}

### URL 이미지

통합하는 사용 사례에는 URL 이미지를 사용합니다:
- 이미지 소스 속성에 Liquid를 포함시켜 동적 이미지를 유동적으로 만듭니다. 예를 들어 이미지 URL로 {% raw %} `https://example.com/images/?imageBanner={{first_name}}` {% endraw %} 을 삽입하여 이미지에 사용자의 이름을 포함할 수 있습니다.
- 웹 서버 또는 공개적으로 액세스 가능한 API에서 직접 이미지를 가져와 [콘텐츠 연결]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/connected_content/) 
- 가져온 CSV 파일 및 API 엔드포인트에서 이미지에 액세스하여 [카탈로그 브레이징하기]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/) 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 이미지 파일 URL 길이 | 최대 2,000자  |
| 이미지 형식          | PNG, JPEG             |
| 파일 크기     |  최대 10MB |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 리치 메시지(이미지 맵)

LINE 리치 메시지는 이미지의 특정 영역을 선택하면 열리는 하나 이상의 링크가 포함된 이미지입니다. 리치 메시지 템플릿을 선택하여 링크가 이미지에 매핑되는 방식을 선택합니다.

사용 사례에는 다음이 포함됩니다:
- 새로 도착한 핸드백 그리드를 각 핸드백의 제품 페이지로 연결되는 링크와 함께 표시합니다.
- 항목을 선택하여 콤보 주문을 시작하는 대화형 메뉴를 표시합니다.
- 사용자가 그리드 사각형을 선택하여 선택할 수 있도록 여러 프로모션을 배치합니다.

![작성기 에디터에서 다양한 샌드위치 사진이 포함된 4각형의 풍부한 메시지를 작성할 수 있습니다.][3]

### 이미지 맵 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 이미지 파일 URL 길이 | 최대 2,000자  |
| 이미지 형식          | PNG(투명 가능), JPEG             |
| 종횡비          | 1:1(너비:높이)
| 파일 크기     |  최대 10MB |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### URI 링크 

| **사양** | **추천 속성** |
|--------------------------|----------------------------|
| 문자 수      | 최대 1,000개 |
| 계획              | HTTP, HTTPS, LINE, 전화 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### 텍스트 

텍스트 서식 있는 메시지에는 최대 400자를 포함할 수 있습니다.

## 카드 기반(캐러셀)

LINE 카드 기반 메시지에서는 사용자가 여러 개의 메시지를 캐러셀처럼 스크롤하여 가장 관심 있는 메시지 또는 카드의 버튼을 선택하여 조치를 취할 수 있습니다.

사용 사례에는 다음이 포함됩니다:
- 특정 메뉴 항목에 대한 프로모션 표시
- 이번 시즌 베스트셀러 재킷 살펴보기
- 키트에 포함된 요리 도구 및 가제트 샘플을 제시합니다.

![작성기 에디터에서 빵과 샌드위치를 홍보하는 카드가 두 장 이상 포함된 카드 기반 메시지입니다.][4]

| 파일 크기                 | 1MB |
