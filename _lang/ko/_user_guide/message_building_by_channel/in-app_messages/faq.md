# 자주 묻는 질문

> 이 문서는 앱 내 메시지에 대한 몇 가지 자주 묻는 질문에 대한 답변을 제공합니다.

### 브라우저 내 메시지란 무엇이며 앱 내 메시지와 어떻게 다릅니까?

브라우저 내 메시지는 웹 브라우저로 전송되는 앱 내 메시지입니다. 브라우저 내 메시지를 만들려면 앱 내 메시지 캠페인 또는 캔버스를 만들 때 **웹 브라우저**를 **보내기** 필드에서 선택해야 합니다. 

### 장치가 오프라인일 때 앱 내 메시지가 표시됩니까?

상황에 따라 다릅니다. 앱 내 메시지는 세션 시작 시 전달되므로 장치가 오프라인으로 전환되기 전에 페이로드를 다운로드할 수 있으며, 오프라인 상태에서도 앱 내 메시지를 표시할 수 있습니다. 페이로드가 다운로드되지 않으면 앱 내 메시지가 표시되지 않습니다.

### 사용자가 이미 장치에 앱 내 메시지 페이로드를 가지고 있고 메시지 만료가 변경되면, 만료가 장치에서 업데이트됩니까?

사용자가 세션을 시작할 때 Braze는 그들이 자격이 있는 앱 내 메시지에 변경 사항이 있는지 확인하고 그에 따라 업데이트합니다. 따라서 만료가 변경되고 세션을 기록하면, 업데이트된 정보와 함께 앱 내 메시지가 장치로 전송됩니다.

### 앱 내 메시지 캠페인을 위한 조용한 시간을 어떻게 설정합니까?

조용한 시간 기능은 앱 내 메시지 캠페인에서 사용할 수 없습니다. 이 기능은 특정 시간 동안 사용자에게 메시지가 전송되지 않도록 하는 데 사용됩니다. 앱 내 메시지 캠페인의 경우, 사용자는 앱 내에서 활동 중일 때만 앱 내 메시지를 받습니다. 

특정 시간에 앱 내 메시지를 보내기 위한 우회 방법으로 다음 샘플 Liquid 코드를 사용하십시오. 이 메시지는 지정된 시간대에서 오후 7시 59분 이후 또는 오전 8시 이전에 인앱 메시지가 표시되면 중단될 수 있습니다.

{% raw %}
```liquid
{% assign time = 'now' | time_zone: ${time_zone} %}{% assign hour = time | date: '%H' | plus: 0 %}
{% if hour > 19 or hour < 8 %}
{% abort_message("Outside allowed time window") %}
{% endif %}
MESSAGE HERE
```
{% endraw %}

### 인앱 메시지의 자격은 언제 계산되나요?

인앱 메시지의 자격은 배달 시점에 계산됩니다. 인앱 메시지가 오전 7시에 전송되도록 예약된 경우, 이 인앱 메시지의 자격은 오전 7시에 확인됩니다.

인앱 메시지가 표시되면, 자격은 인앱 메시지가 다운로드되고 트리거된 시점에 따라 달라집니다.

### 템플릿 인앱 메시지는 무엇인가요?

인앱 메시지는 **표시하기 전에 캠페인 자격 재평가**가 선택되거나 메시지에 다음 Liquid 태그가 존재하는 경우 템플릿 인앱 메시지로 전달됩니다:

- `canvas_entry_properties`
- `connected_content`
- SMS 변수는 {% raw %}`{sms.${*}}`{% endraw %}과 같습니다.
- `catalog_items`
- `catalog_selection_items`
- `event_properties`

이는 세션 시작 중에 장치가 전체 메시지 대신 해당 인앱 메시지의 트리거를 수신함을 의미합니다. 사용자가 인앱 메시지를 트리거하면, 사용자의 장치는 실제 메시지를 가져오기 위해 네트워크 요청을 합니다.

{% alert note %}
장치가 인터넷에 접근할 수 없는 경우 메시지가 전달되지 않습니다. Liquid 로직이 해결되는 데 너무 오랜 시간이 걸리면 메시지가 전달되지 않을 수 있습니다.
{% endalert %}

### 내 아카이브된 인앱 메시지 캠페인이 여전히 인앱 메시지 노출을 전달하는 이유는 무엇인가요?

이는 인앱 메시지 캠페인이 활성화되었을 때 세그먼트 기준을 충족한 사용자에게 발생할 수 있습니다.

이를 방지하기 위해 캠페인 설정 중에 **표시하기 전에 캠페인 자격 재평가**를 선택하세요. 

### Braze는 "1일 후"로 설정된 인앱 메시지 만료를 어떻게 계산하나요?

Braze는 사용자가 메시지를 받을 자격이 생긴 후 24시간을 1일의 만료 시간으로 계산합니다.
