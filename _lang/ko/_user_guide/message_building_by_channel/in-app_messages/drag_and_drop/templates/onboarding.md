# 온보딩 흐름

> **온보딩 흐름** 인앱 메시지 템플릿을 사용하여 캠페인 전략에 도움이 되는 사용자 속성, 인사이트 및 선호도를 수집하세요. 

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## 온보딩 플로우 만들기

### 1단계: 템플릿 선택

드래그 앤 드롭 인앱 메시지를 만들 때는 템플릿에 대한 **온보딩 플로우를** 선택한 다음 **메시지 구축을** 선택합니다. 이 템플릿은 모바일 앱과 웹 브라우저 모두에서 지원됩니다.

온보딩 플로우 템플릿이 포함된 인앱 메시지 편집기.]({% image_buster /assets/img/drag_and_drop/templates/onboarding_editor.png %})

### 2단계: 메시지 스타일 설정하기

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### 3단계: 온보딩 흐름 순서 커스텀하기

온보딩 플로우 구축을 시작하려면 **페이지** 섹션을 사용하여 플로우에 메시지를 추가하거나 삭제하세요. 원하는 메시지 개수를 정한 후 각 페이지에서 버튼을 선택합니다. 그런 다음 사이드 메뉴를 사용하여 사용자가 버튼을 선택하면 어디로 이동할지(플로우를 종료할지, 다음 메시지로 이동할지)를 선택할 수 있습니다.

사용자가 흐름을 올바르게 이동할 수 있도록 메시징이 순차적으로 연결되어 있는지 확인하세요. 방법에 대한 단계는 [함께 연결하기 페이지를]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together) 참조하세요.

### 4단계: 메시지 스타일 지정하기

드래그 앤 드롭 [인앱 메시지 구성 요소를]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) 사용하여 온보딩 흐름의 모양과 느낌을 커스텀할 수 있습니다.
