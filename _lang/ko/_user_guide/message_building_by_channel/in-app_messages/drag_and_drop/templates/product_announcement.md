# 제품 발표

> 인앱 메시지 드래그 앤 드롭 편집기를 사용하여 신제품 출시에 대한 인지도를 높일 수 있습니다.

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## 제품 공지 사항 만들기

### 1단계: 템플릿 선택

드래그 앤 드롭 인앱 메시지를 만들 때는 템플릿에 대해 **제품 공지를** 선택한 다음 **메시지 구축을** 선택합니다. 이 템플릿은 모바일 앱과 웹 브라우저 모두에서 지원됩니다.

제품 공지를 위한 템플릿이 포함된 인앱 메시지 편집기입니다.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement.png %})

### 2단계: 메시지 스타일 설정하기

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### 3단계: 버튼 컴포넌트 커스텀하기

제품 공지를 구축하려면 에디터에서 버튼 컴포넌트를 선택합니다. 그런 다음 사이드 메뉴를 사용하여 사용자가 버튼을 선택하면 어디로 이동하는지 선택합니다. 기본값으로 템플릿은 메시지를 닫지만 앱의 특정 페이지(예: 공지에 기능이 있는 제품의 페이지)로 이동하도록 선택할 수 있습니다.

버튼 요소를 커스텀할 수 있는 사이드 메뉴가 있는 인앱 메시지 편집기.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement_button.png %})

**페이지** 섹션에서 제품 공지에 메시지를 추가한 다음 순차적으로 서로 연결할 수도 있습니다. 예를 들어, 제품의 기능을 간략하게 설명하는 메시지 시퀀스를 구성하고 최종 사용자를 제품 페이지로 이동시키는 버튼으로 끝낼 수 있습니다. [페이지 함께 연결하기에서]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together) 그 방법을 알아보세요.

### 4단계: 메시지 스타일 지정하기

드래그 앤 드롭 [인앱 메시지 구성 요소를]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) 사용하여 제품 공지의 모양과 느낌을 커스텀할 수 있습니다.
