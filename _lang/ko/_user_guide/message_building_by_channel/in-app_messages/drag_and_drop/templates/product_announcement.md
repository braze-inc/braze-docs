# 제품 발표

> 앱 내 메시지 드래그 앤 드롭 편집기를 사용하여 신제품 출시에 대한 인지도를 높일 수 있습니다.

{% multi_lang_include drag_and_drop/templates.md section='SDK 요구 사항' %}

## 제품 공지 사항 만들기

### 1단계: 템플릿 선택

드래그 앤 드롭 인앱 메시지를 만들 때 템플릿에 대한 **제품 공지를** 선택한 다음 **메시지 작성을** 선택합니다. 이 템플릿은 모바일 앱과 웹 브라우저 모두에서 지원됩니다.

![The in-app message editor with the template for a product announcement.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement.png %})

### 2단계: 메시지 스타일을 설정하세요

{% multi_lang_include drag_and_drop/templates.md section='메시지 스타일' %}

### 3단계: 버튼 컴포넌트 사용자 지정

제품 공지를 작성하려면 에디터에서 버튼 컴포넌트를 선택합니다. 그런 다음 사이드 메뉴를 사용하여 사용자가 버튼을 선택하면 어디로 이동하는지 선택합니다. 기본적으로 템플릿은 메시지를 닫지만 앱의 특정 페이지(예: 공지사항에 소개된 제품 페이지)로 이동하도록 선택할 수 있습니다.

![The in-app message editor with a side menu for customizing the button element.]({% image_buster /assets/img/drag_and_drop/templates/product_announcement_button.png %})

**페이지** 섹션에서 제품 공지사항에 메시지를 추가한 다음 순차적으로 서로 연결할 수도 있습니다. 예를 들어, 제품의 기능을 간략하게 설명하는 메시지 시퀀스를 구성하고 사용자를 제품 페이지로 이동시키는 버튼으로 끝낼 수 있습니다. [페이지 함께 연결하기에서]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together) 그 방법을 알아보세요.

### 4단계: 메시지를 스타일링하세요

Customize the look and feel of your product announcement using the drag-and-drop [in-app message components]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components).
