# 모달 인앱 메시지

> 모달은 기기 화면 중앙에 화면 오버레이와 함께 표시되어 백그라운드에서 앱이 눈에 띄도록 도와줍니다. 사용자에게 세일이나 경품 행사를 이용하도록 은근히 제안하는 데 적합합니다.

이미지와 텍스트 추천을 자세히 설명하는 두 개의 모달 인앱 메시지가 나란히 표시됩니다. 자세한 내용은 다음 섹션을 참조하세요.]({% image_buster /assets/img/modal-spec.png %}){: style="max-width: 801px; border: none;"}

## 더 큰 화면

태블릿 또는 데스크톱 브라우저에서는 다음 스크린샷과 같이 모달 인앱 메시지가 여전히 앱 화면 중앙에 표시됩니다.

큰 화면에 표시되는 모달 인앱 메시지. 휴대폰 화면과 마찬가지로 메시징은 화면 중앙에 배치됩니다.]({% image_buster /assets/img/modal-large-viewport.png %}){: style="max-width: 800px; border: none;"}
