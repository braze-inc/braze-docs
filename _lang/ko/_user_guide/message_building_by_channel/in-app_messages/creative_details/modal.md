# 모달 인앱 메시지

> 모달은 기기의 화면 중앙에 나타나며 화면 오버레이가 있어 배경의 앱과 구분됩니다. 이것들은 사용자가 판매 또는 경품 행사를 이용하도록 은근히 제안하는 데 완벽합니다.
