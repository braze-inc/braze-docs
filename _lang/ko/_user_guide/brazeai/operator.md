# <sup>BrazeAITM</sup> 운영자

> <sup>BrazeAITM</sup> 오퍼레이터는 Braze 대시보드에 구축된 AI 기반 어시스턴트입니다. 운영자가 워크플로 내에서 답변, 문제 해결 안내 및 모범 사례를 제공합니다.

{% alert important %}
<sup>BrazeAITM</sup> 운영자는 기능이 제한된 비공개 베타 버전입니다. 시작하는 데 도움이 필요하면 고객 성공 매니저에게 문의하세요.
{% endalert %}

## 운영자 정보

오퍼레이터는 Braze 대시보드에 구축된 AI 어시스턴트입니다. 워크플로 내에서 질문에 답하고, 다음 단계를 제안하고, 작업을 안내합니다.

베타 버전에서는 운영자가 **질문** 모드만 지원합니다. 할 수 있습니다:

- Braze 설명서에서 답변 찾기
- [페이지 인식 컨텍스트를](#page-aware-context) 사용하여 문제 해결
- 모범 사례 및 온보딩 가이드 알아보기

### 서브프로세서 또는 타사 공급자로서의 모델 공급자

고객이 Braze 서비스를 통해 Braze가 제공하는 LLM 제공자와의 통합을 이용하는 경우("Braze 제공 LLM"), 해당 Braze 제공 LLM의 제공자는 고객과 Braze 간의 데이터 처리 부록(DPA) 약관에 따라 Braze 하위 처리자 역할을 수행합니다. BrazeAI Operator는 OpenAI와 통합됩니다.

고객이 자체 API 키를 가져와 Braze AI 운영자와 통합하기로 선택한 경우, 고객과 Braze 간의 계약에 정의된 대로 고객 자체 LLM 구독의 제공자는 타사 제공자로 간주됩니다. 

### 내 데이터는 어떻게 사용되어 OpenAI로 전송되나요?

Braze가 OpenAI를 활용하는 것으로 식별하는 Braze AI 기능을 통해 AI 출력을 생성하기 위해("출력"), Braze는 사용자의 프롬프트, 대시보드에 표시되는 콘텐츠 및 쿼리와 관련된 워크스페이스 데이터를 해당되는 경우 OpenAI로 전송합니다("입력"). [OpenAI의 API 플랫폼 약정에](https://openai.com/enterprise-privacy/) 따라, Braze를 통해 OpenAI의 API로 전송된 데이터는 OpenAI 모델을 학습하거나 개선하는 데 사용되지 않습니다. 귀하와 Braze 사이에서 Output은 귀하의 지적 재산입니다. Braze는 그러한 결과물에 대해 저작권 소유권을 주장하지 않습니다. Braze는 일반적으로 출력을 포함하여 AI가 생성한 콘텐츠와 관련하여 어떠한 종류의 보증도 하지 않습니다.

## 운영자와 대화하는 방법

프롬프트를 사용하여 운영자와 소통합니다. 가장 좋은 방법은 동료나 친구에게 말할 때처럼 자연스럽게 말하는 것입니다. 프롬프트는 간단한 질문부터 복잡한 요청까지 다양합니다:

- **간단합니다:** 사용자가 사이트에서 쇼핑하는 동안 장바구니 유기 이메일을 받지 않게 하려면 어떻게 해야 하나요?
- **복잡합니다:** 메시징의 `abort_message` 태그에 중단을 일으킨 사용자 속성이 포함되도록 하려면 어떻게 해야 하나요?

운영자는 단계별 지침, Braze 문서 링크, 일반 언어 설명을 제공할 수 있습니다. 질문이 명확하고 구체적일수록 답변이 더 유용해집니다. 

### Best practices

운영자는 검색 엔진이 아니라 대화라고 생각하세요. 일반적으로 짧고 자연스러운 메시지가 가장 효과적입니다.

- **구체적으로 설명하세요:** "캔버스에 대해 알려주세요" 대신 "캔버스에서 행동 경로를 사용하려면 어떻게 하나요?"라고 질문하세요.  
- **후속 조치를 사용합니다:** 첫 번째 답변이 필요한 답변이 아니라면 명확한 질문을 하세요. 운영자가 답변을 구체화할 수 있습니다.
- **컨텍스트에 의존하세요:** 운영자는 사용자가 Braze에서 어떤 페이지에 있는지 알고 있습니다. 작업 중인 페이지에서 연산자를 열면 가장 관련성 높은 결과를 얻을 수 있습니다.

## 피처

베타 버전에는 다음과 같은 기능이 포함되어 있습니다:

### GPT 모델

이러한 GPT 모델 중에서 선택하여 오퍼레이터의 다양한 요청 유형에 사용할 수 있습니다:

- [GPT-5 나노](https://platform.openai.com/docs/models/gpt-5-nano)
- [GPT-5 미니](https://platform.openai.com/docs/models/gpt-5-mini)
- [GPT-5](https://platform.openai.com/docs/models/gpt-5)
- [GPT-5.1](https://platform.openai.com/docs/models/gpt-5.1) (기본값)

![드롭다운을 통해 다양한 GPT 모델을 선택할 수 있습니다.]({% image_buster /assets/img/operator/operator_model.png %}){:style="max-width:70%"}

### 페이지 인식 컨텍스트

운영자는 사용자가 Braze에서 작업 중인 페이지를 이해하고 해당 컨텍스트에 따라 맞춤형 응답을 제공할 수 있습니다. 예를 들어 캔버스를 구축하는 동안 오퍼레이터를 열면 사용자가 현재 위치를 설명할 필요 없이 캔버스와 관련된 단계를 제안하거나 안내를 제공할 수 있습니다. 

### 추천 프롬프트

운영자를 열면 시작하는 데 도움이 되는 몇 가지 제안 메시지가 표시됩니다. 시작하려면 하나를 선택하거나 직접 질문을 입력하세요.

### 추론 보기

연산자는 추론 단계가 ' **추론됨**'이라는 레이블이 붙은 접을 수 있는 섹션에 표시됩니다. 드롭다운을 선택하여 이러한 섹션을 확장하고 운영자가 어떻게 답변에 도달했는지 확인합니다.

![운영자가 어떻게 답변했는지에 대한 자세한 내용이 포함된 '추론됨' 드롭다운이 확장되었습니다.]({% image_buster /assets/img/operator/operator_reasoning.png %}){:style="max-width:50%"}

### 추천 작업

경우에 따라 운영자가 다음 단계를 추천하고 Braze 대시보드에서 관련 페이지로 바로 연결되는 링크를 제공하는 경우도 있습니다. 예를 들어 이메일 반송률에 대해 문의하는 경우 운영자는 **전달 가능성 센터** 페이지로 연결할 수 있습니다. 이러한 바로가기를 사용하면 수동으로 탐색할 필요 없이 더 빠르게 조치를 취할 수 있습니다.

### 생성 중지

운영자가 응답을 생성하는 동안에는 **보내기** 버튼이 **중지** 버튼으로 바뀝니다. 응답을 조기에 종료하려면 **중지를** 선택합니다.

### 채팅 기록 지우기

대화를 초기화하려면 **채팅 기록 지우기를** 선택합니다. 이렇게 하면 현재 콘텐츠가 제거되므로 새로 시작할 수 있습니다.

### 패널 최대화 및 최소화

**최대화** 버튼을 사용해 오퍼레이터를 확장하면 더 쉽게 읽을 수 있고, **최소화하면** Braze에서 계속 작업하는 동안 패널을 접어서 보관할 수 있습니다.

### 피드백 보내기

각 응답 하단에 있는 좋아요 또는 싫어요 버튼을 사용하여 빠른 피드백을 제공할 수 있습니다. 이는 운영자의 답변을 개선하는 데 도움이 됩니다.

## 문제 해결

| Issue | 문제 해결 |
| --- | --- |
| 응답 없음 | 페이지를 새로고침하고 운영자 패널을 다시 열어 보세요. |
| 주제에서 벗어난 답변 | 질문을 좀 더 구체적으로 재구성하세요. 문의하려는 기능이나 워크플로우를 언급하세요. |
| Error messages | 운영자가 콘텐츠를 스트리밍할 수 없는 경우 "다시 시도하세요"라는 메시지가 표시될 수 있습니다. 운영자를 일시적으로 사용할 수 없거나 연결이 중단되었을 수 있습니다. 몇 분 후에 다시 시도하세요. |

## Limitations

오퍼레이터는 Braze를 탐색하고 보다 효율적으로 작업을 수행할 수 있도록 설계되었지만, 현재 몇 가지 유의해야 할 제한 사항이 있습니다:

### 데이터에 액세스할 수 없음

운영자는 사용자가 Braze에서 수행하는 작업의 컨텍스트에 액세스할 수 있지만, 운영자는 Braze에 저장된 회사의 데이터에 대해 쿼리하거나 답변을 반환할 수 없습니다. 예를 들어 다음과 같은 요청에는 응답할 **수 없습니다**:

- "작년의 모든 이메일 캠페인 목록을 주세요."
- "지난 분기 참여도가 가장 높았던 세그먼트를 보여주세요."
- "내 캔버스의 성능/성과를 분석하고 개선 사항을 제안하세요."

### 베타 안정성

비공개 베타 버전인 운영자는 간혹 오류, 중단 또는 불완전한 기능이 발생할 수 있습니다.

질문이 지원되는지 확실하지 않은 경우에는 분석이나 과거 데이터를 가져오는 대신 운영자가 Braze 대시보드 내에서 탐색하거나 조치를 취하는 데 어떻게 도움을 줄 수 있는지에 대한 질문으로 표현해 보세요.

### 보낸 메시지 수

운영자에게 보낼 수 있는 메시징 수에는 제한이 있습니다. 쿼리에는 기본값인 GPT-5 미니 또는 GPT-5 나노를 사용하고 더 복잡한 작업에는 GPT-5를 신중하게 사용하는 것이 좋습니다.
