# Overview

> 이 참조는 데이터 소스 연결, 오케스트레이션 설정 및 의사 결정 에이전트 설계를 포함하여 Decisioning Studio 설정에 관련된 단계에 대한 개요를 제공합니다.

BrazeAI Decisioning Studio™는 모든 비즈니스 메트릭을 최적화하는 의사 결정 에이전트를 설계하고 배포할 수 있게 해줍니다. 의사 결정 에이전트는 특정 비즈니스 목표를 충족하도록 맞춤 구성된 설정입니다.

이를 위해 데이터 소스를 연결하고 오케스트레이션을 설정하며 의사 결정 에이전트를 설계해야 합니다.

{% alert tip %}
Decisioning Studio Pro 고객의 경우, AI 전문가 서비스 팀이 최적의 성능을 위해 Decisioning Studio 설정을 지원합니다.
{% endalert %}

## Decisioning Studio 설정

Decisioning Studio를 설정하려면 다음 단계를 완료해야 합니다:

### 1단계: 데이터 소스 연결

고객 프로필 및 참여 데이터를 연결하여 생성하는 의사 결정 에이전트가 각 고객이 누구인지 및 그들의 행동을 이해할 수 있도록 합니다.

일반적으로 Decisioning Studio의 초기 설정 중에 데이터 소스를 한 번만 연결하면 됩니다. 나중에 사용 사례를 확장하면 새로운 데이터 소스를 추가해야 할 수도 있습니다.

{% alert tip %}
[Braze Data Platform]({{site.baseurl}}/user_guide/data/braze_data_platform)에 이미 있는 모든 데이터는 자동으로 Decisioning Studio에서 사용할 수 있습니다.
{% endalert %}

자세한 안내는 귀하의 Decisioning Studio 등급에 대한 설명서를 참조하십시오:
- [Decisioning Studio Go: 데이터 소스 연결]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Decisioning Studio Pro: 데이터 소스 연결]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### 2단계: 오케스트레이션 설정

Decisioning Studio를 고객 참여 플랫폼(CEP)과 통합하여 에이전트가 작업을 조정할 수 있도록 합니다. CEP는 에이전트의 결정에 따라 고객에게 개인화된 경험을 제공하는 데 사용되는 플랫폼입니다.

일반적으로 이 오케스트레이션은 한 번만 설정하면 됩니다.

자세한 안내는 귀하의 Decisioning Studio 등급에 대한 설명서를 참조하십시오:
- [Decisioning Studio Go: 오케스트레이션]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Decisioning Studio Pro: 오케스트레이션]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### 3단계: 에이전트를 설계하세요

결정 에이전트를 구성하여 극대화하고자 하는 결과와 에이전트가 이를 달성하기 위해 취할 수 있는 행동을 정의하세요. 에이전트 설계에 대한 자세한 안내는 [결정 에이전트 설계]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/)를 참조하세요.

티어별 안내:
- [Decisioning Studio Go: 에이전트를 설계하세요]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Decisioning Studio Pro: 에이전트를 설계하세요]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
결정 스튜디오 프로 고객을 위해, AI 결정 서비스 팀이 에이전트를 설계하고 출시하는 데 지원할 것입니다.
{% endalert %}

### 4단계: 결정 에이전트를 출시하세요

결정 에이전트를 출시하고 비즈니스 결과를 위해 지속적으로 학습하고 최적화하도록 하세요.

자세한 안내는 귀하의 Decisioning Studio 등급에 대한 설명서를 참조하십시오:
- [Decisioning Studio Go: 에이전트를 출시하세요]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Decisioning Studio Pro: 에이전트를 출시하세요]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
