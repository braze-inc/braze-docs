# 인사이트 보고서

> 인사이트는 블록 선택과 같이 액션 뱅크의 다양한 추천 옵션이 어떻게 생성되는지 보여줍니다. 두 가지 인사이트 보고서가 있습니다: **에이전트 선호도**와 **SHAP**.

{% tabs local %}
{% tab agent preferences %}
**에이전트 선호도** 보고서는 계절적 트렌드를 파악하고 액션 뱅크에서 선택 항목의 관련성을 평가하여 업데이트에 대한 정보에 기반한 의사결정을 안내합니다.

![특정 기간 동안 다양한 추천 옵션이 얼마나 자주 선택되었는지 비교하는 막대 차트를 보여주는 에이전트 선호도 보고서. 차트에는 여러 색상의 막대가 표시되며, 각 막대는 액션 뱅크의 추천 옵션을 나타냅니다. y축은 선택된 비율(%), x축은 옵션 이름을 나타냅니다.]({% image_buster /assets/img/decisioning_studio/reporting_insights_agent_preferences.png %})

이 보고서에 대한 자세한 내용은 다음 표를 참조하세요:

| 필드 | 설명 |
|-------|-------------|
| 차원 | 채널, 캠페인 또는 플랫폼과 같이 결과를 구성하는 데 사용되는 속성입니다. |
| 비교 그룹 | 보고서에서 비교하려는 그룹입니다. 여러 비교 그룹을 선택할 수 있습니다. |
| 파라미터 | 열기, 클릭 수 또는 전환율과 같이 해당 속성에 적용되는 측정기준입니다. |
| 세그먼트 | Braze에서 생성한 [오디언스 세그먼트]({{site.baseurl}}/user_guide/engagement_tools/segments/)입니다. |
| 옵션             | 액션 뱅크에서 선택된 특정 추천 옵션입니다. |
| 설명        | 옵션이 나타내는 내용에 대한 간단한 설명입니다.            |
| 선택 횟수  | 옵션이 선택된 총 횟수입니다.         |
| 선택 비율(%)   | 전체 선택 중 이 옵션이 선택된 비율입니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation"}

{% tab SHAPs %}
**SHAP** 보고서는 Shapley Additive exPlanations(SHAP) 모델을 사용하여 각 기능 또는 변수가 추천 에이전트에 어떻게 기여하는지 정량화합니다. 차트의 각 점은 하나의 SHAP 값을 나타내며, 점의 분포는 해당 기능이 미치는 방향적 영향을 전반적으로 파악하는 데 도움을 줍니다.

![여러 색상의 막대가 다양한 기능 또는 변수를 나타내는 수평 막대 그래프를 표시하는 SHAP 보고서 차트. 각 막대는 추천 에이전트에 대한 기능의 영향을 보여주며, x축은 SHAP 값, y축은 최근성, 빈도, 채널과 같은 기능 이름을 나타냅니다. 차트는 각 기능이 에이전트의 예측에 긍정적 또는 부정적으로 기여하는 방식을 시각화합니다.]({% image_buster /assets/img/decisioning_studio/reporting_insights_shaps.png %})
