# BrazeAI 결정 스튜디오™ 프로

## 결정 스튜디오 프로에 대하여

결정 스튜디오 프로는 결정 스튜디오 고보다 더 유연하고 더 강력합니다. 프로에는 전체 액션 뱅크 사용자 정의, 매출, 전환 또는 참여와 같은 모든 메트릭에 대한 최적화 기능이 포함되어 있으며, 모든 고객 참여 플랫폼(CEP)과 호환됩니다.

프로 에이전트를 설정하려면 Braze의 전방 배치 데이터 과학 팀인 AI 결정 서비스 팀과 협력해야 하며, 이는 최대 가치를 위해 에이전트 설계를 안내하고 지원합니다.

결정 스튜디오 고와 결정 스튜디오 프로의 차이에 대해 더 알아보려면 [결정 스튜디오 고 대 결정 스튜디오 프로]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro)를 참조하십시오.
