# BrazeAI 의사 결정 스튜디오™ 프로

## 디시전 스튜디오 프로 정보

디시전킹 스튜디오 프로는 디시전킹 스튜디오 고보다 더 유연하고 강력합니다. Pro에는 전체 액션 뱅크 커스텀, 모든 측정기준(예: 매출, 전환 또는 참여)에 대한 최적화 기능이 포함되어 있으며, 모든 고객 참여 플랫폼(CEP)과 호환됩니다.

Pro 상담원을 설정하려면 비즈니스의 가치를 극대화하기 위해 상담원 설계 전반을 안내하고 지원하는 AI 의사 결정 서비스 팀(Braze의 전진 배치된 데이터 과학 팀)과의 협업이 필요합니다.

디시전킹 스튜디오 Go와 디시전킹 스튜디오 프로의 차이점에 대해 자세히 알아보려면 디시전킹 [스튜디오 Go와 디시전킹 스튜디오 프로를]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro) 참조하세요.
