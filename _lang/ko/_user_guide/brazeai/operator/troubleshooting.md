# BrazeAI Operator 문제 해결

> BrazeAI Operator<sup>TM</sup>의 일반적인 문제와 해결 방법.

## 일반적인 문제

| 문제 | 해결 방법 |
| --- | --- |
| 답변이 없음 | 페이지를 새로 고치고 Operator 패널을 다시 엽니다. |
| 관련 없는 답변 | 질문을 더 구체적으로 작성합니다. 해당 기능이나 워크플로를 언급합니다. |
| 오류 메시지 | Operator가 콘텐츠를 스트리밍할 수 없으면 "다시 시도" 프롬프트가 표시될 수 있습니다. Operator가 일시적으로 사용 불가이거나 연결이 끊어졌을 수 있습니다. 몇 분 후 다시 시도합니다. |
| 일일 사용 한도 도달 | 조직 전체 사용 한도에 도달했습니다. 한도가 초기화될 때까지 기다립니다(24시간마다). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 제한 사항

- **데이터 액세스:** Operator는 활성 페이지의 콘텐츠를 볼 수 있습니다. 과거 캠페인, 카탈로그, 사용자 정의 속성 등 일부 조직별 데이터도 가져올 수 있습니다. 그러나 현재 새 보고서를 생성하거나 보고서 쿼리를 실행할 수는 없습니다.
- **사용 한도:** Operator는 24시간마다 초기화되는 조직 전체 일일 사용 한도가 적용됩니다. 한도에 도달하면 채팅에 "일일 사용 한도 도달" 메시지가 표시되며, 초기화될 때까지 추가 요청을 할 수 없습니다.
