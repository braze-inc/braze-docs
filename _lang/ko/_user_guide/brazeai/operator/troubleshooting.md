# 문제 해결 BrazeAI 운영자

> BrazeAI 운영자<sup>TM</sup>에 대한 일반적인 문제 및 해결책.

## 일반적인 문제

| Issue | 문제 해결 |
| --- | --- |
| 응답 없음 | 페이지를 새로 고치고 운영자 패널을 다시 열어보세요. |
| 주제와 관련 없는 답변 | 질문을 더 구체적으로 재구성하세요. 질문하는 기능이나 워크플로우를 언급하세요. |
| Error messages | 운영자가 콘텐츠를 스트리밍할 수 없는 경우, "다시 시도" 프롬프트가 나타날 수 있습니다. 운영자가 일시적으로 사용할 수 없거나 연결이 끊겼을 수 있습니다. 몇 분 후에 다시 시도하세요. |
| 일일 사용 한도 초과 | 회사 전체 사용 한도에 도달했습니다. 한도가 재설정될 때까지 기다리세요 (24시간마다 재설정됨). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Limitations

- **데이터 접근:** 운영자는 활성 페이지의 내용을 볼 수 있습니다. 과거 캠페인, 카탈로그 및 커스텀 속성과 같은 회사 특정 데이터를 조회할 수도 있습니다. 그러나 현재 새로운 보고서를 생성하거나 보고 쿼리를 실행할 수는 없습니다.
- **사용 한도:** 운영자는 24시간마다 재설정되는 회사 전체 일일 사용 한도가 있습니다. 한도가 초과되면 채팅에 "일일 사용 한도 초과" 배너가 나타나고 한도가 재설정될 때까지 추가 요청을 할 수 없습니다.
