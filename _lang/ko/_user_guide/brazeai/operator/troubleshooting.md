# BrazeAI Operator 문제 해결

> BrazeAI Operator<sup>TM</sup>에 대한 일반적인 문제 및 해결책.

## 일반적인 문제

| 문제 | 문제 해결 |
| --- | --- |
| 응답 없음 | 페이지를 새로고침하고 Operator 패널을 다시 열어보세요. |
| 주제와 관련 없는 답변 | 질문을 더 구체적으로 재구성하세요. 질문하려는 기능이나 워크플로우를 언급하세요. |
| 오류 메시지 | Operator가 콘텐츠를 스트리밍할 수 없는 경우, "다시 시도" 프롬프트가 나타날 수 있습니다. Operator가 일시적으로 사용할 수 없거나 연결이 중단되었을 수 있습니다. 몇 분 후에 다시 시도하세요. |
| 일일 사용 한도 초과 | 회사 전체 사용 한도에 도달했습니다. 한도가 재설정될 때까지 기다리세요(24시간마다 재설정됨). |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 제한 사항

- **데이터 액세스:** Operator는 드래그 앤 드롭 편집기 내부의 콘텐츠를 제외하고 활성 페이지의 내용을 볼 수 있습니다. 과거 캠페인, 카탈로그, 커스텀 속성과 같은 회사별 데이터를 조회할 수도 있습니다. 그러나 현재 새로운 보고서를 생성하거나 리포트 실행 쿼리를 수행할 수는 없습니다.
- **사용 한도:** Operator는 24시간마다 재설정되는 회사 전체 일일 사용 한도가 있습니다. 한도에 도달하면 채팅에 "일일 사용 한도 초과" 배너가 나타나며, 한도가 재설정될 때까지 추가 요청을 할 수 없습니다.
