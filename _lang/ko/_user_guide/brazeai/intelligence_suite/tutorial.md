# Intelligence Suite 튜토리얼

> Braze Intelligence Suite를 처음 사용하시나요? 이 튜토리얼로 시작하세요. 일반 정보는 [Intelligence Suite]({{site.baseurl}}/user_guide/brazeai/intelligence_suite/)를 참조하세요.

## 튜토리얼: 패스트푸드

SandwichEmperor라는 패스트푸드점에서 일하며 기간 한정 신메뉴 로얄 로스트를 홍보한다고 가정합니다. Intelligence Suite의 두 기능을 사용해 Canvas에서 맞춤 액션을 보냅니다.

### 1단계: 알림 시점에 Intelligent Timing 사용

Intelligent Timing으로 사용자의 앱 및 각 메시지 채널과의 과거 상호작용을 분석하고, 각 사용자에게 로얄 로스트를 프로모션할 최적의 시간을 자동으로 선택합니다. 어떤 사용자는 오후에, 어떤 사용자는 저녁에 액션을 받습니다.

과거 상호작용이 충분하지 않은 사용자에게는 대체 시간을 지정합니다. 모든 사용자에게 가장 인기 있는 앱 사용 시간입니다.

![메시지 단계의 Intelligent Timing 전달 설정.]({% image_buster /assets/img/intelligence_suite1.png %})

### 2단계: 메시지 선택에 Intelligent Selection 사용

실제 프로모션 메시지에는 Intelligent Selection으로 로얄 로스트용 푸시, 이메일, SMS 세 가지를 테스트합니다. Intelligent Selection은 하루 두 번 모든 프로모션 메시지 성과를 분석하고 점진적으로 최고 메시지를 더 많이, 나머지를 더 적게 보냅니다.

Intelligent Selection이 최고 메시지를 판단할 만큼 데이터가 쌓이면, 해당 메시지가 향후 전송의 100%에 사용됩니다.

![Intelligent Selection이 활성화된 Canvas의 A/B 테스트 영역.]({% image_buster /assets/img_archive/canvas_intelligent_selection.png %})

### 3단계: Canvas 시작

Intelligent Timing과 Intelligent Selection으로 로얄 로스트 액션의 시점과 메시지를 최적화했습니다. Canvas를 시작하고 전송이 사용자 선호에 맞춰 조정되는 것을 확인할 수 있습니다.
