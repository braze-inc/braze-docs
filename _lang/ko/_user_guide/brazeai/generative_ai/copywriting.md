# BrazeAI로 카피 생성하기

> AI 카피라이팅 어시스턴트는 간단한 제품 이름이나 설명을 OpenAI가 소유한 타사 제공업체 GPT 카피 생성 도구에 전달하여 메시징에 사용할 수 있는 사람이 작성한 것과 유사한 마케팅 카피를 생성합니다. 이 기능은 Braze 대시보드의 대부분의 메시지 작성기에서 기본값으로 사용할 수 있습니다.

## 카피 생성

### 1단계: AI 카피라이터 실행

메시지 작성기에서 <i class="fa-solid fa-wand-magic-sparkles"></i> **AI 카피라이터 실행**을 선택합니다.

인앱 메시지 드래그 앤 드롭 편집기에서 텍스트 블록을 선택하고 블록 도구 모음에서 <i class="fa-solid fa-wand-magic-sparkles" title="AI 카피라이터"></i>를 선택합니다.

### 2단계: 세부 정보 입력

입력 필드에 제품 이름 또는 설명을 입력한 다음 대략적인 출력 길이를 선택합니다.

채널별 모범 사례에 따라 특정 채널의 출력 길이를 선택하거나 짧게(1문장), 중간(2~3문장) 또는 길게(1단락) 중에서 선택할 수 있습니다.

### 3단계: 추가 커스터마이즈(선택 사항)

카피를 추가로 커스터마이즈하려면 다음과 같이 할 수 있습니다:

- **브랜드 가이드라인 적용:** [BrazeAI<sup>TM</sup>으로 브랜드 가이드라인을 생성한]({{site.baseurl}}/user_guide/brazeai/generative_ai/brand_guidelines) 후에는 이를 사용하여 카피를 생성할 수 있습니다.
- **톤 선택:** 각 톤은 다른 스타일로 카피를 생성합니다. 브랜드 보이스에 가장 잘 어울리는 톤을 선택하세요.
  
  톤을 선택하면 OpenAI에 전송되는 프롬프트에 스타일 지시가 추가되므로, 입력 내용, 채널 길이, 브랜드 가이드라인, 모델에 따라 정확한 출력이 달라질 수 있습니다. 
  
  각 톤이 기본적으로 의도하는 바는 다음과 같습니다:
  - **격식체:** 더 전문적이고 세련된 표현입니다. 완전한 문장, 더 정중한 언어, 최소한의 속어를 사용합니다.
  - **직설적:** 더 직접적이고 간결합니다. 형용사가 적고, "마케팅 과장"이 적으며, 행동 유도 문구가 더 명확합니다.
  - **캐주얼:** 더 편안하고 대화체입니다. 친근한 표현, 간단한 단어, 가벼운 에너지를 사용합니다.
  - **개인적:** 더 1:1이고 공감적입니다. "당신"을 더 많이 사용하며, 특히 작성 중인 메시지에 {% raw %}`{{${first_name}}}`{% endraw %}와 같은 개인화를 추가하면 더 맞춤화된 느낌을 줄 수 있습니다.
  - **시선을 끄는:** 더 주목을 끄는 스타일입니다. 임팩트 있는 표현, 높은 에너지, 강력한 후크와 CTA를 사용합니다(다른 톤보다 더 "프로모션"처럼 읽히는 경우가 많습니다).
  - **세련된:** 더 격조 있고 정제된 언어입니다. 덜 캐주얼하고, 더 "프리미엄" 포지셔닝을 합니다.
  - **프로페셔널:** 비즈니스적이고 명확합니다. 격식체보다 더 현대적이고 친근하면서도 권위를 유지합니다.
  - **수동적:** 더 부드럽고 덜 강압적인 언어입니다. 직접적인 명령이 적고, 더 제안적인 표현을 사용합니다.
  - **긴급한:** 즉시성과 시간 민감성을 강조합니다. 더 강력한 CTA, 마감일, 희소성 단서를 사용합니다.
  - **흥미진진한:** 더 에너지 넘치고 열정적입니다. 긍정적인 감정과 축하를 강조합니다(시선을 끄는 톤의 후크 중심 접근 방식보다 더 하이프에 초점을 맞추는 경우가 많습니다).
 
  
- **과거 캠페인 데이터 참조:** 활성화하면 캠페인 또는 캔버스 단계를 통해 전송된 이전 모바일 푸시 알림이 새 카피를 생성하는 데 스타일 참조로 사용됩니다. 자세한 내용은 [과거 캠페인 데이터 사용](#past-campaign-data)을 참조하세요.
- **카피 자동 번역:** 카피에 다른 출력 언어를 선택할 수 있습니다. 생성된 콘텐츠는 해당 언어로 출력됩니다.

### 4단계: 카피 생성

완료했으면 **생성**을 선택합니다. 제공한 정보를 사용하여 GPT가 카피를 작성하도록 프롬프트합니다. 응답은 OpenAI에서 가져와서 제공됩니다. 자세한 내용은 [내 데이터는 어떻게 사용되어 OpenAI로 전송되나요?](#ai-policy)를 참조하세요.

## 과거 캠페인 데이터 정보 {#past-campaign-data}

푸시를 출력 길이로 사용할 때, **과거 캠페인 데이터 참조**를 선택하면 무작위로 선택된 이전 모바일 푸시 캠페인이 OpenAI에 전송되어 GPT가 카피 생성의 기초로 사용할 수 있습니다. 현재 AI 카피라이터는 Liquid 구문이 없는 푸시 캠페인을 OpenAI에 전송합니다. 이 기능을 활용하지 않으려면 이 확인란을 선택하지 않은 상태로 두세요. Braze와 OpenAI가 데이터를 사용하는 방법에 대한 자세한 내용은 다음 섹션을 참조하세요. 

[브랜드 가이드라인]({{site.baseurl}}/user_guide/brazeai/generative_ai//brand_guidelines/)과 함께 사용하는 경우 브랜드 가이드라인과 과거 캠페인 데이터가 모두 최종 결과물에 통합됩니다.
