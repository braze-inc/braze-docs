# <sup>BrazeAITM으로</sup> 브랜드 가이드라인 생성하기

> 커스텀 브랜드 가이드라인을 통해 AI 생성 카피의 스타일을 브랜드의 목소리와 개성에 맞게 조정하세요.

## 브랜드 가이드라인 생성 {#steps}

AI 카피라이팅 어시스턴트에서 브랜드 가이드라인을 작성하려면 다음 단계를 따르세요. You can also create brand guidelines on the [Brand Guidelines]({{site.baseurl}}/user_guide/administrative/app_settings/brand_guidelines/) settings page.

### 1단계: 브랜드 가이드라인 작성

1. 메시지 작성기에서 다음을 찾아 선택합니다. <i class="fa-solid fa-wand-magic-sparkles" title="AI 카피라이터"></i> 를 클릭하여 [AI 카피라이팅 도우미를]({{site.baseurl}}/user_guide/brazeai/generative_ai/ai_copywriting/#access) 엽니다.
2. **브랜드 가이드라인 적용**을 선택한 다음 **브랜드 가이드라인 생성**을 선택하세요.

!['브랜드 가이드라인 적용' 드롭다운이 '브랜드 가이드라인 만들기' 버튼에 초점을 맞춰 확장되었습니다.]({% image_buster /assets/img/ai_copywriter/create_brand_guideline_button.png %}){:style="max-width:75%"}

{: start="3"}

3. 이 지침에 대한 이름을 입력하세요. 이것은 이전 선택에서 보게 될 레이블입니다.
4. **이 브랜드 가이드라인을 언제 사용할 것인가?**에 대해, 동료(그리고 미래의 본인)가 이 가이드라인을 사용하는 맥락을 이해할 수 있도록 세부 사항을 추가하세요.
5. If you want this to be the default brand guideline for the current workspace, check **Use as default brand guideline**.

![브랜드 가이드라인 생성 보기]({% image_buster /assets/img/ai_copywriter/manual_brand_guidelines.png %} "브랜드 가이드라인")

### 2단계: 브랜드 성격을 설명하세요

**브랜드 개성**의 경우, 무엇이 브랜드를 독특하게 만드는지 생각해보세요. 특성, 가치, 목소리 및 브랜드를 정의하는 모든 원형을 포함하세요. 다음은 고려해야 할 몇 가지 특성입니다.

| **특징**       | **정의**                                                                       | **예시**                                                        |
|--------------------------|--------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| 평판               | 시장 내에서 브랜드가 어떻게 인식되기를 원하는지.                               | 우리는 업계에서 가장 신뢰할 수 있고 고객 중심의 브랜드로 알려져 있습니다. |
| 성격 특성       | 브랜드의 성격을 설명하는 인간적인 특성.                     | 우리 브랜드는 친근하고 다가가기 쉬우며 항상 긍정적입니다.          |
| 값                   | 브랜드의 행동과 결정을 이끄는 핵심 가치.                           | 우리는 지속 가능성, 투명성, 그리고 공동체를 중요하게 여깁니다.            |
| 미분          | 경쟁사와 차별화되는 브랜드의 독특한 특성.                         | 우리는 개인화된 고객 서비스를 제공하여 두드러집니다. |
| 브랜드 목소리              | 브랜드가 사용하는 의사소통의 어조와 스타일.                                 | 우리의 목소리는 캐주얼하면서도 정보 전달에 충실하여 너무 격식적이지 않으면서도 명확성을 보장합니다. |
| 브랜드 원형          | 브랜드의 페르소나를 나타내는 전형 (영웅, 창조자 등).    | 우리는 항상 새로운 도전과 모험을 추구하는 '탐험가' 원형을 구현합니다. |
{: .reset-td-br-1 .reset-td-br-2 .reset-td-br-3 role="presentation" }

### 3단계: 언어를 정의하세요(선택 사항)

**제외사항**의 경우, 귀하의 브랜드와 맞지 않는 언어나 스타일을 나열하세요. 예를 들어, "비꼬는 것," "부정적인 태도," 또는 "거만한" 어조를 피하고 싶을 수 있습니다.

### 4단계: 가이드라인 테스트

가이드라인이 어떻게 작동하는지 테스트하세요. **가이드라인 테스트**를 확장하여 예시 복사본을 생성하고 필요에 따라 조정하세요.

![이메일 제목에 봄 세일에 대한 프로모션으로 브랜드 가이드라인을 테스트합니다.]({% image_buster /assets/img/ai_copywriter/test_brand_guidelines.png %})

### 5단계: 가이드라인 저장

가이드라인에 만족하면 **브랜드 가이드라인 저장**을 선택하세요. 새로운 지침은 향후 사용을 위해 워크스페이스에 저장됩니다.

{% alert important %}
복사본의 언어와 상관없이 출력 언어를 변경할 수 있지만, Braze나 OpenAI는 번역 품질을 보장하지 않습니다. 항상 번역을 사용하기 전에 테스트하고 검증하세요.
{% endalert %}

## 기존 가이드라인 편집하기

기존 브랜드 가이드라인을 편집하기:

1. AI 카피라이팅 어시스턴트를 열어보세요.
2. 변경하려는 브랜드 가이드라인을 적용하세요. 필드 근처에 버튼이 나타납니다.
3. **가이드라인 편집**을 선택합니다.
