# <sup>BrazeAITM을</sup> 사용한 콘텐츠 QA

> 맞춤법 오류, 문법 문제, 부적절한 어조 또는 불쾌한 언어를 전송하기 전에 발견할 수 있도록 <sup>BrazeAITM으로</sup> 콘텐츠를 QA하는 방법을 알아보세요.

## 지원되는 기능

콘텐츠 품질을 개선하는 데 도움이 되는 다음 기능이 지원됩니다:

| 기능                     | 설명 |
|----------------------------|-------------|
| 맞춤법 및 문법 검사 | 메시징의 맞춤법 및 문법 오류를 자동으로 확인합니다. 콘텐츠의 전반적인 정확도를 높이기 위해 수정 사항을 제안하고 권장 사항을 제공합니다. |
| 톤 분석              | 메시징의 어조를 평가하여 잠재적인 문제를 식별합니다. 의도한 어조가 원하는 커뮤니케이션 스타일과 일치하도록 하고 오해나 의도치 않은 불쾌감을 피하는 데 도움이 됩니다. |
| 모욕적인 언어 탐지 | 메시징에서 불쾌감을 줄 수 있거나 부적절한 언어를 검색하여 콘텐츠를 수정하고 정중한 커뮤니케이션을 유지할 수 있습니다. |
| 실수로 콘텐츠 확인   | 테스트 사용자에게 렌더링되지 않은 Liquid 코드를 포함하여 의도치 않게 추가되었을 수 있는 코드, 마크업 언어 또는 테스트 메시지를 감지합니다. |
| 다국어 지원     | OpenAI에서 공식적으로 지원되지는 않지만 GPT는 [여러 언어를](https://openai.com/research/gpt-4#:~:text=GPT%2D4%203%2Dshot%20accuracy%20on%20MMLU%20across%20languages) 이해할 수 있습니다. Braze는 사본의 언어나 현지화에 대한 정보를 OpenAI로 전송할 때 전달하지 않으므로 작성하는 언어에 따라 결과가 달라질 수 있습니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation"}

## <sup>BrazeAITM을</sup> 사용하여 콘텐츠 QA하기

{% alert note %}
이 기능은 현재 SMS, Android 푸시, iOS 푸시 및 기존 인앱 메시지에만 사용할 수 있습니다.
{% endalert %}

1. 모바일 푸시, SMS 또는 기존 인앱 메시지를 작성한 후 **테스트** 탭으로 이동합니다.
2. **AI를 사용한 콘텐츠 QA** 섹션을 찾습니다.
3. **콘텐츠 테스트를** 클릭합니다.

테스트 탭의 AI를 사용한 콘텐츠 QA 섹션에서 확인하세요.]({% image_buster /assets/img/content_qa_ai.png %})

## 모범 사례

다음 사항을 고려하여 AI를 통한 콘텐츠 QA를 최대한 활용할 수 있습니다:

- **메시지를 교정합니다:** 콘텐츠 검사기는 오류를 식별하는 데 도움이 될 수 있지만 여전히 콘텐츠를 수동으로 교정하는 것이 필수적입니다. AI가 생성한 제안을 유용한 가이드로 활용하되, 정확성을 보장하기 위해 직접 판단하세요.
- **톤 분석을 이해합니다:** 톤 분석 결과는 주관적이며 AI 모델의 이해를 기반으로 합니다. 유용한 인사이트를 제공할 수 있지만, 의도한 어조와 대화 맥락을 고려하여 적절하게 조정하세요.
- **신고된 불쾌감을 주는 언어를 다시 확인하세요:** 모욕적인 언어 감지는 강력하게 설계되었지만 간혹 오탐으로 표시될 수 있습니다. 플래그가 지정된 섹션을 주의 깊게 검토하고 필요에 따라 적절하게 변경하세요.
