# 선택 항목

> 선택 항목은 캠페인에서 각 사용자에 대한 메시지를 개인화하는 데 사용할 수 있는 데이터 그룹입니다. 선택 항목을 사용하면 기본적으로 카탈로그의 특정 열을 기반으로 커스텀 필터를 설정하게 됩니다. 여기에는 브랜드, 크기, 위치, 추가된 날짜 등에 대한 필터가 포함될 수 있습니다. 항목이 먼저 충족해야 하는 기준을 정의하여 사용자에게 표시되는 항목을 제어할 수 있습니다.<br><br>이 페이지에서는 카탈로그에서 선택 항목을 생성하고 사용하는 방법에 대해 설명합니다.

[카탈로그를]({{site.baseurl}}/user_guide/data/activation/catalogs/) 생성한 후에는 Braze 캠페인이나 추천에 선택 항목을 통합하여 카탈로그 데이터를 추가로 참조할 수 있습니다.

예제 카탈로그의 선택 섹션입니다.]({% image_buster /assets/img_archive/catalog_selections1.png %})

## 알아두어야 할 사항

- 카탈로그당 최대 30개의 선택 항목을 만들 수 있습니다.
- 선택 항목당 최대 10개까지 필터를 추가할 수 있습니다.
- 선택 항목은 Braze 카탈로그 데이터에서 추천을 구체화하는 데 유용합니다. 영감을 얻고 싶다면 [아이템 추천 정보에서]({{site.baseurl}}/user_guide/brazeai/recommendations/) 사용 사례를 확인하세요.

## 선택 항목 만들기

선택 항목을 만들려면 다음과 같이 하세요.

1. **카탈로그로** 이동하여 목록에서 카탈로그를 선택합니다.
2. **선택** 탭을 선택하고 **선택 항목 만들기를** 클릭합니다.
3. 선택 항목에 이름과 설명(선택 사항)을 입력합니다.
4. **필터 필드에서 필터**링할 카탈로그 열을 선택합니다. 1,000자를 초과하는 문자열 필드는 필터로 선택할 수 없습니다.
5. 관련 연산자(예: "같음" 또는 "같지 않음")와 속성을 선택하여 필터 기준 정의를 완료합니다.
6. **정렬 유형** 섹션에서 결과 정렬 방식을 결정합니다. 기본적으로 결과는 특별한 순서 없이 반환됩니다. 특정 필드를 기준으로 정렬을 지정하려면 **정렬 순서 무작위화를** 끄고 **정렬 필드** 및 **정렬 순서(오름차순** 또는 **내림차순** )를 지정합니다.
7. **결과 제한** 섹션에 결과를 입력합니다(최대 50개).
8. **선택 항목 생성을** 선택합니다.

### 테스트 및 미리 보기

선택 항목을 만든 후 **사용자 미리 보기** 섹션을 사용하여 임의의 사용자 또는 특정 사용자에 대해 선택 항목이 반환하는 결과를 확인할 수 있습니다. 개인화를 사용하는 선택 항목의 경우 사용자를 선택한 후에만 미리 보기를 볼 수 있습니다.

### 선택 결과의 Liquid

카탈로그에서 커스텀 속성 및 커스텀 이벤트와 같은 Liquid를 사용하면 선택한 사용자마다 다른 결과가 반환될 수 있습니다. 

{% alert note %}
연결된 콘텐츠 Liquid는 이 필터 설정에서 지원되지 않습니다.
{% endalert %}

속성이 Liquid 커스텀 속성으로 설정된 카탈로그 선택에 대한 필터 설정.]({% image_buster /assets/img_archive/catalog_selections7.png %})

## 메시징에서 선택 항목 사용하기

선택 항목을 만든 후 Liquid로 메시지를 개인화하여 해당 카탈로그에서 필터링된 항목을 삽입하세요. 메시지 작성기에 있는 개인화 창에서 Braze가 자동으로 Liquid를 생성하도록 할 수 있습니다:

1. 개인화를 지원하는 모든 메시지 작성기에서 다음을 선택합니다. <i class="fa-solid fa-circle-plus" style="color: #12aec5;" title="개인화 추가하기"></i> 을 선택하여 개인화 창을 엽니다.
2. **개인화 유형에서** **카탈로그 항목을** 선택합니다.
3. 카탈로그 이름을 선택합니다.
4. **항목 선택 방법에서** **선택** **항목** **사용을** 선택합니다.
4. 목록에서 원하는 항목을 선택합니다.
5. **표시할 정보의** 경우 카탈로그에서 각 항목에 포함할 필드를 선택합니다.
6. **복사** 아이콘을 선택하고 메시지에서 필요한 위치에 Liquid를 붙여넣습니다.

다음 선택 항목이 있는 개인화 추가 모달이 표시됩니다: '개인화 유형'은 '카탈로그 항목', '카탈로그 이름'은 '게임', '선택 유형'은 '선택 항목', '제목'은 "game_selection", '표시할 정보'는 "description_en" 을 입력합니다.]({% image_buster /assets/img_archive/catalog_selections6.png %}){: style="max-width:70%;"}

## 사용 사례

음식 배달 서비스를 운영하면서 가장 최근에 본 음식 카테고리를 기반으로 특정 음식 선호도를 가진 사용자에게 개인화된 메시지를 보내고 싶다고 가정해 봅시다. 

식사 이름, 가격, 이미지, 식사 카테고리에 대한 식사 배달 서비스의 정보가 포함된 카탈로그를 사용하여 사용자가 가장 최근에 본 카테고리를 기준으로 세 가지 식사를 추천하는 선택 항목을 만들 수 있습니다.

제품 유형을 식사로 식별하는 필터와 카테고리를 가장 최근에 본 것으로 식별하는 필터 두 개가 있는 식사 배달 서비스 선택의 예입니다. 선택은 세 가지 결과가 반환되는 순서를 무작위로 지정하도록 설정됩니다.]({% image_buster /assets/img_archive/catalog_selections2.png %}){: style="max-width:90%;"}

캠페인에서 이 카탈로그와 선택 항목을 사용하려면 캠페인 구축의 메시지 구성 섹션에서 **개인화 추가** 모달을 사용하세요. 이 예에서는 식사 배달 서비스의 정보가 포함된 카탈로그와 가장 최근에 본 카테고리를 기반으로 한 식사 추천을 선택했습니다. 이를 통해 식사 이름과 가격을 표시할 수 있습니다. 메시지를 더 구축하려면 선택 항목을 사용하여 첫 번째 추천 식단 이미지를 추가할 수도 있습니다.

메시지 구성 섹션에서 "recommendations_be_recent_category" 을 선택하면 "당신은 이 높은 등급의 식사를 좋아할 것입니다!"라는 헤더가 있는 콘텐츠 카드가 표시됩니다.]({% image_buster /assets/img_archive/catalog_selections3.png %}){: style="max-width:90%;"}

예를 들어 가장 최근에 본 카테고리가 '치킨'인 사용자가 있다고 가정해 보겠습니다. 개인화 설정과 콘텐츠 카드 캠페인을 사용하여 이 사용자에게 치킨이 포함된 세 가지 식사 추천을 보낼 수 있습니다.

숯불에 구운 레몬 치킨 이미지와 사용자가 가장 최근에 본 카테고리를 기반으로 치킨이 포함된 세 가지 식사 추천 목록이 포함된 콘텐츠 카드입니다.]({% image_buster /assets/img_archive/catalog_selections4.png %}){: style="max-width:90%;"}

동일한 개인화를 사용하여 가장 최근에 본 카테고리가 '소고기'인 사용자에게 세 가지 식사 추천을 보낼 수도 있습니다.

소고기 스트로가노프 이미지가 포함된 콘텐츠 카드와 사용자가 가장 최근에 본 카테고리를 기반으로 소고기가 포함된 두 가지 추천 메뉴 목록이 표시됩니다.]({% image_buster /assets/img_archive/catalog_selections5.png %}){: style="max-width:90%;"}
