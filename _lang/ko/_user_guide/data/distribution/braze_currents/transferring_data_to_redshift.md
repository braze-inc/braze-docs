# Redshift로 데이터 전송하기

> [Amazon Redshift는](https://aws.amazon.com/redshift/) Amazon S3와 함께 Amazon Web Services에서 실행되는 인기 있는 데이터 웨어하우스입니다. 커런츠의 Braze 데이터는 Redshift로 바로 전송하기 쉽도록 구조화되어 있습니다.

추출, 변환, 로드(ETL) 프로세스를 통해 Amazon S3에서 Redshift로 데이터를 전송하는 방법에 대한 자세한 내용은 커런츠 예제 [GitHub 리포지토리에서](https://github.com/Appboy/currents-examples) 확인할 수 있습니다.

{% endalert %}
