# 커스텀 이벤트 집계 내보내기

> 이 페이지에서는 대시보드의 **커스텀** 이벤트 보고서 페이지에서 시간 경과에 따른 하나 이상의 커스텀 이벤트 발생 현황을 볼 수 있습니다. 커스텀 이벤트 또는 시간별 커스텀 이벤트에 대한 자세한 통계를 보려면 특정 세그먼트별로도 데이터를 볼 수 있는 옵션이 있습니다.

**분석에서** **커스텀 이벤트 보고서를** 찾을 수 있습니다.

커스텀 이벤트]({% image_buster /assets/img_archive/Export_events.png %})

다음 CSV를 내보낼 수 있습니다:

- 날짜별 커스텀 이벤트
    - (선택 사항) 다양한 세그먼트를 위한 커스텀 이벤트
- 시간대별 커스텀 이벤트
    - (선택 사항) 다양한 세그먼트를 위한 커스텀 이벤트
- MAU별 커스텀 이벤트
- 다양한 세그먼트를 위한 커스텀 이벤트
- KPI 공식에 따른 커스텀 이벤트
    - (선택 사항) 다양한 세그먼트를 위한 커스텀 이벤트

{% alert tip %}
CSV 및 API 내보내기에 대한 도움말은 [내보내기 문제 해결을]({{site.baseurl}}/user_guide/data/export_braze_data/export_troubleshooting/) 참조하세요.
{% endalert %}
