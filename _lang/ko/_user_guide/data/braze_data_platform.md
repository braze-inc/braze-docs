# Braze 데이터 플랫폼 개요

> 고객 데이터 플랫폼은 세 가지 데이터 관련 작업을 기반으로 고객 생애주기 전반에 걸쳐 개인화되고 영향력 있는 경험을 창출할 수 있도록 지원하는 포괄적이고 구성 가능한 데이터 기능 및 파트너 통합 세트입니다: [데이터 통합]({{site.baseurl}}/user_guide/data/unification), [데이터 활성화]({{site.baseurl}}/user_guide/data/activation) 및 [데이터 배포]({{site.baseurl}}/user_guide/data/distribution).

고객 데이터 플랫폼의 여러 기능을 조합하여 데이터를 활용하면 고객의 행동에 실시간으로 반응하는 의미 있는 타겟팅 메시지를 만들 수 있습니다.

## 데이터 통합 

API] 및 [SDK]를 사용하여 모든 소스에서 퍼스트파티 데이터를 수집하고 통합하세요. 또한 [Cloud 데이터]({{site.baseurl}}/user_guide/data/unification/cloud_ingestion) 수집과 같은 기본 제공 수집 도구를 활용하여 데이터 웨어하우스 또는 파일 스토리지 솔루션에서 Braze로 직접 간단한 통합을 만들거나, [데이터 변환을]({{site.baseurl}}/user_guide/data/unification/data_transformation) 사용하여 웹훅 통합을 구축하고 관리하여 데이터를 수집하여 Braze로 전송할 수 있습니다.

## 데이터 활성화

데이터를 정리하고, 정리하고, 사용할 수 있도록 준비하세요. 여기에는 고객 프로필과 세그먼트를 통해 고객의 행동과 선호도를 실시간으로 파악하는 것이 포함됩니다. 타겟팅된 메시지를 만들 때 [보고서 측정기준 용어집을]({{site.baseurl}}/user_guide/data/activation/report_metrics) 참조하여 고객이 이러한 개인화된 경험에 어떻게 반응하는지 식별할 수 있습니다.

## 데이터 배포

데이터를 스트리밍하고 외부 시스템으로 내보내 다음 단계의 인사이트와 의사 결정을 내릴 수 있습니다. Braze [커런츠를]({{site.baseurl}}/user_guide/data/distribution/braze_currents) 사용해 이벤트 데이터를 데이터 웨어하우스로 스트리밍하여 비즈니스 인텔리전스 도구를 강화하세요.
