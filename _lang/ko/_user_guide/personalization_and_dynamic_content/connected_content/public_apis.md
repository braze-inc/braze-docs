# 연결된 콘텐츠에 공개 API 사용

> 연결된 콘텐츠에 사용할 수 있는 공개적으로 사용 가능한 다양한 API가 있습니다. 공개 API를 사용하면 메시징에 커스텀 데이터를 삽입할 수 있습니다. 이 페이지에는 연결된 콘텐츠에 사용할 수 있는 공개 API 목록이 정리되어 있습니다. 하지만 연결된 콘텐츠의 다양한 잠재적 용도를 제공하는 더 많은 API가 있습니다.  

공유할 API가 있다면 [성공@braze.com](mailto:success@braze.com) 으로 문의해 주세요!

{% alert note %}
공개 API에는 사용 제한 및 요금 제한이 적용될 수 있습니다. API 설명서를 읽고 사용 목적에 대해 API 제공업체에 문의하세요.
{% endalert %}
