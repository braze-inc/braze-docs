## 필수 조건

[다국어 지원]({{site.baseurl}}/multi_language_support/)을 편집하고 관리하려면 워크스페이스에 대한 다음 [사용자 권한]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/user_permissions/)이 필요합니다:

- 현지화 설정 보기
- 현지화 설정 편집
- 현지화 설정 삭제

메시지에 로케일을 추가하려면 "캠페인 편집" 권한이 필요합니다.

{% alert important %}
다국어 지원은 현재 얼리 액세스 단계에 있습니다. 이 얼리 액세스에 참여하고 싶으시면 Braze 계정 매니저에게 문의하세요.
{% endalert %}

**테스트** 탭의 **사용자로 메시지 미리보기** 드롭다운에서 **커스텀 사용자**를 선택하고 다른 언어를 입력하여 메시지가 예상대로 번역되는지 미리 확인하세요.

## 자주 묻는 질문

#### 내 로케일 중 하나에서 번역된 사본을 변경할 수 있나요?
네. 먼저 CSV에서 편집한 다음 파일을 다시 업로드하여 번역된 사본을 변경하세요.

#### 번역 태그를 중첩할 수 있나요?
아니요.

#### 번역 태그에 HTML 스타일을 추가할 수 있나요?
네. 하지만 HTML 스타일이 콘텐츠와 함께 번역되지 않았는지 반드시 확인하세요.

#### Braze는 어떤 유효성 검사 또는 추가 확인을 수행하나요?

| 시나리오                                                                                                                                                 | Braze의 유효성 검사                                                                                            |
|----------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| 번역 파일에 현재 메시지와 관련된 로케일이 누락되었습니다.                                                                               | 이 번역 파일은 업로드되지 않습니다.                                                                       |
| 번역 파일에 현재 이메일 메시지의 Liquid 번역 태그 내 텍스트와 같은 텍스트 블록이 누락되었습니다.                                | 이 번역 파일은 업로드되지 않습니다.                                                                       |
| 번역 파일에 현재 이메일 메시지의 텍스트 블록과 일치하지 않는 기본값 텍스트가 포함되어 있습니다.                                          | 이 번역 파일은 업로드되지 않습니다. 다시 업로드를 시도하기 전에 CSV에서 이 문제를 수정하세요.               |
| 번역 파일에 **다국어 지원** 설정에 존재하지 않는 로케일이 포함되어 있습니다.                                                           | 이러한 로케일은 Braze에 저장되지 않습니다.                                                                      |
| 번역 파일에 현재 메시지에 존재하지 않는 텍스트 블록(예: 번역이 업로드될 당시의 현재 초안)이 포함되어 있습니다. | 현재 메시지에 존재하지 않는 텍스트 블록은 번역 파일에서 Braze로 저장되지 않습니다. |
| 해당 로케일이 이미 번역 파일의 일부로 메시지에 업로드된 후 메시지에서 로케일을 제거합니다.                           | 로케일을 제거하면 메시지의 해당 로케일과 관련된 모든 번역이 제거됩니다.                   |
