{% if include.metric == "Total Revenue" %}
<i>총 매출</i>은 설정된 주요 전환 기간 내에 캠페인 수신자로부터 발생한 총 매출(달러)입니다.
{% endif %}

{% if include.metric == "Unique Clicks" %}
<i>고유 클릭</i> 수는 메시지 내에서 한 번 이상 클릭한 수신자의 고유한 수이며 <a href='https://braze.com/docs/help/help_articles/data/dispatch_id/'>dispatch_id</a>로 측정됩니다.
{% endif %}

{% if include.metric == "Unique Dismissals" %}
<i>고유 닫기</i> 수는 캠페인에서 콘텐츠 카드를 닫은 사용자 수입니다. 사용자가 캠페인에서 콘텐츠 카드를 여러 번 해제하는 것은 하나의 고유한 해제를 나타냅니다.
{% endif %}

{% if include.metric == "Unique Recipients" %}

{% endif %}

{% endif %}
