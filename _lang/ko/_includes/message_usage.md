# 메시지 사용량 대시보드

> 메시지 사용량 대시보드에서는 SMS 및 WhatsApp 크레딧 사용량에 대한 셀프 서비스 인사이트를 제공하여 계약 할당량과 비교한 과거 및 현재 사용량을 종합적으로 볼 수 있습니다. 이러한 인사이트를 통해 혼란을 줄이고 초과 위험을 방지하기 위한 조정에 도움을 받을 수 있습니다.

{% alert important %}
메시지 사용량 대시보드는 현재 얼리 액세스 중입니다. 얼리 액세스에 참여하려면 Braze 계정 매니저에게 문의하세요.
{% endalert %}

**메시지 사용량** 대시보드는 세 개의 섹션으로 나뉩니다:
- [크레딧 사용 개요](#credit-usage-overview)
- [SMS/MMS](#smsmms) 
- [WhatsApp](#whatsapp)

**설정** > **청구** > **메시지 사용량으로** 이동하여 대시보드에 액세스합니다.

{% alert note %}
[이전 탐색을]({{site.baseurl}}/navigation) 사용하는 경우 계정 아이콘을 선택한 다음 **구독 및 사용량을** 선택하면 이 페이지를 찾을 수 있습니다.
{% endalert %}

## 메시지 크레딧 사용 개요

**메시지 크레딧 사용량 개요** 에서는 크레딧을 사용하는 모든 채널의 사용량에 대한 개요를 제공합니다. 전체 크레딧 할당량 대비 진행 상황을 확인하고 활성 계약 및 계약 기간에 대한 세부 정보를 확인할 수 있습니다.

이 페이지는 메시지 크레딧 계약을 사용 중이거나 WhatsApp을 구매한 경우 표시됩니다. 메시지 크레딧을 사용하는 채널은 **크레딧 계약 개요에** 표시됩니다.

{% alert note %}
WhatsApp을 구매했지만 메시지 크레딧 계약을 체결하지 않은 경우에도 기존 WhatsApp 계약이 청구되는 방식이므로 WhatsApp에 대한 크레딧 소비가 계속 표시됩니다. 이는 메시지 크레딧 계약을 맺은 경우에만 크레딧을 소비하는 기존 SMS와는 다릅니다.
{% endalert %}

**메시지 크레딧 사용량 개요** 데이터는 **크레딧 계약 개요에** 표시되는 계약 기간으로 제한됩니다. **크레딧 기간** 이외의 날짜 범위에서는 필터링할 수 없습니다.

### 계약 기간 동안의 메시지 크레딧 사용량

**계약 기간 동안의 메시지 크레딧 사용량** 그래프에는 선택한 기간 동안의 사용량이 표시됩니다. 이 차트의 세분성은 선택한 기간에 따라 달라집니다. 차트 오른쪽 상단에 있는 메뉴를 선택하여 내보내기 옵션을 선택합니다.

![크레딧 사용량, 크레딧 계약 개요, 계약 기간 동안의 크레딧 소비에 대한 섹션이 있는 메시지 크레딧 사용량 개요 대시보드입니다.][1]{: style="max-width:80%;"}

## SMS 및 MMS

**SMS/MMS 크레딧 사용량에는** SMS/MMS 채널의 사용 내역이 표시됩니다. 데이터 테이블의 열은 SMS/MMS가 크레딧 채널인지 여부에 따라 달라집니다. SMS/MMS가 크레딧 채널인 경우 크레딧 **비율** 및 **크레딧** 열이 추가로 표시되며 해당 국가별 요율과 소비된 크레딧이 표시됩니다. 또한 상위 레벨 타일에는 선택한 날짜 범위의 총 SMS 및 릴리스된 경우 MMS 사용량이 표시됩니다.

**국가** 또는 **SMS 유형별로** 필터링할 수 있는 필터를 사용할 수 있습니다.

![높은 수준의 데이터에 대한 타일과 계정별 사용량 섹션이 있는 SMS/MSS 크레딧 사용량.][2]{: style="max-width:80%;"}

**메시지 크레딧 사용량 개요와** 달리 이 섹션에는 이전 계약 기간의 과거 데이터가 포함되어 있습니다. 

{% alert note %}
비크레딧과 메시지 크레딧 사용량을 모두 포함하는 날짜 범위를 선택할 수 있습니다. 이 경우 메시지 크레딧을 초과하여 발생한 소비는 크레딧 **비율** 및 **크레딧** 열에 `—` (null)로 표시됩니다.
{% endalert %}

![SMS/MMS 크레딧 사용량 테이블에 null 값이 있습니다.][3]{: style="max-width:80%;"}

## WhatsApp

**WhatsApp 크레딧 사용량에는** WhatsApp 채널의 사용 내역이 표시됩니다. 타일에는 총 WhatsApp 크레딧 사용량이 표시되며, **계정별 사용량** 섹션에서 필터를 적용하여 데이터 테이블 결과를 특정 작업 공간으로 제한하여 세분화할 수 있습니다.

### 필터

데이터를 필터링할 수 있습니다:
- 국가
- WhatsApp Business 계정
- Braze 워크스페이스
- 대화 카테고리 유형
- 지역

![WhatsApp 크레딧 사용량에는 총 크레딧 사용량 타일과 계정별 사용량 표가 표시됩니다.][4]{: style="max-width:80%;"}

## 알아두어야 할 사항

{% alert important %}
**메시지 사용량** 대시보드에 표시되는 데이터는 계약 수준이며 개별 대시보드 회사 또는 워크스페이스로 범위가 지정되지 않습니다. 이 데이터는 대시보드 내의 모든 워크스페이스, 그리고 잠재적으로 모든 대시보드(여러 대시보드가 있는 경우)의 사용량을 반영합니다.
{% endalert %}

- 기본 데이터는 매일 주기로 제공되며, 데이터 테이블은 동부 표준시 기준 오전 3시, 오전 9시, 오후 12시, 오후 6시에 새로 고쳐집니다. 
- 브레이즈는 표준 반올림 방법을 따릅니다. 숫자는 가장 가까운 10번째 자리까지 반올림됩니다.
