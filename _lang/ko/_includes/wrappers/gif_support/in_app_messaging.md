{% tab Android %}
기본 [Braze Android SDK](https://github.com/braze-inc/braze-android-sdk)를 사용하여 인앱 메시지에 애니메이션 GIF를 추가할 수 있습니다. 기본적으로 Android SDK는 HTML 인앱 메시지를 사용하여 GIF를 표시합니다.

다른 모든 인앱 메시지 유형의 경우 사용자 지정 이미지 라이브러리를 사용해야 합니다. 자세한 내용은 [Android 인앱 메시징을 참조하세요: GIF]({{site.baseurl}}/developer_guide/in_app_messages/gifs/?sdktab=android).
{% endtab %}

{% tab iOS %}
기본 [Braze Swift SDK를](https://github.com/braze-inc/braze-swift-sdk) 사용하여 인앱 메시지에 애니메이션 GIF를 추가할 수 있습니다. 기본적으로 모든 Braze 인앱 메시지는 GIF를 지원합니다. 전체 안내는 [튜토리얼을 참조하세요: Swift 인앱 메시지용 GIF 지원](https://braze-inc.github.io/braze-swift-sdk/tutorials/braze/c3-gif-support).
{% endtab %}
