{% tabs local %}
{% tab locations %}
**위치** 페이지에서 지오펜스를 인에이블먼트하려면 다음과 같이 하세요:

1. Braze에서 **오디언스** > **위치로** 이동합니다.
2. 작업 공간에서 지오펜싱이 인에이블된 앱의 수는 매핑 아래에 나열됩니다. 예를 들어 지오펜싱이 일부 앱에만 인에이블먼트된 경우 읽을 수 있습니다: **지오펜싱이 인에이블된 앱 5개 중 2개**. 추가 앱을 인에이블하려면 지도 아래에서 현재 개수를 선택하세요.
3. 지오펜스를 인에이블먼트할 앱을 선택한 다음 **완료를** 선택합니다 **.**

![Braze 위치 페이지의 지오펜스 옵션.]({% image_buster /assets/img_archive/enable-geofences-locations-page.png %})
{% endtab %}

{% tab app settings %}
**앱 설정** 페이지에서 지오펜스를 인에이블하려면 다음과 같이 하세요:

1. Braze에서 **설정** > **앱 설정으로** 이동합니다.
2. 지오펜스를 인에이블하려는 앱을 선택합니다.
3. **지오펜스 인에이블먼트에** 체크한 다음 **저장을** 선택합니다 **.**

![Braze 설정 페이지에 있는 지오펜스 체크박스.]({% image_buster /assets/img_archive/enable-geofences-app-settings-page.png %})
