# 2025년 2월 4일 릴리스

## Braze 문서 개선

### 데이터 및 분석 개편
원하는 정보를 더 쉽게 찾을 수 있도록 기존에 "데이터 및 분석" 아래에 있던 문서를 [데이터]({{site.baseurl}}/user_guide/data)와 [분석]({{site.baseurl}}/user_guide/analytics)으로 분리했습니다. 

### 개발자 가이드
[Braze 개발자 가이드]({{site.baseurl}}/developer_guide/home) 전반에 걸쳐 대규모 정리를 수행했으며, 여러 페이지에 나뉘어 있던 "사용 방법"을 하나의 페이지로 통합했습니다.

모든 Braze SDK에 대한 참조 설명서와 리포지토리를 나열하는 새로운 [SDK 참조 페이지]({{site.baseurl}}/developer_guide/references)도 추가되었습니다.

#### 언리얼 엔진 Braze SDK
언리얼 엔진 Braze SDK GitHub 리포지토리 README의 모든 콘텐츠를 Braze 문서의 [전용 섹션]({{site.baseurl}}/developer_guide/sdk_integration/?sdktab=unreal%20engine)으로 마이그레이션하고 재작성했습니다.

## 데이터 유연성

### API 사용량 대시보드

{% multi_lang_include release_type.md release="General availability" %}

[API 사용량 대시보드]({{site.baseurl}}/user_guide/analytics/dashboard/api_usage_dashboard)를 통해 Braze로 들어오는 REST API 트래픽을 모니터링하여 REST API 사용 트렌드를 파악하고 잠재적인 문제를 해결할 수 있습니다.

### 커스텀 속성에 태그 추가

{% multi_lang_include release_type.md release="General availability" %}

"이벤트, 속성, 구매 관리" 권한이 있는 경우 커스텀 속성을 생성한 후 [태그를 추가]({{site.baseurl}}/user_guide/data/custom_data/custom_attributes#adding-tags)할 수 있습니다. 그런 다음 태그를 사용하여 속성 목록을 필터링할 수 있습니다.

### 카탈로그 선택 및 비동기 카탈로그 필드 엔드포인트 

{% multi_lang_include release_type.md release="General availability" %}

다음 엔드포인트가 정식으로 출시되었습니다:
* [POST: 카탈로그 필드 생성]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/post_create_catalog_fields)
* [DELETE: 카탈로그 필드 삭제]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/delete_catalog_field)
* [DELETE: 카탈로그 선택 삭제]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/delete_catalog_selection)
* [POST: 카탈로그 선택 생성]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/post_create_catalog_selections)

### 이메일 주소를 사용하여 캠페인 또는 캔버스 트리거하기

{% multi_lang_include release_type.md release="General availability" %}

이제 이메일 주소로 수신자를 지정하여 [캠페인]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/targeting_users/)과 [캔버스]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/?tab=target%20audience#step-2c-set-your-target-entry-audience)를 트리거할 수 있습니다.

### API를 통해 전화번호로 사용자 식별하기

{% multi_lang_include release_type.md release="General availability" %}

이제 별칭 및 이메일 주소 외에도 전화번호를 사용하여 [`/users/identify` API 엔드포인트]({{site.baseurl}}/api/endpoints/user_data/post_user_identify)를 통해 사용자를 식별할 수 있습니다.

### SAML 추적 가져오기
SAML SSO 관련 문제를 고객지원팀과 보다 효율적으로 해결하는 데 도움이 되는 [SAML 추적을 얻는 방법에 대한 단계]({{site.baseurl}}/user_guide/administrative/access_braze/single_sign_on/set_up#obtaining-a-saml-trace)를 추가했습니다.
 
### 지역별 데이터 센터
Braze가 새로운 지역으로 서비스를 확장함에 따라, 운영 접근 방식을 명확히 하기 위해 [Braze 데이터 센터에 대한 문서]({{site.baseurl}}/user_guide/data/data_centers)를 추가했습니다.

 
## 창의력 발휘
 
### 가격 인하 알림 및 재입고 알림

{% multi_lang_include release_type.md release="General availability" %}

이제 캔버스와 카탈로그를 통해 [재입고 알림]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/back_in_stock_notifications)을 설정하여 품목이 재입고되었을 때 고객에게 알릴 수 있습니다.

또한 카탈로그와 캔버스에서 가격 인하 알림을 설정하여 품목의 가격이 하락했을 때 고객에게 알리는 [가격 인하 알림]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/price_drop_notifications)을 생성할 수도 있습니다.

### 선택 미리보기 

{% multi_lang_include release_type.md release="General availability" %}

선택을 생성한 후, 무작위 사용자 또는 특정 사용자에 대해 [선택이 반환할 내용을 확인]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/selections/#test-and-preview)할 수 있습니다.

### Liquid를 포함한 카탈로그 항목 템플릿화 

{% multi_lang_include release_type.md release="General availability" %}

Liquid를 포함하는 [카탈로그 항목을 템플릿화]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/using_catalogs/#using-liquid)할 수 있습니다.

### 캔버스 템플릿
[선호도 설문조사를 통한 사용자 온보딩]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/preference_survey)과 [더블 옵트인을 통한 이메일 가입 생성]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/email_signup)을 위한 새로운 캔버스 템플릿을 추가했습니다.

### B2B를 위한 Salesforce Sales Cloud로 리드 관리
B2B 마케터가 Braze를 활용하는 방법 중 하나는 Salesforce Sales Cloud와의 통합입니다. 이 [사용 사례]({{site.baseurl}}/user_guide/getting_started/b2b_use_cases/b2b_salesforce_sales_cloud)를 구현하는 방법에 대해 자세히 알아보세요.
 
## 강력한 채널

### 억제 목록

{% multi_lang_include release_type.md release="Beta" %}
 
[억제 목록]({{site.baseurl}}/user_guide/engagement_tools/segments/suppression_lists)은 메시지를 절대 수신하지 않을 사용자 그룹을 지정합니다. 관리자는 세분화와 동일한 방식으로 세그먼트 필터를 사용하여 사용자 그룹을 좁히는 억제 목록을 생성할 수 있습니다.

## 새로운 Braze 파트너십

### Constructor - 동적 콘텐츠
[Constructor]({{site.baseurl}}/partners/ecommerce/product_search_recommendations/constructor/)는 AI와 머신 러닝을 사용하여 전자상거래 및 소매 웹사이트를 위한 개인화된 검색, 추천 및 탐색 경험을 제공하는 검색 및 제품 발견 플랫폼입니다.
 
### Trustpilot - 동적 콘텐츠
[Trustpilot]({{site.baseurl}}/partners/trustpilot/)은 고객이 피드백을 공유하고 리뷰를 관리하고 응답할 수 있게 해주는 온라인 리뷰 플랫폼입니다.
