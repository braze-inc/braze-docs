# 2021년 7월

## 간단한 설문조사 인앱 메시지

[간단한 설문조사 인앱 메시지 템플릿을]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/traditional/templates/simple_survey/) 사용하여 캠페인 전략에 도움이 되는 사용자 속성, 인사이트 및 선호도를 수집하세요. 예를 들어, 사용자에게 앱 사용 방법을 묻거나 개인 선호도에 대해 자세히 알아보거나 특정 기능에 대한 만족도를 물어볼 수 있습니다. 이 설문조사 템플릿은 모바일 앱과 웹 브라우저 모두에서 지원됩니다.

## 리퀴드 사용 사례 라이브러리

Liquid로 한 가지 일을 하는 방법이 궁금하신가요? 영감을 찾고 계신가요? 기념일, 생일부터 플랫폼 타겟팅 등 다양한 사용 사례 모음을 보려면 [Liquid 사용 사례 라이브러리]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/liquid/liquid_use_cases/)를 확인하세요.

## 이메일 추적

이제 사용자 프로필별로 오픈 픽셀 추적 및 클릭 추적을 비활성화할 수 있습니다. 이러한 유연성을 통해 고객은 개별 고객 프로필에서 더 이상 추적을 원하지 않는다고 표시할 수 있는 지역 개인정보 보호법을 준수할 수 있습니다. 자세한 내용은 [이메일 추적]({{site.baseurl}}/user_guide/analytics/tracking/email_tracking/) 도움말을 참조하세요.

## SDK 데이터 수집 옵션

Braze SDK를 통합하여 유연한 데이터 수집을 가능하게 하는 방법에 대한 자세한 내용은 [SDK 데이터 수집 옵션]({{site.baseurl}}/user_guide/data/user_data_collection/sdk_data_collection/)에 대한 새 설명서에서 확인할 수 있습니다.

## SSL 개요

새로운 [SSL 개요]({{site.baseurl}}/user_guide/message_building_by_channel/email/email_setup/ssl/) 문서에서 Braze의 SSL에 대해 자세히 알아보고, SSL이 중요한 이유와 SSL 인증서를 획득하는 방법을 알아보세요.

## 앰플리튜드 사용자 프로필 API 엔드포인트

Amplitude의 고객 프로필 API는 Amplitude의 고객 프로필을 제공합니다. 여기에는 사용자 속성, 계산된 사용자 속성, 해당 사용자를 포함하는 코호트의 코호트 ID 목록 및 권장 사항이 포함됩니다. 자세한 내용은 [Amplitude 고객 프로필 API 엔드포인트]({{site.baseurl}}/partners/data_and_analytics/customer_data_platform/amplitude/amplitude_user_profile_api/)를 참조하세요.

## 캠페인 세부 정보 엔드포인트 

[`/campaigns/details` 엔드포인트에]({{site.baseurl}}/api/endpoints/export/campaigns/get_campaign_details/) 인앱 메시지 채널을 위한 전용 `message` 응답이 추가되었습니다. 이에 대한 문서는 [여기에서]({{site.baseurl}}/api/endpoints/export/campaigns/get_campaign_details/#messages) 확인할 수 있습니다.
