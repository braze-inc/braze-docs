# 2020년 3월

## 사용자 지정 속성 필터 동작

대시보드의 [필터가 개선되어]({{site.baseurl}}/user_guide/data_and_analytics/custom_data/custom_attributes/#dates) 보다 직관적인 동작을 포함할 수 있게 되었습니다.
특정 사용자 정의 속성 필터의 작동 방식에 두 가지 중요한 변경 사항이 있습니다. 

사용자 지정 속성 필터 변경 사항은 필터에 반영됩니다: 
- **X일** 전
- 향후 **X일** 미만
- 반복 이벤트의 날짜<br>

세그먼트에 이러한 필터를 사용하는 경우 이러한 변경 사항을 고려하여 세그먼트를 다시 조정하는 것이 좋습니다.
