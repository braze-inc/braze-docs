# 2020년 2월

## 리텐션 보고서

Braze는 이제 캠페인을 위한 [리텐션 리포트를]({{site.baseurl}}/user_guide/analytics/reporting/retention_reports/) 제공합니다. 이 기능은 특정 캠페인에서 메시지를 받은 사용자에 대한 사용자 리텐션을 측정하는 데 도움이 됩니다. 리텐션 보고서는 대시보드 내 **캠페인 분석** 페이지에서 편리하게 확인할 수 있습니다. 

![보고서 전문]({% image_buster /assets/img/retention_report_full_report.png %})

{% alert important %}
2020년 1월 13일 현재 이 기능은 EU 지역에 배포된 클라이언트나 HIPPA 준수 클러스터에서는 아직 사용할 수 없습니다. 전체 플랫폼에 기능을 지속적으로 배포하면서 해당 지역에서도 이 보고서를 사용할 수 있도록 할 예정입니다.
{% endalert %}
