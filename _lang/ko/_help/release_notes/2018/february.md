# 2018년 2월

## iOS 푸시 배지 수

이제 Braze에서 푸시 작성기 내에서 [배지 수를 업데이트할][89] 수 있습니다.
각 푸시 메시지에 대해 해당 알림이 트리거하는 배지 수를 지정할 수 있습니다.

## 이메일 주소를 사용하여 API를 통해 사용자 내보내기

이제 이메일 주소를 지정하여 [API를 통해 사용자 프로필 데이터를 내보낼][88] 수 있습니다.
이 내보내기에는 해당 이메일 주소와 연결된 모든 프로필이 포함됩니다.

## 이메일 템플릿 API

이제 [API를 통해 이메일 템플릿][87]을 만들고 업데이트할 수 있습니다. 각 템플릿에는 다른 API 호출에서 참조할 수 있는 **email_template_id**가 있습니다.

## REST API 키 권한

이제 [여러 개의 REST API 키][86]를 만들고 각각에 대한 액세스 권한을 구성할 수 있습니다. 각 키는 특정 엔드포인트에 대한 액세스 권한을 부여하도록 구성할 수 있습니다.

또한 주어진 REST API 키에 대해 REST API 요청을 할 수 있는 [IP 주소][85] 및 서브넷의 화이트리스트를 지정할 수도 있습니다.
