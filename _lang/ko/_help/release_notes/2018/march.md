# 2018년 3월

## 링크 템플릿

이제 이메일 메시지의 모든 링크에 [매개변수를 추가하거나]({{site.baseurl}}/user_guide/message_building_by_channel/email/link_templates/#link-templates) URI를 미리 추가하는 템플릿을 만들 수 있습니다. 모든 링크에 Google 애널리틱스 쿼리 매개변수를 추가하고 싶으신가요? 시작하세요!
