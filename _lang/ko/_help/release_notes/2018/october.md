## 캠페인의 정확한 잠재고객 통계 계산하기

이제 **캠페인 분석으로** 이동하여 오디언스에 대한 정확한 통계를 계산할 수 있습니다. **타겟 사용자** 섹션의 바닥글에서 정확한 **통계 계산**을 클릭하면 정확한 오디언스 통계가 채워집니다. 계산하기 전에 캠페인을 저장해야 합니다(초안 캠페인은 초안으로 저장됩니다).

## Windows 8 지원 중단

Braze는 2018년 10월 10일부터 Windows 8을 더 이상 지원하지 않습니다.

## 파트너십 허브

이제 Braze 플랫폼의 통합에서 **통합** 키 및 지침과 함께 통합 목록을 찾을 수 있습니다.

## 이메일 분석 계산

Braze는 이제 이메일 분석의 정확도를 크게 향상시키기 위해 이메일 전송 파트너(ESP)의 이벤트 데이터를 사용하여 모든 이메일 분석을 계산하고 있습니다. 이 솔루션은 오픈 소스 데이터베이스 솔루션인 Postgres를 활용하여 데이터 무결성을 보장합니다.

{% alert important %}
고유 열람 수와 고유 클릭 수는 현재 이메일 전송 파트너가 제공하는 집계 데이터에 따라 달라집니다. 이번 릴리스에 도입된 것과 동일한 인프라를 사용하여 이러한 고유성 통계를 계산하는 작업이 진행 중입니다.
{% endalert %}

## 작곡가 패널 컨트롤

메시지 작성기 컨트롤이 아이콘과 연관된 문구를 포함하도록 개편되어 사용성과 탐색 기능이 향상되었습니다.

## 커런츠용 Azure

이제 커런츠를 사용하는 Braze 고객은 [Azure]({{site.baseurl}}/partners/braze_currents/data_storage_integrations/partners/microsoft_azure_blob_storage/)를 잠재적인 통합 대상으로 볼 수 있습니다.

이제 이메일 제목과 푸시 제목의 입력 상자를 확장할 수 있습니다.
