## 2018년 11월

### 캔버스 항목 설정 마법사

작업 누락과 그로 인한 오류를 방지하기 위해 캔버스 UI가 간소화됩니다. 특히 캔버스 구성은 이제 **캠페인** 마법사의 디자인과 유사하게 마법사에 표시됩니다. 자세한 내용은 [캔버스 만들기]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/) 단계를 확인하세요!

이 새로운 기능에 대해 더 자세히 알고 싶으신가요? [엔트리 마법사](https://learning.braze.com/the-new-canvas-entry-step/264889/scorm/20z5ij5ublxbk)에서 Braze학습 과정을 확인하세요!

### 인앱 메시지 웹 모달 CSS

[웹 모달 CSS 편집기]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/create/#web-modal-css)가 업데이트되어 새로운 [CSS 템플릿]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/in_app_message_color_templates/#css-template)을 사용하여 인앱 메시지의 모양을 더 잘 제어할 수 있게 되었습니다.
