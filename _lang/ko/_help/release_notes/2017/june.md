# 2017년 6월

## 외부 API로 업데이트

외부 API를 업데이트하여 Apple 푸시용 collapse_id 필드를 지원하므로 고객에게 정확하게 메시지를 보낼 수 있습니다. [Braze 메시징 API][49]에 대해 자세히 알아보세요.

## 웹 푸시 알림으로 업데이트

웹 푸시 알림 미리 보기에 드롭다운과 올바른 이미지 높이 렌더링을 포함한 업데이트가 추가되어 웹 푸시 알림을 더 쉽게 테스트하고 보낼 수 있습니다. [웹 푸시에][47] 대해 자세히 알아보세요.

## 보안 설정 개선

고객이 비밀번호 복잡도 규칙, 대시보드 IP 화이트리스트, 2단계 인증을 설정할 수 있는 기능을 추가하여 더 많은 제어 기능을 제공합니다. [보안 설정][46]에 대해 자세히 알아보세요.

## 동시 CSV 가져오기

이제 여러 개의 CSV를 업로드하고 동시에 처리할 수 있으므로 더 빠르게 작업할 수 있습니다. [사용자 가져오기][50]에 대해 자세히 알아보세요.

## 캔버스 템플릿 속성

캔버스 단계에서 보낸 메시지에 [템플릿 속성][45]을 포함할 수 있는 기능이 추가되어 더욱 효율적으로 작업할 수 있습니다.

## 타겟팅 업데이트

이제 날짜와 시간을 필터링할 때 시간 단위를 선택할 수 있어 보다 정확하게 메시지를 타겟팅할 수 있습니다. [필터][44]에 대해 자세히 알아보세요.
