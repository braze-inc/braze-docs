# 2017년 2월

## 출시 후 캔버스 단계 추가

이제 캔버스가 시작된 후에도 기존 단계에 [캔버스 단계를 추가][23]할 수 있습니다.

## 대량 CSV 참여 보고서용

이제 [참여 보고서][22]를 사용하여 캠페인 데이터 CSV를 대량으로 내보낼 수 있습니다. 캔버스 데이터는 참여 보고서를 통해서도 내보낼 수 있습니다.

## 마지막으로 받은 인앱 메시지로 필터

더 개인화된 사용자 타겟팅을 위해 이제 "마지막으로 받은 인앱 메시지"를 시간 기반 필터로 사용할 수 있습니다. Braze를 사용한 사용자 타겟팅에 대해 자세히 알아보려면 [세분화][21]를 방문하세요.

## 이벤트 사용자 로그 업그레이드

이제 업그레이드된 [이벤트 사용자 로그][20]를 통해 SDK 통합에서 무슨 일이 일어나고 있는지 쉽게 확인할 수 있습니다.

## 뉴스피드 템플릿

이제 {% raw %} {{card.${name}}} {% endraw %} 및 {% raw %}  {{card.${api_id}}}{% endraw %} 두 개의 뉴스피드 카드 필드가 있습니다. 이 카드들은 Liquid 템플릿에 추가될 수 있습니다. [개인화된 메시징 및 템플릿][19]에 대해 자세히 알아보세요.
