# 2017년 9월

## 참여 보고서의 새로운 기능

이제 [참여 보고서를][72] 사용하여 특정 기간 동안 캠페인에 대한 지표를 집계할 수 있습니다. 예를 들어, 분기별 총 오픈 횟수 또는 캠페인 또는 캔버스의 전체 기간 동안의 총 클릭 횟수를 내보낼 수 있습니다. 여러분이 해야 할 일은 간단합니다:
- 데이터를 내보낼 기간을 선택합니다,
- 한 명 이상의 수신자에게 정기적으로 전송하는 인게이지먼트 보고서를 예약합니다.
- 태그를 기반으로 캠페인과 캔버스를 보고서에 추가합니다.

## 사용자 프로필 페이지 업데이트

[사용자 프로필 페이지가][73] 업데이트되었습니다.

## 해제하려면 사용자 조치가 필요한 웹 푸시 알림

이제 Chrome 웹 푸시에 대해 메시지 닫기 동작을 설정하여 수신자가 메시지와 상호 작용해야만 메시지가 해제되도록 할 수 있습니다. 이 기능을 사용하려면 웹 SDK 버전 1.6.13 이상이 필요합니다.

## 이메일 프리헤더

이제 Braze 내에서 이메일 메시지를 작성할 때, **정보 보내기** 섹션에 프리헤더를 쉽게 삽입할 수 있습니다.

## 원시 이벤트 내보내기를 위한 새로운 API 엔드포인트

원시 이벤트 내보내기에 특정 날짜가 로드되었는지 쿼리할 수 있는 새로운 [API 엔드포인트인][71]/raw_data/status를 추가했습니다. 특정 날짜의 원시 데이터를 사용할 수 있는지 확인하여 디버깅 및 자동화를 지원하는 데 사용할 수 있습니다.
