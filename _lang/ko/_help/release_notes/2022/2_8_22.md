# 2022년 2월 8일 출시

## 캔버스 작업 경로

[작업 경로]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/action_paths/)를 사용하면 사용자의 작업에 따라 사용자를 정렬할 수 있습니다. 작업 경로로 가능한 작업:
- 특정 작업을 기반으로 사용자 경로를 사용자 지정합니다. 
- 이 평가 기간 동안 사용자의 행동에 따라 다음 경로의 우선순위를 정하기 위해 주어진 기간 동안 사용자를 보류합니다. 

액션 경로에 대한 액세스는 모든 Braze 인스턴스에 천천히 적용될 예정입니다. 자세한 내용은 Braze 계정 관리자에게 문의하세요.

## API 엔드포인트 참조 페이지

사용 가능한 모든 Braze 엔드포인트에 대한 빠른 참조 페이지를 찾고 계신가요? [API 엔드포인트 색인을]({{site.baseurl}}/api/endpoints/) 방문하세요.

## Transifex v2 및 v2.5 API 사용 중단
[트랜시펙스는]({{site.baseurl}}/partners/message_personalization/localization/transifex/#transifex) 2022년 4월 7일부터 API 버전 2와 2.5를 더 이상 지원하지 않고 버전 3으로 전환할 예정입니다. 이 날짜 이후에는 v2 및 v2.5가 더 이상 작동하지 않으며 관련 요청이 실패합니다. 이 파트너 통합의 사용자는 이 날짜 이전에 API 버전을 업데이트하시기 바랍니다.

## 사용 가능한 사용자 지정 속성 데이터 유형
곧 출시됩니다! 커스텀 속성에 대한 오브젝트 및 오브젝트 배열 데이터 유형 지원은 2022년 봄에 제공될 예정입니다.
