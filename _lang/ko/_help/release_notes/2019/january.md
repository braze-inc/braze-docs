# 2019년 1월

새해를 맞이한 것을 환영합니다!

## 푸시 타임 투 라이브(TTL)

계정의 **설정 관리에서** [푸시 TTL 설정]({{site.baseurl}}/user_guide/administrative/app_settings/push_settings/) 탭을 클릭하여 디바이스가 오프라인 상태일 때 재전송을 시도하는 시간을 관리할 수 있습니다.

## 커넥티드 콘텐츠 IP 화이트리스트

Braze는 내부 팀의 인프라 업그레이드를 통해 이제 모든 클러스터에서 연결된 콘텐츠에 대한 IP 화이트리스트를 제공할 수 있게 되었음을 알려드리게 되어 기쁩니다. 향후에는 비유럽연합 클러스터를 위한 IP를 추가할 계획입니다.
