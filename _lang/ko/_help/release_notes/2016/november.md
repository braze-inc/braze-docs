# 2016년 11월

## 연결된 오디언스

연결된 오디언스는 Braze에서 사전 구성 없이 사용자 기반의 하위 집합을 사용하여 메시지를 즉시 타겟팅할 수 있도록 합니다. 예를 들어, 미디어 앱은 고객이 즐겨찾는 모든 카테고리나 TV 프로그램을 추적하기 위해 커스텀 속성 배열을 사용할 수 있습니다. 연결된 오디언스를 사용하면 커스텀 CMS를 Braze와 연결하여 새로운 에피소드가 출시될 때마다 쇼를 즐겨찾기한 사람에게 자동으로 푸시 알림 또는 이메일을 보낼 수 있습니다.

[연결된 오디언스 객체]({{site.baseurl}}/api/objects_filters/connected_audience/)를 참조하거나 [연결된 오디언스](https://www.braze.com/blog/connected-audiences/)에 대한 블로그를 확인하세요.
