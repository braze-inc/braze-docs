# 커스텀 속성 또는 이벤트 데이터 유형 변경

커스텀 속성 또는 이벤트의 데이터 유형을 변경하려면 Braze 대시보드에서 **데이터 설정**으로 이동한 다음 **커스텀 속성** 또는 **커스텀 이벤트**을 선택하십시오.

{% alert note %}
[이전 탐색]({{site.baseurl}}/navigation)을 사용하는 경우 **커스텀 이벤트** 및 **커스텀 속성** 페이지는 **설정 관리** 아래에 있습니다.
{% endalert %}

![커스텀 속성 탭에서 속성 또는 데이터 유형을 편집][1]

커스텀 속성 또는 이벤트의 데이터 유형을 변경해야 하는 경우(예: `time`을 `string`으로 변경), 다음을 고려합니다.

- 변경된 속성 또는 이벤트를 사용하는 세그먼트, 캠페인, 캔버스 또는 기타 위치의 관련 필터는 자동으로 업데이트되지 않습니다. 속성을 수정하기 전에 속성을 세그먼트나 필터에서 사용하는 캠페인이나 캔버스를 중지하고, 속성을 참조하는 필터에서 속성을 제거해야 합니다.
- 사용자 데이터는 소급하여 업데이트되지 않습니다. 변경된 속성이 데이터 유형 변경 전에 고객 프로필에 있었다면, 해당 값은 여전히 이전 데이터 유형일 것입니다. 이로 인해 변경된 속성을 포함하고 있는 세그먼트에서 사용자가 제외될 수도 있습니다. 필터는 새 데이터 유형을 찾아 나서게 되지만, 프로필에 여전히 이전 데이터 유형이 있는 경우라면, 해당 사용자는 지금부터는 세그먼트에서 제외됩니다. 이러한 사용자를 적절한 세그먼트에 다시 포함하려면 먼저 업데이트가 필요합니다. [`users/track` 엔드포인트]({{site.baseurl}}/api/endpoints/user_data/post_user_track/)를 사용하여 이 작업을 수행할 수 있습니다.
- 새로운 데이터는 새로운 데이터 유형이 아니면 허용되지 않습니다. 예를 들어, 변경된 속성에 대한 이전 데이터 유형을 포함하는 `users/track` 엔드포인트에 대한 API 호출은 허용되지 않습니다. 새로운 데이터 유형을 호출해야 합니다.

{% alert important %}
자동 감지가 커스텀 속성 데이터 유형을 업데이트하지 못하도록 방지하는 기능은 현재 초기 액세스 중입니다. 이 얼리 액세스에 참여하려면 고객 성공 매니저에게 문의하세요.
{% endalert %}

_마지막 업데이트 날짜: 2024년 2월 8일_
