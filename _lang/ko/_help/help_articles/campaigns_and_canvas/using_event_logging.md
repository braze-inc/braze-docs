# 이벤트 로깅 사용

Braze 통합 관련 문제를 해결하는 데 도움이 되도록 익명 사용자 프로필과 [이벤트 사용자 로그]({{site.baseurl}}/user_guide/administrative/app_settings/developer_console/event_user_log_tab)를 설정할 수 있습니다. 익명 프로필을 설정하는 방법에 대한 단계는 [테스트 사용자 추가하기]({{site.baseurl}}/user_guide/administrative/app_settings/developer_console/internal_groups_tab/#adding-test-users)를 참조하세요.

## 로깅 정보

이벤트 로깅을 사용하여 익명 사용자의 동작을 테스트하세요. 이는 테스트 중인 앱이 이메일을 수집하지 않는 경우 사용자 ID를 식별하는 데 특히 유용할 수 있습니다. Braze와 디바이스의 IP 주소를 사용하여 해당 디바이스를 테스트 사용자로 추가할 수 있습니다.

익명의 사용자를 찾을 수 있는 좋은 방법입니다. 또한 이 정보를 사용하여 Braze로 전송되는 데이터를 테스트하고 불일치 여부를 확인할 수 있습니다. 이 보기에서 데이터의 델타가 Braze로 전송되는지 여부를 확인할 수 있습니다. 이벤트가 기록될 때마다 이메일 주소 또는 푸시 토큰이 전송되면 모든 데이터가 Braze로 전송됩니다.

아직도 도움이 필요하신가요? [지원 티켓]({{site.baseurl}}/braze_support/)을 여세요.
