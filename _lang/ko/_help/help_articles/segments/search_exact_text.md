# 정확한 텍스트 검색

특정 항목을 검색하면 검색어가 포함된 모든 결과를 표시하기 때문에 Braze에서 가끔 너무 많은 결과가 표시될 수 있습니다.

정확한 텍스트를 검색하려면 검색어를 `[ ]`대괄호로 묶으세요.

예를 들어 **세그먼트** 페이지에서 `[Purchase]` 을 검색하면 "구매"라는 단어가 포함된 세그먼트는 표시되지만 "구매자" 또는 "구매"는 표시되지 않습니다.
