### 코드 샘플

```html
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
```

코드 샘플에서 사용자가 조정해야 할 수 있는 매개변수나 요소를 정의해야 합니다. 많은 사용자가 복사하여 붙여넣기만 합니다.
