# Wunderkind

> [Wunderkind](https://www.wunderkind.co)는 고성능 1:1 이메일 전송을 통해 이커머스 매출을 확장하는 플랫폼입니다. Wunderkind의 식별 기술을 사용하면 브랜드가 여러 기기에서 더 많은 익명 사용자를 인식할 수 있으며, 실행 가능한 이메일 주소까지 파악할 수 있습니다. Wunderkind는 평균적으로 웹사이트 트래픽의 3~5%에서 최대 40~60%까지 식별 비율을 확장합니다.

Wunderkind와 Braze의 통합을 통해 성능 향상을 분석하고 더 많은 익명 사용자를 식별할 수 있으며, Braze를 통해 전송되는 일대일 메시지와 Braze에 직접 추가된 연락처를 크게 확장할 수 있습니다.

## 통합

통합 설정에 도움을 받으려면 WunderKind 팀에 문의하세요. Wunderkind에서 API를 통해 이메일 배포 및 가입 상태를 보낼 수 있도록 Braze ESP 또는 API 자격 증명을 제공할 것을 사용자에게 요청합니다. 

연결되면 Wunderkind에서 다음을 수행합니다.
- 예비 테스트를 수행하여 안정적인 연결을 확인하고 Wunderkind 측에서 ESP로 이메일이 배포될 수 있는지 확인합니다. 
- 시리즈의 각 부분에 대해 Braze UI 내에서 개별 캠페인을 생성하여 렌더링된 이메일을 전송할 수 있습니다.
- 적절한 API 엔드포인트를 사용하여 Braze를 통해 렌더링된 이메일 콘텐츠를 실시간으로 전송하기 위한 통합을 개발합니다.
- 테스트 환경에서 광범위한 품질 보증을 수행하여 이메일이 올바르게 전달되고 데이터가 예상대로 Braze에 유입되는지 확인합니다.
