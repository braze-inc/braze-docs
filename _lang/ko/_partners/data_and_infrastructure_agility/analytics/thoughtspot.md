# ThoughtSpot

> ThoughtSpot은 최신 분석 클라우드이자, 최신 데이터 스택에 실시간 분석을 제공하는 차세대 분석 플랫폼으로, 이를 통해 동료, 파트너 및 고객이 데이터를 유용한 인사이트로 전환할 수 있습니다.

Braze와 ThoughtSpot 통합은 Braze 사용자가 워크시트 및 모델의 사전 구축된 템플릿을 사용하여 사용자 행동 분석을 가속화할 수 있도록 하는 ThoughtSpot TML 블록을 활용합니다. 이 통합을 통해 사용자는 Braze 상호 작용 데이터 전반을 무제한으로 검색하고 유용한 인사이트를 발견할 수 있습니다. 

## 필수 조건

Braze에서 ThoughtSpot을 사용하기 시작하려는 경우 데이터를 클라우드 데이터 웨어하우스로 전송해야 ThoughtSpot이 실시간 쿼리를 수행할 수 있습니다.

| 요구 사항 | 설명 |
| ----------- | ----------- |
| ThoughtSpot 계정 | 이 파트너십을 활용하려면 ThoughtSpot 계정이 필요합니다. |
| 클라우드 데이터 웨어하우스| Braze 데이터는 Braze 커런츠를 사용하여 클라우드 데이터 웨어하우스에 저장됩니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## TML 블록

Braze 사용자는 모든 디지털 상호 작용 데이터를 쉽게 액세스하고 검색할 수 있습니다. 우리의 템플릿은 사용자가 미리 만들어진 시각화 및 워크시트를 통해 신속하게 분석을 설정할 수 있도록 합니다. 검색, 드릴다운 및 spotIQ를 통해 웹사이트의 유입 및 사용자 행동을 분석하세요.

## 통합

### 1단계: ThoughtSpot 연결 

ThoughtSpot 인스턴스에 로그인하고 Braze 커런츠를 사용하여 Braze에서 가져온 각 테이블에 Embrace 연결을 만드십시오.

#### 2단계: TML 가져오기

Thoughtspot에 워크시트 및 라이브보드를 위한 압축 파일을 가져와 오류 없이 가져왔는지 확인하십시오. 

가져온 후에는 라이브보드를 검색하고 사용자 지정할 수 있습니다.
