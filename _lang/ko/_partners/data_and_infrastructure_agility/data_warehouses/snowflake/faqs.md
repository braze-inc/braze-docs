# 자주 묻는 질문

### Snowflake 데이터 공유를 사용할 때 PII 데이터 난독화가 가능한가요?
아니요, 현재로서는 지원되지 않습니다.

### 같은 지역 또는 다른 지역에 대한 데이터 공유가 필요한가요?
다음 시나리오에서는 동일한 지역에 대한 데이터 공유를 사용합니다:
- Snowflake 계정이 US-EAST-1(AWS)에 있고 Braze 대시보드 리전이 미국에 있습니다.
- Snowflake 리전은 EU-CENTRAL-1에 있고 Braze 대시보드 지역이 EU에 속해 있습니다.

그렇지 않으면 리전 간 데이터 공유를 사용합니다. 

### 새 Snowflake 계정으로 전환할 때 데이터 공유는 어떻게 해야 하나요?
기존 Snowflake 계정과 연결된 기존 데이터 공유를 삭제한 다음, 새 계정에 대한 새 공유를 생성할 수 있습니다. 모든 기록 데이터는 새 공유에서 사용할 수 있습니다. 

### 데이터 공유에 데이터가 표시되지 않는 이유는 무엇인가요?
데이터 공유를 생성할 때 잘못된 Snowflake 계정 ID를 사용했을 수 있습니다. 데이터 공유 대시보드의 계정 ID는 Snowflake 계정에서 `CURRENT_ACCOUNT()`의 출력과 일치해야 합니다.

공유가 여러 리전에 걸쳐 있는 경우 데이터를 즉시 사용하지 못할 수도 있습니다. 데이터 양에 따라 데이터가 해당 리전과 동기화되는 데 몇 시간이 걸릴 수 있습니다.
