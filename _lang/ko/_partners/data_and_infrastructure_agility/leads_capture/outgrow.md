# 아웃그로우

> [Outgrow는](https://outgrow.co/) 퀴즈, 계산기, 설문조사 및 기타 유형의 매력적인 콘텐츠를 제작하여 사용자 데이터와 인사이트를 수집할 수 있는 대화형 콘텐츠 플랫폼입니다. 브레이즈와 아웃그로우의 통합을 통해 아웃그로우의 사용자 데이터를 자동으로 브레이즈로 전송하여 고도로 개인화된 타겟팅 캠페인을 진행할 수 있습니다.

인터랙티브 콘텐츠에 Braze와 Outgrow 통합을 사용하면 다음과 같은 이점이 있습니다:

- **향상된 개인화**: Braze의 사용자 지정 속성에 매핑할 수 있는 Outgrow 퀴즈, 설문조사, 계산기에서 데이터를 수집하세요. 이 데이터를 통해 정밀한 세분화와 개인화된 캠페인을 진행할 수 있습니다.
- **실시간 데이터 동기화**: Braze에서 아웃그로우 데이터를 실시간으로 수신하여 사용자 인사이트에 따라 즉시 조치를 취할 수 있습니다. 이를 통해 사용자의 가장 최근 상호 작용을 기반으로 적시에 후속 조치를 취하거나 개인화된 메시지를 보낼 수 있습니다.
- **간소화된 데이터 관리**: Outgrow와 Braze 간의 데이터 전송을 자동화하여 수동 데이터 내보내기 및 가져오기를 없애고 데이터 불일치를 줄이며 시간을 절약하세요.
- **사용자 경험 개선**: 사용자 인사이트를 활용하여 더욱 관련성 높은 경험을 만들어 만족도, 리텐션, 평생 가치를 높일 수 있습니다.
- **유연한 타겟팅 및 세분화**: 아웃그로우 데이터를 사용하여 Braze에서 세분화하여 특정 상호작용(예: 퀴즈 점수 또는 설문조사 응답)을 기반으로 사용자를 타겟팅하여 사용자의 공감을 이끌어내는 캠페인을 만들 수 있습니다.

## 필수 조건

아웃그로우와 브레이즈 연동을 설정하기 전에 다음 사항이 있는지 확인하세요:

| 요구 사항 | 설명 |
|-------------|-------------|
| **계정 성장** | 대화형 콘텐츠 및 데이터 전송 설정을 구성하고 관리하기 위해 등록된 Outgrow 계정 |
| **브레이즈 계정** | REST API 자격 증명에 액세스할 수 있는 Braze 계정 |
| **API 키** | 사용자 데이터 전송을 활성화할 수 있는 `users.track` 권한이 있는 Braze의 API 키 |
| **Braze의 사용자 지정 속성** | 퀴즈 점수, 세그먼트 등 아웃그로우 응답을 캡처하기 위해 Braze에서 설정한 사용자 지정 속성(예: 퀴즈 점수, 세그먼트 등) |

## 통합

다음 단계에 따라 Braze와 Outgrow 통합을 구성하세요:

### 1단계: Braze API 키 생성

1. Braze 계정에서 **개발자 콘솔** > **API 설정으로** 이동합니다.
2. **새 API 키 생성을** 선택합니다.
3. API 키의 이름을 지정하고 `users.track` 권한을 켜고 API 키를 저장합니다.

### 2단계: Outgrow에서 Braze 통합 구성하기

1. Outgrow 계정에 로그인합니다.
2. 대시보드에서 **통합으로** 이동합니다.
3. 사용 가능한 연동 목록에서 **Braze를** 선택합니다.
4. **Braze API 키와** **REST API 엔드포인트 URL을** 입력합니다:
   - **API 키**: Braze에서 생성한 API 키를 입력합니다.
   - **REST 엔드포인트 URL**: Braze 인스턴스의 엔드포인트를 입력합니다(예: `https://rest.iad-01.braze.com`).
5. **저장을** 선택하여 통합을 켭니다.

### 3단계: Outgrow 데이터를 Braze 속성에 매핑하기

Outgrow에서는 퀴즈 결과, 사용자 지정 세그먼트 또는 참여 점수 등의 대화형 콘텐츠의 응답을 Braze 사용자 지정 속성에 매핑할 수 있습니다.

1. 브레이즈에 대한 아웃그로우 **통합 설정에서** 어떤 아웃그로우 응답을 브레이즈 속성에 매핑할지 정의합니다.
2. 선택한 각 응답이 Braze의 사용자 지정 속성과 일치하는지 확인하세요. For example:
   - 퀴즈 점수는 `outgrow_quiz_score` 로 이동합니다.
   - 사용자 지정 세그먼트는 `outgrow_custom_segment` 로 매핑됩니다.
3. 매핑 설정을 저장합니다.

### 4단계: 통합 테스트

통합을 구성한 후 테스트를 실행하여 데이터가 Outgrow에서 Braze로 제대로 전송되는지 확인합니다.

1. 퀴즈나 계산기와 같은 Outgrow 경험을 게시하고 테스트 사용자로 완료합니다.
2. Braze 계정에서 **사용자 프로필** 섹션으로 이동하여 업데이트된 속성을 확인합니다(예: `outgrow_quiz_score` 또는 `outgrow_custom_segment`).
3. 데이터가 적절한 사용자 지정 속성 아래에 올바르게 입력되었는지 확인합니다.

## 세분화 및 타겟팅을 위해 Braze에서 아웃그로우 데이터 사용하기

### 아웃그로우 데이터로 Braze에서 세그먼트 만들기

통합을 통해 아웃그로우 응답에서 채워진 사용자 지정 속성을 기반으로 Braze 세그먼트를 만들 수 있습니다.

1. Braze에서 **참여** > **세그먼트로** 이동하여 **새 세그먼트 만들기를** 선택합니다.
2. 세그먼트의 이름을 지정하고 아웃그로우 데이터를 기반으로 필터를 설정합니다. For example:
   - `outgrow_quiz_score` 으로 필터링하여 특정 임계값 이상의 점수를 받은 사용자를 타겟팅합니다.
   - `outgrow_custom_segment` 으로 필터링하여 특정 아웃그로우 정의 세그먼트에 속하는 사용자를 타겟팅합니다.
3. 캠페인과 캔버스에서 사용할 수 있도록 세그먼트를 저장하세요.

### 아웃그로우 정의 세그먼트로 캠페인 시작하기

아웃그로우 데이터에서 생성된 사용자 지정 세그먼트를 사용하여 Braze 캠페인을 개인화하고 대화형 콘텐츠에 대한 사용자의 반응을 기반으로 사용자를 타겟팅할 수 있습니다. 이렇게 하여 더욱 개인화된 사용자 환경을 만들려면 다음 단계를 따르세요:

1. Braze에서 **참여** > **캠페인으로** 이동합니다.
2. **캠페인 생성을** 선택하고 캠페인 유형(이메일, 푸시, 인앱 메시지 또는 기타)을 선택합니다.
3. 오디언스 타겟팅 단계에서 아웃그로우 속성에서 생성된 세그먼트(예: 특정 퀴즈 점수 또는 세그먼트가 있는 사용자)를 선택합니다.
4. 캠페인 콘텐츠와 설정을 사용자 지정한 다음 실행합니다.

## 일반적인 문제 해결

| 이슈 | 솔루션 |
|-------|----------|
| **데이터가 Braze로 전송되지 않습니다.** | Outgrow 연동 설정에서 API 키와 엔드포인트 URL이 올바른지 확인합니다. API 키에 `users.track` 권한이 켜져 있는지 확인합니다. |
| **잘못된 데이터 매핑** | 매핑된 각 아웃그로우 응답이 유효한 Braze 사용자 지정 속성에 해당하는지, 그리고 속성 이름이 정확히 일치하는지 확인하세요. |
| **세그먼트가 올바르게 필터링되지 않음** | Braze의 사용자 지정 속성이 제대로 설정되어 있고 데이터를 수신하고 있는지 확인하세요. 세그먼트 필터 로직을 다시 확인하세요. |

## 추가 고려 사항

- **데이터 개인정보 보호**: 플랫폼 간에 사용자 데이터를 전송할 때 데이터 개인정보 보호 규정(예: GDPR 및 CCPA)을 준수하세요.
- **요금 제한**: 아웃그로우 데이터는 실시간으로 Braze로 전송되지만, 대량의 데이터에는 Braze API 속도 제한이 적용될 수 있습니다. 트래픽이 많은 경험을 위해 적절히 계획하세요.
- **사용자 지정 속성 구성**: 이 통합에 사용된 Braze 사용자 지정 속성이 아웃그로우에서 전송된 데이터를 캡처하도록 올바르게 구성되었는지 확인합니다.

추가 지원이 필요한 경우 [Outgrow 설명서를](https://support.outgrow.co/docs/configuring-native-integration-between-outgrow-braze) 참조하거나 Outgrow 지원팀에 문의하세요.
