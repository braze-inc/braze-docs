# B.Layer

> [B.Layer](https://blayer.phiture.com) 는 모바일 앱의 CRM 팀이 코딩 없이도 간단하고 빠르게 맞춤형 디자인의 인앱 메시지를 만들 수 있도록 도와주는 Phiture의 인앱 메시지 빌더입니다. 

Braze와 B.Layer의 통합을 통해 B.Layer 인앱 메시지 빌더를 활용하여 zip 파일 또는 인라인 HTML로 Braze에 내보낼 수 있는 온브랜드 인앱 메시지를 작성할 수 있습니다. 이 통합에는 추가적인 개발자 리소스가 필요하지 않으므로 시간과 예산을 절약할 수 있습니다.

![][2]

## 사용 사례

B.Layer에서는 제품 추천 슬라이더, 멀티스크린 온보딩 또는 설문조사, NPS, 이메일 캡처, 특별 오퍼 등 다양한 기능을 빌드하고 실험할 수 있는 여러 가능성을 제공합니다.

Lifesum, Blinkist, OnX Hunt 등의 브랜드와 협력하여 추가 리소스 없이도 사용자 경험을 개선할 수 있도록 지원하고 있습니다. 또한 앱 혁신 부문에서 APS 어워드 2022의 최종 후보에 올랐습니다.

## 통합

### 1단계: 인앱 메시지 생성

#### 브랜드 색상 및 글꼴 설정

B.Layer의 페이지 상단에 있는 햄버거 메뉴에서 **브랜드 자산 > 브랜드 자산 추가**를 클릭합니다. 여기에서 브랜드 색상과 글꼴을 지정할 수 있습니다.
모든 준비가 완료되었습니다. 이제 인앱 메시지 디자인을 시작할 수 있습니다.

![][4]

#### 인앱 메시지 디자인

인앱 메시지를 디자인하려면 단일 인앱 메시지를 선택합니다. 그런 다음 메시지 스타일을 지정하고 필요한 구성 요소를 추가합니다. 각 구성요소를 조정할 수 있습니다.

![][5]

### 인앱 메시지 다운로드

완료되면 메시지를 다운로드합니다. 메시지는 ZIP 또는 인라인 HTML로 다운로드할 수 있습니다. 

### 2단계: B.Layer 사용자 지정 코드 추가

Braze에서 사용자 지정 코드 인앱 메시지를 생성합니다. ZIP 파일이 있는 경우 코드 섹션 위의 상자에 파일을 끌어다 놓습니다. 인라인 HTML 파일이 있는 경우 인라인 HTML을 HTML 섹션에 붙여넣습니다.

![][6]

## 버튼 추적

B.Layer 를 사용하면 버튼 상호작용이나 텍스트 입력을 Braze 속성으로 기록할 수 있습니다. 편집기 내에서 이 작업을 수행할 수 있습니다. 대표적인 예로 NPS 설문조사를 들 수 있습니다.

B.Layer 는 사용자가 입력한 링크(예: `?button=0`)에 추가된 Braze 버튼 추적을 사용합니다. 이렇게 하면 캠페인의 분석 부분에서 버튼 클릭 수를 확인할 수 있습니다.
