# Voucherify

> [Voucherify](https://www.voucherify.io/)는 사용자 참여와 유지율을 높이는 개인화된 캠페인과 로열티 프로그램을 가능하게 하는 올인원 프로모션 플랫폼입니다. 

_이 통합은 Voucherify에 의해 유지됩니다._

## 통합 정보

<iframe src="https://player.vimeo.com/video/745340934?h=17ceae8c3c" width="640" height="360" frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>

Braze와 Voucherify 통합을 통해 고유 코드를 사용하여 프로모션 캠페인을 성장시킬 수 있습니다:

- [연결된 콘텐츠]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_fetching_data_through_braze_connected_content): Braze 연결된 콘텐츠를 통해 Braze 캠페인에 고유 코드를 추가하세요. 이 기능을 사용하면 Voucherify 할인 쿠폰, 기프트 카드 캠페인, 로열티 카드 및 추천 코드를 사용할 수 있습니다.
- [커스텀 속성]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_distribution_with_braze_custom_attributes): 커스텀 속성을 통해 Voucherify 고유 쿠폰, 기프트 카드, 로열티 카드 및 추천 코드를 Braze의 고객 프로필에 할당할 수 있습니다. 결과적으로 이메일 캠페인에서 첨부된 코드와 속성을 보내고 사용자와 공유할 수 있습니다.
- [프로모션 코드 목록]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_using_braze_promotion_codes_list): Voucherify에서 생성된 프로모션 코드를 사용하고 Braze에 업로드하세요.

## 필수 조건

| 요구 사항 | 설명 |
| ----------- | ----------- |
|Voucherify 계정 | 이 파트너십을 활용하려면 Voucherify 계정이 필요합니다. |
| Braze REST API 키 | `users.track` 권한이 있는 Braze REST API 키. <br><br> Braze 대시보드의 **설정** > **API 키**에서 생성할 수 있습니다. |
