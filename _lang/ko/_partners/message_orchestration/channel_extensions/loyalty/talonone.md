# Talon.One

> [Talon.One](https://talon.one/) 는 모바일 마케팅 CRM을 위한 맞춤형 인센티브를 제공하며 상황에 맞는 1대1 쿠폰, 추천, 할인, 로열티 캠페인을 빠르고 효율적으로 시작할 수 있습니다.

Braze와 Talon.One 통합을 사용하면 Talon.One 에서 생성한 코드를 Braze 커넥티드 콘텐츠를 통해 잠재고객에게 전송하여 로열티 또는 쿠폰 프로그램을 한 단계 더 발전시킬 수 있습니다.


## 필수 조건

| 요구 사항 | 설명 |
| ----------- | ----------- |
|Talon.One 계정 | 이 파트너십을 이용하려면 Talon.One 계정이 필요합니다. |
|Talon.One API 키 | Talon.One 에서 **설정** > **개발자 설정에서** 통합을 위한 Braze 타사 API 키를 생성합니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

{% alert warning %}
Talon.One에서는 분당 최대 2,500개의 메시지 사용량 제한을 **_요구_**합니다. 이 사용량 제한은 Braze 대시보드에서 [수정]({{site.baseurl}}/user_guide/engagement_tools/campaigns/testing_and_more/rate-limiting/#delivery-speed-rate-limiting)할 수 있습니다.
{% endalert %}

## 통합

1. 통합 설정 방법, Talon.One API 쿠폰 엔드포인트 사용 방법, Braze 메시지에 필요한 연결된 콘텐츠 템플릿을 찾을 수 있는 위치에 대한 지침은 [Talon.One 설명서](https://docs.talon.one/docs/dev/technology-partners/braze)를 참조하세요.
2. 로열티 포인트 및 추천 등 Talon.One에서 제공하는 다른 기능을 활용하려면 다음 문서를 참조하세요.
  - [Braze 로열티 포인트 추가](https://docs.talon.one/docs/dev/technology-partners/braze/adding-loyalty-points-braze)
  - [Braze에서 로열티 원장 가져오기](https://docs.talon.one/docs/dev/technology-partners/braze/receiving-loyalty-ledger-braze)
  - [Braze를 통해 쿠폰 생성](https://docs.talon.one/docs/dev/technology-partners/braze/creating-coupons-braze)
  - [Braze를 통해 추천 생성](https://docs.talon.one/docs/dev/technology-partners/braze/creating-referrals-braze)
  - [Braze를 통해 생일 프로모션 만들기](https://docs.talon.one/docs/dev/technology-partners/braze/bday-promotion-braze)
