# Survicate

![Braze 이메일에서 임베드된 HTML 설문조사(첫 번째 질문)의 예제.][2]{: style="float:right;max-width:40%;border:0; margin-left:8px;"}

> [Survicate][1]는 순고객추천지수 및 고객 피드백 설문조사 소프트웨어로, 퍼널의 모든 단계에서 지속적인 사용자 인사이트를 얻을 수 있도록 도와줍니다. 

Braze와 Survicate 통합을 통해 이메일에 설문조사 링크를 포함하거나 설문조사 스니펫을 직접 삽입하여 응답률을 높일 수 있습니다. 설문조사가 완료되면 Survicate로 돌아와 설문조사 응답자의 속성과 응답을 식별하고 분석합니다.

## 필수 조건

| 요구 사항 | 설명 |
| ----------- | ----------- |
| Survicate 계정 | 이 파트너십을 활용하려면 Survicate 계정이 필요합니다. |

## 사용 사례

Survicate는 더 현명한 결정을 내리고 더 나은 고객 경험을 제공하기 위해 유용한 인사이트를 발견하는 데 도움이 되는 다양한 사용 사례를 다룰 수 있습니다:
- 고객 만족 측정
- 순추천고객지수
- 제품 피드백 수집
- 사용자 또는 시장 조사를 수행하다

## 통합

Survicate를 사용하면 Braze에서 몇 분 안에 피드백 수집을 시작할 수 있으며, 이를 가동 및 실행하는 데 기술적인 기술이 필요하지 않습니다.

### 1단계: Survicate에서 설문조사를 작성하세요

Survicate 계정에 로그인하고 **새 설문조사 만들기** 버튼을 클릭하세요. **이메일 또는 공유 가능한 링크 설문조사** 유형을 선택하고, 설문조사를 처음부터 디자인하거나 100개 이상의 사용 가능한 설문조사 템플릿 중에서 선택하세요.

### 2단계: 설문조사를 Braze에 연결하세요

설문조사가 준비되면 설문조사 작성기에서 **설정** 탭으로 이동하여 **응답자 식별** 필드 아래 도구 드롭다운 목록에서 **Braze**를 선택합니다. 이제 응답자에게 연락처를 물어볼 필요가 없습니다.

![][3]{: style="max-width:70%;" }

### 3단계: 설문조사를 공유하세요

**공유** 탭으로 이동하여 설문조사 배치의 선호 옵션을 선택하세요. 직접 링크를 얻거나(나중에 Braze에서 버튼 또는 하이퍼링크 아래에 숨길 수 있음) HTML 코드를 사용하여 이메일 본문에 첫 번째 질문을 삽입할 수 있습니다.

![][4]{: style="max-width:70%;" }

### 4단계: Braze에서 설문조사를 이메일 캠페인에 추가하세요

Braze에서 설문조사의 링크 또는 HTML 코드를 이메일 캠페인의 콘텐츠에 추가하세요. 고객 설문 조사를 시작하고 Survicate에서 결과를 분석하세요!
