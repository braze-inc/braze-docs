# Iterate

> [Iterate](https://iteratehq.com)는 브랜드에 맞춘 디자인과 느낌의 사용자 친화적인 지능형 리서치 툴을 제공하여 고객에 대해 더 쉽게 파악할 수 있도록 합니다.

_This integration is maintained by Iterate._

## 통합 정보

Iterate와 Braze의 통합을 통해 제품 또는 캠페인 내에서 Iterate 설문조사를 기본적으로 원활하게 제공할 수 있습니다. 설문조사 응답은 Braze에서 커스텀 사용자 속성으로 기록할 수 있으므로 사용자에 대한 완전한 이해 기반을 구축하거나 강력한 새로운 오디언스와 세그먼트를 생성할 수 있습니다.

앱이나 웹사이트에 Braze SDK를 설치하면 Braze에서 제공되는 세분화 및 타겟팅 도구를 사용하여 트리거 또는 사용자 지정 세그먼트에 따라 특정 오디언스에게 인앱 메시지를 통해 설문조사를 전달할 수 있습니다. Iterate 설문조사는 이메일 캠페인에 직접 삽입하거나 푸시 또는 기타 캠페인 유형에 링크로 포함할 수도 있습니다.

## 사용 사례

Iterate를 사용하면 거의 모든 유형의 데이터를 수집할 수 있습니다. 개인 정보(이름, 나이, 이메일), 성과 데이터(NPS, 고객 만족도, 별점), 선호도(선호하는 기기, 선호하는 커뮤니케이션 빈도), 성격(좋아하는 책, 개, 고양이 등)에 이르기까지 다양한 정보를 수집할 수 있습니다. 어떤 종류의 데이터를 수집할 것인지, 어떤 오디언스를 구축할 것인지는 전적으로 사용자에게 달려 있습니다.

## 통합

### 시작하기: Braze와 Iterate 연결

Iterate 계정에 로그인하고 **회사 설정** 페이지에서 Braze REST 엔드포인트와 REST API 키를 추가합니다.

### 인앱 메시지로 설문조사 제공

#### 1단계: 설문조사 만들기

설문조사를 생성하기 전에 Iterate 설정에서 **인앱 메시지 설문조사 사용** 토글을 켭니다.

다음으로, Iterate에서 새 설문조사를 생성하고 관련 설문조사 질문을 추가합니다. 적절한 경우 설문조사 전에 표시할 안내 메시지를 포함할 수도 있습니다. 설문조사 유형으로 **Braze 인앱 메시지를 통해 보내기를** 선택합니다.

설문조사가 완료되면 **게시** 탭에서 복사 아래의 코드 스니펫을 **복사하여 임베드 코드를 붙여넣습니다**.

#### 2단계: 설문조사 공유

Braze에서 새 인앱 메시징 캠페인을 생성하고 메시징 유형으로 **커스텀 코드**를 선택한 다음, 코드 스니펫을 메시지에 붙여넣습니다. 그런 다음, 클릭 시 메시지 행동으로 **사용자가 해제할 때까지 대기**를 선택합니다.

다른 인앱 메시징 캠페인과 마찬가지로 캠페인을 계속 설정하고, 전달 방법을 선택한 후 오디언스를 타겟팅합니다.

### 이메일 또는 푸시를 통해 설문조사 제공

#### 1단계: 설문조사 만들기

Iterate서 새 이메일 또는 링크 설문조사를 생성하고 관련 설문조사 질문을 추가합니다. 질문을 작성하고 디자인을 사용자 지정한 후 **설문조사 전송 > 통합 > Braze**를 선택합니다.

그러면 Braze에 응답을 보내기 위한 구성 옵션이 표시됩니다. 통합을 켜면 해당 설문조사에 대한 응답을 Braze로 전송할 수 있습니다. 

#### 2단계: 설문조사 공유

설문조사는 두 가지 방법으로 공유할 수 있습니다. 첫 번째 질문을 메시지에 포함하거나 Iterate 플랫폼에서 설문조사에 대한 직접 링크를 포함시키는 것입니다.

![Iterate 링크 옵션][2]

- **코드 삽입**
  - **설문조사 보내기** 탭의 Braze 통합 섹션에 있는 **이메일 임베드 코드** 아래에 있는 코드 스니펫을 복사합니다. 설문조사의 시작 부분을 표시할 Braze 이메일의 HTML에 코드를 삽입합니다. 
  - 설문조사 문항을 렌더링하는 데 문제가 있거나 형식이 잘못 표시되는 경우 메시지 작성기의 **정보 보내기** 탭으로 이동하여 **인라인 CSS를** 선택 취소해야 합니다.
- **링크 포함**
  - **설문조사 보내기** 탭의 Braze 통합 섹션에 있는 **설문조사 링크** 아래에 있는 링크를 복사합니다. 링크({% raw %}`?user_braze_id={{${braze_id}}}`{% endraw %})에 포함된 Liquid는 전송 시 각 사용자에 대해 자동으로 교체됩니다.

### 다음 단계: 후속 캠페인 구축

사용자가 응답하면 프로필에 실시간 데이터가 채워집니다. 이 데이터는 사용자를 세분화하고 개인화된 후속 캠페인을 보내는 데 사용할 수 있습니다. 예를 들어 "우리 제품을 좋아하시나요?"라는 질문을 보낸 경우 "예" 또는 "아니요"라고 응답한 사용자 지정 사용자 속성 `Do you enjoy our products?` 을 가진 사용자 세그먼트를 만들고 이러한 사용자를 타겟팅할 수 있습니다.

## Braze 커스텀 이벤트

사용자가 설문조사 질문에 답하면 Iterate는 Braze 내에서 `survey-question-response`라는 이름의 커스텀 이벤트를 트리거합니다. 커스텀 이벤트를 통해 원하는 수와 유형의 후속 캠페인을 트리거할 수 있습니다.

## 사용자 속성 이름 사용자 지정

기본적으로 질문에 대해 생성된 사용자 속성은 프롬프트와 동일합니다.
경우에 따라 이를 사용자 지정하고 싶을 수도 있습니다. 이렇게 하려면 **설문조사 만들기** 단계에서 **사용자 속성 이름 사용자 지정** 드롭다운을 클릭하고 원하는 사용자 지정 이름을 입력합니다.
