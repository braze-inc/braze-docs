# 인류학

> [Anthropic은](https://www.anthropic.com/) 다양한 언어 작업에 유용하고 정직하며 안전하도록 구축된 차세대 AI 비서인 Claude를 개발하는 AI 안전 및 연구 회사입니다.

_이 통합은 Anthropic에서 유지 관리합니다._

## 통합 정보

Braze와 Anthropic 통합을 통해 Anthropic API 키를 Braze에 연결하여 커스텀 AI 에이전트를 구축할 때 Claude 모델을 사용할 수 있습니다. 이 통합을 통해 상담원은 개인화된 사본을 생성하고, 실시간으로 의사 결정을 내리거나, Anthropic의 Claude 모델을 사용하여 카탈로그 필드를 업데이트할 수 있습니다.

## 필수 조건

| Requirements | 설명 |
|---|---|
| API 키가 있는 익명 계정 | API 키가 있는 Anthropic 계정입니다. 도움이 필요하면 관리자 또는 [Anthropic 지원팀에](https://support.anthropic.com/) 문의하세요. |
| Braze instance | [API 개요 페이지]({{site.baseurl}}/api/basics/#endpoints) 또는 Braze 온보딩 매니저에서 Braze 인스턴스를 찾을 수 있습니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integration

Anthropic API 키를 Braze에 연결하려면:

1. Braze 대시보드에서 **파트너 통합** > **기술 파트너로** 이동하여 Anthropic을 찾습니다.
2. Anthropic에서 API 키를 입력합니다.
3. Select **Save**.

저장한 후 상담원 콘솔에서 [커스텀 상담원을 만들]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) 때 클로드 모델을 선택할 수 있습니다.

통합과 관련된 문제나 질문이 있는 경우 [Anthropic 지원팀에](https://support.anthropic.com/) 문의하세요.
