# 구글 제미니 자리

> [구글 제미니는](https://deepmind.google/technologies/gemini/) 텍스트, 코드, 이미지 전반에 걸쳐 진행된 추론을 결합하여 브랜드가 더욱 스마트하고 개인화된 경험을 제공할 수 있도록 지원하는 구글의 AI 모델 제품군입니다.

_이 통합은 Google에서 유지 관리합니다._

## 통합 정보

Braze와 Google Gemini 통합을 통해 Google Gemini API 키를 Braze에 연결하여 커스텀 AI 에이전트를 구축할 때 Gemini 모델을 사용할 수 있습니다. 이 통합을 통해 상담원은 개인화된 사본을 생성하고, 실시간 의사 결정을 내리거나, Google의 Gemini 모델을 사용하여 카탈로그 필드를 업데이트할 수 있습니다.

## 필수 조건

| Requirements | 설명 |
|---|---|
| Gemini API 키가 있는 Google 클라우드 계정 | Gemini API 키가 있는 Google Cloud 계정입니다. 도움이 필요하면 관리자 또는 [Google Cloud 지원팀에](https://cloud.google.com/support) 문의하세요. |
| Braze instance | [API 개요 페이지]({{site.baseurl}}/api/basics/#endpoints) 또는 Braze 온보딩 매니저에서 Braze 인스턴스를 찾을 수 있습니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integration

Google Gemini API 키를 Braze에 연결하려면:

1. Braze 대시보드에서 **파트너 통합** > **기술 파트너로** 이동하여 Google Gemini를 찾습니다.
2. Google에서 받은 API 키를 입력합니다.
3. Select **Save**.

저장 후 상담원 콘솔에서 [커스텀 상담원을 만들]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) 때 Gemini 모델을 선택할 수 있습니다.

통합과 관련된 문제나 질문이 있는 경우 [Google Cloud 지원팀에](https://cloud.google.com/support) 문의하세요.
