# OpenAI

> [OpenAI는](https://openai.com/) 자연어 이해 및 생성을 지원하는 GPT와 같은 고급 AI 모델을 생성하여 브랜드가 의미 있는 고객 인터랙션을 구축하고 확장할 수 있도록 지원합니다. 

_이 통합은 OpenAI에서 유지 관리합니다._

## 통합 정보

Braze와 OpenAI 통합을 통해 OpenAI API 키를 Braze에 연결하여 커스텀 AI 에이전트를 구축할 때 OpenAI 모델을 사용할 수 있습니다. 이 통합을 통해 상담원은 OpenAI의 대규모 언어 모델을 사용하여 개인화된 사본을 생성하고, 실시간 의사 결정을 내리거나, 카탈로그 필드를 업데이트할 수 있습니다.

## 필수 조건

| Requirements | 설명 |
|---|---|
| API 키가 있는 OpenAI 계정 | API 키가 있는 OpenAI 계정. 문제나 질문이 있는 경우 관리자 또는 [OpenAI 지원팀에](https://help.openai.com/) 문의하세요. |
| Braze instance | [API 개요 페이지]({{site.baseurl}}/api/basics/#endpoints) 또는 Braze 온보딩 매니저에서 Braze 인스턴스를 찾을 수 있습니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## Integration

OpenAI API 키를 Braze에 연결하려면:

1. Braze 대시보드에서 **파트너 통합** > **기술 파트너로** 이동하여 OpenAI를 찾습니다.
2. OpenAI에서 API 키를 입력합니다.
3. Select **Save**.

저장 후 상담원 콘솔에서 [커스텀 상담원을 만들]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) 때 OpenAI 모델을 선택할 수 있습니다.

통합과 관련된 문제나 질문이 있는 경우 [OpenAI 지원팀에](https://help.openai.com/) 문의하세요.
