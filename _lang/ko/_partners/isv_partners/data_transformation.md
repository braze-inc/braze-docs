# Braze 데이터 변환

> Braze [데이터 변환]({{site.baseurl}}/data_transformation/)은 파트너 플랫폼에서 웹훅을 수집하고, 이를 통해 고객은 Braze 고객 프로필의 속성, 이벤트, 구매 등 원하는 사용자 데이터로 해당 웹훅의 페이로드를 변환하는 매핑을 정의할 수 있습니다.

## 데이터 변환 기반 통합의 모습

데이터 변환 기능을 기반으로 하는 파트너 통합으로, 공개 설명서를 통해 고객과 공유되는 변환 코드 템플릿이 있을 수 있습니다.

상호 고객의 경우 다음과 같을 수 있습니다.

1. 플랫폼에 로그인하여 웹훅을 설정합니다.
2. Braze 팀과 협력하여 Braze 데이터 변환에 액세스하고 Braze 대시보드 내에서 새로운 변환을 생성합니다.
3. 변환에 의해 생성된 URL이 복사됩니다.
4. Braze로 돌아가서 복사한 변환 URL로 테스트 웹훅을 보냅니다.
5. Braze에서 변환 코드 템플릿을 복사하여 붙여넣습니다.
6. 변환을 지원합니다.
7. 활성화된 경우, 사용자들은 Braze 사용자 검색 도구를 통해 웹훅을 기반으로 사용자 프로필이 업데이트되었는지 확인하고 원하는 대로 변환 코드를 편집할 수 있습니다.

{% alert tip %}
변환 코드 예제를 구축할 때는 Braze로 전송되는 웹훅 유형별로 변환을 생성하는 것이 좋습니다.
{% endalert %}
