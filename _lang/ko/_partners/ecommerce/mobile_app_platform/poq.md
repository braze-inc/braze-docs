# Poq

[Poq](https://poqcommerce.com/)는 기업이 완전한 네이티브 iOS 및 Android 앱을 빠르게 출시, 관리, 확장할 수 있도록 지원하여 커머스를 촉진하고 브랜드 약속을 실현하는 고성능 모바일 경험을 제공합니다.

*이 통합은 Poq에서 유지 관리합니다.*

## 통합 소개

Poq와 Braze 통합을 사용하면 네이티브 모바일 앱 경험을 Braze의 고객 참여 플랫폼과 연결할 수 있습니다. Braze iOS 및 Android SDK를 사용하여 Braze가 Poq 기반 앱에 직접 통합되므로, 모바일에서 세분화, 개인화, 타겟 캠페인 전달이 가능합니다.

## 지원 항목

다음 채널 및 기능이 지원됩니다.

| 채널/기능 | 설명 |
| :---- | :---- |
| 푸시 알림 | 리치 푸시 알림을 포함하여 Braze를 통해 앱 사용자에게 타겟팅된 개인화된 푸시 알림을 전송합니다. |
| 인앱 메시지 | 네이티브 앱 경험 내에서 Braze 인앱 메시지를 표시하여 고객 여정의 적절한 순간에 사용자의 참여를 유도합니다. |
| 분석 및 이벤트 추적 | Poq는 전체 고객 여정을 포괄하는 종합적인 앱 이벤트 세트를 Braze로 전송합니다. 이 데이터는 Braze에서 세분화, 캠페인 트리거, 개인화를 지원합니다. |
| 콘텐츠 카드 및 메시지 받은편지함 | 앱 내에서 전용 메시지 받은편지함 화면, Poq의 동적 콘텐츠 페이지 빌더로 구축된 페이지, 또는 앱 전체의 사전 정의된 슬롯에 영구적인 Braze 콘텐츠 카드를 전달합니다. 콘텐츠 카드는 Braze에서 구성된 키-값 페어를 사용하여 필터링됩니다. |
| 배너 | Poq의 동적 콘텐츠 페이지 빌더로 구축된 페이지 또는 앱 전체의 사전 정의된 슬롯에 Braze 배너를 표시합니다. 배너 콘텐츠는 배치 ID를 사용하여 가져옵니다. |

## 필수 조건

이 통합을 사용하려면 다음이 필요합니다.

| 요구 사항 | 설명 |
| :---- | :---- |
| Poq 앱 | 이 통합을 활용하려면 Poq 앱이 필요합니다. |
| Braze 앱 API 키 | 각 플랫폼(iOS 및 Android)에 대한 Braze 앱 API 키가 필요합니다. Braze 대시보드에서 **설정** > **API 및 식별자** > **앱 식별자**로 이동하여 키를 얻을 수 있습니다. |
