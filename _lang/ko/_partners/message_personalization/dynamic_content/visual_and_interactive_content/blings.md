# 블링

> [Blings는](https://www.blings.io/) 차세대 개인화된 비디오 플랫폼으로, 채널 전반에 걸쳐 실시간 데이터 중심의 인터랙티브 비디오 경험을 대규모로 제공할 수 있도록 지원합니다.  

_이 통합은 Blings에서 유지 관리합니다._

## 필수 조건  

| Requirement     | 설명                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| 블링 계정  | 이 파트너십을 이용하려면 블링스 계정이 필요합니다.         |  

## Integration  

### 1단계: 블링스 HTML 스니펫 가져오기

{% tabs %}
{% tab Blings business and free plans %}

#### 블링 비즈니스 및 무료 요금제

블링스 앱에서 HTML 스니펫을 직접 찾아 복사하세요.

1. 선택한 MP5 프로젝트의 **연결** 탭으로 이동합니다.
2. Blings **연결** 페이지에서 해당 변수에 Braze Liquid 태그를 추가합니다. 태그는 HTML 스니펫의 값을 동적으로 채웁니다. 

![블링 HTML 스니펫.]({% image_buster /assets/img/blings/blings_connect_audience.png %}){: style="max-width:70%;"}  

{% endtab %}
{% tab Blings Enterprise plan %}

#### Blings 엔터프라이즈 요금제

블링스 담당자에게 HTML 스니펫을 요청하세요. 

{% endtab %}
{% endtabs %}

### 2단계: Create a Braze campaign  

Braze에서 새 이메일 또는 인앱 메시지 캠페인을 생성하고 Blings HTML 스니펫을 삽입합니다.

![Blings Braze 캠페인.]({% image_buster /assets/img/blings/blings_braze_campaign.png %})  

### 3단계: 테스트 및 출시  

Braze에서 캠페인을 미리 보고 개인화된 필드가 올바르게 채워지는지 확인합니다. 그런 다음 MP5 캠페인을 대규모로 배포하세요.

![Blings Braze 미리 보기.]({% image_buster /assets/img/blings/blings_braze_preview.png %}){: style="max-width:70%;"}
