# Playable

> [Playable][1]을 사용하면 Braze 이메일 캠페인에 자동 재생 비디오 콘텐츠를 추가할 수 있습니다.

_이 통합은 Playable에서 유지 관리합니다._

## 통합 정보

Braze와 Playable 통합을 통해 최고의 콘텐츠(고품질 비디오)를 최고의 오디언스(이메일)에게 전달하여 받은편지함 내에서 자동으로 재생되는 흥미로운 고품질 콘텐츠로 클릭률과 클릭 후 측정기준을 높일 수 있습니다.

## 필수 조건

| 요구 사항 | 설명 | 
| ----------- | ----------- |
| Playable 계정 | 이 파트너십을 활용하려면 Playable 계정이 필요합니다. 만약 Playable 계정이 없으시다면, [여기][signup]에서 가입하세요.
비디오 콘텐츠 | Playable에 비디오 파일을 업로드하거나 Facebook, Instagram, YouTube, X(이전의 Twitter), TikTok 등과 같은 웹사이트에서 비디오 URL을 제공합니다. |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 구현

### 1단계: Playable에 비디오 추가

Playable 플랫폼에서 비디오 파일을 업로드하거나 Facebook, Instagram, YouTube, X(이전의 Twitter), TikTok 등에서 비디오 URL을 제공하여 비디오를 추가합니다.

### 2단계: Playable에서 embed 코드 복사

업로드되면 Playable은 Braze 캠페인에 삽입하는 경우 열 때 자동 재생할 비디오를 이메일에서 임제드하는 코드를 생성합니다. 이메일을 열면 Playable 서버는 이메일 클라이언트, 기기, 화면 크기 및 네트워크 상태에 따라 비디오에 대한 최적의 버전을 제공합니다.

{% alert tip %}
동영상은 iPhone Mail, Gmail, Apple Mail, iOS용 Outlook, Android용 Outlook, Mac용 Outlook 및 Windows용 최신 버전의 Outlook 365를 포함하여 98%가 넘는 받은 편지함에서 자동 재생됩니다. 레거시 Windows용 Outlook 사용자에게는 대신 정적 이미지가 표시됩니다.
{% endalert %}

### 3단계: Braze에 임베드 코드를 붙여넣기

마지막으로, 코드를 Braze 이메일 캠페인에 붙여넣고, 이메일 캠페인의 디자인, 테스트 및 게시를 계속 진행합니다.
