# 자주 묻는 질문

> 이 문서에서는 오디언스 동기화에 대해 몇 가지 자주 묻는 질문에 대한 답변을 제공합니다.

### 오디언스 동기화 파트너 대시보드에 오디언스가 채워지는 데 얼마나 걸리나요? 

오디언스를 채우는 데 걸리는 시간은 특정 파트너에 따라 다릅니다.

모든 네트워크는 Braze의 요청을 처리하고 사용자를 매칭하려고 시도합니다. 이 과정은 일반적으로 6-48시간 정도 소요됩니다.

각 오디언스 동기화 파트너에 대한 설명서의 문제 해결 섹션에서 특정 시간 범위를 확인할 수 있습니다. 

### 오디언스 동기화에서 사용할 수 있는 퍼스트 파티 데이터 유형에는 어떤 것이 있나요?

각 파트너에 사용되는 특정 필드는 파트너의 요구 사항에 따라 다를 수 있습니다. 

예를 들어, Facebook에 오디언스 동기화를 구성할 경우 이메일, 전화, 이름, 성 등 다양한 퍼스트파티 필드를 사용할 수 있지만, Snapchat에서는 이메일, 전화 또는 모바일 광고주 ID 중 하나만 선택할 수 있습니다. 

동기화하도록 선택할 수 있는 사용자 필드는 Braze 표준 속성 및 모바일 광고 ID와 상관된다는 점에 유의하세요. SDK 또는 API를 통해 이 데이터를 적절하게 전달해야 합니다. 

### 내 데이터를 각 오디언스 동기화 파트너로 전송하도록 처리 중일 때 어떤 상황이 발생하나요?

오디언스 동기화 대상으로 전송하도록 선택한 데이터는 정규화됩니다. 파트너마다 API 요구 사항에 따라 데이터 정규화에 대한 사양이 다를 수 있으므로 자세한 내용은 각 파트너별 엔드포인트를 검토하세요.

또한 Braze는 오디언스 동기화 파트너와 사용자를 동기화하기 전에 모든 데이터를 해시 처리하여 모든 PII가 SHA256을 사용하여 해시되도록 합니다.

### 오디언스 동기화를 생성하고 관리할 때 발생할 수 있는 가장 일반적인 오류는 무엇인가요?

- **유효하지 않은 토큰**<br>
  - 일반적인 원인으로는 특정 광고 네트워크에 로그인하기 위해 비밀번호를 변경했거나 자격 증명이 만료된 경우 등이 있습니다.
  - 이 문제를 해결하려면 해당 파트너 페이지로 이동하여 계정 연결을 끊었다가 다시 연결하면 됩니다.
- **잠재 고객 규모가 너무 낮음**<br>
  - 이 오류는 일반적으로 오디언스에서 사용자를 제거하는 오디언스 동기화 단계를 생성한 경우에 발생합니다. 오디언스 크기가 0에 가까워지면 네트워크에서 오디언스 크기가 너무 작아서 서비스를 제공할 수 없다는 플래그를 지정할 수 있습니다. 
  - 이 문제를 해결하려면 오디언스 규모를 완전히 고갈시키지 않는 범위에서 정기적으로 사용자를 추가 및 제거하는 오디언스 동기화 전략을 고려해야 합니다.
- **대상자가 존재하지 않음**<br>
  - 이 오류는 오디언스 동기화 단계에서 존재하지 않는 오디언스를 사용하기 때문에 발생합니다. 오디언스에 액세스하는 데 필요한 권한이 없는 경우에도 이 기능이 트리거될 수 있습니다. 
  - 이 문제를 해결하려면 오디언스 동기화 구성에서 활성 오디언스를 추가하거나 새 오디언스를 생성합니다.
- **광고 계정 액세스 시도**<br>
  - 이 오류는 선택한 광고 계정 및/또는 대상에 대한 권한이 없는 경우 발생합니다.
  - 이 문제를 해결하려면 광고 계정의 관리자와 협력하여 적절한 액세스와 권한을 확보합니다. 
- **잘못된 설정**<br>
  - 이 오류는 일치시킬 광고 계정, 오디언스 또는 사용자 필드를 포함하여 캔버스에서 특정 오디언스 동기화 대상을 구성하지 않은 경우 트리거됩니다. 
  - 이 문제를 해결하려면 시작하기 전에 각 파트너의 구성을 완료합니다.
- **서비스 약관**<br>
  - Facebook과 같은 일부 오디언스 동기화 대상의 경우, 오디언스 동기화 기능을 사용하려면 광고 네트워크의 특정 서비스 약관에 동의해야 합니다. 이 오류는 적절한 약관에 동의하지 않은 경우 트리거됩니다. 
  - 이 문제를 해결하려면 각 파트너의 필수 약관을 수락했는지 확인합니다. 특히 Facebook의 경우 [Facebook 문제 해결을](https://www.braze.com/docs/partners/canvas_steps/facebook_audience_sync/#troubleshooting) 검토하세요.
