## Décision Studio Go vs. Decisioning Studio Pro 

Decisioning Studio propose deux niveaux : Go et Pro. Chaque niveau est conçu pour répondre à des besoins et des cas d'utilisation différents.

### Décisions Studio Go

Go est idéal pour les équipes qui se lancent dans le décisionnel par intelligence artificielle. Il comprend

- Configuration créative en libre-service avec un agent décisionnel préconçu
- Mesure du succès axée sur les indicateurs de clics
- Compatibilité avec trois plateformes d'engagement client (CEP) : Braze, Salesforce Marketing Cloud et Klaviyo

### Decisioning Studio Pro

Pro offre la suite complète des fonctionnalités de Decisioning Studio pour les cas d'utilisation avancés. Les fonctionnalités principales sont les suivantes :

- Une équipe dédiée aux services décisionnels de l'intelligence artificielle pour vous accompagner de la conception de l'agent décisionnel jusqu'à l'état stable.
- L'indicateur de succès peut être n'importe quel indicateur commercial (pas seulement les indicateurs).
- Possibilité de connecter n'importe quelle source de données client pour la prise de décision.
- Suite complète de rapports et d'informations
- Modèles d'orchestration étendus
