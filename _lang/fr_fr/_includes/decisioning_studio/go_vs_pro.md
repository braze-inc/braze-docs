## Studio de prise de décision Go vs. Studio de prise de décision Pro 

Decisioning Studio propose deux niveaux : Veuillez vous rendre sur Pro. Chaque niveau est conçu pour répondre à différents besoins et cas d'utilisation.

### Studio de prise de décision Go

Go est idéal pour les équipes qui se lancent dans la prise de décision basée sur l'intelligence artificielle. Il comprend :

- Configuration créative en libre-service avec un agent de décision prédéfini
- Indice de succès axé sur les clics
- Compatibilité avec trois plateformes d'engagement client (CEP) : Braze, Salesforce Marketing Cloud et Klaviyo

### Studio de prise de décision Pro

Pro offre la suite complète des fonctionnalités de Decisioning Studio pour les cas d'utilisation avancés. Les principales fonctionnalités sont les suivantes :

- Une équipe dédiée aux services de prise de décision par intelligence artificielle pour vous accompagner depuis la conception de l'agent de prise de décision jusqu'à la stabilisation du système.
- Les indicateurs de réussite peuvent être n'importe quel indicateur commercial (pas seulement les clics).
- Capacité à connecter n'importe quelle source de données client pour la prise de décision
- Suite complète de rapports et d'informations
- Modèles d'orchestration étendus
