# Utiliser les API publiques pour le contenu connecté

> Il existe une variété d'API accessibles au public qui peuvent être utilisées pour le contenu connecté. Les API publiques vous permettent d'insérer des données personnalisées dans les messages. Cette page compile une liste d'API publiques qui pourraient être utilisées pour le contenu connecté. Cependant, il existe de nombreuses autres API, qui offrent une grande variété d'utilisations potentielles du contenu connecté.  

Contactez-nous à [success@braze.com](mailto:success@braze.com) pour nous faire savoir si vous avez une API à partager !

{% alert note %}
Les API publiques peuvent être soumises à des restrictions d'utilisation et à des limites de débit. Veillez à lire les documentations de l'API et à contacter les fournisseurs d'API au sujet de l'utilisation que vous comptez en faire.
{% endalert %}

## Nouvelles et informations

|	 API 	| Description |
| --------- | --- |
| [OpenWeatherMap](http://openweathermap.org/api) | Fournit des données météorologiques actuelles, des prévisions à 5 et 16 jours et des données historiques. |
| [Recherche d'articles dans le NYT](https://developer.nytimes.com/docs/articlesearch-product/1/overview) | Fournit des données sur les articles du NYT, notamment le titre, le sujet, l'URL, la date, le résumé, etc. |
| [The Guardian API](http://open-platform.theguardian.com/documentation/) | Fournit les données de l'article Guardian qui comprennent le titre, le sujet, l'URL, la date, le résumé, etc.|
