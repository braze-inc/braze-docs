# Flux d'onboarding

> Utilisez le modèle de message in-app du **flux d'onboarding** pour collecter les attributs, informations et préférences des utilisateurs qui alimentent votre stratégie de communication. 

{% multi_lang_include drag_and_drop/templates.md section='SDK requirements' %}

## Créer un flux d'onboarding

### Étape 1 : Choisissez votre modèle

Lorsque vous créez un message in-app par glisser-déposer, sélectionnez **Flux d'onboarding** pour votre modèle, puis sélectionnez **Créer un message.** Ce modèle est pris en charge à la fois pour les applications mobiles et les navigateurs web.

L'éditeur de messages in-app avec le modèle de flux d'onboarding.]({% image_buster /assets/img/drag_and_drop/templates/onboarding_editor.png %})

### Étape 2 : Définissez les styles de vos messages

{% multi_lang_include drag_and_drop/templates.md section='message style' %}

### Étape 3 : Personnalisez l'ordre de votre flux d'onboarding

Pour commencer à créer votre flux d'onboarding, utilisez la section **Pages** pour ajouter ou supprimer des messages à votre flux. Une fois que vous avez le nombre d'envois de messages souhaité, sélectionnez les boutons sur chaque page. Utilisez ensuite le menu latéral pour sélectionner l'endroit où les utilisateurs se rendent lorsqu'ils cliquent sur le bouton : ils quittent le flux ou passent au message suivant.

Vérifiez que vos messages sont connectés dans un ordre séquentiel afin que les utilisateurs se déplacent correctement dans le flux. Pour savoir comment procéder, consultez la page [Connecter les pages ensemble.]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/create/?tab=adding%20pages#step-3a-connect-pages-together)

### Étape 4 : Donnez du style à votre message

Personnalisez l'aspect et la convivialité de votre flux d'onboarding à l'aide des [composants de message in-app par]({{site.baseurl}}/user_guide/message_building_by_channel/in-app_messages/drag_and_drop/style_settings/#message-components) glisser-déposer.
