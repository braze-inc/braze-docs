# Informations requises pour la demande de 10DLC

Si vous avez acheté de l'aide auprès de Braze pour enregistrer votre 10DLC, vous devrez nous envoyer toutes les informations requises énumérées à l'e-mail ci-dessous, comme convenu dans votre contrat avec nous.

Veuillez inclure toutes les captures d'écran et pièces jointes pertinentes et envoyez le tout à [smsapplications@braze.com](mailto:smsapplications@braze.com). Si vous avez des questions sur le 10DLC, consultez le [guide SMS 10DLC de Bra]({{site.baseurl}}/assets/pdf/SMS_10DLC_Guide.pdf) ze ou contactez votre gestionnaire satisfaction client de Braze.
