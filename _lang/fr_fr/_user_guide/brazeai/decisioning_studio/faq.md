# Foire aux questions

> Cet article répond aux questions les plus fréquemment posées sur Decisioning Studio.

### Qu'est-ce qu'un agent décisionnel ?

Un agent décisionnel est une configuration personnalisée pour BrazeAI Decisioning Studio™, conçue sur mesure pour répondre à un objectif métier spécifique. Celle-ci est définie par l'indicateur de réussite, les indicateurs et les options que vous choisissez. L'agent décisionnel découvre automatiquement l'action optimale pour chaque client afin de maximiser votre indicateur commercial choisi.

### Quels sont les indicateurs que je peux optimiser ?

Vous pouvez optimiser pour tout indicateur métier qui s'aligne sur vos objectifs, comme le chiffre d'affaires, les conversions, le revenu moyen par utilisateur (ARPU), la valeur vie client (CLV), le bénéfice, les renouvellements de contrats ou tout autre indicateur clé de performance métier.

### Que sont les dimensions dans Decisioning Studio ?

Les dimensions peuvent être considérées comme les *types de leviers* que l'agent décisionnaire peut actionner pour maximiser l'indicateur de réussite. Les dimensions typiques sont l'offre, la ligne d'objet, la création, le canal ou l'heure d'envoi.

### Qu'est-ce qu'une banque d'action ?

La banque d'actions définit les *options spécifiques* auxquelles l'agent décisionnaire a accès pour chaque dimension "levier". Par instance, pour une dimension de canal, vous définirez les canaux spécifiques auxquels l'agent décisionnel a accès. Pour une dimension d'offre, vous définissez les offres spécifiques que l'agent décisionnaire peut tester.

### L'agent décisionnel peut-il prendre des mesures que je n'ai pas configurées ?

Non. L'agent décisionnel ne peut entreprendre que les actions que vous configurez et ajoutez à la banque d'actions. Cela signifie que toutes les actions possibles sont définies par les combinaisons de ce que vous mettez dans la banque d'actions.

### Qu'est-ce qu'une contrainte ?

Les contraintes limitent les actions de l'agent décisionnaire afin qu'il respecte les règles essentielles de l'entreprise. Il peut s'agir, par exemple, d'empêcher qu'une offre spécifique soit sélectionnée pour les clients d'une zone géographique non éligible, ou de fixer un budget maximum à dépenser par l'agent décisionnaire.

### Quelle est la différence entre Decisioning Studio Go et Decisioning Studio Pro ?

Decisioning Studio Pro inclut l'assistance des services de décision en matière d'intelligence artificielle de l'équipe de science des données déployée en avant-première de Braze, qui vous aidera à concevoir et à configurer votre agent pour maximiser les résultats de votre entreprise.
