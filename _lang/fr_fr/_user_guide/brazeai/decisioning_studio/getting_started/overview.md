# Aperçu

> Cette référence présente un aperçu des étapes nécessaires à la configuration de Decisioning Studio, notamment la connexion des sources de données, l'orchestration et la conception de votre agent de décision.

BrazeAI Decisioning Studio™ vous permet de concevoir et de déployer des agents décisionnels qui optimisent tous les indicateurs commerciaux. Un agent décisionnel est une configuration personnalisée conçue pour répondre à un objectif métier spécifique.

Pour ce faire, il est nécessaire de connecter les sources de données, de configurer l'orchestration et de concevoir vos agents décisionnels.

{% alert tip %}
Pour les clients Decisioning Studio Pro, votre équipe de services en intelligence artificielle vous assistera dans la configuration de Decisioning Studio afin d'obtenir des performances optimales.
{% endalert %}

## Configuration de Decisioning Studio

Pour configurer Decisioning Studio, veuillez suivre les étapes suivantes :

### Étape 1 : Connecter les sources de données

Connectez les profils clients et les données d'engagement afin que les agents décisionnaires que vous créez comprennent qui est chaque client et comment il se comporte.

En règle générale, il suffit de connecter vos sources de données une seule fois, lors de la configuration initiale de Decisioning Studio. Si vous élargissez vos cas d'utilisation ultérieurement, il se peut que vous deviez ajouter de nouvelles sources de données.

{% alert tip %}
Toutes les données déjà présentes dans la [plateforme Braze Data]({{site.baseurl}}/user_guide/data/braze_data_platform) sont automatiquement disponibles dans Decisioning Studio.
{% endalert %}

Pour obtenir des instructions détaillées, veuillez consulter la documentation relative à votre niveau de Decisioning Studio :
- [Studio de prise de décision Go : Connecter les sources de données]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/connect_data_sources/)
- [Studio Pro de prise de décision : Connecter les sources de données]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/connect_data_sources/)

### Étape 2 : Configurer l'orchestration

Effectuez l'intégration de Decisioning Studio à votre plateforme d'engagement client (CEP) afin de permettre à vos agents d'effectuer l'orchestration des actions. Le CEP est la plateforme utilisée pour offrir des expériences personnalisées à vos clients en fonction des décisions prises par l'agent.

En général, il est nécessaire de configurer cette orchestration une seule fois.

Pour obtenir des instructions détaillées, veuillez consulter la documentation relative à votre niveau de Decisioning Studio :
- [Studio de prise de décision Go : Configurer l'orchestration]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/set_up_orchestration/)
- [Studio Pro de prise de décision : Configurer l'orchestration]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/set_up_orchestration/)

### Étape 3 : Concevez vos agents

Veuillez configurer vos agents décisionnels afin de définir les résultats que vous souhaitez optimiser et les mesures que l'agent peut prendre pour les atteindre. Veuillez consulter [la section Conception d'agents décisionnels]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/getting_started/designing_decisioning_agents/) pour obtenir des conseils détaillés sur la conception d'agents.

Pour obtenir des conseils spécifiques à chaque niveau :
- [Studio de prise de décision Go : Concevez votre agent]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/design_your_agent/)
- [Studio Pro de prise de décision : Concevez votre agent]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/design_your_agent/)

{% alert tip %}
Pour les clients de Decisioning Studio Pro, votre équipe de service en intelligence artificielle vous assistera dans la conception et le lancement de vos agents décisionnels.
{% endalert %}

### Étape 4 : Lancez votre agent décisionnel

Lancez votre agent décisionnel et laissez-le apprendre et s'optimiser en continu pour améliorer les résultats de votre entreprise.

Pour obtenir des instructions détaillées, veuillez consulter la documentation relative à votre niveau de Decisioning Studio :
- [Studio de prise de décision Go : Veuillez lancer votre agent.]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_go/launch_your_agent/)
- [Studio Pro de prise de décision : Veuillez lancer votre agent.]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/decisioning_studio_pro/launch_your_agent/)
