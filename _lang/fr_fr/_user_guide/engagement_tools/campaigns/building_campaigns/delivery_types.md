En complément de ces articles, nous vous recommandons également de consulter notre cours d'apprentissage sur <a href='https://learning.braze.com/campaign-setup-delivery-targeting-conversions'>la configuration des campagnes à</a> Braze.<br><br>Le choix du mode de diffusion de votre campagne est crucial pour développer une campagne efficace. Heureusement, avec Braze, vous pouvez contrôler avec précision quand et comment vos campagnes sont envoyées. <br><br>Tout marketeur avisé sait que le timing est essentiel, c'est pourquoi Braze propose plusieurs options de planification qui vous permettront d'atteindre les utilisateurs précisément au bon moment. Une grande flexibilité peut toutefois entraîner une incertitude quant au type de planification qui correspond le mieux aux objectifs de votre campagne. Pour vous aider à tirer le meilleur parti de Braze, nous avons rédigé ces articles pratiques qui passent en revue vos options de planification, les meilleures pratiques et les cas d'utilisation."
