### Onboarding

Imaginons que votre restaurant veuille aider les utilisateurs onboarding à effectuer leur première réservation. Étant donné que ce canvas est destiné à l'onboarding, le moment idéal pour le lancer serait au début de la session pour tous les nouveaux clients. Pour atteindre rapidement et efficacement votre audience, vous pouvez utiliser le canal de communication par SMS.

\![]({% image_buster /assets/img_archive/canvas_outline_onboarding.png %}){: style="max-width:90%;"}

### Vente incitative

La vente incitative de vos abonnements peut aussi être encouragée en créant et en envoyant des canevas efficaces. Par exemple, si vous souhaitez mettre à niveau les utilisateurs actifs qui sont sur une version gratuite de votre appli, vous pouvez créer un Canvas basé sur des actions pour le déclencher lorsqu'un client a atteint l'événement personnalisé "3 heures de streaming". À l'aide d'une étape Message, vous pouvez inviter ces clients à s'inscrire à vos abonnements premium.

\![]({% image_buster /assets/img_archive/canvas_outline_upsell.png %}){: style="max-width:90%;"}

### Paniers abandonnés

Les entreprises de vente au détail se trouvent souvent dans l'obligation de rappeler à leurs clients qu'ils ont effectué des achats incomplets. Avec un canvas basé sur l'action, vous pouvez envoyer un rappel à tous les clients enregistrés pour qu'ils achètent les articles de leurs paniers abandonnés. Vous pouvez également tester la réceptivité de vos clients à votre message avec différents délais.

\![]({% image_buster /assets/img_archive/canvas_outline_cart.png %}){: style="max-width:90%;"}

### Ressources personnalisées

Vous pouvez utiliser les toiles pour informer les clients sur les ressources. Par exemple, pour les compagnies aériennes, vous pouvez créer un canvas qui stimule les clients ayant réservé un voyage dans les trois jours en planifiant un e-mail hebdomadaire contenant les informations sur leur vol et les questions les plus fréquemment posées à l'aéroport.

\![]({% image_buster /assets/img_archive/canvas_outline_resource.png %}){: style="max-width:90%;"}
