# OpenAI

> [OpenAI](https://openai.com/) crée des modèles d'intelligence artificielle avancés, comme GPT, qui permettent de comprendre et de générer du langage naturel, donnant aux marques les moyens de créer et de mettre à l'échelle des interactions significatives avec les clients. 

_Cette intégration est maintenue par OpenAI._

## À propos de l'intégration

L'intégration Braze et OpenAI vous permet de connecter votre clé API OpenAI à Braze afin d'utiliser les modèles OpenAI lorsque vous créez des agents d'intelligence artificielle personnalisés. Grâce à cette intégration, vos agents peuvent générer des copies personnalisées, prendre des décisions en temps réel ou mettre à jour les champs du catalogue en utilisant les grands modèles de langage d'OpenAI.

## Conditions préalables

| Exigences | Description |
|---|---|
| Compte OpenAI avec clé API | Un compte OpenAI avec une clé API. Contactez votre administrateur ou le [support d'OpenAI](https://help.openai.com/) pour tout problème ou question. |
| Instance de Braze | Vous pouvez trouver votre instance Braze sur la [page d'aperçu de l'API]({{site.baseurl}}/api/basics/#endpoints) ou auprès de votre gestionnaire onboarding Braze. |

## Intégration

Pour connecter votre clé API OpenAI à Braze :

1. Allez dans **Intégrations partenaires** > **Partenaires technologiques** dans le tableau de bord de Braze et trouvez OpenAI.
2. Saisissez votre clé API d'OpenAI.
3. Sélectionnez **Enregistrer**.

Après avoir enregistré, vous pouvez sélectionner des modèles OpenAI lors de la [création d'un agent personnalisé]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) dans la console d'agent.

Contactez le [support d'OpenAI](https://help.openai.com/) pour tout problème ou question concernant votre intégration.
