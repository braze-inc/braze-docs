# Google Gemini

> [Google Gemini](https://deepmind.google/technologies/gemini/) est la famille de modèles d'intelligence artificielle de Google qui combine un raisonnement avancé à travers le texte, le code et les images pour aider les marques à offrir des expériences plus intelligentes et plus personnalisées.

_Cette intégration est gérée par Google._

## À propos de l'intégration

L'intégration Braze et Google Gemini vous permet de connecter votre clé API Google Gemini à Braze afin d'utiliser les modèles Gemini lorsque vous créez des agents d'intelligence artificielle personnalisés. Grâce à cette intégration, vos agents peuvent générer des textes personnalisés, prendre des décisions en temps réel ou mettre à jour les champs du catalogue à l'aide des modèles Gemini de Google.

## Conditions préalables

| Exigences | Description |
|---|---|
| Compte Google Cloud avec clé API Gemini | Un compte Google Cloud avec une clé API Gemini. Pour obtenir de l'aide, contactez votre administrateur ou l'[assistance Google Cloud.](https://cloud.google.com/support) |
| Instance de Braze | Vous pouvez trouver votre instance Braze sur la [page d'aperçu de l'API]({{site.baseurl}}/api/basics/#endpoints) ou auprès de votre gestionnaire onboarding Braze. |

## Intégration

Pour connecter votre clé API Google Gemini à Braze :

1. Allez dans **Intégrations partenaires** > **Partenaires technologiques** dans le tableau de bord de Braze et trouvez Google Gemini.
2. Saisissez votre clé API de Google.
3. Sélectionnez **Enregistrer**.

Après avoir enregistré, vous pouvez sélectionner les modèles Gemini lors de la [création d'un agent personnalisé]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) dans la console d'agent.

Contactez l'[assistance de Google Cloud](https://cloud.google.com/support) pour tout problème ou question concernant votre intégration.
