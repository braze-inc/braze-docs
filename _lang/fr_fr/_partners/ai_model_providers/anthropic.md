# Anthropique

> [Anthropic](https://www.anthropic.com/) est une entreprise de sécurité et de recherche en intelligence artificielle qui développe Claude, un assistant d'intelligence artificielle de nouvelle génération créé pour être utile, honnête et sûr pour un large éventail de tâches langagières.

_Cette intégration est maintenue par Anthropic._

## À propos de l'intégration

L'intégration Braze et Anthropic vous permet de connecter votre clé API Anthropic à Braze afin d'utiliser les modèles de Claude lorsque vous créez des agents d'intelligence artificielle personnalisés. Grâce à cette intégration, vos agents peuvent générer des textes personnalisés, prendre des décisions en temps réel ou mettre à jour les champs du catalogue en utilisant les modèles d'Anthropic Claude.

## Conditions préalables

| Exigences | Description |
|---|---|
| Compte Anthropic avec clé API | Un compte Anthropic avec une clé API. Pour obtenir de l'aide, contactez votre administrateur ou le [service d'assistance d'Anthropic](https://support.anthropic.com/). |
| Instance de Braze | Vous pouvez trouver votre instance Braze sur la [page d'aperçu de l'API]({{site.baseurl}}/api/basics/#endpoints) ou auprès de votre gestionnaire onboarding Braze. |

## Intégration

Pour connecter votre clé API Anthropic à Braze :

1. Allez dans **Intégrations partenaires** > **Partenaires technologiques** dans le tableau de bord de Braze et trouvez Anthropic.
2. Saisissez votre clé API d'Anthropic.
3. Sélectionnez **Enregistrer**.

Après avoir enregistré, vous pouvez sélectionner les modèles Claude lors de la [création d'un agent personnalisé]({{site.baseurl}}/user_guide/brazeai/agents/creating_agents/) dans la console d'agent.

Contactez le [support d'Anthropic](https://support.anthropic.com/) pour tout problème ou question concernant votre intégration.
