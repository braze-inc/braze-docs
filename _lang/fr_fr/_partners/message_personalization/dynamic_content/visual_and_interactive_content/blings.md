# Blings

> [Blings](https://www.blings.io/) est une plateforme vidéo personnalisée de nouvelle génération qui vous permet de proposer des expériences vidéo en temps réel, interactives et axées sur les données sur l'ensemble des canaux, à grande échelle.  

_Cette intégration est maintenue par Blings._

## Conditions préalables  

| Condition     | Description                                                                 |
|-----------------|-----------------------------------------------------------------------------|
| Compte Blings  | Un compte Blings est nécessaire pour bénéficier de ce partenariat.         |  

## Intégration  

### Étape 1 : Obtenez votre extrait de code HTML Blings

{% tabs %}
{% tab Blings business and free plans %}

#### Blings business et plans gratuits

Emplacement/localisation et copie de votre extrait de code HTML directement dans l'application Blings.

1. Accédez à l'onglet **Connecter** du projet MP5 sélectionné.
2. Ajoutez les étiquettes Liquid de Braze aux variables correspondantes dans la page Blings **Connect**. Les étiquettes rempliront dynamiquement les valeurs de l'extrait de code HTML. 

![Extrait de code HTML de Blings.]({% image_buster /assets/img/blings/blings_connect_audience.png %}){: style="max-width:70%;"}  

{% endtab %}
{% tab Blings Enterprise plan %}

#### Plan d'entreprise Blings

Demandez l'extrait de code HTML à votre conseiller Blings. 

{% endtab %}
{% endtabs %}

### Étape 2 : Créez une campagne Braze  

Dans Braze, créez une nouvelle campagne d'e-mails ou de messages in-app et insérez l'extrait HTML Blings.

![Campagne Blings Braze.]({% image_buster /assets/img/blings/blings_braze_campaign.png %})  

### Étape 3 : Test et lancement  

Prévisualisez la campagne dans Braze pour confirmer que les champs personnalisés se remplissent correctement. Ensuite, déployez votre campagne MP5 à grande échelle.

![Aperçu de Blings Braze.]({% image_buster /assets/img/blings/blings_braze_preview.png %}){: style="max-width:70%;"}  

## Obtenir de l'aide  

Pour toute question ou pour demander votre extrait de code, contactez Blings à [support@blings.io](mailto:support@blings.io) ou référez-vous au [centre d'aide de Blings](https://blings.gitbook.io/blings-knowledge-base/documentation).
