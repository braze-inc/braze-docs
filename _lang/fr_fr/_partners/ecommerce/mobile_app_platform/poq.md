# Poq

[Poq](https://poqcommerce.com/) permet aux grandes entreprises de lancer, gérer et faire évoluer rapidement des applications iOS et Android entièrement natives, offrant des expériences mobiles performantes qui stimulent le commerce et donnent vie à la promesse de votre marque.

*Cette intégration est maintenue par Poq.*

## À propos de l'intégration

L'intégration de Poq et Braze vous permet de connecter votre expérience sur l'application mobile native à la plateforme d'engagement client de Braze. Grâce aux SDK iOS et Android de Braze, Braze est intégré directement dans une application propulsée par Poq, permettant la segmentation, la personnalisation et la distribution ciblée de campagnes sur mobile.

## Fonctionnalités prises en charge

Les canaux et fonctionnalités suivants sont pris en charge.

| Canal/Fonctionnalité | Description |
| :---- | :---- |
| Notifications push | Envoyez des notifications push ciblées et personnalisées aux utilisateurs de l'application via Braze, y compris des notifications push riches. |
| Messages in-app | Affichez des messages in-app de Braze au sein de l'expérience native de l'application pour engager les utilisateurs au bon moment de leur parcours. |
| Analytique et suivi des événements | Poq envoie un ensemble complet d'événements applicatifs à Braze, couvrant l'intégralité du parcours client. Ces données alimentent la segmentation, les déclencheurs de campagnes et la personnalisation dans Braze. |
| Cartes de contenu et boîte de réception | Diffusez des Cartes de contenu persistantes de Braze au sein de l'application, soit dans un écran dédié de boîte de réception, soit sur des pages créées avec le générateur de contenu dynamique de Poq, ou encore dans des emplacements prédéfinis à travers l'application. Les Cartes de contenu sont filtrées à l'aide de paires clé-valeur configurées dans Braze. |
| Bannières | Affichez des Bannières Braze au sein de l'application, soit sur des pages créées avec le générateur de contenu dynamique de Poq, soit dans des emplacements prédéfinis à travers l'application. Le contenu des bannières est récupéré à l'aide d'un ID d'emplacement. |

## Conditions préalables

Les éléments suivants sont nécessaires pour utiliser cette intégration.

| Requis | Description |
| :---- | :---- |
| Application Poq | Une application Poq est requise pour bénéficier de cette intégration. |
| Clés API d'application Braze | Une clé API d'application Braze est requise pour chaque plateforme (iOS et Android). Vous pouvez les obtenir depuis **Paramètres** > **API et identifiants** > **Identifiants d'application** dans le tableau de bord de Braze. |
