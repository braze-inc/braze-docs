> Vous souhaitez migrer vers le SDK Swift iOS à partir de notre ancien SDK Objective-C ? C'est simple. Consultez notre [guide de migration](https://braze-inc.github.io/braze-swift-sdk/documentation/braze/appboy-migration-guide/) pour plus de détails.

Après avoir effectué l'intégration à l'aide de l'une des options répertoriées, suivi les étapes pour [terminer l'intégration]({{site.baseurl}}/developer_guide/platform_integration_guides/swift/initial_sdk_setup/completing_integration/) et activé d'autres personnalisations du SDK (facultatif), vous pouvez intégrer, activer et personnaliser des canaux et des fonctionnalités supplémentaires pour répondre aux besoins de vos futures campagnes.  

{% alert note %} Carthage n'est pas pris en charge en tant que gestionnaire de paquets Swift pour le SDK Swift iOS. {% endalert %}
