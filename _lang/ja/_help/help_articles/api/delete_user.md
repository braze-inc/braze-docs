# API によるユーザーの削除

[Braze REST API を使用してユーザーを削除すると][1]、次のデータが削除 (無効化) されます。
-ユーザーが持っていたすべての属性
-メールアドレス
-電話番号
-外部ユーザ ID
-性別
-カントリー
-言語

[Braze REST API を使用してユーザーを削除すると][1]、次のイベントが発生します。
-ユーザープロファイルは匿名化されています。
-[ライフタイムユーザー数は][2]、新たに匿名化されたユーザーを考慮して更新されます。	
-匿名化されたユーザーは、集計されたコンバージョン率には引き続きカウントされます。匿名ユーザーのカスタムイベント数と購入数は更新されません。

## メールアドレスが共有されている複数のプロフィール

たとえば、同じメールアドレスを共有する複数のユーザープロファイルを統合したいとします。 

これらのユーザープロファイルを統合するには:

 1. 電子メールアドレスが重複しているユーザーを特定します。 
 2. 1 つのプロファイルのすべての属性をエクスポートします。 
 3. これらの属性を API または CSV 経由でユーザープロファイルにインポートします。 
 4. API を使用してユーザーを削除します。つまり、重複したユーザーと上記のデータを基本的に削除します。
