# 2017年1月

## メールメッセージの新しいコンバージョンイベント

"キャンペーンからのメール開封 "または "キャンペーンからのメールクリック "のコンバージョンイベントを指定できるようになった。[キャンペーン配信タイプについて][16]学習する。

## コントロールグループ付きCSVエクスポート
ユーザーキャンペーンCSVをエクスポートする際、ユーザーデータにコントロールグループを含めることができるようになった。[ダッシュボードデータのエクスポートについて][17]学習する。

## ウェルカムメールとダブルオプトインメール
新しく作成された "メールアドレスを追加しました "アクションベースのトリガーを使用することで、ユーザーが初めてメールアドレスを追加した際に、ダブルオプトインやウェルカムメールを簡単に送信できるようになった。[メールのベストプラクティスについて][18]学習する。

## 配信停止メールヘッダー
サブスクリプションステータスが "subscribe/opted in users "または "opted in users only "のList-Unsubscribeヘッダーを設定できるようになった。これにより、Gmailの送信者名の横に「配信停止」リンクが追加され、配信率を向上させることができる。

## ユーザビリティの更新
ボタンをクリックすると、キャンバスのヘッダーが最小化され、Brazeのユーザープロファイルでキャンバスやキャンペーンのフルネームにカーソルを合わせることができる。
