# 2017年11月

## Brazeの通知設定

会社設定ページのBraze[通知設定][77]センターを使用すると、毎週の分析レポート、キャンペーン送信時の通知、プッシュ認証エラーなど、Brazeからのさまざまなアラートをメールで受け取るようにチームのユーザーを設定することができます。Webhookを設定して、[Slackなど][78]別のチャネルに通知を表示することもできる。

## メディアライブラリーの更新

**テンプレート＆メディア**」ページから、複数のアセットやアセットフォルダを[メディアライブラリに][79]アップロードできるようになったほか、タグの適用や複数のアセットの削除など、一括アクションを実行できるようになった。

## キャンバスの最初のステップのためのBranching

バリアントの最初のステップに Branch を作成できるようになった。この機能により、バリアントの最初のステップとして複数のメッセージを送信できるようになり、さらにオーディエンスにターゲットを絞る機会が開封される。
