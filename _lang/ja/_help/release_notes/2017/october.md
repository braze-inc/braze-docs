# 2017年10月

## プッシュ通知ストーリー

iOSとAndroidでは、複数のページで[通知を作成][74]できるようになった。各ページには画像、クリック動作、任意のタイトルとサブタイトルが含まれる。この機能にはiOS SDK 3.2.1とAndroid SDK 2.2.0が必要である。この新製品にアクセスするには、顧客サクセスマネージャー、アカウントマネージャーに連絡するか、[サポートチケットを][support]開封する。

## Currents

[Currentsは][75]、データウェアハウス、ビジネスインテリジェンスツール、データ管理プラットフォームなどのテクノロジーに直接接続するための、あらかじめ構築された顧客データ用のリアルタイムの大容量データ接続の設定を表示する。Amazon Redshift、Amplitude、データマネジメントプラットフォームのSegmentとmParticleとの統合を開始する。この新製品にアクセスするには、顧客サクセスマネージャー、アカウントマネージャーに連絡するか、[サポートチケットを][support]開封する。

## Oktaシングルサインオン（2017年10月）

企業は、Oktaを使用してBrazeへのアクセスをコントロールし、仲間のBrazeユーザーにOkta認証情報を使用してログインすることを許可または強制することができる。この新製品にアクセスするには、顧客サクセスマネージャー、アカウントマネージャーに連絡するか、[サポートチケットを][support]開封する。

> 更新だ：Oktaへのアクセス方法については、[2018年5月のリリースノートに]({{site.baseurl}}/help/release_notes/2017/august/#may-2018)記載されている。

## メールフッターの長さ制限の更新

[メールのフッターの][76]上限が100kbになった。この更新はメールのベストプラクティスに沿ったもので、本文が400kbを超えるメールはスパムになりやすいからだ。
