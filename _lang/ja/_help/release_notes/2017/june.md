# 2017年6月号

## 外部 API の更新

Apple プッシュの collapse\_id フィールドをサポートするように外部 API を更新し、顧客に正確にメッセージを送信できるようになりました。[Braze messaging API][49]の詳細をご覧ください。

## Web プッシュ通知の更新

ドロップダウンや正しい画像の高さのレンダリングなど、Web プッシュ通知のプレビューの更新が追加され、Web プッシュ通知のテストと送信が簡単になりました。[詳しくは、Web プッシュ][47]についての記事をご覧ください。

## セキュリティ設定の改善

パスワードの複雑さのルール、ダッシュボードの IP ホワイトリスト、2 要素認証をお客様が設定できる機能が追加され、より詳細に制御できるようになりました。詳しくは、 [セキュリティ設定][46]をご覧ください。

## CSVの同時インポート

複数のCSVをアップロードして同時に処理できるようになり、より迅速に作業できるようになりました。[詳しくは、ユーザーのインポート][50]についての記事をご覧ください。

## キャンバス テンプレートの属性

キャンバス ステップから送信されるメッセージに [テンプレート属性][45] を含める機能が追加され、効率が向上しました。

## ターゲティングの更新

日付と時刻をフィルター処理するときに時間単位として時間を選択できるようになり、メッセージをより正確にターゲットにできるようになりました。[詳しくは、フィルター][44]についての記事をご覧ください。
