# 2017年3月

## 新しいリターゲティングフィルター

新しい「キャンバスからのコンバージョン」フィルターを追加し、よりシームレスにリターゲティングするオプションを提供する。[リターゲティングフィルターの][27]使用について詳しく学習する。

## Webhook テンプレート

Webhookメッセージをテンプレートとして保存して簡単に再利用したり、Twilioを使ったSMSやFacebook Messengerのようなチャットボットのようなプラットフォーム用にあらかじめ設定されたテンプレートから始めることができるようになった。[Webhookの作成について][26]詳しく学習する。

## エンゲージメントレポートの複数メールサポート

[エンゲージメントレポートを][25]複数のメールアドレスに送信できるようになった。

## アクションベースの配信で時間を指定する

遅延アクションベースのトリガーを使用する場合、遅延後の特定の時間に送信するオプションが追加された。この機能はキャンバスとキャンペーンで利用できる。[アクションベースの配信キャンペーンについて][24]学習する。
