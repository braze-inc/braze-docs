# 2017年3月

## 新しいリターゲットフィルタ

新しい"Converted from Canvas" filter を追加し、よりシームレスなターゲット変更オプションを提供しました。詳細については、[ターゲット変更フィルタ][27] を使用してください。

## Webhookテンプレート

Webhook メッセージをテンプレートとして保存し、簡単に再利用したり、Twilio を使用したSMS やFacebook Messenger などのチャットボットのようなプラットフォーム用に事前設定されたテンプレートから始めることができるようになりました。[webhook][26]の作成について詳しく説明します。

## エンゲージメントレポートの複数のメールサポート

[エンゲージメントレポート][25]が複数のメールアドレスに送信されるようになりました。

## アクションベースの配信の時間の指定

これで、遅延アクションベースのトリガを使用する場合に、遅延後に特定の時間に送信するオプションがあります。この機能は、キャンバスとキャンペーンで使用できます。詳細については、[アクションベースの配信キャンペーン][24]を送信してください。
