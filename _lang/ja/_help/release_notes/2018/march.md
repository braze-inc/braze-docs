# 2018年3月

## リンクテンプレート

メールメッセージ内のすべてのリンクに[パラメータを追加][91]したり、URIを事前に追加したりする[テンプレートを][91]作成できるようになった。すべてのリンクにGoogle Analyticsのクエリパラメータを追加したい？頑張れ！

## 開発者向けユニバーサルAPN証明書

これで、開発環境で使用する[APN証明書を設定][90]できる。
