# 2018年3月

## リンクテンプレート

メールメッセージ内のすべてのリンクに[パラメータを追加][91]したり、URIを事前に追加したりする[テンプレートを][91]作成できるようになりました。すべてのリンクにGoogle Analyticsのクエリパラメータを追加しますか？頑張れ！

## 開発ビルド用のユニバーサルAPN証明書

これで、開発環境で使用する[APNs証明書を設定][90]できます。
