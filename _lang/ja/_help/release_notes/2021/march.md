# 2021年3月

## セグメントエクステンション

[Segment Extension]({{site.baseurl}}/user_guide/engagement_tools/segments/segment_extension)は、過去365日間のカスタムイベントと購入行動に基づいて、ユーザのより正確なリストをターゲットにできるようにすることで、既存のセグメンテーション機能を拡張します。これらの拡張機能リストが生成されると、それらをセグメント内のフィルタとして含める/除外することができます。セグメント拡張を作成するときに、24時間ごとにリストを再生成するように指定することもできます。

## A2P 10DLC

A2P 10DLC とは、ビジネスが標準の10 桁のロングコード(10DLC) 電話番号を介してApplication-to-Person (A2P) タイプのメッセージングを送信できるようにする、米国のシステムのことです。10 桁のロングコードは、従来、Person-to-Person(P2P)トラフィック用に設計されており、ビジネスは、制限されたスループットと高められたフィルタリングによって制約されます。現在および/または長いコードを使用しているすべてのお客様は、10DLCのロングコードを登録する必要があります。詳細については、[A2P 10DLC]({{site.baseurl}}/user_guide/message_building_by_channel/sms/sms_setup/short_and_long_codes/#application-to-person-10-digit-long-codes-a2p-10dlc)の記事を参照してください。

## グローバルコントロールグループ

[Global Control Groups]({{site.baseurl}}/user_guide/engagement_tools/testing/global_control_group/) を使用すると、時間の経過に伴うメッセージング作業の全体的な影響を分析できます。これらのグループは、メッセージングを受信したユーザの動作を、受信しなかったユーザの動作と比較することで、マーケティングキャンペーンとキャンバスがセッションやカスタムイベントの上昇をもたらす仕組みを理解するのに役立ちます。
