# 2019年6月

## Snowflake提携

BrazeはSnowflakeとのパートナーシップを発表できることを誇りに思う！

Snowflake は、あらゆるデータ、あらゆるユーザーのために特別に設計された SQL クラウドデータウェアハウスです。Snowflake 独自の特許取得済みアーキテクチャにより、すべてのデータを集約し、迅速な分析を可能にし、すべてのユーザーにデータドリブン型のインサイトを提供することが容易になります。

Brazeは、Snowflake の Data Exchange を活用して、プレゼンスを構築し、新規顧客を見つけ、成長を続ける Snowflake の顧客基盤を通じてリーチを拡大します。

このパートナーシップについて詳しくは、[こちら](https://www.braze.com/perspectives/article/snowflake-partner-announcement)または [Braze のドキュメント]({{site.baseurl}}/partners/snowflake/)をご覧ください。
