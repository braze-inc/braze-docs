# タイムラインレポート

> タイムラインレポートは、パフォーマンス指標と共に主要なイベントの視覚的な記録を提供します。これらのイベントには、エージェントの実行、設定の変更、ガードレールの更新などが含まれます。

アノテーションはアップリフトチャート上および専用のタイムラインタブに直接表示されるため、過去の変更を追跡する必要なく、結果の変動に対する即座のコンテキストを得ることができます。

![パフォーマンス指標の時系列チャートを表示するタイムラインレポート。エージェントの実行、設定の変更、ガードレールの更新などの主要なイベントがタイムラインに沿ってアイコンとして表示されています。チャートの下には、日付、タイプ、ラベル、詳細、チャートでの表示の列を持つイベント一覧テーブルがあります。]({% image_buster /assets/img/decisioning_studio/reporting_timeline.png %})

2つのグループ間のパフォーマンスを比較するには、ドロップダウンを使用して目的の比較条件を選択します。詳細については、以下の表を参照してください。

| フィールド | 説明 |
|-------|-------------|
| 日付 | イベントが発生した日付です。 |
| タイプ | システム更新、エージェントの実行、設定の変更など、イベントのカテゴリです。 |
| ラベル | イベントに付けられた名前または識別子です。 |
| 詳細 | イベントを説明する追加情報です。 |
| チャートでの表示 | イベントが関連するチャートに表示されるかどうかを示します。 |
