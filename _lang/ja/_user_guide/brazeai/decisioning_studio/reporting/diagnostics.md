# 診断レポート

> 診断レポートには、**送信**と**受信**の2種類のレポートタイプがあります。

{% tabs local %}
{% tab outbound %}
送信診断レポートは、オーディエンス全体で生成およびアクティベートされたおすすめの日次ボリュームを表示します。配信の問題を特定したり、アクティベーションの急増や減少を追跡したり、メッセージが期待どおりに適切なグループに届いていることを確認するために使用します。

![送信診断レポート。さまざまなオーディエンスグループに対して生成およびアクティベートされたおすすめの日次ボリュームを追跡する折れ線グラフを表示しています。グラフには「生成済み」と「アクティベート済み」のラベルが付いた2本の線が表示され、Y軸はおすすめの数値、X軸は日付を示しています。凡例では各線を色で識別できます。インターフェイスにはグラフの上に日付範囲とオーディエンス選択のドロップダウンフィルターがあります。]({% image_buster /assets/img/decisioning_studio/reporting_diagnostics_outbound.png %})

{% endtab %}

{% tab inbound %}

受信診断レポートは、BrazeAI<sup>TM</sup>へのデータフィードの健全性を監視します。各アセットのファイル数、サイズ、行数などの詳細を追跡し、データが期待どおりに流入していることを確認し、エージェントやキャンペーンに影響を与える前に問題をトラブルシューティングするのに役立ちます。

ドロップダウンを使用して、平均ファイルサイズやファイル数など、さまざまなグラフ指標を選択できます。

![受信診断レポート。BrazeAI<sup>TM</sup>に配信されたデータアセットの日次ファイル数と平均ファイルサイズを追跡する折れ線グラフを表示しています。グラフには「ファイル数」と「平均ファイルサイズ（MB）」のラベルが付いた2本の線が表示され、Y軸は値、X軸は日付を示しています。グラフの上には日付範囲とデータアセット選択のドロップダウンフィルターがあります。]( {% image_buster /assets/img/decisioning_studio/reporting_diagnostics_inbound.png %} )

受信レポートの各指標の詳細については、以下の表を参照してください。

| フィールド | 説明 |
|-------|-------------|
| データアセット | 配信されたデータセットまたはファイルの名前。 |
| 日付 | データが受信された日付。 |
| 最終配信時刻 | データが配信された最新の時刻。 |
| ファイル数 | 受信されたファイルの合計数。 |
| 最大ファイルサイズ（MB） | 受信された最大ファイルのサイズ（メガバイト単位）。 |
| 平均ファイルサイズ（MB） | 受信されたすべてのファイルの平均サイズ（メガバイト単位）。 |
| ファイル行数 | 配信されたファイルに収められている行の合計数。 |
