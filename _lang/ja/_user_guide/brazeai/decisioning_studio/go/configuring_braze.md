# Braze での設定

> BrazeAI Decisioning Studio™Go用に設計されたBrazeでAPI キャンペーンを作成します。

## ステップ 1: キャンペーンを作成 

1. Braze ダッシュボードで、**メッセージング**> **キャンペーン** に移動します。
2. **キャンペーンの作成**を選択します。
3. キャンペーン型については、**API キャンペーン**を選択します。
4. キャンペーン名を入力します。例として、"Decisioning Studio Go Email"があります。

!["Decisioning Studio Go Email"という名前のAPI キャンペーン。]({% image_buster /assets/img/decisioning_studio_go/api_campaign_name.png %})

{: start="5"}
5. メッセージング チャネルの場合は、**Eメール**を選択します。

![API キャンペーンのメッセージング チャネルを選択します。]({% image_buster /assets/img/decisioning_studio_go/select_api_campaign.png %})

{: start="6"}
6. **追加オプション**で、**ユーザーがキャンペーン**を受信できるように再び許可するチェックボックスを選択します。
7. 再利用可能になるまでの時間は、**1**と入力し、ドロップダウンから**Hours**を選択します。

![選択した原薬キャンペーンの再適格性]({% image_buster /assets/img/decisioning_studio_go/additional_options.png %})

{: start="8"}
8. **キャンペーンの保存**を選択します。

## ステップ 2:キャンペーン ID をコピーアンドペーストする

API キャンペーンで、** キャンペーン ID** をコピーします。次に、BrazeAI Decisioning Studio™Goポータルに移動し、**キャンペーンID**を貼り付けます。

![コピー&ペーストするメッセージバリエーションIDの例。]({% image_buster /assets/img/decisioning_studio_go/campaign_id.png %})

## ステップ 3:メッセージバリエーションIDをコピーして貼り付け

API キャンペーンで、**メッセージバリエーションID** をコピーします。次に、BrazeAI Decisioning Studio™Goポータルに移動し、**メッセージバリエーションID**を貼り付けます。
