# 意思決定エージェントの設計

> このリファレンス記事では、意思決定エージェントの設計と設定に関する主要な概念とベストプラクティスについて説明します。

## 意思決定エージェントについて

意思決定エージェントの設計は、Decisioning Studio の設定における最初のステップです。意思決定エージェントが判断を下せるようにするには、最大化したい結果を定義し、そのためにエージェントが取れるアクションを定義する必要があります。

### 主要な概念

以下の用語は、Decisioning Studio ガイド全体で参照されます。

| 用語 | 定義 |
| --- | --- |
| **意思決定エージェント** | 意思決定エージェントとは、特定のビジネス目標を達成するためにカスタマイズされた、BrazeAI Decisioning Studio™ のカスタム設定です。これは、選択した成功指標、ディメンション、およびオプションによって定義されます。 |
| **成功指標** | 最適化したい具体的なビジネス指標です。例えば、収益、コンバージョン、またはユーザーあたりの平均収益（ARPU）などがあります。これは意思決定エージェントがそのアクションを通じて最大化を目指す指標です。 |
| **ディメンション** | ディメンションとは、意思決定エージェントが成功指標を最大化するために操作できる*手段の種類*と考えることができます。典型的なディメンションには、オファー、件名、クリエイティブ、チャネル、送信時間などが含まれます。 |
| **アクションバンク** | アクションバンクは、意思決定エージェントが各ディメンション「レバー」に対してアクセス可能な*具体的な選択肢*を定義します。例えば、チャネルディメンションについては、意思決定エージェントがアクセスできる特定のチャネルを定義します。オファーディメンションでは、意思決定エージェントがテストできる特定のオファーを定義します。 |
| **制約** | 一般的に、意思決定エージェントはアクションバンクに設定したアクションの任意の組み合わせを実行できます。ただし、重要なビジネスルールを遵守させるため、意思決定エージェントのアクションを制限する制約を定義することもできます。例えば、特定のオファーが対象外の地域にいる顧客に選択されないようにしたり、意思決定エージェントが支出できる最大予算を設定したりすることが挙げられます。 |

{% alert important %}
意思決定エージェントは、*あなたが*設定してアクションバンクに追加したアクションのみを実行できます。つまり、可能なすべてのアクションは、アクションバンクに入れたものの組み合わせによって定義されます。
{% endalert %}

## 意思決定エージェントの設計方法

意思決定エージェントを設定する際には、主に4つの設計要素について検討する必要があります。

### 「目標」：成功指標を定義する

> エージェントに最大化させたい結果は何ですか？

成功指標とは、エージェントが最適化を目指すビジネス成果です。これはビジネス目標と直接一致させるべきです。クリック数や開封率のような代理指標ではなく、収益、コンバージョン、ARPU、顧客生涯価値といった実際のビジネス成果を選びましょう。

### 「誰に」：オーディエンスを選択する

> 意思決定エージェントは誰にエンゲージしますか？

エージェントが対応するオーディエンスを定義します。これは全顧客、特定のセグメント（例えばロイヤルティプログラム会員）、あるいはライフサイクルの特定の段階にある顧客（例えば最近購入した顧客や解約リスクのあるサブスクライバー）が対象となります。

### 「何を」：アクションバンクを設定する

> エージェントは結果を導くために、どのような選択肢から選ぶことができますか？

アクションバンクは、エージェントが操作できるすべての手段を定義します。つまり、チャネル、オファー、タイミング、頻度といったディメンションと、各ディメンション内の具体的な選択肢です。エージェントはこれらのオプションのさまざまな組み合わせを試行し、各顧客にとって最適な方法を見つけます。

### 「どのように」：制約を設定する

> エージェントはどのようなルールに従うべきですか？

制約とは、エージェントが従わなければならないルールです。例えば、対象外の地域にいる顧客に対して特定のオファーが選択されないようにしたり、意思決定エージェントが支出できる最大予算を設定したりすることが挙げられます。

## ベストプラクティスと事例

意思決定エージェントの効果を最大化するには、以下を実践してください。

- ビジネス目標や目的に密接に関連する成功指標を選びましょう。例えば、収益、コンバージョン、ARPU などです。
- 成功指標に最も大きな影響を与える可能性が高いディメンション、つまりテストする「レバー」に焦点を当てましょう。例えば、オファー、件名、クリエイティブ、チャネル、送信時間などです。
- 各ディメンションについて、成功指標に最も大きな影響を与える可能性が高い選択肢を選びましょう。例えば、メールと SMS の比較や、毎日と毎週の頻度の比較などです。
