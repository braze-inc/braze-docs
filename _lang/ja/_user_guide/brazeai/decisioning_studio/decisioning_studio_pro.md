# BrazeAI Decisioning Studio™プロ

## デシジョニング・スタジオ・プロについて

Decisioning Studio Proは、Decisioning Studio Goよりも柔軟で強力だ。Proには、アクションバンクの完全なカスタマイズ、あらゆる指標（収益、コンバージョン、エンゲージメントなど）での最適化機能が含まれ、あらゆるカスタマーエンゲージメントプラットフォーム（CEP）と互換性がある。

Proエージェントの設定には、当社のAI意思決定サービスチーム（Brazeの前方展開データサイエンスチーム）との協力が必要であり、このチームは、お客様のビジネスに最大限の価値をもたらすよう、エージェントの設計全体を指導し、支援する。

Decisioning Studio GoとDecisioning Studio Proの違いについては、[Decisioning Studio GoとDecisioning Studio Proを]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro)参照のこと。
