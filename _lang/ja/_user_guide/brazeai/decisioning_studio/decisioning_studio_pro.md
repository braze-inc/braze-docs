# BrazeAI Decisioning Studio™ Pro

## Decisioning Studio Proについて

Decisioning Studio Proは、Decisioning Studio Goよりも柔軟性が高く、より強力である。Pro版には完全なアクションバンクのカスタマイズ機能、あらゆる指標（収益、コンバージョン、エンゲージメントなど）の最適化機能が含まれ、あらゆるカスタマーエンゲージメントプラットフォーム（CEP）と互換性がある。

プロエージェントの設定には、当社のAI意思決定サービスチーム（Brazeの前線に配置されたデータサイエンスチーム）との連携が必要だ。このチームがエージェント設計の全工程を指導・支援し、貴社のビジネスに最大の価値をもたらす。

Decisioning Studio Go と Decisioning Studio Pro の違いについて詳しく学習したい場合は、[「Decisioning Studio Go と Decisioning Studio Pro の比較]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro)」を参照せよ。
