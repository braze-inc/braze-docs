# よくある質問

> この記事では、Decisioning Studioに関するよくある質問とその回答を紹介する。

### 意思決定エージェントとは何か？

意思決定エージェントは、BrazeAI Decisioning Studio™のカスタム設定であり、特定のビジネス目標を満たすためにオーダーメイドされる。これは、選択した成功指標、ディメンション、オプションによって定義される。意思決定エージェントは、選択したビジネス指標を最大化するために、すべての顧客に対して最適なアクションを自動的に発見する。

### どのような指標で最適化できるのか？

収益、コンバージョン、ユーザー1人当たりの平均収益（ARPU）、顧客生涯価値（CLV）、利益、契約更新、その他のビジネスKPIなど、目標に沿ったあらゆるビジネス指標に対して最適化できる。

### Decisioning Studioのディメンジョンとは何か？

次元は、意思決定エージェントが成功指標を最大化するために引くことのできる*レバーの種類と*考えることができる。典型的なディメンションには、オファー、件名、クリエイティブ、チャネル、送信時間などがある。

### アクションバンクとは何か？

アクションバンクは、各次元「レバー」に対して、意思決定エージェントがアクセスできる*特定のオプションを*定義する。例えば、チャネル・ディメンジョンでは、意思決定エージェントがアクセスできる特定のチャネルを定義する。オファーディメンジョンでは、意思決定エージェントがテストできる特定のオファーを定義する。

### 意思決定エージェントは、私が設定していないアクションを取ることができるのか？

決定エージェントは、あなたが設定し、アクションバンクに追加したアクションしか取ることができない。つまり、可能なアクションはすべて、アクションバンクに入れたものの組み合わせで決まるということだ。

### 制約とは何か？

制約は、重要なビジネスルールを尊重するために、意思決定エージェントのアクションを制限する。例えば、対象外の地域の顧客に特定のオファーが選択されないようにしたり、意思決定エージェントが使用できる最大予算を設定したりすることができる。

### Decisioning Studio GoとDecisioning Studio Proの違いは？

Decisioning Studio Proには、Brazeの前方展開データサイエンスチームによるAIデシジョニングサービスサポートが含まれており、ビジネス成果を最大化するためのエージェントの設計と設定をサポートする。詳細については、[Decisioning Studio GoとDecisioning Studio Proを]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro)参照のこと。
