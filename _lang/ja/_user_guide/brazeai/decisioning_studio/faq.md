# よくある質問

> この記事は、Decisioning Studioに関するよくある質問への回答を提供する。

### 意思決定エージェントとは何か？

意思決定エージェントとは、特定のビジネス目標を達成するためにカスタマイズされた、BrazeAI Decisioning Studio™ のカスタム設定である。これは、選択した成功指標、次元、およびオプションによって定義される。意思決定エージェントは、選択したビジネス指標を最大化するため、顧客ごとに最適なアクションを自動的に発見する。

### どの指標を最適化できるのか？

目標に沿ったあらゆるビジネス指標に対して最適化できる。例えば、収益、コンバージョン、ユーザーあたりの平均収益（ARPU）、顧客生涯価値（CLV）、利益、契約更新、その他のビジネスKPIなどだ。

### Decisioning Studioにおけるディメンションとは何か？

次元とは、意思決定エージェントが成功指標を最大化するために操作できる*手段の種類*と考えることができる。典型的な要素には、オファー、件名、クリエイティブ、チャネル、送信時間などが含まれる。

### アクションバンクとは何か？

アクションバンクは、意思決定エージェントが各次元「レバー」に対してアクセス可能な*具体的な選択肢*を定義する。例えば、チャネル次元については、意思決定エージェントがアクセスできる特定のチャネルを定義する。オファー次元では、意思決定エージェントがテストできる特定のオファーを定義する。

### 意思決定エージェントは、私が設定していないアクションを取ることができるのか？

いいえ。意思決定エージェントは、あなたが設定しアクションバンクに追加したアクションのみを実行できる。これは、可能なすべてのアクションが、アクションバンクに入れるものの組み合わせによって定義されることを意味する。

### 制約とは何か？

制約は、意思決定エージェントが重要なビジネスルールを遵守するよう、そのアクションを制限する。例えば、特定のオファーが対象外の地域にいる顧客に選択されないようにしたり、意思決定エージェントが支出できる最大予算を設定したりすることが挙げられる。

### Decisioning Studio GoとDecisioning Studio Proの違いは何だ？

Decisioning Studio Proには、Brazeの先進的なデータサイエンスチームによるAI意思決定サービスのサポートが含まれる。これにより、ビジネス成果を最大化するためのエージェント設計と設定が可能となる。詳細については、[Decisioning Studio Go と Decisioning Studio Pro]({{site.baseurl}}/user_guide/brazeai/decisioning_studio/#decisioning-studio-go-vs-decisioning-studio-pro) の比較を参照のこと。
