# [![Braze Learning コース]({% image_buster /assets/img/bl_icon3.png %})](https://learning.braze.com/most-engaged-channel){: style="float:right;width:120px;border:0;" class="noimgborder"}インテリジェントチャネルフィルター

> **インテリジェントチャネル** フィルター（旧 **Most Engaged**）は、選択したメッセージチャネルが「ベスト」チャネルであるオーディエンスの一部を選びます。

## チャネルフィルターについて

![選択可能なチャネルがドロップダウンで表示されたインテリジェントチャネルフィルター。]({% image_buster /assets/img/intelligent_channel_filter.png %}){: style="float:right;max-width:40%;margin-left:10px;margin-top:10px;border:0"}

ここでベストとは、ユーザー履歴を踏まえエンゲージメントの可能性が最も高いチャネルを指します。チャネルには、メール、SMS、WhatsApp、ウェブプッシュ、モバイルプッシュ（対応する OS やデバイスを含む）を指定できます。

インテリジェントチャネルは、過去 6 か月の活動において、メッセージのやり取り（開封またはクリック）数を受け取ったメッセージ数で割った比率で、ユーザーごと・チャネルごとのエンゲージメント率を算出します。チャネルはそれぞれのエンゲージメント率でランク付けされ、最も高いチャネルがそのユーザーの「Most Engaged」になります。

メッセージが送信されるたび、またはユーザーがメッセージに反応するたびに、数秒以内にエンゲージメント率が再計算されます。1 通のメッセージに対する反応は 1 回だけカウントされます（同じメールの開封とクリックは 1 回のエンゲージメントとして扱われます）。

インテリジェントチャネルフィルターを有効にするには、メール・ウェブプッシュ・モバイルプッシュのいずれかのキャンペーン作成時に **ターゲットオーディエンス** で **インテリジェントチャネル** フィルターを選択します。

{% alert important %}
SMS チャネルのエンゲージメント率を算出するには、[SMS リンク短縮]({{site.baseurl}}/user_guide/message_building_by_channel/sms_mms_rcs/link_shortening/) で高度なトラッキングとクリックトラッキングを有効にしてください。このトラッキングがないと、[同率時の扱い]({{site.baseurl}}/user_guide/brazeai/intelligence_suite/intelligent_channel/#tie-breaking) により、エンゲージメント率 0% の SMS がインテリジェントチャネルとして選ばれる場合があります。
{% endalert %}

## 「データ不足」オプション

Braze が「ベスト」チャネルを判定するには、十分なデータが必要です。ユーザーは、利用可能な 3 チャネルのうち少なくとも 2 つで、チャネルあたり 3 通以上のメッセージを受け取っている必要があります（開封は必須ではありません）。

十分なメッセージを受け取っていないユーザーは、このフィルターの「データ不足」に該当します。その場合、いずれのチャネルでもターゲットにできます。

例えば、プッシュを好むユーザーとデータ不足のユーザーの両方に同じプッシュを送りたい場合は、インテリジェントチャネルフィルターで **モバイルプッシュ** を選び、**OR** で 2 つ目のインテリジェントチャネルフィルターを **データ不足** に設定します。メールを好むユーザー向けには、インテリジェントチャネルをメールにした別キャンペーンで対応できます。

![モバイルプッシュまたはデータ不足のインテリジェントチャネルフィルター。]({% image_buster /assets/img/intelligent_example.png %}){:style="border:none"}

{% alert note %}
[フリークエンシーキャップ]({{site.baseurl}}/user_guide/engagement_tools/campaigns/testing_and_more/rate-limiting/#delivery-rules) を無視するキャンペーンや Canvas ステップは、インテリジェントチャネルの対象外となり、データ要件に含まれません。
{% endalert %}

同率時の扱い、届かないチャネル、オーディエンスサイズなどのベストプラクティスは、左側の目次または Braze ダッシュボードのヘルプから同じ記事の詳細版を参照してください。
