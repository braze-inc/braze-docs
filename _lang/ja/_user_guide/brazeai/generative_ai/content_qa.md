# BrazeAI<sup>TM</sup> によるコンテンツ QA

> <sup>BrazeAITMで</sup>コンテンツをQAする方法を学習し、スペルミス、文法の問題、不適切なトーン、不快な言葉遣いを、送信を押す前にキャッチできるようにしよう。

## サポートされている機能

コンテンツの品質向上のために、以下の機能がサポートされています。

| 機能                     | 説明 |
|----------------------------|-------------|
| スペルと文法のチェック | メッセージのスペルミスや文法ミスを自動的にチェックします。コンテンツの全体的な正確性を向上させるために、修正項目を提案し、推奨事項を提示します。 |
| トーン分析              | メッセージのトーンを評価し、潜在的な問題を特定します。意図したトーンを目的のコミュニケーションスタイルと一致させ、誤解や意図しない無礼を避けるために役立ちます。 |
| 失礼な言葉遣いの検出 | 失礼な、または不適切な言葉遣いがないかメッセージをスキャンするため、コンテンツを修正して丁寧なコミュニケーションを維持できます。 |
| 偶発的なコンテンツのチェック   | 意図せずに追加されたコード (テストユーザーに対してレンダリングされなかった Liquid コードを含む)、マークアップ言語、またはテストメッセージが含まれている場合に検出します。 |
| 多言語サポート     | OpenAI は公式にサポートしていませんが、GPT は[複数の言語](https://openai.com/research/gpt-4#:~:text=GPT%2D4%203%2Dshot%20accuracy%20on%20MMLU%20across%20languages)を理解できます。Brazeは、コピーをOpenAIに送信する際に、コピーの言語やローカライゼーションに関する情報を一切渡さないので、あなたが書いている言語によって結果が異なる可能性があることを覚えておいてほしい。 |

## BrazeAI<sup>TM</sup> を使用したコンテンツの QA

{% alert note %}
この機能は、現時点では SMS チャネル、Android プッシュ通知チャネル、iOS プッシュ通知チャネルのみで利用できます。
{% endalert %}

1. モバイルプッシュまたは SMS メッセージを作成したら、[**テスト**] タブに移動します。
2. [**AI によるコンテンツ QA**] セクションを見つけます。
3. [**コンテンツのテスト**] をクリックします。

![[テスト] タブの [AI によるコンテンツ QA] セクション。]({% image_buster /assets/img/content_qa_ai.png %})

## ベストプラクティス

AI によるコンテンツ QA の機能を最大限に活用するために、以下の点を考慮してください。

- **メッセージの校正:**コンテンツチェッカーはエラーの特定に役立ちますが、それでもコンテンツの校正を手作業で行うことが不可欠です。AI が生成した提案は有用なガイドとして頼りになりますが、正確さを期すために、最終的な判断は自分でしてください。
- **トーン分析の解釈:**トーン分析の結果は主観的で、AI モデルの判断力に基づいています。有用なインサイトを提供することもできますが、意図するトーンと会話の文脈を考慮して、適切な調整を行ってください。
- **フラグが設定された失礼な言葉遣いのダブルチェック:**失礼な言葉遣いの検出は高い信頼性で設計されていますが、誤検知によりフラグを設定する可能性があります。フラグが設定された箇所を注意深く見直し、必要に応じて適切な変更を加えてください。
