# BrazeAIでコピーを生成する

> AI のコピーライティングアシスタントは、OpenAI が所有するサードパーティプロバイダーの GPT コピー生成ツールに製品名や簡単な説明を渡し、メッセージングで使用できる、人間が書いたように自然なマーケティングコピーを生成します。この機能は、Braze ダッシュボード内のほとんどのメッセージ作成画面でデフォルトで使用できます。

## コピーの生成

### ステップ 1:AI コピーライターを起動する

メッセージ作成画面から、<i class="fa-solid fa-wand-magic-sparkles"></i>[**AI コピーライターを起動**] を選択します。

アプリ内メッセージのドラッグ＆ドロップエディターで、テキストブロックを選択し、ブロックのツールバーで [<i class="fa-solid fa-wand-magic-sparkles" title="AI コピーライター"></i>] を選択します。

### ステップ 2:詳細を入力する

入力フィールドに製品名または説明を入力し、出力のおおよその長さを選択します。

チャネル固有のベストプラクティスに基づいて、出力長に応じた特定のチャネルを選択したり、短文 (1 文)、中文 (2～3 文)、長文 (1 段落) のいずれかを選択したりできます。

### ステップ 3:さらにカスタマイズする（オプション）

コピーをさらにカスタマイズするには、以下の方法があります。

- **ブランドガイドラインを適用する:**[BrazeAI<sup>TM</sup> でブランドガイドラインを生成]({{site.baseurl}}/user_guide/brazeai/generative_ai/brand_guidelines)したら、そのガイドラインを使用してコピーを生成できます。
- **トーンを選択する:**各トーンによって、異なる文体のコピーが生成されます。ブランドボイスに最もマッチするトーンを選択してください。
  
  トーンを選択すると、OpenAI に送信されるプロンプトにスタイル指示が追加されるため、入力内容、チャネルの長さ、ブランドガイドライン、モデルによって出力結果が異なる場合があります。
  
  各トーンがデフォルトで意図する効果は以下のとおりです。
  - **フォーマル:** よりプロフェッショナルで洗練された表現です。完全な文章、より丁寧な言葉遣い、スラングを最小限に抑えます。
  - **ストレート:** より直接的で簡潔な表現です。形容詞が少なく、「マーケティング的な誇張」を抑え、行動喚起がより明確です。
  - **カジュアル:** よりリラックスした会話調の表現です。親しみやすいフレーズ、シンプルな言葉、軽やかなエネルギーが特徴です。
  - **パーソナル:** より1対1で共感的な表現です。「あなた」をより多く使い、特にメッセージ作成時に {% raw %}`{{${first_name}}}`{% endraw %} のようなパーソナライゼーションを追加すると、よりカスタマイズされた印象を与えます。
  - **目を引く:** より注目を集める表現です。インパクトのあるフレーズ、高いエネルギー、強力なフックと CTA が特徴です（他のトーンよりも「プロモーション」的な印象になることが多いです）。
  - **洗練:** より格調高く、上品な言葉遣いです。カジュアルさを抑え、より「プレミアム」なポジショニングを目指します。
  - **プロフェッショナル:** ビジネスライクで明確な表現です。フォーマルよりもモダンで親しみやすく、権威性を維持します。
  - **控えめ:** より柔らかく、押しつけがましくない表現です。直接的な命令を減らし、提案的なフレーズを多く使います。
  - **緊急:** 即時性と時間的な緊迫感を強調します。より強力な CTA、期限、希少性の手がかりが特徴です。
  - **エキサイティング:** よりエネルギッシュで熱意のある表現です。ポジティブな感情やお祝いを強調します（「目を引く」のフック重視のアプローチよりも、盛り上がりを重視する傾向があります）。
 
  
- **過去のキャンペーンデータを参照する:**有効にすると、キャンペーンまたはキャンバスステップで送信された過去のモバイルプッシュ通知が、新しいコピーを生成するための文体の参考として使用されます。詳細については、「[過去のキャンペーンデータの使用](#past-campaign-data)」を参照してください。
- **コピーを自動翻訳する:**コピーの出力言語として別の言語を選択できます。生成されたコンテンツはその言語で出力されます。

### ステップ 4:コピーを生成する

完了したら、[**生成**] を選択します。入力された情報を使用して、GPT にコピーの作成を指示します。応答は OpenAI から取得され、ユーザーに提供されます。詳細については、「[データはどのように使用されて、OpenAI に送信されるのですか?](#ai-policy)」を参照してください。

この機能は、OpenAI の[コンテンツポリシー](https://beta.openai.com/docs/usage-guidelines/content-policy)に違反する攻撃的なコンテンツに関する応答をフィルターで排除します。
{% endalert %}

## 過去のキャンペーンデータについて {#past-campaign-data}

プッシュを出力長として使用する場合、[**過去のキャンペーンデータを参照する**] を選択すると、ランダムに選択された以前のモバイルプッシュキャンペーンが OpenAI に送信され、GPT がそのコピー生成の基礎として使用できるようになります。現在、AI コピーライターは Liquid 構文を含まないプッシュキャンペーンを OpenAI に送信します。この機能を活用しない場合は、このボックスをオフのままにしてください。Braze および OpenAI でのデータの使用方法の詳細については、次のセクションを参照してください。 

[ブランドガイドライン]({{site.baseurl}}/user_guide/brazeai/generative_ai//brand_guidelines/)と組み合わせて使用すると、ブランドガイドラインと過去のキャンペーンデータの両方が最終出力に組み込まれます。
