# BrazeAI Operatorでサポートチケットを提出する

> Brazeダッシュボードを離れることなく、BrazeAI Operator<sup>TM</sup>から直接サポートチケットを提出する方法を説明します。

## チケットの提出方法

1. Operatorに質問し、<i class="fa-regular fa-circle-question"></i> **サポートチケットを提出**を選択します（またはOperatorにチケットの提出を依頼します）。
2. 問題のタイトルと説明を入力します。
3. 提案された添付ファイルと含めるコンテキストを確認します。
4. 確認して**送信**を選択します。

## 自動的に含まれるコンテキスト

Operatorからチケットを提出すると、以下のコンテキストが自動的に含まれます。

- Operatorチャットセッションの最近のメッセージ
- Operatorを開いた時点のアクティブページのページコンテキスト
- ワークスペース名
- 関連するログとセッションの詳細

テキストフィールドを使用するか、ファイルを添付することで、手動でコンテキストを追加できます。

## 迅速な解決のためのヒント

- 問題を再現するためのステップの詳細な説明を追加してください。
- 関連するエラーメッセージや予期しない動作を含めてください。
- 送信前に自動提案された添付ファイルを確認してください。
