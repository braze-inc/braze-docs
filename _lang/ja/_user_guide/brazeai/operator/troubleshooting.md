# BrazeAI Operator のトラブルシューティング

> BrazeAI Operator<sup>TM</sup> のよくある問題と対処方法です。

## よくある問題

| 問題 | 対処方法 |
| --- | --- |
| 応答がない | ページを再読み込みし、オペレータパネルを再度開いてください。 |
| 話題がずれた回答 | 質問をより具体的にし直してください。対象の機能やワークフローに触れます。 |
| エラーメッセージ | オペレータがコンテンツをストリーミングできない場合、「再試行」プロンプトが表示されることがあります。オペレータが一時的に利用できないか、接続が切れている可能性があります。数分後に再試行してください。 |
| 1日の利用上限に達した | 会社全体の利用上限に達しています。リセット（24時間ごと）までお待ちください。 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 制限事項

- **データアクセス:** オペレータは表示中のページの内容を参照できます。過去のキャンペーン、カタログ、カスタム属性など、一部の会社固有データも参照できます。ただし、新規レポートの作成やレポート用クエリの実行は現時点ではできません。
- **利用上限:** オペレータには会社全体の1日あたりの利用上限があり、24時間ごとにリセットされます。上限に達すると、チャットに「1日の利用上限に達しました」と表示され、リセットまで新しいリクエストはできません。
