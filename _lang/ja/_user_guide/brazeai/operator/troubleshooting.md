# BrazeAI オペレーターのトラブルシューティング

> BrazeAI Operator<sup>TM</sup>のよくある問題とソリューション。

## よくある問題

| 問題 | トラブルシューティング |
| --- | --- |
| 返答がない | ページを更新して、オペレーターパネルを再度開封してみるといい。 |
| 脱線した回答 | 質問をもっと具体的に言い換えるんだ。質問されている機能やワークフローについて言及する。 |
| エラーメッセージ | オペレーターがコンテンツをストリーミングできない場合、「もう一度試す」というプロンプトが表示されることがある。オペレーターが一時的に利用できないか、接続が中断された可能性がある。数分後に再試行せよ。 |
| 1 日の使用上限を超えました | 会社全体での使用制限に達した。制限がリセットされるのを待て（24時間ごとにリセットされる）。 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 制限事項

- **データアクセス：**オペレーターはアクティブなページの内容を見ることができる。また、過去のキャンペーンやカタログ、カスタム属性など、企業固有のデータも参照できる。ただし、現時点では新しいレポートを生成したり、レポートクエリを実行したりすることはできない。
- **使用制限：**オペレーターには、会社全体で適用される1日あたりの使用制限があり、24時間ごとにリセットされる。制限に達した場合、チャットに「1日の使用制限を超えました」というバナーが表示され、制限がリセットされるまでそれ以上のリクエストはできない。
