# BrazeAI Operator のトラブルシューティング

> BrazeAI Operator<sup>TM</sup>のよくある問題とソリューション。

## よくある問題

| 問題 | トラブルシューティング |
| --- | --- |
| 応答がない | ページを更新して、Operator パネルを再度開いてみてください。 |
| 的外れな回答 | 質問をより具体的に言い換えてください。質問対象の機能やワークフローについて明示してください。 |
| エラーメッセージ | Operator がコンテンツをストリーミングできない場合、「もう一度試す」というプロンプトが表示されることがあります。Operator が一時的に利用できないか、接続が中断された可能性があります。数分後に再試行してください。 |
| 1日の使用上限を超えました | 会社全体での使用上限に達しました。上限がリセットされるまでお待ちください（24時間ごとにリセットされます）。 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

## 制限事項

- **データアクセス:** Operator はアクティブなページのコンテンツを確認できますが、ドラッグ＆ドロップエディター内のコンテンツは対象外です。また、過去のキャンペーン、カタログ、カスタム属性など、会社固有のデータも参照できます。ただし、現時点では新しいレポートを生成したり、レポートクエリを実行したりすることはできません。
- **使用上限:** Operator には会社全体で適用される1日あたりの使用上限があり、24時間ごとにリセットされます。上限に達した場合、チャットに「1日の使用上限を超えました」というバナーが表示され、上限がリセットされるまでそれ以上のリクエストはできません。
