# 推進コードの作成

> キャンペーンs とキャンバスで昇格コードs を作成する方法について説明します。

## 昇格コード一覧の作成 {#create}

### ステップ 1: 新しいリストを作成する

ダッシュボードで、**データ設定** > **プロモーションコード**に移動し、**プロモーションコードリストの作成**を選択します。

![プロモーションコードを作成するボタンです。]({% image_buster /assets/img/promocodes/promocode1.png %})

### ステップ 2:詳細を入力する

1. プロモーションコードのリストに名前を付け、任意の説明を追加する。
2. 次に、プロモーション・コードのコード・スニペットを作成する。 

コードスニペットを作成する際に考慮すべき詳細を次に示します。

- 保存後にコードの抜粋コードを変更することはできません。
- スニペットは大文字と小文字を区別する。たとえば、システムは"Birthday_promo" と"birthday_promo" を2 つの異なるスニペットとして認識します。
- この一連のプロモーションコードを参照するには、Liquid でスニペット名を使用します。
- コードスニペットが別のリストでまだ使用されていないことを確認します。

![コードのスニペット"springSale2025" とコードのスニペット"spring25" という名前のプロモーションリスト。]({% image_buster /assets/img/promocodes/promocode3.png %}){: style="max-width:80%"}

### ステップ 3:プロモーションコードオプションの選択

各プロモーションコード・リストには、作成時に設定された有効期限と時間がある。最大有効期限は、リストを作成または編集している日から6 か月間です。

その時間内であれば、有効期限を何度でも変更・更新することができる。この有効期限のアプリは、この一覧に追加されたすべてのコードにあります。期限切れになると、コードsはBrazeから削除され、その一覧のコードの抜粋を呼び出すすべてのメッセージは送信されません。

![残りのすべてのコードs が2025年4 月30 日の午前12 時に期限切れになる期限設定を表示します。]({% image_buster /assets/img/promocodes/promocode4.png %}){: style="max-width:80%"}

また、オプションでカスタマイズした閾値アラートを設定するオプションもある。設定されている場合、これらの警告は指定された受信者をメールします。これは、リストがこのリスト内の使用可能な昇格コードs で低レベルで実行されている場合、または昇格コードの一覧が期限切れに近い場合のいずれかです。受信者は1日1回通知されます。

![たとえば、"マーケティング@abc.com" に通知するしきい値アラートです。プロモーションコードの一覧が5 日間で期限切れになった場合です。]({% image_buster /assets/img/promocodes/promocode5.png %}){: style="max-width:80%"}

### ステップ 4: プロモーションコードのアップロード

Braze ではコードの作成や償還が管理されません。つまり、プロモーションコードを CSV ファイルに生成し、Braze にアップロードする必要があります。 

CSV ファイルが次のガイドラインに従っていることを確認します。

- プロモーションコードの列も含まれています。
- 1行に1つのプロモーションコードがある。

[Voucherify]({{site.baseurl}}/partners/ecommerce/loyalty/voucherify/) との組み込み統合または [Talon.One]({{site.baseurl}}/partners/ecommerce/loyalty/talonone/) を使用して、プロモーションコードを作成しエクスポートすることができます。

{% alert important %}
最大ファイルサイズは 100 MB、最大リストサイズは 20 MM の未使用コードです。間違ったファイルがアップロードされた場合は、新しいファイルをアップロードして前のファイルを置き換えます。
{% endalert %}

1. アップロードが完了したら、「**リストを保存**」を選択し、入力したすべての詳細とコードを保存する。

!["springsale&quotという名前のCSVファイル。正常にアップロードされました。]({% image_buster /assets/img/promocodes/promocode7.png %})

{:start="2"}
2\.save を選択すると、**Import History** に新しい行アプリが表示されます。
3\.インポートが完了したかどうかを確認するためにテーブルを更新するには、テーブルの上部にある <span style="font-size: 14px;margin-bottom: .5rem;height: 16px;width: 16px;" class="fas fa-sync" ></span> [**同期**] を選択します。

![アップロード edになる過程でのプロモーションコードs。]({% image_buster /assets/img/promocodes/promocode8.png %})

{% alert note %}
大きいファイルをインポートするには数分かかります。待つ間、インポートが進行している間はページを離れて他の作業をすることができます。インポートが終了すると、ステータスはテーブルの**Complete**に変わります。
{% endalert %}

## プロモーションコード一覧を更新する

リストを更新するには、既存のリストを1つ選択する。名前、説明、リストの有効期限、しきい値アラートを変更できる。また、新しいファイルをアップロードし、「**Update List**」を選択することで、リストにさらにコードを追加することもできる。一覧内のすべてのコードs は、インポート日に関係なく、同じ有効期限を持ちます。

{% alert important %}
プロモーションコードは削除できません。
{% endalert %}

### 不正な昇格コード一覧の変更 

間違った昇格コードs でCSVファイルをアップロードし、**リストを保存** を選択した場合は、次のいずれかの方法で解決できます。

- リスト全体を非推奨にします。キャンペーンs、キャンバス、またはテンプレートs で、現行のプロモーションコード一覧の使用を中止します。次に、正しいコードs でCSVファイルをアップロードし、メッセージングで使用します。
- 間違ったコードs を使用します。不正なプロモーションコード一覧から、不正なコードがすべて使用されるまで、プロモーションコードをプレースホルダに送信するキャンペーンを作成します。次に、正しい昇格コードsを同じ一覧にアップロードします。
