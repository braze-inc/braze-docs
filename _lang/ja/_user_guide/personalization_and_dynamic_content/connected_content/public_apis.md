# パブリックAPI

> コネクテッド・コンテンツに利用できるAPIは、さまざまなものが公開されている。パブリックAPIでは、メッセージにカスタムデータを挿入することができます。コネクテッド・コンテンツに使用できるパブリックAPIのリストをまとめた。しかし、世の中にはもっと多くのAPIがあり、コネクテッド・コンテンツの潜在的な用途を幅広く提供している。  

共有するAPIをお持ちの方は、[success@braze.com](mailto:success@braze.com)までご連絡ください！

{% alert note %}
公開APIは利用制限や料金制限を受けることがある。 必ずAPIドキュメントを読み、APIプロバイダーに使用目的について問い合わせること。
{% endalert %}

| --------- | --- |

[| ガーディアン API][9]｜ガーディアンの記事データ（見出し、トピック、URL、日付、抄録など）を提供します。
