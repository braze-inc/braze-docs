# テンプレートを管理する

> テンプレートをアーカイブまたは複製すると、テンプレートの整理と管理が容易になります。この参考記事では、Braze ダッシュボードの [**テンプレート**] セクションでテンプレートをアーカイブおよび複製する方法について説明します。

## テンプレートを複製する

### 個々のテンプレートを複製する

![][8]{: style="float:right;max-width:15%;margin-left:15px;"}

個々のテンプレートを複製するには、<i class="fas fa-cog"></i>個々のテンプレートの歯車アイコンをクリックし、ドロップダウンメニューから [**複製**] を選択します。
<br><br>

{% alert note %}
[コンテンツブロック]({{site.baseurl}}/user_guide/engagement_tools/templates_and_media/content_blocks/)テンプレートの場合、ドラフトコピーが作成されます。他のすべてのテンプレートでは、新しい複製コピーが自動的に作成されます。
{% endalert %}

### 複数のテンプレートを複製する

{% raw %}

テンプレート名の横にあるチェックボックスを選択すると、複数のテンプレートを複製できます。まず、テンプレートを選択し、表示される [**複製**] ボタンをクリックします。

重複したテンプレートは、[**最終編集日時**] 列をソートすることで検索できます。デフォルトでは、新しいテンプレートには `Copy of {{original template name}}` という名前が付けられます。

{% endraw %}

![ユーザーが2つのテンプレートを選択して [複製] をクリックすると、合計4つのテンプレートが生成され、テンプレートが最後に編集された時間順にソートされることを示す GIF。] [9]

## テンプレートをアーカイブする

{% alert note %}
[古いナビゲーション]({{site.baseurl}}/navigation)を使用している場合、「**テンプレート**」は「**テンプレートとメディア**」です。
{% endalert %}

### 個々のテンプレートをアーカイブする

![拡張設定] ドロップダウンメニューには次の3つのオプションが表示されます。[編集]、[アーカイブ]、[複製]。[アーカイブ] オプションが強調表示されています。][10]{: style="float:right;max-width:20%;margin-left:15px;"}

個々のテンプレートをアーカイブするには、テンプレートグリッド画面の設定アイコンをクリックし、[**アーカイブ**] を選択します。

### 複数のテンプレートをアーカイブする

複数のテンプレートをアーカイブするには、アーカイブする各テンプレートの横にあるチェックボックスを選択します。複数のテンプレートを選択したら、[**選択した項目をアーカイブ**] をクリックします。テンプレートグリッドの [**表示**] で [**アーカイブ済み**] を選択すると、アーカイブされたテンプレートを見つけることができます。

![保存済みのドロップアンドドロップメールテンプレート] セクションには、選択した2つのテンプレートが表示されます。「プレミアムを試すテンプレート」と「ウェルカムテンプレート」。[選択した項目をアーカイブ] ボタンはユーザーによって強調表示されます。] [11]

{% alert important %}
現在、[リンクテンプレート]({{site.baseurl}}/user_guide/message_building_by_channel/email/link_templates/#link-templates)をアーカイブすることはできません。
{% endalert %}
