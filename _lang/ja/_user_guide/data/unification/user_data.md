{% alert important %}
Braze は、セッション数が500万を超えるユーザー（「ダミーユーザー」）を禁止またはブロックし、SDK イベントの取り込みを停止します。これは通常、誤ったインテグレーションの結果であるためです。正当なユーザーでこのような事態が発生した場合は、Braze アカウントマネージャーにご連絡ください。
{% endalert %}
