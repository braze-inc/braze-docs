{% alert important %}
Brazeは、セッション数が 500 万を超えるユーザー (「ダミーユーザー」) の禁止またはブロックを行い、SDK イベントの取り込みを停止します。これは通常、誤った連携の結果であるためです。正規ユーザーに対して禁止またはブロックが発生しているとわかった場合は、Braze アカウントマネージャーに連絡してください。
{% endalert %}
