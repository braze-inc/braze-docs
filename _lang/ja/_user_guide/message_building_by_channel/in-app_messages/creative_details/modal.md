# モダルアプリ内メッセージ

> モダルは、デバイスの画面の中央に表示され、バックグラウンドでアプリから目立つようにする画面オーバーレイが表示されます。これらは、ユーザがセールやプレゼンテーションを利用することを、あまり微妙に示唆するものではありません。
