# 10DLCアプリケーションに必要な情報

10DLCを登録するためにBrazeからヘルプを購入した場合、当社との契約で合意された通り、以下のメールに記載されているすべての必要情報を当社に送信する必要がある。

関連するスクリーンショットや添付ファイルを含め、すべてを[smsapplications@braze.com](mailto:smsapplications@braze.com) までお送りいただきたい。10DLCについてご質問がある場合は、[Braze SMS 10DLCガイドを]({{site.baseurl}}/assets/pdf/SMS_10DLC_Guide.pdf)ご覧になるか、Brazeカスタマーサクセスマネージャーまでお問い合わせください。
