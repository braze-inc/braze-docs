# セルフサービスのカスタムドメイン

> このページでは、Brazeダッシュボードで独自のカスタムドメインを設定する方法について説明する。カスタムドメインを使用すると、汎用的な短縮リンクやBrazeドメイン（`brz.ai`）ではなく、自社ブランドのアイデンティティを反映した短縮リンクを利用できる。これにより、SMSリンクに対するユーザーの信頼性とキャンペーンのエンゲージメントが向上する。

セルフサービスのカスタムドメイン機能により、SMS、RCS、WhatsApp用の独自ドメインを、Brazeダッシュボードから直接設定・管理できる。最大10個のカスタムドメインを、一箇所で簡単に追加・監視・管理できる。

## セルフサービス型カスタムドメインの利点

- **簡素化された設定：****会社設定**ページでドメインを設定すれば、設定時間を短縮できる。
- **透明性の向上：**ダッシュボードのバナーを通じて、ドメインの設定ステータスに関するリアルタイムの更新情報を受け取る。
- **事前通知：**カスタムドメインが接続された時や設定エラーが発生した時に、すぐに通知を受け取る。

## ドメイン要件

- ドメインは、お客様が調達し、所有し、管理しなければなりません。これはGoDaddy、Amazon Route 53、Google Domainsなどのドメイン登録業者を通じて行うことができる。
- この機能に使用するドメインは次の通りでなければならない：
  - 独自のもの（あなたの Web サイトのドメインとは異なる）
  - Webコンテンツをホストすることはできない
    - 独自のサブドメインも使える。例えば、ドメインには  や  `braze.com``sms.braze.com`といったサブドメイン`whatsapp.braze.com`を持つ可能性がある。

## カスタムドメインの委任

カスタムドメインをBrazeに委任する必要がある。これにより、当社のリンク短縮サービスおよびクリックトラッキングサービスとの適切なルーティングとインフラ互換性を実現できる。ドメインを Braze に委任すると、サービスが失効しないように、証明書の更新が自動的に処理されます。 

## カスタムドメインを追加する

1. Brazeでは、**会社設定**＞**SMS/RCSとメッセージングアプリドメイン**に移動する。
![SMS/RCSおよびメッセージングアプリのドメイン」ページには、いくつかのドメインがリストされている。]({% image_buster /assets/img/main_page.png %})

{: start="2"}
2\.**「ドメインを追加」**を選択して、新しいカスタムドメインの設定を開始する。
3\.購入したカスタムドメインをアプリ内の入力欄に入力する。入力欄は既存の検証ロジックで適切な形式かどうかを確認する。その後**「次へ」**を選択し「**送信**」する。

![「SMS/RCSおよびメッセージングアプリドメイン」ページにある「ドメインを追加」ボタン。]({% image_buster /assets/img/custom_domain_button.png %}){: style="max-width:70%;"}

{: start="4"}
4. 技術チーム（開発やITなど）に、表示されるCloudflareのDNSレコードの詳細でDNS設定を更新させること。技術チームは45日以内にこれらの詳細でDNSレコードを更新しなければならない。
  - DNSレコードを更新するのに追加の時間が必要な場合、プロセスを再起動してドメイン用の新しいDNSレコードセットを生成できる。

BrazeはDNS設定を約30分ごとに確認し、更新の有無をチェックする。

![DNSレコードの設定を完了させるには、以下の3つのステップを実行する必要がある。]({% image_buster /assets/img/dns_record.png %})

{% alert note %}
ドメインの進捗は自動的に保存される。途中で終了する必要がある場合、**SMS/RCSおよびメッセージングアプリのドメイン**ページで保留中のドメインエントリを選択すれば、後で再開できる。
{% endalert %}

### 継続的な管理と使用

ドメインの確認が完了すると、カスタムドメインは**SMS/RCSおよびメッセージングアプリドメイン**ページの表にステータス表示付きで表示される。接続済みドメインは、複数のサブスクリプショングループやワークスペース、SMS、RCS、WhatsAppチャネルをまたいで即時利用可能だ。

![カスタムドメインとステータスのリスト。]({% image_buster /assets/img/custom_domain_statuses.png %}){: style="max-width:60%;"}

アクティブなドメインに問題が発生した場合、ライブ監視がBrazeダッシュボードで通知する。これによりカスタムリンクは使用可能な状態を維持できる。問題が発生した場合は、アプリ内のエラー詳細を参照するか、Braze[サポートに]({{site.baseurl}}/braze_support/)連絡して支援を受けよ。

## サブスクリプショングループへのカスタムドメインの割り当て

設定後、カスタムドメインは1つまたは複数のSMS、RCS、WhatsAppのサブスクリプショングループに割り当てられる。

1. **オーディエンス**＞**サブスクリプショングループ管理**へ移動する。
2. リストから自分のサブスクリプショングループを見つけて選択する。
3. **サブスクリプショングループの**詳細で、**リンク短縮ドメイン**としてカスタムドメインを選択する。

![サブスクリプショングループの設定により、リンク短縮ドメインを選択できます。]({% image_buster /assets/img/custom_domain.png %})

リンク短縮機能を有効にして送信されたキャンペーンは、SMS、RCS、またはWhatsAppのサブスクリプショングループに関連付けられた割り当てドメインを使用する。

![[メッセージ] ボックスのドメインとは異なる短縮リンクドメインがある SMS メメッセージ作成画面のプレビュー。]({% image_buster /assets/img/custom_domain2.png %})

## よくある質問

### 委任されたドメインは複数のサブスクリプショングループ間で共有できますか？

はい。1 つのドメインを複数のサブスクリプショングループで使用できます。そのためには、各サブスクリプショングループが関連付けられるドメインを選択します。

### 委任されたドメインは複数のワークスペース間で共有できますか?

はい。ドメインは、複数のワークスペース内のサブスクリプショングループに関連付けることができます。ワークスペースが同じ会社内に含まれていることが前提です。

### カスタムドメインはいくつ追加できるのか？

ダッシュボードごとに最大10個のカスタムドメインを追加できる。

### DNSレコードを45日以内に更新しなかったらどうなるのか？

CloudflareのDNSレコードの詳細は45日後に期限切れとなるが、同じドメインで設定プロセスを再開すれば、Brazeが新しいDNSレコードを生成し、設定期間を延長できる。

### DNS更新中にエラーが発生した場合、通知されるのか？

はい。エラーが発生した場合、Brazeダッシュボードにバナーが表示され、問題の詳細と解決のステップが記載される。 

### カスタムドメインを複数のチャネルで使えるか？

はい。カスタムドメインが確認されると、ダッシュボード内の全ワークスペースにわたるSMS、RCS、WhatsAppのすべてのサブスクリプショングループで使用できる。 

### 質問があったり、さらにサポートが必要だったらどうすればいいか？

カスタムドメインの設定と管理に関する詳細なガイダンス（トラブルシューティングステップや技術要件を含む）については、[サポートに連絡すること]({{site.baseurl}}/braze_support/)。
