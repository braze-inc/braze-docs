# Application-to-Person 10桁のロングコード

> A2P 10DLCは、企業が標準の10桁のロングコード(10DLC)電話番号を介してApplication-to-Person(A2P)タイプのメッセージングを送信できるようにする米国のシステムを指します。これらの登録されたロングコードには、標準のロングコードよりも高いスループット、優れた配信品質、および改善されたコンプライアンスが付与されます。

{% alert important %}
現在、米国のロングコードを持っている、または米国の顧客に送信するすべての顧客は、10DLCのロングコードを登録する必要があります。これを怠ると、すべてのメッセージのフィルタリングが厳しくなります。この申請プロセスには4〜6週間かかります。
{% endalert %}

## なぜ必要なのか

10DLCサービスは、特にロングコードを使用したA2Pメッセージングを容易にするために作成されました。歴史的に、ロングコードは個人間(P2P)メッセージングを目的としていましたが、マーケティング上の理由で使用されると、スループットの制限とフィルタリングの強化によってビジネスが制約される原因となっていました。 

10DLCは、以下を提供することで、これらの問題を軽減するのに役立ちます。
- **より高いスループット**:10DLC 番号は、通常のロングコードよりも多くのメッセージをサポートします。
- **配信品質の向上**:10DLC 番号は A2P トラフィック用に指定されているため、これらの番号で送信されたメッセージは、通常のローカル ロング コードで送信されたメッセージよりも受信者に到達する可能性が高く、通信事業者によってフィルタリングまたは拒否される可能性が低くなります。
- **コンプライアンスの向上**:商用テキストメッセージにローカルロングコードを使用することは、 [CTIA](https://api.ctia.org/wp-content/uploads/2019/07/190719-CTIA-Messaging-Principles-and-Best-Practices-FINAL.pdf) ガイドラインに違反します。10DLC番号はマスメッセージング用に指定されており、ブランドはショートコードに頼らずに業界の規制に準拠することができます。
- **予算にやさしい**:10DLCは、SMSの送信を開始したい、または少量のSMSを送信したい企業に最適なオプションです。1日に100,000件を超える大量のメッセージを送信するブランドには、ショートコードの使用をおすすめします。 

2019年以降、通信事業者は商用メッセージングに10DLCを採用し始めており、VerizonとAT&Tは現在10DLCをサポートしており、すべての主要な通信事業者がまもなく追随すると予想されます。短期的には不便が生じるかもしれませんが、長期的には、顧客は不要なメッセージから消費者を保護しながら、より良い配信率を享受できます。 

## 知っておきたいこと

### アクセス

A2P 10DLCへのロングコードの登録には4〜6週間かかります。

### コスト 

A2P 10DLCへの登録には、いくつかの種類の料金が含まれる場合があります。

|料金の種類 |説明 |
| -------- | ---------- |
|参加費 |米国のすべての主要なネットワークでブランドとユースケースを登録する際に適用されるわずかな料金。|
|二次審査料 |ブランドは、 [ブランド信頼スコア](#trust-score) に異議を申し立て、全体的なスループットを向上させるための二次審査プロセスをリクエストできます。このプロセスには料金がかかります。|
|キャリア料金 |10DLC 登録後にユーザーに送信される送信 SMS および MMS メッセージに対して通信事業者が請求する料金。2021 年 10 月 1 日以降、未登録のトラフィック (標準のロングコード) では、登録済みのトラフィック (10DLC) よりもキャリア手数料が高くなります。|
{: .reset-td-br-1 .reset-td-br-2}

Twilio 10DLCの記事を参照して、最新の [料金見積もり](https://support.twilio.com/hc/en-us/articles/1260803965530-What-pricing-and-fees-are-associated-with-the-A2P-10DLC-service-)をご確認ください。

### スループット

10DLC のメッセージスループットは、ブランドの信頼スコア、毎日のメッセージ制限、メッセージングのユースケースなど、いくつかの要因によって異なります。

#### ブランド信頼スコア {#trust-score}

Campaign Registry(TCR)は、レピュテーションアルゴリズムを使用して会社に関連する特定の基準をレビューし、各ブランドのメッセージングスループットを決定する信頼スコアを割り当てるサードパーティエージェンシーです。この信頼スコアは、顧客が US 10DLC メッセージングに登録するときに割り当てられます。信頼スコアが高いほど、1秒あたりのメッセージ数(MPS)が向上します。 

|    |トラストスコア |AT&T |T-Mobileの |ベライゾン |
| --- | ----------- | ---- | -------- | ------- |
|高 |75〜100 |75メガピクセル|75メガピクセル|75メガピクセル|
|ミディアム |50〜74 |40メガピクセル|40メガピクセル|40メガピクセル|
|該当なし |1-49 |4メガピクセル |4メガピクセル |4メガピクセル |
{: .reset-td-br-1 .reset-td-br-2 .reset-td-br-3  .reset-td-br-4}

{% alert tip %}
Russel 3000 Indexに上場している企業には、10DLCの登録と審査を経て、高いスループットとブランド信頼スコアが与えられます。
{% endalert %} 

#### 毎日のメッセージ制限

毎日の制限は、ブランドの信頼スコアに応じて 2,000 から 200,000 メッセージの範囲で、すべてのロングコードに適用されます。高いブランド信頼スコアは、毎秒 60 件のメッセージのスループットをもたらしますが、通信事業者が設定した 1 日のメッセージ制限は引き続き適用されます。つまり、ブランドの毎日のピークメッセージが課せられた毎日の制限よりも高い場合、ショートコードの方が適しているということです。 

#### メッセージングのユースケース

スループットは、選択したメッセージングのユースケースの種類によっても影響を受けます。ほとんどのお客様は、標準的なマーケティングまたは混合マーケティングのユースケースに分類されます。他のあまり一般的でないユースケースは、異なるスループット値の影響を受ける可能性があります。

ユースケースによって、最大スループットを達成するために必要な信頼スコアは異なります。次の表に、標準的なユースケースと一般的なユースケースの信頼スコア範囲を示します。緊急サービスや慈善団体などの特殊なユースケースについては、 [Twilioのドキュメント](https://support.twilio.com/hc/en-us/articles/1260803225669-Message-throughput-MPS-and-Trust-Scores-for-A2P-10DLC-in-the-US)を参照してください。

|標準的な使用例 |説明 |
| ------------------ | ----------- |
|マーケティング |セールや期間限定のオファーなどのプロモーションコンテンツ。|
|混合 |カスタマーケアなど、複数のユースケースをカバーするキャンペーン。|
|高等教育 |高等教育機関向けのキャンペーン。|
|投票と投票 |顧客調査など、非政治的な世論調査や投票。|
|PSAの |特定のトピックに関する意識を高めるためのPSA。|
|カスタマーケア |サポート、アカウント管理、その他の顧客とのやり取り。|
|配信通知 |配信メッセージのステータス。|
|アカウント通知 |アカウントのステータスに関する通知。|
|2FAの|OTPなどのアカウント検証の認証。|
|セキュリティアラート |侵害されたシステムの通知。|
|詐欺アラート |不正行為の可能性があるアクティビティに関するメッセージ。|

## 次のステップ

10DLCにまだ登録していないお客様は、カスタマーサクセスマネージャーと協力してロングコードを登録する必要があります。**2021 年 10 月 1 日以降、お客様がロングコードの登録に失敗した場合、ロングコードを使用する A2P 送信者は、すべてのメッセージに対して大量のフィルタリングが発生します。**カスタマーサクセスマネージャーに連絡して、10DLC登録を開始してください。
