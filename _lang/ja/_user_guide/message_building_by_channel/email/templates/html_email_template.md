# HTML メールテンプレートのアップロード

> Brazeダッシュボードでは、独自のHTMLメールテンプレートをアップロードして保存し、後でキャンペーンで使用することができます。エディターを使用して [メールテンプレートを作成する]({{site.baseurl}}/user_guide/message_building_by_channel/email/creating_an_email_template/) こともできます。

## 前提 条件 {#upload-requirements}

まず、HTMLメールテンプレートを作成する必要があります。これは、次の内容を含むZIPファイルである必要があります。

* 1 つの HTML ファイル (メールの本文)
* HTML ファイルで参照される画像のフォルダー
* 画像ファイルが 50 個未満
* 5 MB 未満であること

## テンプレートのアップロード

### ステップ 1:メールテンプレートエディターに移動します

**[テンプレート**] > **[メールテンプレート**] に移動します。

{% alert note %}
[古いナビゲーション]({{site.baseurl}}/navigation)を使用している場合、このページは **[Engagement** > **Templates & Media** > **Email Templates にあります**。
{% endalert %}

### ステップ 2:アップローダーを開く

「 **テンプレート・タイプ** 」セクションで、「 **HTMLエディタ** 」を選択し、「 **基本HTMLテンプレートから開始**」セクションまでスクロールします。**[ファイルから**] を選択します。

### ステップ 3:テンプレートをアップロードする

「 **ファイルからアップロード** 」をクリックし、コンピューターからテンプレートを選択します。「 [前提条件](#upload-requirements) 」セクションを参照して、テンプレートがアップロード要件を満たしていることを確認してください。

#### テンプレートのアップロードエラーのトラブルシューティング

HTML テンプレート ファイルのアップロード時に受信する可能性のある電子メール エラー メッセージがいくつかあります。エラーが発生した場合は、次の表で一般的な問題と推奨される修正方法を参照してください。

|エラー |修正 |
|------|---|
|.zip 5 MB |ファイルサイズを小さくして、もう一度アップロードしてみてください。|
|.zipが壊れています|ファイルを確認して、もう一度アップロードしてみてください。|
|HTML の欠落 |HTMLファイルをZIPファイルに追加して、もう一度アップロードしてみてください。|
|複数の HTML|HTML ファイルの 1 つを削除して、もう一度アップロードしてみてください。|
|5 MB を超える画像|画像の数を減らして、もう一度アップロードしてみてください。|
|エクストラ画像|HTML ファイルで参照されていない追加の画像がファイル内に存在する可能性があります。これにより失敗エラーは発生しませんが、余分な画像は破棄されます。これらの画像がHTMLファイルで参照されることになっている場合は、コンテンツを確認し、エラーを修正して、もう一度アップロードしてみてください。|
|画像が見つからない|HTMLファイルで参照されている画像があるが、それらの画像がZIPファイルの画像フォルダに含まれていない場合、ファイルエラーが表示されます。ファイルを調べてエラー(スペルミスなど)を修正するか、不足している画像をZIPファイルに追加して、もう一度アップロードしてみてください。|
{: .reset-td-br-1 .reset-td-br-2}

### ステップ 4: テンプレートを終了して保存する

必ず **[テンプレートの保存**] をクリックしてテンプレートを保存してください。これで、選択したキャンペーンやキャンバスでこのテンプレートを使用する準備が整いました。

{% alert note %}
既存のテンプレートを編集しても、そのテンプレートの以前のバージョンを使用して作成されたキャンペーンには反映されません。
{% endalert %}
