# HTMLメールテンプレートのアップロード

> Brazeダッシュボードでは、独自のHTMLメールテンプレートをアップロードし、後でキャンペーンに使用するために保存できます。また、エディターを使って[メールテンプレートを作成する]({{site.baseurl}}/user_guide/message_building_by_channel/email/templates/email_template/)こともできます。

## 前提条件 {#upload-requirements}

まず、HTMLメールテンプレートを作成する必要があります。これは、以下を含むZIPファイルでなければなりません：

* 1つのHTMLファイル — メールの本文
* HTMLファイルで参照される画像のフォルダ
* 画像ファイル数50未満
* 5&nbsp;MB未満であること

## テンプレートをアップロードする

### ステップ 1:メールテンプレートエディターに移動する

[**テンプレート**] > [**メールテンプレート**] に移動します。

### ステップ 2:アップローダーを開く

[**テンプレートタイプ**] セクションで [**HTMLエディター**] を選択し、[**基本HTMLテンプレートから開始**] セクションまでスクロールダウンします。[**ファイルから**] を選択します。

### ステップ 3:テンプレートをアップロードする

[**ファイルからアップロード**] を選択し、コンピューターからテンプレートを選択します。[前提条件](#upload-requirements)セクションを参照し、テンプレートがアップロード要件を満たしていることを確認してください。

### ステップ 4:テンプレートを完成させて保存する

必ず [**テンプレートを保存**] を選択してテンプレートを保存してください。これで、このテンプレートを任意のキャンペーンやキャンバスで使用する準備が整いました！

{% alert note %}
既存のテンプレートに編集を加えた場合、その変更はそのテンプレートの旧バージョンを使用して作成されたキャンペーンには反映されません。
{% endalert %}

## APIキャンペーンでテンプレートを使用する {#api_for_upload_email_templates}

APIキャンペーンでメールを使用するには、`email_template_id` が必要です。これは、Brazeで作成されたメールテンプレートの下部にあります。

## トラブルシューティング

HTMLテンプレートファイルをアップロードする際に、いくつかのメールエラーメッセージが表示されることがあります。エラーが発生した場合、一般的な問題と推奨される修正方法については、以下の表を参照してください：

| エラー | 修正方法 |
|------|---|
|`.zip が5&nbsp;MB超`| ファイルサイズを小さくして、再度アップロードしてください。|
|`.zip が破損`| ファイルを確認し、再度アップロードしてください。 |
|`HTMLが見つからない`| HTMLファイルをZIPファイルに追加し、再度アップロードしてください。|
|`HTMLが複数ある`| HTMLファイルの1つを削除し、再度アップロードしてください。|
|`画像が5&nbsp;MB超`| 画像の数を減らして、再度アップロードしてください。 |
|`余分な画像`| HTMLファイルで参照されていない画像がファイル内に含まれている可能性があります。これは失敗エラーにはなりませんが、余分な画像は破棄されます。これらの画像がHTMLファイルで参照されるべきものであれば、コンテンツを確認し、エラーがあれば修正して、再度アップロードしてください。|
|`画像が見つからない`| HTMLファイル内で参照されている画像が、ZIPファイルの画像フォルダに含まれていない場合、ファイルエラーが発生します。ファイルを確認し、エラー（スペルミスなど）を修正するか、不足している画像をZIPファイルに追加して、再度アップロードしてください。|
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

HTMLキャンペーン、メールメッセージを含むキャンバスステップ、またはテンプレートのファイルをWindowsマシンでダウンロードする場合、`|`（パイプ文字）はサポートされていないため、ZIPファイルからダウンロード内容を展開するには別のアプリケーションを使用する必要がある場合があります。
