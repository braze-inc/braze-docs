# Gmailプロモーション設定

> [Gmail モバイルの [プロモーション] タブ](https://developers.google.com/gmail/promotab/)を使用すると、マーケターは件名やプリヘッダーの情報だけでなく、「カード」の注釈をを使用してより多くの情報を送信できます。Braze には、メールキャンペーンからカードを作成するためのツールが組み込まれています。

## 前提条件

まず、お客様のドメインとサブドメインを Google の [プロモーション] タブのアウトリーチチーム (<a href="mailto:p-promo-outreach@google.com">p-promo-outreach@google.com</a>) に転送して、Gmail の許可リストに追加してもらいます。これにより、Gmail の [プロモーション] タブの商品カルーセルなど、豊富な画像を表示する機能を使用できます。

## Brazeでカードを構築する

次のステップに従い、メール キャンペーンのGmail プロモーションカードを構築します。エディタの**Content** セクションから移動すると、**Gmail Promotion** タブのフィールドs とインフォメーションがリセットされます。プロモーションカードの設定を完了し、HTML コードを失わないように生成されたHTMLを複製します。

### ステップ 1: メールキャンペーンを作成する

まず、[メールキャンペーンを作成]({{site.baseurl}}/user_guide/message_building_by_channel/email/html_editor/creating_an_email_campaign/)し、編集環境として**HTMLコードエディタを**選択する。

### ステップ 2:Gmailプロモーションカードに詳細を追加する

次に、HTMLエディタの**「コンテンツ」**セクションに行き、「**Gmailプロモーション**」タブを選択する。**基本情報**」のフィールドに記入し、「**HTMLコードを生成**」を選択する。これにより、**HTML コードを`<Head>`**にコピー&ペーストする、Gmail Promoタブカードのスクリプトが生成されます。

![カードのビルド方法のサンプル。]({% image_buster /assets/img/create-gmail-promo.png %})

### ステップ 3:Gmailプロモーションカードをカスタマイズする

Gmailプロモーションカードに割引オファー、ディールカード、プロモーションカード、またはすべてのオプションを含めるかどうかを選択する。

{% tabs %}
{% tab Discount offer %}

割引オファーを設定すると、割引の有効な日付を指定できます。 

1. **割引オファーの**トグルを選択する。
2. **Offerには**、割引の簡単な概要を入力する。例えば、「20％オフ」である。
3. **コードには**、ユーザーがチェックアウト時に適用する必要のあるプロモーションコードを追加する。
4. 次に、割引オファーの開始日時を選択する。
5. 割引キャンペーンを特定の時期に終了させるか、それとも終了させないかを決定する。

![割引オファーのオファー値、コード、および開始日時を指定するオプション。]({% image_buster /assets/img/gmail_promo_discount_details.png %}){: style="max-width:70%;"}

{% endtab %}
{% tab Deal Cards %}

ディールカードを使って、メール本文の上部に主要なディール情報を直接掲載する。これにより、受信者はオファーの詳細を素早く理解し、アクションを起こすことができる。例えば、ディールカードを使って期間限定のキャンペーンを宣伝し、ユーザーがメール内で詳細を検索する手間を省くことができる。

1. **ディールカードの**トグルを選択する。
2. **Offerには**、割引の簡単な概要を入力する。例えば、「シューズ全品20％オフ」である。
3. (オプション）**コードには**、ユーザーがチェックアウト時に適用する必要のあるプロモーションコードを追加する。
4. 以下のURLの少なくとも1つを入力する。 
-  **オファーページのURL**特定のオファーのランディングページのURL。これで「Shop now」（または同様の）ボタンが作成される。このURLをディールカードに記載することをお勧めする。 
- **マーチャント・ホームページのURL**メイン・ホームページのURL。このフィールドは、特定のオファーページのURLが利用できない場合にのみ使用する。
5. (オプション）オファーの開始日を追加する。
6. オファーは特定の時間に終了させるか、終了させないかを決める。

![ディールカードのオファーの価値、コード、開始日時を指定するオプション。]({% image_buster /assets/img/gmail_promo_deal_cards.png %}){: style="max-width:70%;"}

{% endtab %}
{% tab Promotion cards %}

製品カルーセルのプロモーションカードは、オファーに画像を表示する場合に役立ちます。また、製品カルーセルの変数をカスタマイズして、画像のプレビュー (各画像が固有) を最大 10 枚含めることもできます。

1. **プロモーションカードの**トグルを選択する。
2. **プロモーションカードの追加を**選択する。製品カルーセルの各画像には、一意の URL を指定し、同じアスペクト比 (4:5, 1:1, 1.91:1) を使用する必要があります。
3. 画像, 写真のURLを含める。
4. **ターゲットURLには**、プロモーションのリンクを追加する。

{% alert tip %}
商品画像をメディアライブラリにアップロードし、URLをコピーして適切なフィールドに貼り付けることをお勧めする。静止画像形式（PNGとJPEG）のみ受け付ける。一部の画像形式 (GIF) は、アップロードされますが意図どおりには表示されません。
{% endalert %}

5. 見出し、通貨、価格、割引額を追加してプロモーションカードをカスタマイズしよう。

| 通貨 | (オプション) 価格の通貨。 |
| 価格 | プロモーションの価格。 |
| 割引値 | 元の価格から割り引かれた金額。 |

### ステップ 4: HTMLコードを生成して貼り付ける

Gmailプロモーションカードを作成したら、**HTMLコードを生成するを**選択する。スクリプトをコピーして、メールの HTML の `<head>` 要素に貼り付けます。 

{% alert tip %}
ドラッグ＆ドロップ・エディターの場合は、生成されたHTMLコードを「**送信設定**」の[カスタム・ヘッドタグ]({{site.baseurl}}/user_guide/message_building_by_channel/email/drag_and_drop/overview/#custom-head-tags)・セクションにコピー＆ペーストする。
{% endalert %}

{% alert warning %}
プロモーションスクリプトは、メールが Gmail の [プロモーション] タブに振り分けられた場合にのみ表示されます。現在、Gmail はアルゴリズムを使用して、メールが振り分けられる場所を決定しています。ただし、ユーザーがお客様からのメールをプロモーションとしてマークした場合、Gmail のアルゴリズムは無視され、メールは自動的に [プロモーション] タブに振り分けられます。
{% endalert %}

## Gmailカードを測定する

Gmailはこれらのカードの分析を返さず、Brazeのようなメールサービスプロバイダー（ESP）は、ヘッダーセクションのリンク（プロモーションカードや製品カルーセルを含む）に独自のリンク追跡を挿入することができない。ただし、セットアップ中にUTMパラメータやユニークコードをURLに追加することはできる。これらのパラメータは、メールサービスプロバイダー（ESP）によって挿入されるのではなく、URL自体の一部であるため、独自のWebサイト分析やコンバージョントラッキングを使用してエンゲージメントを追跡することができる。メールサービスプロバイダー（ESP）レベルのクリックトラッキングは、これらのリンクでは利用できない。

### 画像、写真を取り入れる

Gmail では、メールに関連した強力な画像でより良い成果が得られています。Gmail では、テキストのみのデザインを使用することは推奨されていません。このスペースが、メールマーケティングに不可欠なビジュアル表現をプレビューに取り込むように設計されているためです。抜き出したテキストの画像を使用したり、複数のキャンペーンで同じ画像を繰り返して使用したりしないでください。

### オファーを説明する

Gmail では、「短パンとシャツの全商品で、1 枚買ったら 1 枚無料か割引」のような文やフレーズを使用することを推奨していません。興味を引かずに読み飛ばされたり、件名と競合したりするためです。このスペースは、お客様のメッセージングに対する顧客のエンゲージメントを獲得するためにのみ使用する必要があります。そのため、「このメールを今すぐ開封してください」や「お買い得情報はここをクリック」のような文章は使用しないでください。件名を繰り返さないようにすることをお勧めします。

## よくある質問

### 自分のプロモーションメッセージで、エンドユーザーの受信トレイにプロモーションカードやプロダクトカルーセルが表示されないのはなぜですか?

Gmailのプロモーションタブに商品カルーセルが表示されるかどうかを決める要因はたくさんある。

注釈内の画像はすべて、品質フィルターで合格する必要があります。商品カルーセルが表示されるためには、注釈に含まれるすべての画像は、推奨される画像アスペクト比であり、高品質で高解像度の商品クローズアップ画像でなければならない。画像, 写真にはテキストをほとんど含まないこと。品質フィルターは不適切なコンテンツも除外するため、画像は家族、ユーザー、および子どもに適したものでなければなりません。

さらにGmailでは、ユーザーのGmailプロモーションタブに表示される商品カルーセルの数にキャップがある。例えば、ユーザーがプロモーションメールに商品カルーセルを使っているブランドをたくさんサブスクライバー登録している場合、Gmailは最終的に商品カルーセルの表示数にキャップをかける。

グーグルのプライバシーおよび安全規則により、アノテーションが機能するためには、アノテーション付きのメールを広く送付する必要があります。Googleのシステムが "大量送信 "として検出するためには、キャンペーンを開始し、少なくとも100人の受信者に送信することが推奨される。画像の URL を受信者によって変えることはできません。

### プロモーションカードまたはプロダクトカローセルのクリックはどのように追跡されますか?

Braze や他のメールサービスプロバイダー (ESP) がヘッダーセクションのリンクにリンクトラッキングを挿入することはできません。つまり、プロモーションカードまたはプロダクトカルーセルのクリックを追跡できません。

### 商品カルーセルを受け取ったユーザーの数を確認する方法はありますか?

いつ誰にカードを表示するかは、Gmail が決定します。したがって、すべての受信者に製品カルーセルが表示されるとは限りません。

### Gmailのプロモーションタブに注釈が表示されないのはなぜか？

アノテーションはGoogleワークスペースではサポートされていない。注釈をプレビューするには、Gmailで個人メールアドレスを作成する。

注釈は**プライマリ・タブや**Gmailモバイルアプリの他のタブではレンダリングされない。ユーザーがメールを開封した後や、`DiscountOffer` のアノテーションタイプを使用している場合、日時がすでに経過していると、アノテーションが表示されない。
