# Gmail プロモーションの設定

> [Gmail モバイルの [プロモーション] タブで][1]は、マーケターは件名やプリヘッダー情報だけでなく、「カード」のアノテーションを使用してより多くの情報を送信できます。Brazeには、メールキャンペーンからカードを作成するのに役立つツールが組み込まれています。

## 前提

まず、ドメインとサブドメインを Google の [プロモーション] タブのアウトリーチ チームに転送し <a href="mailto:p-promo-outreach@google.com">、Gmail</a> の許可リストに追加してもらい p-promo-outreach@google.com。これにより、Gmail の [プロモーション] タブの商品カルーセルなど、豊富な画像を表示する機能を使用できます。

## Brazeでカードを作る

メール キャンペーン用の Gmail プロモーション カードを作成するには、次の手順に従います。エディタの [ **コンテンツ** ] セクションから移動すると、[ **Gmail のプロモーション** ] タブのフィールドと情報がリセットされます。プロモーションカードの設定を完了し、生成された HTML をコピーして、HTML コードを失わないようにします。

1. [メールキャンペーンを作成する][7]をクリックし、編集エクスペリエンスとして **HTMLエディター** を選択します。
2. HTML エディタの [ **コンテンツ** ] セクションに移動し、[ **Gmail のプロモーション]** タブを選択します。
3. 「 **基本情報**」の下のフィールドに入力し、「 **HTMLコードを生成」**をクリックします。これは、[ **HTMLコードをコピーして貼り付ける] `<Head>`** セクションにあるGmailプロモーションタブカードのスクリプトを生成するのに役立ちます。<br> ![カードの作り方の一例][2]
4. Gmail プロモーション カードに割引特典のみを含めるか、プロモーション カードを含めるか、またはその両方を含めるかを選択します。<br> ![割引オファーとプロモーションカードを含めるオプション。[10]{: style="max-width:50%;"}
5. スクリプトをコピーして、メールのHTMLの要素に `<head>` 貼り付けます。

{% alert warning %}
プロモーション スクリプトは、メールが Gmail の [プロモーション] タブに届いた場合にのみ表示されます。現在、Gmail ではアルゴリズムを使用してメールの配信先が決定されています。ただし、ユーザーがメールをプロモーションとしてマークした場合、Gmail のアルゴリズムは無視され、メールは自動的に [プロモーション] タブに表示されます。
{% endalert %}

### 割引オファーを含める

割引オファーを設定すると、割引の有効な日付を指定できます。割引オファーを決定したら、開始日時を選択します。割引オファーは、特定の時間に終了するか、終了しないかを選択できます。

![割引オファーのオファー値、コード、開始日時を指定するオプション。[11]{: style="max-width:50%;"}

### 商品カルーセルのカスタマイズ

商品カルーセルのプロモーションカードは、出品商品に画像を提供するのに役立ちます。また、商品カルーセルの変数をカスタマイズして、各画像が一意である最大10個の画像プレビューを含めることもできます。

![Mottoという会社の商品カルーセルの例で、「ベストセラーの靴下がセール中です」という見出しのメールに、靴下の画像が3枚掲載され、割引価格が表示されます。[9]{: style="max-width:40%;"}

|カスタマイズ可能な変数 |説明 |
|---|---|
|画像URL |画像の URL。商品カルーセルの各画像には一意のURLを設定し、同じアスペクト比(4:5、1:1、1.91:1)を使用する必要があります。|
|ターゲットURL |プロモーションのリンク。|
|ヘッドライン |(オプション)プロモーションに関する 1 文または 2 文の説明。プレビュー画像の下に表示されます。|
|通貨 |(オプション)価格の通貨。|
|価格 |プロモーションの価格。|
|割引額 |元の価格から割引された金額。|
{: .reset-td-br-1 .reset-td-br-2}

{% alert tip %}
商品画像をメディアライブラリにアップロードし、URLをコピーして適切なフィールドに貼り付けることをお勧めします。静止画像形式(PNGおよびJPEG)のみが受け入れられます。一部の画像形式(GIF)はアップロードされますが、期待どおりに表示されません。
{% endalert %}

### おすすめの方法

一般的には、これらの [Gmail が推奨するおすすめの方法][8] に従ってください。 

{% alert tip %}
このスクリプト内でLiquidを使用できますが、エラーを回避するために、メッセージングを可能な限りテストすることを強くお勧めします。

#### 画像の組み込み

Gmail では、メール メッセージに関連する強い画像でより良い結果が得られています。Gmailは、メールマーケティングに不可欠な視覚的な言語をプレビューに取り入れるように設計されているため、テキストのみのデザインを使用することはお勧めしません。テキストが途切れた画像を使用したり、複数のキャンペーンで画像を繰り返すことはしないでください。

#### オファーの説明

Gmail では、「You Can Buy 1 Get 1 Free or Discounts on All Shorts and Shirts」などの文章やフレーズは、クリップして目を引かず、件名と競合する可能性があるため、推奨していません。このスペースは、メッセージで顧客を惹きつけるためにのみ使用する必要があるため、「このメールを今すぐ開く」や「取引についてはここをクリック」のような表現は避けてください。件名を繰り返さないことをお勧めします。

## よくある質問

### プロモーションメッセージで、エンドユーザーの受信トレイにプロモーションカードや商品カルーセルが表示されないのはなぜですか?

商品カルーセルが Gmail の [プロモーション] タブに表示されるかどうかは、さまざまな要因で決まります。

注釈内のすべての画像は、品質フィルターを通過する必要があります。商品カルーセルにデータを入力するには、アノテーション内のすべての画像が、推奨される画像のアスペクト比、高品質、または高解像度のクローズアップ商品画像であることが重要です。画像には、テキストがほとんどまたはまったく含まれていない必要があります(推奨)。品質フィルターは不適切なコンテンツもフィルタリングするため、画像は家族、ユーザー、子供に優しいものでなければなりません。

さらに、Gmail では、ユーザーの [Gmail プロモーション] タブに表示される商品カルーセルの数に密度の上限があります。たとえば、ユーザーがプロモーション メールで商品カルーセルを使用する多くのブランドに登録している場合、Gmail では最終的に表示される商品カルーセルの数に上限が設けられます。

Google のプライバシーと安全に関する規制により、アノテーションを機能させるには、アノテーション付きのメールを広く送信する必要があります。キャンペーンを開始し、少なくとも100人の受信者に送信して、Googleのシステムが「大量送信」として検出できるようにすることをお勧めします。 画像の URL は、受信者間で異なる場合があります。

### プロモーションカードや商品カルーセルのクリックはどのように追跡されますか?

Braze やその他の ESP は、ヘッダー セクションのリンクにリンク トラッキングを挿入できません。つまり、プロモーションカードや商品カルーセルでクリックをトラッキングすることはできません。

### 商品カルーセルを受け取ったユーザーの数を確認する方法はありますか?

カードを表示するタイミングと相手は Gmail で決定されるため、すべての受信者に商品カルーセルが表示される保証はありません。
