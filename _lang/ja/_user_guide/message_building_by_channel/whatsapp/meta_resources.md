## WhatsApp製品の更新 

### 2024年2月:マーケティング会話実験
*最終更新日2024年2月*

2024年2月6日から、WhatsAppは、より価値ある経験を創造し、ブランドのマーケティング対話に対する顧客エンゲージメントを最大化するために、インドの消費者から始まる新しいアプローチをテストしています。これには、読まれにくい少数の会話から始まり、ある期間にユーザーがブランドから受け取るマーケティング会話の数を制限することが含まれる。

### 2023年10月:テンプレートペーシング 
*最終更新日2023年10月*

2023年10月12日から、WhatsAppはマーケティングメッセージのための「テンプレートペーシング」という概念を導入している。キャンペーン対象者全体にメッセージを同時に送信する代わりに、「テンプレートペーシング」は、最初はメッセージを小規模なユーザのサブセットに配信し、残りのメッセージを送信する前にキャンペーン対象者からリアルタイムのフィードバックを収集します。 

「ペース制限」(送信されるメッセージの初期サブセット)は、テンプレートによって異なります。最初の送信後、WhatsApp は残りのメッセージを最大30 分間保持します。この保持期間中、顧客のフィードバックに基づいてテンプレートの品質を評価します。フィードバックが正の場合、高品質のテンプレートを示し、残りのメッセージを配信します。フィードバックが否定的な場合、残りの未配信メッセージが削除され、顧客の大部分からのさらなる否定的なフィードバックが防止され、潜在的な品質強制の問題(電話番号の品質評価の影響など)を回避できます。 

WhatsAppは、テンプレートのペーシングにおけるテンプレートの品質を評価するために、テンプレートの一時停止と同じシステムを使用することに注意してください。そのため、テンプレートペーシング中に配信されなかったメッセージ(低品質のテンプレートのため)は、より大きなスケールで一時停止されたメッセージと同じです。 

最終的には、このアップデートにより、より迅速なフィードバックループ(テンプレートの一時停止では時間または日数に対して30分)が提供されるため、テンプレートを調整し、より良いカスタマーエクスペリエンスを提供できます。

**このアップデートについてさらにご質問がある場合は、メタパートナーの担当者までお問い合わせください。**

### 2023年6月:メッセージング実験 
*最終更新日2023年6月*

2023年6月14日から、Metaは、消費者の経験とエンゲージメントにマーケティングメッセージがどのように影響するかを評価するために、WhatsAppプラットフォームに新しい実験手法を導入しています。この実験は、WhatsApp Business API with Brazeで送信されるマーケティングメッセージに影響を与える可能性があります。

MetaはWhatsAppプラットフォームでこのような実験を続けるつもりだ。詳細については、[Metaのドキュメント](https://developers.facebook.com/docs/whatsapp/on-premises/guides/experiments?content_id=86oue5PtwEgcBJl)を参照してください。

**WhatsAppの実験はマーケティングメッセージのみに影響します。**この実験は、マーケティング・テンプレート・メッセージの配信に影響を与える可能性がある。ユーティリティおよび認証テンプレートは、実験に影響を与えることなく引き続き配信されます。

実験では、MetaはWhatsAppの消費者の約1%を無作為に参加者として選んでいる。選択した場合、以下のいずれかが当てはまらない限り、Meta はマーケティングメッセージテンプレートをこれらのコンシューマーに配信しません。

- 消費者が過去24時間以内に返信した場合。
- 既存のマーケティング対話が開いている場合、または
- 過去72時間以内に消費者がWhatsApp Adをクリックした場合。

## よくある質問 {#faq}

### 自分のマーケティングメッセージがメタの実験の影響を受けたかどうかは、どうすればわかるだろうか。

実験のためにメッセージが配信されない場合、特定のエラー・コードがアクティビティー・ログおよびCurrents 内に表示されます。また、このメッセージは障害としてカウントされ、Braze ダッシュボード内のすべてのレポート全体でWhatsApp Failures メトリクスに組み込まれます。これらのメッセージには課金されません。

この130472エラーコードには、「ユーザー番号は実験の一部」と表示されます。 WhatsApp Cloud API エラーコードの詳細については、[Meta のドキュメント](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes?content_id=8SJRLBEjYGvXO9k) を参照してください。

### Metaの実験から外すことはできますか?

いいえ、Metaは実験のオプトアウトを許可しません。すべてのWhatsApp Business APIプロバイダーとユーザーは、このメタ実験の対象となります。

### 後でテンプレートを再送してもいいですか?

この実験には決まった時間はない。このように、消費者は実験に従うことができる。

### Metaの実験のためにマーケティングメッセージが配信されない場合、どうすればよいですか?

メール、SMS、プッシュ通知、アプリ内メッセージなど、他のBraze チャネルを使用して、目的のユーザに類似したコンテンツを含むメッセージを送信することをお勧めします。
