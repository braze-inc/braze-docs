# Braze ダッシュボード

> Braze ダッシュボードは、[dashboard.braze.com](https://dashboard.braze.com/) のWeb インターフェイスです [dashboard.braze.eu](https://dashboard.braze.eu/)

Braze ダッシュボードを使用して、キャンペーンのプランニング、メッセージの起動と管理、オーディエンス インサイトの探索、セグメンテーションの調整、単一のインターフェイスからのリアルタイムのパフォーマンスとエンゲージメントの測定基準の確認を行います。

## ダッシュボードに接続する

開始するには、[ Braze アカウントにサインインします]({{site.baseurl}}/user_guide/administrative/access_braze/accessing_your_account)。ダッシュボードおよび権限内のページへのアクセスは、割り当てられた[ユーザー権限]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/user_permissions/#list-of-permissions) に基づいて特定のアクションを実行します。権限に関するヘルプが必要な場合は、Braze管理者にお問い合わせください。

## Brazeの操作

Braze のナビゲーションは、デバイスを問わず機能やコンテンツに効率的にアクセスできるように設計されています。Braze ダッシュボードには、地球ヘッダーとサイドナビゲーションの2段階のナビゲーションがあります。

全体的なヘッダーは、ほとんどの場合、スクリーンの上部に表示されます。以下のような必須の工具や設定にすばやくアクセスできます。

- 検索
- サポートとコミュニティの連携
- [ダッシュボード言語]({{site.baseurl}}/user_guide/administrative/access_braze/language/)
- 通知
- 取引先設定
- [BrazeAI Operator™]({{site.baseurl}}/user_guide/brazeai/operator/)

### サイドナビゲーションの使用

左側の縦のメニューでは、機能別にBrazeの道具が整理され、最も使用頻度の高いアイテムが手の届く範囲に収まります。メインメニュー項目を選択すると、スタックされた垂直レイアウトにそのオプションが表示されます。 

![Braze ダッシュボード のワークスペース切り替える]({% image_buster /assets/img/workspace_switcher.png %}){: style="max-width:35%;float:right;margin-left:15px"}

#### ワークスペース切り替える

サイドナビゲーションの上部にあるワークスペース 切り替えるer では、Brazeインスタンス内のさまざまなワークスペース間を移動できます。有効なワークスペースが強調表示されます。

[ワークスペース]({{site.baseurl}}/user_guide/getting_started/workspaces) は、ブランド、地域、製品ライン、またはチーム別にコンテンツを整理するのに役立ちます。それぞれのワークスペースには、独自のデーター、キャンペーンs、および設定s が含まれます。ワークスペース s 間でアクセスできます。たとえば、あるワークスペースでは編集アクセス権があり、別のワークスペースでは表示専用アクセス権があるとします。

#### サイドナビゲーションの最小化

特にキャンバスのデザインなどの作業中に、視覚的な混乱を軽減するために、サイドナビゲーションパネルを最小限に抑えることができます。**最小化メニュー**を押して折りたたみます。最小化した場合でも、アイコンの上にマウスを置くと、メニュー項目名のツールチップが表示されます。これにより、ワークスペースを清潔に保ちながら、工具間を素早く移動できます。

![メニューアイコンの最小化と最大化]({% image_buster /assets/img/minimize_expand_menu.png %}){: style="max-width:60%;border:none"}

#### レスポンシブなナビゲーション

ナビゲーションは、さまざまなスクリーンサイズにシームレスにを適合させます。小さい画面では、サイドナビゲーションは自動的に折りたたまれます。必要に応じて、<i class="fa-solid fa-bars" aria-label="ナビゲーションメニュー"></i>を押してメニューを開きます。 

![小さい画面では、サイドナビゲーションは自動的に折りたたまれます。メニューアイコンを開封するとナビゲーションオプションが表示されます。]({% image_buster /assets/img/navigation/navigation_small_screens.png %}){: style="max-width: 80%;border:none"}

## ダッシュボードを検索する

ヘッダーにあるグローバル検索バーは、Braze ダッシュボード全体でコンテンツを見つける最も高速な方法です。検索インターフェイスを開封し、必要なものにジャンプします。 

![検索語句を入力しないグローバル検索開封。最近開封したページが表示されます。]({% image_buster /assets/img/navigation/search_recently_opened.png %})

最近開封した内容アプリが検索バーの下に表示されます。これには、最近操作したキャンペーン、キャンバス、テンプレート、またはページが含まれます。これにより、作業に簡単に戻ることができます。

### 検索できる項目

次の項目とアクションを検索できます。

- キャンペーン名
- キャンバス名
- コンテンツブロック
- セグメント名
- メールテンプレート名
- Braze内のページ(シノニムを含む)

{% alert tip %}
完全に一致するテキストを検索するには、検索語を引用符 ("") で囲みます。例えば、“すべてのユーザー“ を検索すると、“すべてのユーザー“ と完全に一致する語句を名前に含むすべての項目が返されます。
{% endalert %}

### コンテンツタイプとステータスタグ

それぞれの結果には、キャンペーン、キャンバス、Segmentなどのコンテンツタイプとそのステータス(有効、アーカイブ、停止)を示すタグが表示されます。

### 有効および下書き内容のフィルター

デフォルトでは、検索には有効、下書き、およびアーカイブされたアイテムが含まれます。**Show active and 下書き only**トグルを使用して、絞り込みを行います。

!["有効と下書きのみを表示"を切り替えます。]({% image_buster /assets/img/navigation/show_active_draft_new.png %})

## ダッシュボードのアクセシビリティ

Braze ダッシュボードは、カラーコントラストのWCAG AA規格に適合したブランドカラーを使用しています。これにより、すべてのユーザーの包括的なエクスペリエンスがサポートされ、アクセシビリティのベストプラクティスと整合します。
