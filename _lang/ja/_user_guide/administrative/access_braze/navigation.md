# ろう付けナビゲーション

ブレーズナビゲーションは、デバイス間で機能やコンテンツに効率的にアクセスできるように設計されています。それを最大限に活用する方法を探ってみましょう。

- **簡易メニュー:**スタックされたサブメニュー内のツールを整理する垂直メニュー構造を使用して、直感的にナビゲートします。
- **強力な検索:**グローバル検索バーを使用して、名前、タイプ、または以前の名前でページまたは項目を検索します。
- **モバイルに配慮したデザイン:**どのデバイスでも、Braze ナビゲーションの全機能にアクセスできます。

{% alert note %}
この更新では、すべてのユーザーを統合ナビゲーションに移行することで、エクスペリエンスを簡素化します。ナビゲーションバージョンを切り替えるオプションは使用できなくなりました。
{% endalert %}

## メニューを使う

縦方向のメニュー構造では、最も使用頻度の高いツールを1 回押すだけで済みます。メインメニュー項目を選択すると、スタックされた垂直レイアウトにそのオプションが表示されます。 

![「"Settings"」ページを選択すると、右側にスタックされた垂直メニューとしてサブメニューが開きます。][1]

## グローバル検索

ヘッダーにあるグローバル検索は、Braze で何かを見つける最も早い方法です。検索バーを選択するか、Windows の場合はCtrl + K を押し、Mac の場合は⌘ + K を押して、検索バーを直接選択します。 

![グローバル検索が開き、検索語句が入力されず、最近開いたページが表示されます。][2]

検索では、更新されたナビゲーションで名前が変更されたページのシノニムが理解されます。たとえば、「Currents」を検索すると、そのページの名前が変更されたため、「Data Export」と表示されます。

![検索結果"currents"以下を表示します。"データエクスポート。以前は'Currents'."][3]

## サイドナビゲーションの最小化

視覚的な混乱を減らし、ワークスペースに集中するには、サイド・ナビゲーション・パネルを最小限に抑えます。**最小化メニュー**を押してナビゲーションを非表示にします。これは、キャンバスでユーザフローを設計するときに特に役立ちます。 

![ユーザは、Braze Canvas の左側のナビゲーションメニューを展開したり折りたたんだりします。フルテキストラベルとアイコンのみの表示を切り替えます。][4]

最小化した場合でも、アイコンの上にマウスポインタを置くと、メニュー項目名のツールチップが表示され、すばやくアクセスできます。

![Braze ナビゲーションを最小化し、"Audience"フォーカスがAudience メニュー項目に移動されたときのツールチップを表示します。][5]{:style="max-width:50%"}

## 応答ナビゲーション

ナビゲーションは、デバイスのサイズに応じてシームレスに調整されます。小さい画面では、サイドナビゲーションは自動的に折りたたまれます。必要に応じて、<i class="fa-solid fa-bars" aria-label="ナビゲーションメニュー"></i>を押してメニューを開きます。 

![小さい画面では、サイドナビゲーションは自動的に折りたたまれます。メニューアイコンをタップすると、ナビゲーションオプションが開きます。][6]

## フィードバックの共有

ナビゲーション、ユーザビリティ、アクセシビリティ、ビジュアルアピール、その他重要なトピックに関するフィードバックを共有できます。グローバルヘッダの**Support**メニューを開き、**Share feedback**を選択して、あなたの考えをお送りください。あなたのインプットは私たちのブレーズの向上に役立ち、私たちはあなたからの意見を大切にします!

![サポートメニューが開き、ブレーズマニュアル、ブレーズラーニング、フィードバックの共有、およびヘルプの取得のオプションが表示されます。][7]{:style="max-width:50%"}
