# Braze許可

> 権限セットの作成、ロールの作成、ユーザー権限の編集、およびユーザー権限のエクスポートの方法について説明します。これにより、ユーザーが必要なワークスペースと機能にのみアクセスできるようになります。

## 権限セットの作成

権限セットを使用して、特定のテーマ領域またはアクションに関連する権限をバンドルします。権限セットは、異なるワークスペースにわたって同じアクセス権を必要とするダッシュボード ユーザーに適用できます。権限セットを作成するには、**Settings** > **Permission Settings** に移動し、**Create permission set** を選択します。各権限については、[権限一覧](#list-of-permissions)を参照してください。

## ロールの作成

ロールを使用すると、ワークスペースのアクセスコントロールと個別のカスタムアクセス許可をバンドルすることで、より多くの構成が可能になります。これは、1 つのダッシュボードに多数のブランドまたは地域ワークスペースがある場合に特に便利です。ロールを使用して、適切なワークスペースにダッシュボード ユーザーを追加し、関連付けられた権限を直接付与することができます。各権限については、[権限一覧](#list-of-permissions)を参照してください。

## 権限設定や役割はチームとどう違うのか？

{% multi_lang_include permissions.md content="Differences" %}

### ユーザー権限をチームに追加する場合の考慮事項

権限をBraze ダッシュボードに保存しようとすると、特にユーザーs をワークスペースから追加または削除したり、チームに追加したりするときに問題が発生することがあります。**Save/Update ユーザー s**ボタンは、ユーザーのパーミッションがワークスペースのパーミッションと同じ場合、グレー表示になることがあります。すべてのユーザーがワークスペース全体と同じ権限を持っている場合、チームを持つメリットはないため、この制約が存在します。

同じ権限を維持しながらチームにユーザーを正常に追加するには、ワークスペースの権限を割り当てないでください。代わりに、チームレベルでのみ権限を割り当てます。

## 限定ユーザー

限定されたユーザーには、社内管理者やワークスペース管理者と比較して制約を受けながら、Braze ダッシュボードの特定の側面を管理することができる特別な権限があります。

| 権限| 制限付きユーザーは、"Dashboard ユーザー&quot の管理;権限がチェックされている場合、他の制限付きユーザーの権限を編集できます。また、新しい制限付きユーザーを作成したり、権限セットを変更したりすることもできます。ただし、会社の管理アカウントを作成または管理することはできません。|
| ロールの制限| 制限されたユーザーに"App Group Admin"を除くすべての権限がある場合でも、通常はワークスペース管理者に付与される他のすべての権限へのアクセス権が付与されます。|
| 権限の可視性| 制限されたユーザーに&quot がある場合;Dashboard ユーザーの管理" Dev などのアプリグループに対してチェックされますが、Prod などの別の権限についてはチェックされません。"ユーザーの管理" プロファイル。|
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### 限られたユーザーの比較

| 限定ユーザー型 | 説明 |
| --- | --- |
| アプリグループ管理者 | アプリグループ管理者には、アプリグループの管理に固有の権限がありますが、社内管理者と同じ権限はありません。制限ユーザは、必要な権限がチェックされている場合、App Group Admins の権限と同様の権限を継承できます。 |
| 会社管理者 | 企業管理者には、ダッシュボード ユーザーを削除する機能など、より広範な権限があります。ただし、自分のアカウントを削除することはできず、そのアクションの別の会社管理者に連絡する必要があります。 |
| 基本的な読み取り専用権限 | テクノロジーパートナーページなど、ダッシュボードの特定の部分にアクセスするには、ユーザー s に基本的な読み取り専用権限が必要です。これには、"Manage External Integrations"を有効にすること、およびAccess Campaigns、Canvases、Cards、Segments、およびMedia Libraryのアクセス権限が含まれます。 |
{: .reset-td-br-1 .reset-td-br-2 role="presentation" }

### 立入制限エラー

"Limited Accessなどのメッセージが表示されることがあります。このPage.&quot にアクセスする権限がありません。このような場合、アカウント管理者は、ユーザーの権限を無効にして再び有効にすることで問題を解決できるかどうかを確認する必要があります。

{% alert note %}
あるダッシュボード ユーザーから別のユーザー権限を結合または読み込むことはできません。
{% endalert %}

## ユーザー権限の編集

ユーザーの現行管理者、企業、またはワークスペースの権限を変更するには、**設定**> **企業ユーザーs**に移動し、名前を選択します。

### 管理者

管理者は、あらゆる会社の設定を変更する機能にアクセスして変更できます。以下の操作を実行できます。

- [[承認] 設定]({{site.baseurl}}/user_guide/engagement_tools/campaigns/managing_campaigns/campaign_approval/#turning-on-campaign-approval)の変更
- 他の[Braze ユーザーs]({{site.baseurl}}/user_guide/administrative/app_settings/manage_your_braze_users/adding_users_to_your_dashboard/#adding-braze-users)を追加、編集、削除、一時停止、または一時停止解除する
- Braze ユーザーs をCSV としてエクスポート

管理者権限の付与または削除を行うには、[**このユーザーは管理者です**] を選択してから、[**ユーザーを更新**] を選択します。

\![選択したユーザーの内容にフォーカスされた管理チェックボックスが付いています。]({% image_buster /assets/img/braze_permissions/admin_level_permissions.png %}){: style="max-width:40%;"}

{% alert warning %}
ユーザーから管理者権限を削除すると、[会社レベル](#company)または[ワークスペースレベル](#workspace)の権限を少なくとも 1 つ割り当てるまで、そのユーザーは Braze にアクセスできなくなります。
{% endalert %}

### 会社

ユーザーの次の会社レベルの権限を管理するには、その権限の横にあるチェックボックスをオンまたはオフにします。終了したら、**ユーザーをアップデート**を選択します。

|許可名|説明|
|----------|-----------|
|会社の設定の管理|ユーザーに、会社の設定の変更を許可します。|
|ワークスペースの作成と削除s|ユーザーに、ワークスペースの作成および削除を許可します。|

### ワークスペース

Braze に属するワークスペースごとに、ユーザーにさまざまな権限を与えることができます。ワークスペースレベルの権限を管理するには、[**ワークスペースと権限を選択**] を選択し、権限を手動で選択するか、[以前に作成した権限セット](#creating-a-permission-set)を割り当てます。

ユーザーに、ワークスペースによって異なる権限を付与する必要がある場合は、必要なだけこのプロセスを繰り返します。各権限については、[権限一覧](#list-of-permissions)を参照してください。

{% subtabs %}
{% subtab Select manually %}

[**ワークスペース**] のドロップダウンから 1 つ以上のワークスペースを選択します。次に、**Permissions** の下で、ドロップダウンから1 つ以上の権限を選択します。これらの権限は、選択したワークスペースにのみ割り当てられます。必要に応じて、このワークスペースの完全な権限を付与する場合は、**Enable Admin Access** を選択できます。

終了したら、**ユーザーをアップデート**を選択します。

\![ワークスペースレベルの権限がBrazeで手動で選択されている。]({% image_buster /assets/img/braze_permissions/workspace_level_permissions_individual.png %})

{% endsubtab %}
{% subtab Assign permission set %}

[**ワークスペース**] のドロップダウンから 1 つ以上のワークスペースを選択します。次に、**Permission Sets**の下で、1つの権限セットを選択します。これらの権限は、選択したワークスペースにのみ割り当てられます。

終了したら、**ユーザーをアップデート**を選択します。

\![ワークスペースレベルの権限は、Braze で設定された権限によって割り当てられます。]({% image_buster /assets/img/braze_permissions/workspace_level_permissions_set.png %})

## ユーザー権限のエクスポート

ユーザーとその権限のリストをダウンロードするには、[**設定**] > [**会社ユーザー**] に移動し、[**ユーザーをエクスポート**] を選択します。すぐに CSV ファイルが自分のメールアドレスに送信されます。

\!["Company Users"ページはBrazeの"Export Users"オプションはフォーカスされています。]({% image_buster /assets/img/braze_permissions/exporting_user_permissions.png %})
