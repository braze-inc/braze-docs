# Braze ユーザーの管理

> ユーザーの追加、一時停止、削除など、会社アカウントのユーザーを管理する方法について学びましょう。

{% alert note %}
このページのいくつかのセクションでは、「**会社ユーザー**」ページを参照しています。[古いナビゲーション]({{site.baseurl}}/navigation)を使用している場合、[**会社ユーザー**] は [**ユーザーを管理**] と呼ばれ、アカウントアイコンの下にあります。
{% endalert %}

## Braze ユーザーの追加

会社アカウントに設定できるユーザー (管理者または限定ユーザー) の数に制限はありません。ユーザーの追加、および関連する権限の設定は、会社の管理者が担当します。ただし、ワークスペースに残っている管理者が 1 人のみの場合、その人は自分の管理者権限を削除できません。

Braze アカウントに新規ユーザーを追加するには、管理者権限があることを確認して、次の手順に従います。

1. [**設定**] > [**会社ユーザー**] に移動します。
2. [**\+ 新規ユーザーを追加**] をクリックします。
3. 指示に従って、メールアドレス、部署、[ユーザーロール]({{site.baseurl}}/user_guide/administrative/manage_your_braze_users/user_permissions/#editing-user-permissions)などの情報を入力します。<br><br>![][2]<br><br>
4. ユーザーのロールを限定する場合は、このユーザーに付与する会社レベルとワークスペースレベルの権限を選択します。<br><br>![][3]

ワークスペースで使用されるメールアドレスはすべて一意である必要があります。つまり、ワークスペースへのアクセス権を持っていたユーザー、または引き続きアクセスできるユーザーに既に関連付けられているメールアドレスを追加しようとすると、エラーメッセージが表示されます。メールアドレスにプラス記号 (+) を追加して作成されたエイリアスを認識する Gmail アカウントの回避策として、メールアドレスに「+example」を追加できます。例えば、`contractor@braze.com` を `contractor+1@braze.com` にすることができます。

### 部署の選択

会社の Braze アカウントに追加した各ユーザーについて、部署を選択する必要があります。

ユーザープロファイルに所属部署を追加すると、Braze の使用方法に関する適切なコミュニケーションを確実に受信できます。該当するサポートとアラートが受信されます。

## Braze ユーザーの一時停止

ユーザーを一時停止すると、そのアカウントが非アクティブ状態になり、ユーザーはログインできなくなりますが、そのアカウントに関連付けられたデータは保持されます。管理者のみが Braze ユーザーの一時停止または停止解除ができます。

ユーザーを一時停止するには、[**設定**] > [**会社ユーザー**] に移動し、ユーザー名を見つけて <i class="fa-solid fa-user-lock"></i> [**一時停止**] をクリックします。

![ユーザーの一時停止][4]

管理者は、リストからユーザー名を選択し、フッターの [**ユーザーを一時停止**] をクリックしてユーザーを一時停止することもできます。

![ユーザー詳細の編集中にユーザーを一時停止する。][5]

## Braze ユーザーの削除

ユーザーを削除するには、[**設定**] > [**会社ユーザー**] に移動し、ユーザー名を見つけて <i class="fa fa-trash-can"></i> [**ユーザーを削除**] をクリックします。

![ユーザーの削除][34]

ユーザーを削除すると、Braze は以下のデータを一切保持しません。

- ユーザーが持っていたすべての属性
- メールアドレス
- 電話番号
- 外部ユーザー ID
- 性別
- 国
- 言語
- その他の類似データ
