# キーと値のペアのエクストラ

`ABKInAppMessage` オブジェクトはキーと値のペアを `extras` として保持できます。これらは、キャンペーン作成時にダッシュボードで指定します。キーと値のペアを使用して、アプリ内メッセージとともにデータを送信し、アプリでさらに処理することができます。
