# サンプルアプリ

> React SDK GitHub リポジトリには、SDK メソッドの多くを実装するサンプルアプリがあります。 

このアプリを参照して、独自の React Native 統合を構築できます。

アプリは[こちら](https://github.com/braze-inc/braze-react-native-sdk/tree/master/BrazeProject)で確認できます。
