# 正規表現リファレンスシート

このページは、一般的なトークン、メタシーケンス、一般的なトークン、グループ定数などを含む正規表現のクイックリファレンスガイドとして機能します。

{% tabs %}
{% tab 一般的なトークン %}

| 一般的なトークン |
| ------------- |
| a、b、cのいずれか1文字 | `[abc]` |
| a、b、c以外の文字 | `[^abc]` |
| 範囲内の文字: a-z | `[a-z]` |
| 範囲外の文字: a-z | `[^a-z]` |
| a-zまたはA-Zの範囲の文字 | `[a-zA-Z]` |
| 任意の単一文字 | `.` |
| 任意の空白文字 | `\s` |
| 任意の非空白文字 | `\S` |
| 任意の数字 | `\d` |
| 任意の非数字 | `\D` |
| 任意の単語文字 | `\w` |
| 任意の非単語文字 | `\W` |
| 囲まれたものをキャプチャ | `(...)` |
| aまたはbのいずれかに一致 | `(a|b)` |
| ゼロまたは1の | `a?` |
| ゼロ以上のA | `a*` |
| 1つ以上のa | `a+` |
| ちょうど3つのa | `a{3}` |
| 3と6の間のa | `a{3,6}` |
| 文字列の開始 | `^` |
| 文字列の終わり | `$` |
| 単語の境界 | `\n` |
| None-word boundary | `\B` |
{: .reset-td-br-1 .reset-td-br-2}
{% endtab %}
