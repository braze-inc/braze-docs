# WhatsApp与信比率(秘密)

> **WhatsApp** は、2 ウェイメッセージングに焦点を当てたチャネルであり、(個々のメッセージの数ではなく) 対話にアンカーします。対話は、企業とエンドユーザーの間の24時間スレッドです。会話内の個々のメッセージの数は、クレジット使用量に影響しません。 

{% alert note %}
メタがWhatsAppの価格設定または梱包を変更した場合、Brazeは、合理的な通知をもってそれに応じて信用比率を変更する権利を留保する。
{% endalert %}

**認証会話**:企業がユーザー s を1 回限りのパスコードs で認証できるようにします。ログイン処理の複数のステップs で認証できる可能性があります(たとえば、取引先の確認、取引先のリカバリ、整合性の問題)。 
  - 認証の対話はケースバイケースでのみサポートされ、Braze は特定のSLA を保証できません。また、Brazeは暗証番号の生成に対応していません。

**マーケティング会話**:プロモーションまたはオファー、情報更新、または顧客の回答またはアクションを受けるための招待を含める。 

**サービス会話**:End-ユーザー は、企業に電文を送信します。ビジネスが応答メッセージを送信すると、24 時間枠が開始されます。

**ユーティリティ会話**:進行中のトランスアクションに関する、合意された具体的な要請、トランスアクションまたは更新を顧客に円滑化する。これには、購入後の通知および定期的な請求明細が含まれる。
