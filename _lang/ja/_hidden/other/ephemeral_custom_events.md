# エフェメラルカスタムイベント

- エフェメラルカスタムイベントには、購入イベントまたは開始セッションと終了セッション (Brazeのドキュメントに記載されている) は含まれません。
- Braze は、SDK 実装ごとに最大12の配置/イベント名に対して、カスタムイベントを Braze のデータポイントとして送信および保存しません。
- エフェメラルイベントソリューションは、Braze の本番 iOS および Android SDK、および Braze がエフェメラルカスタムイベントソリューションを有効にしたプラットフォーム用の将来の Braze 本番 SDK に展開されます。これにより、お客様はこのような本番 SDK を使用できるようになり、将来の機能やバグ修正と同期しなくなります。これにより、顧客は、他のカスタムプロセスを必要とせずに、将来の SDK 更新を実行できるようになります。
