# 10DLC申請に必要な情報

10DLCの登録に関してBrazeから購入するお客様は、お客様とBrazeの間で適用される契約で合意されたとおり、下記の電子メールで要求されたすべての情報に対する回答をBrazeに提出する必要があります。

関連するスクリーンショットや添付ファイルを含む情報は、 [smsapplications@braze.com](mailto:smsapplications@braze.com) に提出する必要があります。10DLCについてご不明な点がございましたら、 [Braze SMS 10DLCガイド]({{site.baseurl}}/assets/pdf/SMS_10DLC_Guide.pdf) をご覧いただくか、Brazeカスタマーサクセスマネージャーにお問い合わせください。

## SMSキャンペーンのご案内

米国のロングコードを含む各SMSサブスクリプショングループを、独自のA2P 10DLCキャンペーン(ユースケースと呼ばれることもあります)に登録する必要があります。米国のロングコードを含むSMSサブスクリプショングループごとに、以下の情報を1回入力してください。	

|分野 |オプション(該当する場合) |説明 |

|キャンペーン名 | | {::nomarkdown}<p>SMS キャンペーン名は、メッセージ、利用規約、およびユーザーがプログラムにサインアップするすべての場所に一貫して表示する必要があります。</p><p>例：</p><ul><li>ろう付けマーキングアラート</li><li>Brazeアカウント通知</li><li>ろう付けカートリマインダー</li></ul>{:/}
