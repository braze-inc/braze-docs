# イノベーションステートメント

{% if site.language == 'ja' %}
{% raw %}
_最終更新日は2025年6月2日です。_

ブレイズは、絶えず新しい機能、製品、およびサービスを作成しています。そのために、Brazeは顧客からのフィードバックを求めるだけでなく、顧客のニーズを予測するよう努めています。

イノベーション戦略の一環として、Brazeは、顧客全般に対してコンサルティングおよび情報を提供するために、ならびに本サービスまたは関連するサービスを構築、提供または改善するために、本サービスに含まれる情報と本サービスの使用方法（ダッシュボード全体での本ダッシュボードユーザーのクリックや動きの分析などを分析します（ただし、当該使用に、他の顧客に対するサービスの提供の際に使用するために、ユーザーのプロフィールを構築もしくは修正すること、またはBrazeが他の情報源から入手したデータを訂正した場合もしくは増強することが含まれないことを前提とします）。

ブレイズはまた、本ダッシュボードユーザーと通信して、本サービスの使用を最適化する方法を教育および通知し、ブレイズの革新的な取り組みを共有します。ブレイズは、本ダッシュボードユーザーに電子メールを送信したり、本サービスを介してユーザーと通信したりする場合があります。顧客は、BrazeのGitHubリポジトリにサブスクライブして、新しいリリースが作成されたときに通知を受け取ることもできます。
ブレイズは、そのイノベーション活動から生じるあらゆる資料に対するすべての権利、権原、および利益を所有します。

Braze は常に新しい機能、特徴、製品、サービスを創造しています。そのために、Braze は顧客からのフィードバックを求めるだけでなく、顧客のニーズを先取りするよう努めています。

イノベーション戦略の一環として、Brazeは、本サービスに含まれる情報および本サービスがどのように使用されているかを分析します（e.g 、ダッシュボードユーザーのクリックおよびダッシュボード内の移動の分析）。これは、顧客一般にコンサルティングおよび分析情報を提供するため、および本サービスまたは関連サービスを構築、提供または改善するためです。
また、Braze は、ダッシュボードのユーザーに対し、本サービスの利用を最適化する方法を教育、通知し、Braze の革新的な取り組みを共有するためにコミュニケーションを行います。Braze は、ダッシュボードユーザーに対してメール送信、またサービスを通じてユーザーと連絡を取ることができます。また、顧客は Braze の GitHub リポジトリに登録することで、新しいリリースが行われた際に通知を受け取ることができます。

Braze は、そのイノベーション活動から生じたすべての資料に関するすべての権利、権原および利益を所有しています。
