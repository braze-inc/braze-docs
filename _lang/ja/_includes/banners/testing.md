{% if include.page == "testing" %}[バナーメッセージを作成する]({{site.baseurl}}/user_guide/message_building_by_channel/banners/create/#compose-a-banner)際は、{% elsif include.page == "campaigns" %}{% endif %} **プレビュー**を選択してバナーをプレビューするか、テストメッセージを送信する。

![バナー作成ツールのプレビュータブ。]({% image_buster /assets/img/banners/select_preview.png %}){: style="max-width:50%;"}

ハードウェアの違いにより、プレビューはユーザーの端末で表示される最終的なレンダリングと同一ではない可能性があることを覚えておいてほしい。

テストメッセージを送信するには、**テスト**受信者としてコンテンツテストグループか、1人以上の個別ユーザーを追加し、その後**「テスト送信」**を選択する。テストメッセージは端末上で最大5分間表示できる。次に**「プレビューリンクをコピー」**を選択すると、ランダムなユーザーに表示されるバナーの見た目を示す共有可能なプレビューリンクが生成され、コピーできる。リンクは7日間有効であり、それ以降は再生成が必要になります。

![バナー作成ツールのプレビュータブ。]({% image_buster /assets/img/banners/preview_banner.png %})

テストバナーを確認する際、以下の点を確認せよ：

- バナー広告キャンペーンは配置先に割り当てられているか？
- 画像やメディアは、対象のデバイス種別や画面サイズで期待通りに表示され、動作するか？
- リンクやボタンは、ユーザーを正しい場所へ誘導しているか？
- 液体は期待通りに機能しますか？Liquidが情報を返さない場合に、デフォルト属性値を考慮しましたか？
- あなたのコピーは明確で簡潔で正しいですか？
