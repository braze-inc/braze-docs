# ダイナミック・イールド

> [Mastercardの会社であるDynamic Yieldは](https://www.dynamicyield.com/)、あらゆる業界の企業が、パーソナライズされたされ、最適化され、同期されたデジタル顧客エクスペリエンスを提供できるよう支援しています。Dynamic [YieldのExperience OSを使用すると](http://www.dynamicyield.com/experience-os)、マーケティング担当者、プロダクトマネージャー、開発者、デジタルチームは、コンテンツ、製品、およびオファーをアルゴリズム的に各顧客照合して、収益と顧客ロイヤルティを加速させることができます。

BrazeとDynamic Yieldのパートナーシップにより、Dynamic Yieldのレコメンテーションおよびセグメンテーションエンジンを活用して、Brazeメッセージに埋め込むことができるエクスペリエンスブロックを作成できます。エクスペリエンスブロックは次のもので構成できます。
- **レコメンデーションブロック**:アルゴリズムとフィルタリングを設定して、メールが開かれたときに伝わるようにユーザーのパーソナライズされたコンテンツを配信します。 
- **動的コンテンツブロック**:さまざまなユーザーにさまざまなプロモーションやメッセージをターゲティングします。ターゲティングは、アフィニティまたはオーディエンスのいずれかに基づいて設定できます。Dynamic Yieldは、メールが開かれたときにどのパーソナライズされたエクスペリエンスを提供するかを決定します。

## 統合

### ステップ1:エクスペリエンスブロックを作成

Dynamic Yieldでエクスペリエンスブロックを作成するには、「**メール」>「エクスペリエンスメール」>「新規作成**」に移動します。

次に、「**エクスペリエンスブロックを作成**」を選択して、Braze メールテンプレートに埋め込む動的コンテンツブロックまたはレコメンデーションブロックをデザインします。<br>![][8]

### ステップ2:メッセージングの下書きを作成

次の画像, 写真は、ビルダーのメールを最初から表示したものです。<br>![][6]

1. 見出し領域に、キャンペーン名、メモ、およびキャンペーンラベルを入力します。<br><br>
2. エクスペリエンスブロックを挿入します。これらのブロックには次のものが含まれます。
  - [推奨事項](#configure-a-recommendations-block):完全にパーソナライズされたおすすめをユーザーに提供するウィジェット。
  - [ダイナミックコンテンツ](#configure-a-dynamic-content-block):さまざまなオーディエンスにさまざまなプロモーションやメッセージをターゲティングします。<br><br>
3. 更新設定:
  - URL パラメータを使用して、分析ソフトウェア内のクリックを追跡します（オプション）。必要に応じて、デフォルト表示にパラメータを追加します。
  - 属性ウィンドウ (7 日間 (デフォルト) または 1 日) を選択します。<br><br>
4. 保存して終了します。コードが生成される前であれば、いつでもメールのすべての要素を編集できます。コードが生成されたら、[コードに影響しないものはすべて編集できます](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FAZPXB6MH094J1MWS5N86FXH)。

### レコメンデーションブロックの設定

レコメンデーションブロックでは、メールが開かれたときに反映されるユーザーのパーソナライズされたコンテンツをソースとするアルゴリズムとフィルタリングを設定できます。 

1. レコメンデーションブロックを編集ペインからメールの本文にドラッグします。<br><br>
2. 目的のアルゴリズム (人気、ユーザーアフィニティ、類似性など) を選択します。選択したアルゴリズムに応じて、追加のオプションが表示されます。 
  - 人気度に基づくおすすめの場合は、検索結果をシャッフルして、視聴者が開封した別のメールから同じおすすめが表示されるのを防ぐことができます。
  - 類似性などの他のアルゴリズムでは、コンテキストに基づいてレコメンデーションを行うため、含める項目を選択する必要があります。これらのアイテムをビルダーに追加したり、[埋め込みコードにマージタグを追加してダイナミックな](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#advanced)したりできます。たとえば、出荷確認メールに類似アイテムを追加するなどです。<br><br>
3. ユーザーすでに購入した製品を除外して、それらの製品を推奨しないようにすることができます。<br><br>
4. [カスタムフィルタールを追加して特定の製品をスロットに固定したり](https://support.dynamicyield.com/hc/en-us/articles/4404013832465-Experience-Email#h_01FAZP4ZWZX1JJ2SH61MB3HVXD)、製品プロパティごとに製品を含めたり除外したりできます。たとえば、コードが5ドル未満の商品を表示したり、ショーツカテゴリの商品のみを表示したりしてください。<br><br>
5. 最後に、レコメンデーションブロックデザインを設定します。これを行うには、アイテムテンプレートを選択し、表示するアイテムの数と行数を設定します。 

### 動的コンテンツブロックの設定
動的コンテンツを使用すると、さまざまなユーザーにさまざまなプロモーションやメッセージをターゲティングできます。ターゲティングは、アフィニティまたはオーディエンスのいずれかに基づいて設定できます。Dynamic Yieldは、メールが開かれたときにどのパーソナライズされたエクスペリエンスを提供するかを決定します。 

1. 動的コンテンツブロックを編集ペインからメールの本文にドラッグします。<br><br> 
2. 最初のバリエーションのテンプレートを選択します。デザイン変数とコンテンツ変数を定義できるようになりました。完了したらバリエーションを保存します。<br>![][4]<br><br> 
3. 動的コンテンツペインでオーディエンス設定します。<br>![][5]<br><br> 
4. 別のバリエーションを追加して、別の特定のオーディエンスまたはすべてのユーザーをターゲットにします。必要に応じて繰り返します。<br><br> 
5. 上矢印と下矢印を使用してバリエーションの優先順位を設定します。<br><br> 
6. 優先順位によって、ユーザー複数のエクスペリエンスを利用できる場合にどのバリエーションが提供されるかが決まります。

### ステップ3:メールを Braze と統合する

この統合により、Dynamic Yieldを利用したパーソナライズされたレコメンデーションウィジェットとダイナミックなコンテンツをBrazeメールキャンペーンに追加できます。これらのキャンペーンをBrazeキャンペーンに埋め込むには、Brazeメールエディターに貼り付ける簡単な埋め込みコードを使用します。

1. エクスペリエンスメールリストページのメールサービスプロバイダー (ESP) 統合アイコンをクリックします。<br><br> 
2. ユーザーの CUID と電子メール ID を挿入する Braze の関連トークン入力します。<br>![][3]
  
メールに満足したら、ステップは Braze に埋め込むコードを生成することです。
1. **エクスペリエンスメールで**、「**コードを生成**」をクリックします。<br><br> 
2. 次に、\[**クリップボードにコピー**] をクリックします。<br>![][1]<br><br> 
3. コード Brazeメールキャンペーンに貼り付けてから、メールキャンペーンデザイン、テスト、公開を続けてください。
