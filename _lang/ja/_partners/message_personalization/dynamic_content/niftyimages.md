# 気の利いた画像

> [NiftyImagesは](https://niftyimages.com)、マーケティング担当者がエンゲージメントと収益を増やしながら、関連性の高い最新の電子メールコミュニケーションをより効率的に送信できるようにするリアルタイムの電子メールパーソナライゼーションソフトウェアです。これはユーザーフレンドリーなセルフサービスツールであり、マーケティング担当者は動的コンテンツを簡単かつ迅速にメールに追加できます。

## 前提条件

NiftyImagesは、統合を必要とせずに、すぐに使用できるBrazeと連携します。[始めるために必要なのはNiftyImagesアカウントだけです](https://niftyimages.com/Signup)。

## サポート対象機能

BrazeのNiftyImagesを活用することで、既存のBrazeパーソナライゼーションタグをNiftyImages URLにマッピングすることで、メールキャンペーン用の動的でパーソナライズされた画像を作成できます。

- プライバシーすべてのデータはNiftyImagesではなくBrazeに保存されます。
- **パーソナライズされた画像:**任意の Braze マージタグを使用して画像をパーソナライズできます。
- **チャートとグラフ:**階層レベル、顧客ステータス、消費金額、ポイントなどを、カスタマイズ可能なチャートやグラフで表示します。
- **マップ:**ユーザーがメールを開いた場所に最も近い場所を使用してマップの画像を表示します。
- **カスタムカウントダウンタイマー:**誕生日、トライアルの有効期限、最終購入日、期限切れの請求書、または最終サインイン日の日付データベース変数を含むユニークなタイマーを表示します。
- **リアルタイムコンテンツ:**おすすめ商品、カート放棄、値下げ、在庫レベル、天気などのリアルタイム画像を表示します。
- **ライブ投票:**ライブ投票を表示してエンゲージメントを促進し、関心度に関する洞察を得ましょう。
- **ルールベースのロジック:**ユーザーデータ、人口統計、行動、場所、時間帯、曜日、起動デバイス、オペレーティングシステムなどに基づいて動的な画像を表示します。

たとえば、これはNiftyImagesが顧客のファーストネームを使用して生成したカスタム画像です。

## 気の利いたイメージの作成

### ステップ1: マージタグを作成する

niftyImages でマージタグを選択し、デフォルト値を入力します。終了したら、[**次へ**] を選択します。

![alt text]({% image_buster /assets/img/niftyimages/2.png %}){: style="max-width:70%;"}

オプションで、データタイプを入力し、[**次へ**] を選択します。

![alt text]({% image_buster /assets/img/niftyimages/3.png %})
{: style="max-width:70%;"}

オプションで、将来使用できるようにタグを保存することもできます。終了したら、[**保存**] を選択してマージタグを作成します。

![alt text]({% image_buster /assets/img/niftyimages/4.png %}){: style="max-width:70%;"}

### ステップ2: 画像をカスタマイズ

画像のフォント、フォントサイズ、位置、色、レイヤーなどをカスタマイズできます。終了したら、画像の URL をコピーします。

![alt text]({% image_buster /assets/img/niftyimages/5.png %})

### ステップ 3 Braze に画像の URL を追加

Braze でキャンペーンまたはキャンバスを開き、NiftyImage の URL を貼り付けます。オプションで、変更内容をプレビューしてLiquidタグを確認できます。

![alt text]({% image_buster /assets/img/niftyimages/6.png %})
