# マイライツィオ

> [Mailizioは](https://mailizio.com/)、直感的なビジュアルエディタを使って、再利用可能でブランドセーフなコンテンツを簡単にデザインできるメール作成・管理プラットフォームである。MailizioとBrazeの統合により、コンテンツブロックとメールテンプレートをエクスポートし、同じアセットからアプリ内メッセージを自動的に生成することができ、迅速かつ完全にコントロールされたキャンペーン展開が可能になる。

_この統合はMailizioによって維持されている。_

## 統合について

MailizioとBrazeの統合により、Mailizioのエディターを使用してダイナミックなメールテンプレートをデザインし、Brazeの設定で使用されているLiquid変数を活用し、キャンペーンを合理的に実行するためにBrazeにプッシュすることができる。

## ユースケース

- キャンペーンやトランザクションメッセージ用に、すぐに送信できるメールテンプレートをBrazeに直接プッシュ。
- 再利用可能なコンテンツモジュール（ヘッダー、フッター、プロモーションなど）を構築し、複数のキャンペーンやチャネルにわたって制作を効率化する。
- メールからアプリ内メッセージを生成する：Mailizioは、メールの関連セクションを識別し、アプリ内キャンペーンで使用するためにHTMLをエクスポートすることができる。
- メールとアプリ内メッセージの両方でBraze互換のLiquid変数を使用してスケールのパーソナライゼーションを行う。
- クリエイティブアセットをMailizioでマネージャーし、Brazeで更新することで、ブランディングの一貫性を保つ。

## 前提条件

| 必要条件 | 説明 |                          
| ----------- | ----------- |  
| Mailizioアカウント | このパートナーシップを利用するには、Mailizioアカウントが必要である。 |  
| Braze REST API キー | 完全な**テンプレート**権限を持つBraze REST API キー。<br><br>Braze REST APIキーは、Brazeダッシュボードの**設定**> APIキーから作成できる。 |  
| Braze RESTエンドポイント | [あなたのRESTエンドポイントURL]({{site.baseurl}}/developer_guide/rest_api/basics/#endpoints)。エンドポイントはインスタンスの Braze URL に応じて異なります。 |

## 統合

Mailizio カスタマーサクセスマネージャーにBraze REST APIキーとクラスターインスタンスを提供する。その後、Mailizioチームが最初の統合設定を行う。

{% alert important %}
これは1回限りのセットアップであり、今後のエクスポートは自動的にこのAPIキーを利用する。
{% endalert %}

### ステップ 1: Mailizioでメールを作成する

Mailizioでは、ドラッグ＆ドロップのエディターを使って、ブランドアイデンティティを反映したメールを作成し、**保存を**クリックすると、作成したメールが保存される。

![ドラッグ＆ドロップエディターのスクリーンショット]({% image_buster /assets/img/mailizio/screenshot_1.png %})

### ステップ 2:Braze へのメールテンプレートのエクスポート

準備ができたら、**ニュースレターのエクスポートを**クリックする。ポップアップで**Braze-メールを**選択し、エクスポートを確認する。

後でコンテンツを更新した場合は、Mailizioから再エクスポートしてBrazeで更新する。

![エクスポート・モーダルのスクリーンショット]({% image_buster /assets/img/mailizio/screenshot_2.png %})

{% alert important %}  
Mailizioの**モジュール**エディタを使用して、同じ方法でコンテンツブロックを作成し、エクスポートすることができる。  
{% endalert %}

## 使用

アップロードしたMailizioテンプレートを、Brazeアカウントの**テンプレート& メディア > メールテンプレートセクションで**探す。これで、このメールテンプレートを使用して、顧客に魅力的なメールメッセージを送信できます。
