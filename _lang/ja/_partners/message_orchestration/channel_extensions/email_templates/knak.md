# クナック

> [Knakは][1]、企業のマーケティングチームが社内で使用するために構築された最初のキャンペーン作成プラットフォームです。ドラッグアンドドロッププラットフォームにより、コーディングや外部の支援を必要とせずに、誰でも数分でブランドに合った美しいメールやランディングページを作成できます。

BrazeとKnakの統合により、完全にレスポンシブメールを数日または数週間ではなく数分または数時間で作成し、すぐに使用できるBrazeテンプレートとしてエクスポートできます。Knakは、外部の代理店やハンドコーディングを必要とせずに、Brazeで管理されているキャンペーンのメール作成をレベルアップしたいマーケティング担当者向けに構築されています。 

## 前提条件

| 必要条件 | 説明 |
| ----------- | ----------- |
| クナックアカウント | このパートナーシップを利用するには、Knakアカウントが必要です。 |
| Braze REST API キー | **完全なテンプレート権限を持つBraze** REST API キーです。<br><br>これは Braze ダッシュボードの **\[設定] > \[**API キー**]** から作成できます。 |
| Braze REST エンドポイント | [あなたの REST エンドポイント URL][2]。エンドポイントは、インスタンスの Braze URL によって異なります。 |
{: .reset-td-br-1 .reset-td-br-2}

## ユースケース

Knakは、コーディングや外部の支援を必要とせずに、メール作成をレベルアップしたいマーケティング担当者向けに構築されています。次のような方に最適です。
- 現在はメール用のシンプルなテンプレートを使用しており、ゲームを盛り上げたいと考えている
- Braze 用のメール作成は外部の機関や開発者に任せる
- アセット作成のクリエイティブコントロールを取り戻し、市場投入までの時間を大幅に短縮したい

## 統合

### ステップ1:インテグレーションを設定する

Knakで、「インテグレーション」>「**プラットフォーム」>「+ 新規インテグレーションを追加**」に移動します。

![\[統合を追加] ボタン][5]

次に、**Braze** プラットフォームを選択し、Braze API キーと REST エンドポイント。「**新規インテグレーションを作成**」をクリックしてインテグレーションを完了します。 

![新しいインテグレーションを作成][6]

### ステップ2:Knak テンプレートを同期

**Knakで、Brazeに同期したいメールを探し、\[**公開**]、\[同期] の順に選択します。**

![カンクインテグレーション 1][8]

次に、メール名を確認し、\[**同期**] をクリックします。

![カンクインテグレーション 2][9]

## インテグレーションを使用する

アップロードした Knak メールは、Braze の \[**エンゲージメント] > \[テンプレートとメディア**] で確認できます。美しく、ブランドイメージに忠実で、レスポンシブも完璧です。唯一の制限はあなた自身の創造性です！
