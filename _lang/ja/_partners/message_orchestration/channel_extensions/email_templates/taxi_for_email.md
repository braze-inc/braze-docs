# Eメール用タクシー

> [メール用タクシー](http://taxiforemail.com/)は、直感的なドラッグアンドドロップビジュアルメールエディタを提供するオンラインメール マーケティングツールです。タクシーは、チームがメール キャンペーンsで簡単にコラボレーションすることを奨励し、コピーライターや編集者がメールsを構築するために必要な情報や情報をコードなく入手できるようにします。

Brazeとタクシーの融合は、タクシーのシンプルでパワフルなシンタックスを活用して、Brazeにインテリジェントなメール テンプレートを作成してエクスポートします。 

## 前提条件

| 要件 | 説明 |
| ------------| ----------- |
| Eメールアカウントのタクシー | この提携の前進タグeをとるためには、電子メールのためのタクシーが必要である。 |
| Braze REST API キー | 完全な**テンプレート**権限を持つBraze REST API キー。<br><br> これは、**Settings** > **API Keys** のBraze ダッシュボードで作成できます。 |
| Braze エンドポイント | [あなたのBraze エンドポイント]({{site.baseurl}}/api/basics/#endpoints)はあなたのBraze ダッシュボード URLに合わせます。<br><br> たとえば、ダッシュボード URL が`https://dashboard-03.braze.com` の場合、エンドポイントは`dashboard-03` になります。 |

## 統合

### ステップ1:タクシーメール テンプレートの作成

タクシープラットフォームにタクシーテンプレートを作成します。テンプレートが作成されたら、<サービスプロバイダー an id="1">組織設定</サービスプロバイダーan> に移動し、<サービスプロバイダー an id="2">メールサービスプロバイダー (ESP) Connectors</サービスプロバイダー an> タブを選択します。

### ステップ2:Brazeコネクターの作成

1. アプリが表示されるダイアログで、**新規**ボタンをクリックし、ドロップダウンから**Braze**を選択します。 
2. **Braze**を押して、Brazeコネクターの設定を変更します。
3. Braze エンドポイントとBraze API キーを入力します。

コネクターフィールドは、正しい権限が与えられた後にカラーを変更します。このフィールドが変わらない場合は、フィールドがリストされている条件に合っていることを確認します。

## 使用

アップロード Taxi テンプレートを、Braze アカウントの** テンプレート s & amp; Media > E メールテンプレートs** セクションで見つけます。これで、このメール テンプレートを使用してメールを顧客 s に送信できるようになりました。
