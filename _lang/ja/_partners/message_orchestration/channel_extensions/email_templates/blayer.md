# B. レイヤー

> [B.LayerはPhiture](https://blayer.phiture.com) 's in-app message builder that helps mobile apps' CRMチームで、カスタムデザインのアプリ内メッセージを簡単かつ迅速に、コーディングなしで作成できます。 

BrazeとB.Layerの統合により、B.Layerアプリ内メッセージビルダーを活用して、ブランドに合わせたアプリ内メッセージを作成し、zipファイルまたはインラインHTMLとしてBrazeにエクスポートできます。この統合は追加の開発者リソースを必要としないため、時間と予算を節約できます。

![][2]

## 前提条件

| 必要条件 | 説明 |
| ----------- | ----------- |
| B. レイヤーアカウント | このパートナーシップを利用するには [B.Layer](https://blayer.phiture.com) アカウントが必要です。 |

## ユースケース

B.Layerには、製品推奨スライダー、マルチスクリーンのオンボーディングまたはアンケート、NPS、メールキャプチャ、特別オファーなど、構築して試す機会が無限にあります。

彼らはLifesum、Blinkist、OnX Huntなどのブランドと協力して、追加のリソースなしでユーザーエクスペリエンスの向上を支援しています。また、APS Awards 2022のアプリイノベーション部門のファイナリストにも選ばれています。

## 統合

### ステップ1:アプリ内メッセージを作成する

#### ブランドカラーとフォントの設定

B.Layerでは、ページ上部のハンバーガーメニューで、\[**ブランドアセット] > \[ブランドアセットの追加**] をクリックします。ここで、ブランドカラーとフォントを割り当てることができます。
これで準備は完了です。これで、アプリ内メッセージデザインを開始できます。

![][4]

#### アプリ内メッセージデザインする

アプリ内メッセージをデザインするには、アプリ内メッセージ 1つ選択します。次に、メッセージのスタイルを設定して、必要なコンポーネントを追加します。各コンポーネントは調整できます。

![][5]

### アプリ内メッセージをダウンロードする

完了したら、メッセージをダウンロードします。メッセージは ZIP またはインライン HTML としてダウンロードできます。 

### ステップ2:B.Layer カスタムコード追加

Braze で、カスタムコードアプリ内メッセージを作成します。ZIP ファイルがある場合は、コードセクションの上のボックスにドラッグ＆ドロップします。インライン HTML ファイルがある場合は、インライン HTML を HTML セクションに貼り付けます。

![][6]

## ボタントトラッキング, 追跡

B.Layer では、ボタンの操作やテキスト入力を Braze 属性として記録できます。これはエディター内で実行できます。一般的な例はNPS調査です。

B.Layer は、入力したリンク (例:) に追加された Braze ボタントトラッキング, `?button=0` 追跡を使用します。これにより、キャンペーン分析部分でボタンのクリック数を確認できます。
