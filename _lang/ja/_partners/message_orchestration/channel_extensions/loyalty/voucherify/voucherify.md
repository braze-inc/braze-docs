# Voucherify

> [Voucherifyは](https://www.voucherify.io/)、ユーザーエンゲージメントリテンションを促進するパーソナライズされたキャンペーンとロイヤルティプログラムを可能にするオールインワンのプロモーションプラットフォームです。 

<iframe src="https://player.vimeo.com/video/745340934?h=17ceae8c3c" width="640" height="360" frameborder="0" allow="autoplay; fullscreen; picture-in-picture" allowfullscreen></iframe>

BrazeとVoucherifyの統合により、以下を使用して独自のコードを送信することで、プロモーションキャンペーンを拡大できます。

- [接続コンテンツ]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_fetching_data_through_braze_connected_content):Braze コネクテッドコンテンツを使用して Braze キャンペーンに固有のコードを追加します。この機能を使用すると、Voucherify割引クーポン、ギフトカードキャンペーン、ロイヤルティカード、および紹介コードを使用できます。
- [カスタム属性]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_distribution_with_braze_custom_attributes):カスタム属性を使用すると、Voucherify独自のクーポン、ギフトカード、ロイヤルティカード、紹介コードをBrazeのユーザーのプロファイルに割り当てることができます。これにより、メールキャンペーンで添付コードや属性を送信し、ユーザーと共有することができます。
- [プロモーションコードリスト]({{site.baseurl}}/partners/message_orchestration/channel_extensions/loyalty/voucherify/voucherify_using_braze_promotion_codes_list):Voucherifyが生成したプロモーションコードを使用して、Brazeにアップロードしてください。

## 前提条件

| 必要条件 | 説明 |
| ----------- | ----------- |
|Voucherify アカウント | このパートナーシップを利用するには、Voucherifyアカウントが必要です。 |
| Braze REST API キー | `users.track`権限のあるBraze REST APIキー。<br><br> これは Braze ダッシュボードの **\[設定] > \[**API キー**]** から作成できます。 |
