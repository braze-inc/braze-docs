## サポートされるデータ型

以下のデータタイプがサポートされている：

<table>
  <thead>
    <tr>
      <th>データ型</th>
      <th>説明</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>数値</td>
      <td><code>1や</code> <code>5.5などの</code>数値。</td>
    </tr>
    <tr>
      <td>string</td>
      <td><code>Hello "</code>や<code>"The Hobbit "</code>などのテキスト値<code>。</code></td>
    </tr>
    <tr>
      <td>ブール値</td>
      <td><code>true</code>または<code>falseと</code>評価される値。</td>
    </tr>
    <tr>
      <td>配列</td>
      <td><code>"red"、"blue"、"green"]の</code>ような値のリスト。</td>
    </tr>
    <tr>
      <td>時刻</td>
      <td>
        日付と時刻の比較に使われるタイムスタンプ値。階層化された時間カスタム属性をフィルターする際に、選択できる：<br><br>
        <ul>
          <li><strong>年中無休</strong>：<code>03-15</code>」のように、月と日だけをチェックして比較する。</li>
          <li><strong>時間</strong>:<code>2023-03-15T12:00:00Zの</code>ように、年を含む完全なタイムスタンプを比較する。</li>
        </ul>
      </td>
    </tr>
    <tr>
      <td>オブジェクト</td>
      <td>キーと値のペアを持つ構造化された値で、<code>{"author"："Tolkien"}</code>.</td>
    </tr>
    <tr>
      <td>オブジェクト配列</td>
      <td>
        <code>[{"title"] のようなオブジェクトのリスト：「ホビット』だ：「デューン"}]</code>.
        詳細については、参照してください。 
        オブジェクト配列
      </td>
    </tr>
  </tbody>
</table>
