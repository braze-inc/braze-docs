# 2025年2月4日リリース

## Braze ドキュメントの改善

### データ＆分析の刷新
お探しの情報を見つけやすくするため、以前は「データ＆分析」の下にまとめられていた記事を[データ]({{site.baseurl}}/user_guide/data)と[分析]({{site.baseurl}}/user_guide/analytics)に分離しました。

### 開発者ガイド
[Braze 開発者ガイド]({{site.baseurl}}/developer_guide/home)の全ドキュメントを大幅にクリーンアップし、複数のページに分散していた「ハウツー」記事を1つのページに統合しました。

また、新しい [SDK リファレンスページ]({{site.baseurl}}/developer_guide/references)では、各 Braze SDK のすべてのリファレンスドキュメントとリポジトリが一覧で確認できます。

#### Unreal Engine Braze SDK
Unreal Engine Braze SDK の GitHub リポジトリの README からすべてのコンテンツを [Braze Docs の専用セクション]({{site.baseurl}}/developer_guide/sdk_integration/?sdktab=unreal%20engine)に移行し、書き直しました。

## データの柔軟性

### API 使用状況ダッシュボード

{% multi_lang_include release_type.md release="General availability" %}

[API 使用状況ダッシュボード]({{site.baseurl}}/user_guide/analytics/dashboard/api_usage_dashboard)を使用すると、Braze への受信 REST API トラフィックを監視して、REST API の使用傾向を把握し、潜在的な問題をトラブルシューティングできます。

### カスタム属性にタグを追加する

{% multi_lang_include release_type.md release="General availability" %}

「イベント、属性、購入の管理」権限があれば、カスタム属性の作成後に[タグを追加]({{site.baseurl}}/user_guide/data/custom_data/custom_attributes#adding-tags)できます。追加したタグを使用して、属性のリストをフィルタリングできます。

### カタログセレクションと非同期カタログフィールドエンドポイント

{% multi_lang_include release_type.md release="General availability" %}

以下のエンドポイントが一般提供になりました。
* [POST: カタログフィールドの作成]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/post_create_catalog_fields)
* [DELETE: カタログフィールドの削除]({{site.baseurl}}/api/endpoints/catalogs/catalog_fields/asynchronous/delete_catalog_field)
* [DELETE: カタログセレクションの削除]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/delete_catalog_selection)
* [POST: カタログセレクションの作成]({{site.baseurl}}/api/endpoints/catalogs/catalog_selections/asynchronous/post_create_catalog_selections)

### メールアドレスを使用してキャンペーンやキャンバスをトリガーする

{% multi_lang_include release_type.md release="General availability" %}

[キャンペーン]({{site.baseurl}}/user_guide/engagement_tools/messaging_fundamentals/targeting_users/)や[キャンバス]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/create_a_canvas/?tab=target%20audience#step-2c-set-your-target-entry-audience)をトリガーする際に、メールアドレスで受信者を指定できるようになりました。

### 電話番号を使用して API 経由でユーザーを識別する

{% multi_lang_include release_type.md release="General availability" %}

[`/users/identify` API エンドポイント]({{site.baseurl}}/api/endpoints/user_data/post_user_identify)を通じてユーザーを識別する際に、エイリアスやメールアドレスに加えて電話番号も使用できるようになりました。

### SAML トレースを取得する
サポートチームと SAML SSO に関する問題をより効率的に解決できるように、[SAML トレースの取得手順]({{site.baseurl}}/user_guide/administrative/access_braze/single_sign_on/set_up#obtaining-a-saml-trace)を追加しました。
 
### 地域別データセンター
Braze が新しい地域へのサービス提供を拡大するのに伴い、運用アプローチを明確にするための [Braze データセンターに関する記事]({{site.baseurl}}/user_guide/data/data_centers)を追加しました。

 
## 創造性を引き出す
 
### 値下げ通知と再入荷通知

{% multi_lang_include release_type.md release="General availability" %}

キャンバスとカタログで[再入荷通知]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/back_in_stock_notifications)を設定することで、商品の再入荷を顧客に通知できるようになりました。

また、カタログとキャンバスで[値下げ通知]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/catalog_triggers/price_drop_notifications)を設定して、商品の価格が下がったときに顧客に通知することもできます。

### セレクションのプレビュー

{% multi_lang_include release_type.md release="General availability" %}

セレクションを作成した後、ランダムなユーザーまたは特定のユーザーに対して[セレクションが返す結果を確認]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/selections/#test-and-preview)できます。

### Liquid を含むカタログアイテムのテンプレート化

{% multi_lang_include release_type.md release="General availability" %}

[Liquid を含むカタログアイテムをテンプレート化]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/catalogs/using_catalogs/#using-liquid)できます。

### キャンバステンプレート
[好みの調査によるユーザーのオンボーディング]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/preference_survey)と、[ダブルオプトインによるメール登録の作成]({{site.baseurl}}/user_guide/engagement_tools/canvas/get_started/braze_templates/email_signup)のための新しいキャンバステンプレートを追加しました。

### B2B 向け Salesforce Sales Cloud によるリード管理
B2B マーケターが Braze を活用する方法の1つとして、Salesforce Sales Cloud との統合があります。この[ユースケース]({{site.baseurl}}/user_guide/getting_started/b2b_use_cases/b2b_salesforce_sales_cloud)の実装方法について詳しくご覧ください。
 
## 強力なチャネル

### 除外リスト

{% multi_lang_include release_type.md release="Beta" %}
 
[除外リスト]({{site.baseurl}}/user_guide/engagement_tools/segments/suppression_lists)は、メッセージを一切受信しないユーザーグループを指定します。管理者は、セグメンテーションと同様にセグメントフィルターを使用して除外リストを作成し、ユーザーグループを絞り込むことができます。

## 新しい Braze パートナーシップ

### Constructor - ダイナミックコンテンツ
[Constructor]({{site.baseurl}}/partners/ecommerce/product_search_recommendations/constructor/) は、AI と機械学習を活用して、e コマースおよび小売の Web サイト向けにパーソナライズされた検索、おすすめ、ブラウジング体験を提供する検索・商品発見プラットフォームです。
 
### Trustpilot - ダイナミックコンテンツ
[Trustpilot]({{site.baseurl}}/partners/trustpilot/) は、顧客がフィードバックを共有でき、レビューの管理や返信が行えるオンラインレビュープラットフォームです。
