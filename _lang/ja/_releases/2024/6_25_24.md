# 2024年6月25日リリース

## データの柔軟性

### APIトリガーキャンペーンの添付ファイル

{% multi_lang_include release_type.md release="General availability" %}

[`/campaigns/trigger/send` エンドポイント]({{site.baseurl}}/api/endpoints/messaging/send_messages/post_send_triggered_campaigns)で添付ファイルがサポートされるようになりました（`/messages/send` エンドポイントでメールの添付ファイルがサポートされるのと同様です）。 

### データウェアハウスの追加サポート

{% multi_lang_include release_type.md release="Early access" %}

Braze [クラウドデータ取り込み (CDI)]({{site.baseurl}}/user_guide/data/cloud_ingestion/connected_sources/) で、BigQuery、Databricks、Redshift、Snowflake がサポートされるようになりました。

### WhatsAppの電話番号移行

Metaの埋め込みサインアップを使用して、WhatsApp Businessアカウント間でWhatsAppの電話番号を移行できます。[WhatsApp電話番号移行]({{site.baseurl}}/user_guide/message_building_by_channel/whatsapp/overview/phone_number_migration)について詳しくお読みください。
 
## 創造性を引き出す

### デバイス別エンゲージメント

{% multi_lang_include release_type.md release="General availability" %}

新しい**デバイス別エンゲージメント**レポートでは、ユーザーがどのデバイスを使ってメールに関与しているかの内訳を確認できます。このデータは、モバイル、デスクトップ、タブレット、およびその他のデバイスタイプにわたるメールエンゲージメントを追跡します。[レポートとメールパフォーマンスダッシュボード]({{site.baseurl}}/user_guide/analytics/dashboard/email_performance_dashboard/)について詳しくご覧ください。

### キャンバスフローのWhatsAppおよびSMS Liquidプロパティ

{% multi_lang_include release_type.md release="General availability" %}

[キャンバスフローでのWhatsAppおよびSMS Liquidプロパティ]({{site.baseurl}}/user_guide/engagement_tools/canvas/create_a_canvas/canvas_entry_properties_event_properties)のサポートが追加されました。アクションパスステップに「SMSインバウンドメッセージを送信しました」または「WhatsAppインバウンドメッセージを送信しました」トリガーが含まれている場合、後続のキャンバスステップにSMSまたはWhatsApp Liquidプロパティを含めることができるようになりました。これは、キャンバスでイベントプロパティが機能する仕組みと同様です。これにより、メッセージを活用して、ユーザープロファイルや会話メッセージングに関するファーストパーティデータを保存し、参照することができます。
 
### 繰り返しキャンバスのパーソナライズドパス

{% multi_lang_include release_type.md release="Early access" %}

キャンバスのパーソナライズドパスでは、コンバージョンの可能性に基づいて、個々のユーザーに対してキャンバスジャーニーの任意のポイントをパーソナライズすることができます。パーソナライズドパスが繰り返しキャンバスでも利用可能になりました。詳細については、[パーソナライズドバリアント]({{site.baseurl}}/user_guide/engagement_tools/canvas/canvas_components/experiment_step/personalized_paths)を参照してください。

### セグメントのトラブルシューティング

セグメントを使用していますか？以下は、覚えておくべき[トラブルシューティングの手順と留意点]({{site.baseurl}}/user_guide/engagement_tools/segments/troubleshooting)です。

アクセシビリティガイドラインをより適切にサポートするために、[Liquidが使用する色分け]({{site.baseurl}}/user_guide/personalization_and_dynamic_content/liquid)を改善しました。

## 強力なチャネル

### SMS地理的許可

{% multi_lang_include release_type.md release="Early access" %}

SMS地理的許可は、SMSメッセージを送信できる国に対する制御を強制することにより、セキュリティを強化し、不正なSMSトラフィックから保護します。管理者は、承認された地域にのみSMSメッセージが送信されるように、国の許可リストを指定できるようになりました。詳細については、[SMS地理的許可]({{site.baseurl}}/sms_geographic_permissions)を参照してください。 

![最も一般的な国が上部に表示される「国の許可リスト」ドロップダウン。]({% image_buster /assets/img/sms/allowlist_dropdown.png %}){: style="max-width:80%;"}

### SMS/MMSのベストプラクティス

オプトアウト監視やトラフィックポンピングに関する推奨事項を含む、[BrazeでのSMS/MMSのベストプラクティス]({{site.baseurl}}/sms_mms_rcs_best_practices/)について詳しくご覧ください。 

### プッシュ配信停止のトラッキング

プッシュ配信停止を追跡するためのヒントについては、新しい[ヘルプ記事]({{site.baseurl}}/help/help_articles/push/push_unsubscribes)をご確認ください。

### Shopify `checkout.liquid` の廃止

Shopify `checkout.liquid` のサポートは2024年8月に非推奨が開始され、2025年8月に終了する予定です。Brazeがこの移行にどのように対応するかについては、[こちら]({{site.baseurl}}/help/release_notes/deprecations/shopify_checkout)をご覧ください。
